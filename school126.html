全方位學習活動
孩子喜歡透過音樂、視覺藝術創作表達自己，因此本校致力發展課外活動，力求多元化：制服隊伍、球類活動、歌詠、舞蹈及兩文四語等興趣小組；「繽紛星期六」則集中培育學生體藝潛能。
辦學宗旨
發揚基督博愛精神，效法若翰洗者體證真理；建立積極的價值觀、力求創新及具承擔的精神，熱愛生命、關心社會、心懷國家；在優良學習環境中，啟動學生的多元智能及創意，主動學習，奠定終身學習的基礎，為社會作貢獻。
學校特色
學校管理
學校管理架構 : 本校由天主教香港教區管理，已成立法團校董會。校監、校長及副校長帶領行政及中層小組管理學校的事務。行政及中層小組分為四大範疇，分別為管理與組織、學與教、校風與學生支援以及特設小組。
法團校董會/學校管理委員會/校董會 : 本校於2014年4月1日成立法團校董會，成員包括辦學團體委任校董八位、獨立校董一位、教員校董兩位、家長校董兩位、校友校董一位及校長，一同肩負管理學校的責任。
環保政策 : 本校校園廣闊，遍植花草樹木，環境翠綠。價值教育組會邀請環保團體到校舉辦講座，鼓勵學生從生活做起，如節約用電，用水，盡量循環再用，為環保出一分力。此外，本校設有飯堂，上課日為學生預備新鮮的蔬果及飯餸，棄用塑膠飯盒及餐具，減少廚餘，為學生提供健康的午膳。
學校關注事項 : 1. 致力培養孩子關愛近人及認真學習的態度。
2. 學校是孩子的學習樂園，各級引入網上學習計劃及多元化課程。
3. 加強照顧個別差異，按學生需要開設輔導教學小組，以便老師在輔導時聚焦跟進學生學習表現。
教學規劃
學習和教學策略 : 1. 早操及早讀課。
2. 週三價值教育課。
3. 週五課外活動、課餘尖子培訓及加油班。
4. 科技探究及校外學習日。
5. 英語閱讀計劃。
6. 年度學習主題：「剛毅自強齊突破　探究求真同創新」。
關鍵項目的發展 : 1. 專題研習—各級按不同主題以個人或小組形式進行探究，配合戶外學習，豐富學習經歷。
2. 運用資訊科技進行—本校設有成長課、價值教育課及週訓，鼓勵學生參與各種公民教育活動，如：升旗隊、制服團隊及基督小先鋒等。以年度主題：「剛毅自強齊突破　探究求真同創新」的生命教育元素滲透各學習領域。此外，亦以「健康校園」為主題，帶出環保、健康、安全校園等訊息。
共通能力的培養 : 提升批判思維及溝通能力，鼓勵學生發揮創意。
學生支援
全校參與照顧學生個別差異 : 1. 積極拔尖補底：培育球類、舞蹈、田徑、數學、英語及視藝等尖子；為學習有需要學生提供個人及小組輔導。
2. 每天設功課指導課。
3. 引入專業支援，如教育心理學家、言語治療師、職業治療師等。
課程剪裁及調適措施 : 設有幼小銜接課程，為小一的學生適應小學學習作準備。課程讓新生和家長認識學校環境，以及本校的課程特色。老師在幼小銜接課程中以主題學習小冊子為學習課業，按學生學習情況滲入校本課程內容作增潤。本校亦著重拔尖補底，於課餘時提供尖子培訓，以發揮學生的潛能；能力較弱的學生，亦可參與輔導課以協助解決學習上的困難。
家校合作及校風
家校合作 : 1. 家長高參與學校。
2. 家校同心，共建樂園。
校風 : 關愛校園，共融互助。
學校發展
學校發展計劃 : 1. 優化學生學習表現。
2. 透過天主教教育五項核心價值，幫助學生建立健康的人生觀。
3. 優化行政架構，加強校政的透明度。
4. 詳情請參考三年發展計劃：http：//www.kbsjb.edu.hk
教師專業培訓及發展 : 1. 認識天主教核心價值。
2. 培育教師身心靈健康。
3. 掌握照顧學生精神健康的技巧。
4. 分析評估數據，了解學生的弱項，以優化學生表現。
5. 運用網上評估工具提升學與教效能。
學費減免 : 不適用。
