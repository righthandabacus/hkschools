辦學宗旨
我們以聖經原則及基督教信念，透過辦學去貫徹教會傳道服務精神；為兒童及青少年提供優質教育，造就人才；致力啟發學生潛能，幫助全人成長，建立品德靈性；培育他們成為良好公民，貢獻社會國家。
學校特色
學校管理
學校管理架構 : 設有明確的行政及監察制度。校內由校長及副校長領導行政小組制定、推行及監察各項政策，各科亦設有科主任關注科目的管理及發展。
法團校董會/學校管理委員會/校董會 : 已成立法團校董會，分別由辦學團體、獨立人士、校友、校長、家長及教師擔任校董。
環保政策 : 致力建立綠化校園及環保政策，推廣校內環保及生態活動，培育關心世界，珍惜大自然的品德。
學校關注事項 : 1. 提升學習動機，加強學習效能。
2. 營造健康的校園生活，培養樂於學習的態度。
教學規劃
學習和教學策略 : 學習設計以學生為中心，著重啟發式教學及探究式學習。又透過合作學習、活動教學、專題研習、各種閱讀策略、思維及研習工具，並加入資訊科技元素，以激發學生的自主學習能力及提升學生的共通能力。
關鍵項目的發展 : 閱讀︰閱讀日、廣泛閱讀、網上閱讀及參與校外活動/比賽，老師及學生於午會分享及推介書籍。
IT：以平板電腦輔助教學，發展數碼藝術、高年級程式編寫及立體繪圖課程、培養正確使用資訊科技態度，將資訊科技知識融入各科學習內。已發展學校內聯網電子家課及校園電視台，從而提升學生自主學習及語文能力。
專題︰每年兩次常識專題，注重提高學生研習技能及培養共通能力。
德育及公民教育︰1. 靈德課程。2. 中華文化活動「基道狀元紅」系列。
3. 專題德育週會。4. 小五服務學習計劃。5. 制服團隊。6. 校本輔導獎勵計劃。
共通能力的培養 : 除課堂設計外，本校亦安排科本活動、服務學習、境外交流活動及基道才藝匯演等，全面提升學生各種共通能力。
學生支援
全校參與照顧學生個別差異 : 本校重點發展資優教育，設立校本卓越尖子培訓計劃，按學生能力和潛質分成不同的小組學習，如科學菁英組、創思組、數學資優培訓組、視藝菁英組、英文大使及小小天才組，讓學生得到適切的栽培，發揮潛能。為照顧學生的學習差異，學校以小班教學形式施教，並設加強輔導教學及輔導教學組。教師透過合作學習，加強對學生的照顧。
課程剪裁及調適措施 : 英文科參加的外籍英語教師組英文閱讀及寫作計劃已推展至小三；中文科、數學科及常識科繼續參考往年教育局校本支援服務給予的意見為課程作調適及剪裁；另各科亦不時為課程作檢討，令課程日趨完善。
家校合作及校風
家校合作 : 家教會以家校合作為目標，定期舉辦各項家長講座、興趣小組和親子活動，同時又成立家長義工隊，以配合校務的推展，與學校共同為學生創建一個優良的學習環境。
校風 : 採用全校參與，訓輔合一方式為學校建立良好校風，訓輔組多年獲關愛校園榮譽獎及仁濟醫院公民教育獎。
學校發展
學校發展計劃 : 1. 提升學生學習動機，加強學習效能。
2. 營造健康及愉快的校園生活，讓學生樂於學習
3. 發展平板電腦輔助教學模式，透過資訊科技提升學與教水平。
4. 常識科加入教育局校本課程發展組常識科教師網絡，發展E-STEM課程。
教師專業培訓及發展 : 每年舉辦多次教師發展日，又鼓勵教師參與校外培訓。校內亦推行課堂研究計劃及開放教室，以提升教學質素。
學費減免 : 不適用。
其他 : 中、英、數、常邀請教育界資深人士提供指導及支援，積極提升學與教水平。
已成立校友會，得到各界專業校友支援。
