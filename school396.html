班級教學模式
本校以活動教學為學習模式，讓學生透過活動式教學學習知識，除常規課程外，教師亦會教導學生利用高階思維的思考方法學習，發展其學習潛能。
班級結構備註
設普通話教中文班，另設有英文拼音及閱讀實踐課，以提昇學生的英語能力。

辦學宗旨
學校秉承校訓「愛主愛人，求道更新」，本著基督精神，讓學生在優良及愉快的環境內，善用資訊科技促進學習，從而發揮個人的多元智能，尊重他人，尊重生命，關心社會，關心國家，接受全人教育。
學校特色
學校管理
學校管理架構 : 法團校董會委任校長處理校務，推行校本管理。
法團校董會/學校管理委員會/校董會 : 法團校董會由辦學團體代表、校長、教師、家長和獨立人士組成。
環保政策 : 1. 園藝小組：推廣校本環境保護課程。
2. 有機花園：提倡有機種植的好處。
3. 魚菜共生系統：這系統體現水及能量的循環。
學校關注事項 : 1. 透過多元化學習活動，推動校園閱讀風氣。2. 善用資訊科技促進學習，建構學生自學能力。3. 透過專題研習，發展共通能力及提升他們對學習的主動性。4. 透過不同類型學習活動，培育學生身心靈健康成長。
教學規劃
學習和教學策略 : 透過聯課活動、小組討論及專題研習，為學生創造多元化活動。除了正規課堂學習，學生參加不同類型的學習活動，如資優教育課程、校園電台製作及科學探究等。
關鍵項目的發展 : 1. 從閱讀中學習–「英文語音閱讀計劃」提昇學生閱讀習慣和興趣。
2. 運用資訊科技進行互動學習–無紙藝術、電子互動白板計劃、校本英文及數學學習平台計劃、校園電視台計劃、我是未來領袖課程。
3. 專題研習–多元化的全方位學習活動。
4. 德育及公民教育–完善的德育及公民教育課程。
共通能力的培養 : 創造力、批判性思考能力、溝通能力及協作能力。
學生支援
全校參與照顧學生個別差異 : 訓輔及學生支援組，採用小一及早識別策略；為低年級學生進行評估及輔導，讓他們掌握基本學習技巧。此外，更設有資優課程、劍橋英語及奧數班，以發展學生潛能。
課程剪裁及調適措施 : -
家校合作及校風
家校合作 : 透過家長日及諮詢會，加強家校溝通。本校有強大支援隊伍，包括：教育心理學家、學生輔導教師和資源教師，為能力不同的學生提供適切的服務。
校風 : 訂立有品校園約章，培養自律精神。推行大哥哥大姐姐計劃，發揮友愛精神。家校合作無間：家長義工隊超過100人。
學校發展
學校發展計劃 : 1. 21世紀技能學習中心–透過中英普演說技巧訓練，提昇學生的演說能力。
2. 無紙藝術–電腦美術及音樂課程。
3. 英文讀寫計劃–提昇學生英文閱讀及寫作能力。
4. 我是未來領袖創意資訊科技教育計劃–網上媒體教育課程。
5. 致力與大專院校合作，進行教學研究，提昇教學效能。
6. 推行中文、英文、數學及常識網上學習，培養學生的自學能力。
7. 推展中英數課堂研究。
8. 與教育局合作發展英文校本課程。
教師專業培訓及發展 : 100%教師獲得大學學士學位，另外57%老師獲得專科碩士學位。
本校教師師資優良，曾四度獲得行政長官卓越教學獎。
學費減免 : 不適用。
其他 : 學生品學兼優，學校比賽屢創佳績。教師曾四度獲得行政長官卓越教學獎。
