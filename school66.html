班級教學模式
我們為成績稍遜的同學加強輔導；成績優異的則開設中、英、數、常拔尖課程等恰切培訓；並於二至六年級部份班別實行普通話教中文。全校實行小班教學，及以合作學習模式上課，以照顧學生的個別差異。

多元學習評估 : 學校設實際應用、態度及價值觀等多元化評估，並把課外活動的表現及參與納入評估範疇內。除設有進展及總結性評估外，亦會進行其他模式的多元化評估。為學生設學習歷程檔案，以紀錄各方面的表現。
分班安排 : 於四至六年級按學生成績及各方面表現分班，目的是照顧不同能力的學生在學習上的差異。

健康校園生活 : 學校設有飯堂，由指定的供應商提供合乎健康飲食標準的午餐，並採用循環再用的餐具，致力推動至『營』的飲食文化及環保意識。
備註 : 設有早讀、圖書課、午間活動、宗教聚會、公民教育、輔導、探究學習及專題研習等。學校為學生提供一個多元的學習環境。
全方位學習活動
課外活動分長期及短期組，如手鈴、手鐘、管樂、戲劇、劍擊、水墨畫及球類活動等，以發掘學生體藝潛能，盡展才華，以達至一人一體藝目標。課後由專業導師帶領劍橋英語、珠心算、跆拳道、武術、銅管樂、鋼琴、舞蹈及戲劇等興趣班。
辦學宗旨
學校透過分享耶穌基督的愛和真理，得以在知識上、成長上、關顧他人–特別那些有極大需要的人士，能夠有所增長。
學校特色
學校管理
學校管理架構 : 實施校本管理，有效完善的自評基制，從多個角度讓各持份者進行自評，優化學校政策。
法團校董會/學校管理委員會/校董會 : 校董會於2006年成立，由校監、辦學團體代表，專業人士、校長、家長及老師代表組成，積極參與學校發展，建立良好顆伙關係。
環保政策 : 積極參與環境保護署的環保校園項目，校內多項設施，例如：電光管等均採用最新環保型號；另外，各班設「環保長」，希望從學生的日常生活著手，潛移默化地滲入環保意識。
學校關注事項 : 2014-2017 年度關注事項：
1. 加強照顧學習差異，促進學生有效學習。
2. 培養學生正確價值觀，建立關愛校園。
教學規劃
學習和教學策略 : 設探究學習週，讓學生進行探究與研習。教師透過教學研究，以提升教學質素。學校突破課堂的框架，設主題學習，透過實地參觀及訪問擴闊學生視野。設小班教學，教師能多照顧個別差異，使學生學得更豐富及具信心。學校設有跨科學習及討論，讓學生把所學的應用於生活。（請參閱學校網頁，有更多資料。）
關鍵項目的發展 : 本校重視多元及網上閱讀，以提升學生的閱讀策略，建立其自學能力。學校的專題研習，結合資訊科技、閱讀，引導學生進入探究。德育及公民教育方面，教師藉平日的課堂滲入正確的價值觀，以培養學生的品德。除此之外，本校十分重視學生有效學習，有效益的家課對學習十分重要，家課亦是一種學生學習經歷。無論在家課設計、擬定內容及學生表現回饋上都不斷求進，適時作出家課調適，以照顧學習差異，教師善用家課以了解學生在汲取知識、掌握技能及培養良好學習態度等各方面的表現，務求促進學與教的發展，提升學習效能。(請參閱學校網頁，有更多資料。)
共通能力的培養 : 藉各類型的教學活動，培養學生共通能力，強調課堂互動，以豐富學生學習的經驗。（請參閱學校網頁，有更多資料。）
學生支援
全校參與照顧學生個別差異 : 學校很重視每位同學的學習，為配合不同能力需要的學生，學校在工作紙及評估方面設有不同類別或深淺項目，讓學生更能因應自己的程度與能力去完成。（請參閱學校網頁，有更多資料。）
課程剪裁及調適措施 : 學校在小二至小六進行中文，英文及數學三科的小組學習及調適課程，並在小二開展『普教中』教學。（請參閱學校網頁，有更多資料。）
家校合作及校風
家校合作 : 家長非常支持學校活動，並擔任義工工作。學校致力推動家長教育，支援家長用關愛培育子女，讓他們在鼓勵和讚美中茁壯成長。
校風 : 師生與家長同心建立和諧而關愛的校園；透過成長課、全方位輔導、成長的天空、非華語課後小組、兒童發展基金、童夢之旅、家校合作及有規劃的活動課，輔助學生成長。
學校發展
學校發展計劃 : 我們是一所學習型的學校，學生透過自發性學習和探究精神建構知識，養成良好的學習習慣，並分階段地深化各種共通能力。我們重視強化學生在兩文三語、自主學習和資訊科技學習上的應用，以加強他們日後的競爭力。課程更注入不同的元素，讓學生從中享受學習的經歷與樂趣。此外，學校亦透過自評及對外的分享，使發展更趨完善。
教師專業培訓及發展 : 學校為教師提供切合學校發展的培訓，例如小班教學策略等等。學校並設優化學與教組，提升教師的教學技巧，以豐富學生在課堂的學習。
學費減免 : 不適用。
其他 : 為發展學生潛能，學校鼓勵學生參加比賽，屢獲殊榮。本校榮獲香港青年音樂匯演管樂團比賽小學初級組金獎、學校舞蹈節中國舞小學高級組甲等獎。於香港聯校音樂協會聯校音樂大賽中，更同時囊括木管樂合奏、手鈴及手鐘公開組、初級組合唱及高級組合唱四項金獎。體育方面，榮獲本區學界田徑賽男丙團體冠軍、校際乒乓球比賽男團季軍及女團殿軍。學術方面，學生於港澳數學奧林匹公開賽中獲得優異成績。
