班級教學模式
全校推行「電子學習」︰小一至小三運用iPad；小四至小六運用平板電腦（一人一機）及雲端科技，並運用ICT工具進行跨學科主題探究式學習。採用「全校參與模式」支援有特殊教育需要學生（協作及抽離式並用）。
班級結構備註
小二至小六：普通話教中文；小一至小三：外籍英語教師識字計劃及 Get Reading Right 拼音課程；小一至小六：校本英文網上學習平台（由閱讀到寫作）。

辦學宗旨
本校秉承慈幼會創辦人聖若望鮑思高提倡的「預防教育法」，藉著理智、宗教和仁愛的信念，透過教職員、家長及神長積極地臨在於學生之中，熱誠地參與學生的學習和課外活動，使學生深深地感受到師長對他們的愛護和教導。
學校特色
學校管理
學校管理架構 : 由法團校董會管理學校，在校長之下有不同科組部門，以管理及執行學校發展和行政工作。詳情請參考本校網頁。
法團校董會/學校管理委員會/校董會 : 成員包括校監、神長、校長、教育界及其他人士。
環保政策 : 積極參與環保活動：回收舊校服、環保飯盒、節能大使及綠化校園計劃。
學校關注事項 : 1. 學生掌握二十一世紀學習、工作及生活必須具備的技能。
2. 實踐鮑聖的預防教育法，把學校塑造成充滿關愛的家。
3. 效法慈幼聖人的精神，培養學生正面積極的價值觀。
教學規劃
學習和教學策略 : 全面推行「電子學習」，為學生裝備二十一世紀須具備的學習及生活技能。藉著雲端科技，全校師生均可享用無縫學習環境，免費接觸到最新的資訊科技及學習軟件。小一至小三運用學校提供的iPad學習，小四至小六學生更可免費擁有個人平板電腦進行學習活動。以中央課程為基礎，再根據學生的程度、興趣及需要訂定校本課程，透過不同學與教策略，如「合作學習」及不同層次的提問技巧，讓學生從分組活動中，透過互動建構知識，發展高階思維能力，培養共通能力，建立正面的價值觀和態度，達致終身學習的果效。本校小一至小六均採用小班教學，以便照顧學生的學習多樣性，部份課堂更增加額外教師進行協作教學。
關鍵項目的發展 : 以ICT帶動學習，讓學生能使用最新的ICT工具(iPad/Tablets)進行學習。學校推行反轉教室學習模式，讓學生掌握自學的技能。電腦科加入電腦編程及3D打印，讓學生掌握未來社會需要的知識及技能。
共通能力的培養 : 學校有系統安排不同的學習活動（專題研習、戶外學習、服務學習、主題學習）。
學校提供不同的課外及課後活動（體藝訓練、電腦課程、話劇、語文興趣班等），讓學生在不同的學習經歷中掌握共通能力。
學校積極參與校外比賽及表演，讓學生在過程中掌握共通能力。
學生支援
全校參與照顧學生個別差異 : 成立「學生學習支援小組」，為有需要的學生提供協助，邀請語言治療師、教育心理學家等協助同學，向家長提供講座，並透過購買服務為同學安排不同的訓練小組，以提昇專注力及社交能力。增聘教師於課堂及課後提供支援，不時為全校教師提供培訓，以提高照顧差異的能力及提供功課及評估調適，並透過不同活動營造校園共融文化。
課程剪裁及調適措施 : 英文科參與教育局校本支援服務，發展符合學生需要的校本英文課程，讓學生更有效建立良好的英文基礎。
家校合作及校風
家校合作 : 除了藉著校訊、通告、「慈幼家書」及網頁等，促進家校溝通外，更透過不同活動，如：家長日、講座、親子旅行、親子運動會、家長興趣班等，提升家校的聯繫。透過班主任課、生活教育活動及內聯網，培養和諧的師生關係。舉辦小一適應課程、畢業營、境外學習活動、各項班際比賽，提升學生對學校的歸屬感。
校風 : 校風純樸，學生守規有禮。透過宗教的薰陶，老師灌輸正確的價值觀。定期的德育活動，建構關愛及共融文化。
學校發展
學校發展計劃 : 1. 廣泛使用最新的ICT (Information and Communication Technology) 軟件和設備，並將之融入學與教活動以提升課堂互動和提高學習動機。
2. 積極發展「電子教學」方案及將ICT的應用普及化，實踐e-Learning for All，並貫串各科學習，以培養學生掌握廿一世紀學習、工作及生活必須具備的技能。
3. 以「預防教育法」強化學生德育及公民教育。
教師專業培訓及發展 : 配合學校發展及學生需要定期進行培訓，全體老師已接受電子化學與教、小班教學策略、照顧有特殊需要學生等培訓。除了教育局指定的3天教師發展日外，校方亦為教師安排不同的電子教學工作坊，老師亦會善用工餘時間參與不同的培訓以提昇個人的專業水平。
學費減免 : 不適用。
其他 : 1. 本校獲Microsoft頒發Showcase School 榮譽。
2. 國際傑出電子教學獎（金獎）2015-16。
3. Hong Kong ICT Awards 2017：Best Student Invention(Silver Award)。
4. 運算思維及編程教育計劃（資源學校）。
