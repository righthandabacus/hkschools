辦學宗旨
1. 學校願景
本校矢志成為香港卓越的基督教學校，為學生提供優質教育，促進全人發展。
2. 學校使命
根據聖公會所推崇的基督教全人教育理念，本校致力於培育和發展學生在德、智、體、群、美、靈六育的成長。
（由於篇幅有限，未能盡錄，詳情請參考本校網頁。）
學校特色
學校管理
學校管理架構 : 本校為聖公宗（香港）小學監理委員會屬校，法團校董會負責監察學校的運作及提供意見。
法團校董會/學校管理委員會/校董會 : 法團校董會定期舉行會議，就校務提供意見和建議。
環保政策 : 採用全校參與及跨學科的方式，以推廣環境教育。
學校關注事項 : 1. 持續優化教學，提升教學質素和效能。
2. 透過學習基督真理，讓孩童在靈性上得以覺悟。
教學規劃
學習和教學策略 : 配合「學生為本」的學習策略，各科課堂教學強調思考和實踐，老師運用小組合作學習、專題研習、服務學習等策略，為同學們提供思考、討論和探究的機會，並發展他們各方面的能力和正確的價值觀。
關鍵項目的發展 : 1. 閱讀計劃：設圖書課及早讀時間；另設校本閱讀計劃及中、英文科閱讀計劃。
2. 電子學習：課室和特別室設置無線網絡，各科使用平板電腦輔助教學；各級設電腦課；成立「IT小先鋒」，推行拔尖課程。
3. 專題研習：小一至小六均推行。
4. 德育及公民教育：設早禱、早會、週會、班主任課和成長課，實施全方位輔導計劃。
共通能力的培養 : 本校着重培育學生的共通能力，尤其重視溝通、協作、創意和批判性思考能力的培育。一至三年級常識科課程加入生活技能訓練。
學生支援
全校參與照顧學生個別差異 : 採用「全校參與」模式，照顧不同學習需要的學生；開設英文及數學輔導班；課後及星期六開設拔尖補底課程；為資優生提供跳升級支援制度。
課程剪裁及調適措施 : 各科進行合適的增刪和調適，以照顧學生的學習差異。本校亦為有情緒、社交或學習障礙的學生提供由校外專業人員主持的課後學習班。近年亦積極開展資優生課程，為語文、數學和常識科資優學生提供額外的學習機會。
家校合作及校風
家校合作 : 本校家長教師會組織完善，積極推動家校合作，定期舉辦講座、康樂和親子活動。家長義工熱心服務，積極參與學校活動，例如「故事爸媽」、運動會和生日會等。
校風 : 1. 校風淳樸。
2. 訓輔合一。
3. 家校合作緊密。
4. 協助小一新生盡快適應校園生活。
學校發展
學校發展計劃 : 1. 啟發創意思維，鼓勵自學。
2. 發展校本課程，推行課程調適及課程統整。
3. 鼓勵學生從閱讀中學習。
4. 加強全方位輔導。
5. 推行「普通話教授中文」計劃。
6. 推行電子學習。
7. 發展資優課程。
教師專業培訓及發展 : 全面推行共同備課和同儕觀課。未來數年的教師發展及培訓將以「照顧學生的特殊學習需要」和「推行電子學習」為重點。
學費減免 : 不適用。
