辦學宗旨
秉承區會以「傳道服務」為目標的「全人教育」辦學精神，向學生提供全面的優質教育服務。承諾營造一個愉快和諧的學習環境，並與區會、家長、教會、社區及有關團體衷誠合作，為學生提供德、智、體、群、 美、靈六育並重的課程和活動。培養學生正確人生觀，為社會培育人材。
學校特色
學校管理
學校管理架構 : 1. 設法團校董會及學校持續發展組定期會議，促進學校發展。
2. 定期舉行行政會議及教職員會議。
法團校董會/學校管理委員會/校董會 : 成員：校監、辦學團體校董、校長、教師校董、家長校董、校友校董及獨立人士校董。
環保政策 : 1. 綠化校園。
2. 節能減碳，循環再用。
學校關注事項 : 1. 提升學教效能，建構學生解難能力。
2. 提升隻學生的自我管理能力。
教學規劃
學習和教學策略 : 1. 跨學科學習。
2. 推行合作學習模式。
3. 主題式學習。
4. 分析學生表現，跟進學生強弱項目。
關鍵項目的發展 : 1. 從閱讀中學習：閱讀計劃、早讀、廣泛閱讀、親子閱讀等。
2. 德育及公民教育：建立正確人生觀及價值觀，如：週會、成長課、關愛活動及獎勵計劃。
3. 專題研習：專研學習重點及評估。
4. 透過資訊科技互動學習：設各科網上學習活動及比賽。
共通能力的培養 : 提升學生溝通、自我管理、批判性思考、創造力及協作能力，讓學生能面對將來社會的需要。
學生支援
全校參與照顧學生個別差異 : 1. 設功課輔導班、情緒／社交訓練及評估。
2. 設拔尖課程及校本課後學習支援。
3. 幼小銜接及升中銜接。
課程剪裁及調適措施 : 功課及考試調適。
家校合作及校風
家校合作 : 1. 家長教師會：常設會議及活動，增強家校溝通。如講座、旅行及參觀。
2. 家長義工：協助午膳值日、校外活動、推廣閱讀、故事爸媽、英語爸媽及課堂組導師等工作。
校風 : 1. 校風淳樸，宣揚基督大愛，提倡關愛文化，師生／家校關係融洽。
2. 學生身心均衡發展，熱心參與社區服務，學術及體藝表現優異。
3. 教師積極發展專業知識，有效提供優質教育。
學校發展
學校發展計劃 : 1. 拓展學生良好的語文能力。
2. 優化課堂，適切地照顧個別差異。
3. 善用評估數據，促進學與教效能。
教師專業培訓及發展 : 合作學習、照顧個別差異及高階思維。
學費減免 : 不適用。
其他 : 拔臣獎學金、劉活川獎書金、學科獎、進步獎、家教會藝術獎、校友會傑出運動員獎等。
