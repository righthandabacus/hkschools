全方位學習活動
本校提供六十多項長期訓練及興趣班，藉此培養學生不同的興趣、發展所長。逢星期五下午舉行生趣課，以豐富學生的生活經驗。三至六年級學生均為公益少年團團員，培訓學生自小養成服務大眾、貢獻社會的良好品格。
辦學宗旨
本校遵行聖經的教導：「不可叫人小看你年輕，總要在言語、行為、愛心、信心、清潔上，都作信徒的榜樣。」（提摩太前書4章12節）致力提供優質教育，讓學生在優良的環境中學習，使他們在「德、智、體、群、靈、美」六方面都得到均衡的發展。
學校特色
學校管理
學校管理架構 : 法團校董會 > 學校發展組 > 學校行政組
法團校董會/學校管理委員會/校董會 : 香港浸信會聯會小學法團校董會
環保政策 : 1. 以校園持續發展為目標，設立專責小組督導各項措施及全校參與。
2. 善用資源：設置回收設施、發展生趣園再生能源系統、舉行種植活動。
3. 推行環境教育活動：設環保大使，鼓勵學生身體力行。
學校關注事項 : 推動生命教育、推動STEM教育、提升學生自主學習能力
教學規劃
學習和教學策略 : 我們著重培養學生積極、主動的學習態度，敢於面對挑戰，誘發他們的潛能和創意，為學校、為社會作出貢獻。推行資優教育和思維教學策略，舉行全方位學習周跨學科專題研習，學生進行研習活動，如訪問、調查、整理及分析數據、給予建議學習日誌等，於成果展示日向來賓滙報學習成果。
關鍵項目的發展 : 1. 各科推動閱讀，設早讀時間，舉辦書展及不同的閱讀推介活動等，鼓勵同學多閱讀。
2. 善用資訊科技，運用平板電腦及電子白板促進學習，增強效能。
3. 舉辦全方位學習周、科技日等，培訓學生研習及探究能力。
4. 設校本輔導主題及每周公民教育課，加強學生國民及德育意識。
共通能力的培養 : 透過各學科學習活動（如：普通話日、英語日、數學日、科技日等）和生趣課的多元化活動，培養學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 採用「合作學習小組」策略，利用小組動力和小老師朋輩輔導提升能力稍遜同學的學習表現和態度。為突出表現的學生進行拔尖活動，同時亦推薦資優學生參加校外資優課程和比賽。設小樹苗班及課後輔導班，分別為有特殊學習需要和成績稍遜的學生提供學習支援。
課程剪裁及調適措施 : -
家校合作及校風
家校合作 : 設有家教會，為家長提供專題講座、戶外活動、興趣班等，促進家校聯繫。家長義工隊協助製作教具、帶領參觀，並擔任故事姨姨與學生分享圖書。
校風 : 本校訓輔合一，透過訂立「好學生約章」、推行「好學生獎勵計劃」及校本輔導主題活動等，培養學生正確價值觀及良好品格。本校又透過「大哥哥大姐姐計劃」，讓學生發揮友愛互助精神。
學校發展
學校發展計劃 : 2016-2019 三年的關注事項
1. 發展自主學習，提昇學生獨立學習的能力。
2. 關注學生成長的不同需要，推行生命教育，本著基督信仰，以建立學生正向的價值觀。
3. 促進家校合作，協力培養學生高尚的品德。
教師專業培訓及發展 : 為配合教育變革及學校發展，本校致力在學校層面創造空間，有系統地安排不同類型教師培訓，讓教師參與集體性持續專業發展活動。學校亦鼓勵教師，按個人的需要進修，發展教師專業。
學費減免 : 不適用。
