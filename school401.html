班級教學模式

辦學宗旨
秉承嗇色園的辦學宗旨，崇奉道、釋、儒三教，以普濟勸善為校訓，培養學生勤奮向學，樂善好施，服務社會的精神，並著重學生德、智、體、群、美五育的均衡發展。
學校特色
學校管理
學校管理架構 : 嗇色園主辦可信學校法團校董會。
法團校董會/學校管理委員會/校董會 : 嗇色園主辦可信學校法團校董會。
環保政策 : 環保大使計劃、綠化校園計劃、廢紙回收及循環再用計劃。
學校關注事項 : 優化學與教策略，使學生學習更具成效。
培育正向價值觀，以「可信人」為己任。
教學規劃
學習和教學策略 : 1. 學校為了擴闊學生的視野，安排不同的實地交流活動，讓學生與外地的學生，進行專題研習及實地考察活動。
2. 跨科目課程統整已推展至各級，於每年三至五月進行，內容包括參觀、戶外學習、綜合活動日、匯演等，藉以發展學生的共通能力。
3. 推行遠程教學活動，讓學生可透過科技與外地學生在不同時段作交流。
4. 以校園電視台為平台，鼓勵學生參與表演，展示才華。
5. 各級推行資優教育，發掘學生不同類別資優潛質。
關鍵項目的發展 : 1. 閱讀計劃︰實施閱讀獎勵計劃，並於每天早上安排15分鐘早讀時間。
2. 資訊科技︰除了在全校推行電腦課外，學校亦自行設計電腦輔助教學軟件，供課堂使用。在午膳及放學後，開放電腦室並有IT大使輪流當值。
3. 在一至六級各科推行專題研習，以常識科為主推行跨學科專題研習。
4. 德育及公民教育：每週設立德育課及週會，培養學生的德育發展。
共通能力的培養 : 培養及提升學生自學能力、思維能力和語文能力。
學生支援
全校參與照顧學生個別差異 : 1. 為一年級學生安排每週二天課後功課輔導班，讓其儘快適應小一生活。
2. 在二至六年級安排課後學科輔導，另於三至六年級設加強輔導班以照顧個別差異。
3. 推展合作學習，讓不同能力之學生均有足夠機會參與課堂活動。
4. 大哥哥大姐姐計劃，讓高年級學生在小息時段照顧低年級學生。
5. 提供言語治療。
課程剪裁及調適措施 : 1. 在五、六年級為成績優異之學生舉辦英文和數學科拔尖班，讓學生儘早適應升中學習模式。
2. 於星期六，安排新來港學童班，協助學生儘快適應本校的校園生活。
3. 中、英文科自編校本教材，配合坊間教材讓學生學習得更全面。
4. 數學科設立關卡制度跟進學生成績。
家校合作及校風
家校合作 : 1. 本校家長教師會已經成立多年，宗旨是加強家長與學校的聯繫及溝通，幫助兒童愉快地成長。恆常活動：聖誕聯歡會、親子旅行等。
2. 家長教育方面，每年安排不同的講座，讓家長明白教導子女之技巧。
3. 家長義工活動：伴讀家長、午間小遊戲、檢查學童體溫、校園種植。
校風 : 1. 協助學生建立「可信人」的價值觀，包括：H-成為一個誠實可靠﹔S-懂得關愛互助﹔P-言行有禮﹔S-靈巧學習的學習者。（HSPS為本校英文名之第一個字母）
2. 鼓勵學生積極學習，也重視學生在德育方面的發展。
3. 設立閱讀獎勵計劃、家長伴讀活動：鼓勵學生閱讀不同類型之書籍，充實個人知識。
4. 設立大哥哥大姐姐計劃：鼓勵學生幫助他人，照顧同伴。
5. 本校於午膳時段之後，輪流安排全校學生參與校園電視台之表演，鼓勵學生展示個人特質，提升學生自信。
學校發展
學校發展計劃 : 1. 提升學生的學習動機及興趣：推行優化課堂教學計劃、深化小班教學、家長教育、提升學生自信心計劃等等。
2. 提高學生英文科語文能力，以助學生將來儘快適應中學之學習環境，活動包括：每日網上閱讀、閱讀馬拉松、筆友會等等。
教師專業培訓及發展 : 1. 教師積極參與課堂優化計劃、小班教學、培養共同備課及同儕觀課之文化，藉此提升學與教之效能。2. 設教師專業發展日、教師分享日及不同範疇之教學進修，讓教師互相交流，促進教學成效。
學費減免 : 不適用。
其他 : 1. 可風可信合作委員會定期召開會議，制訂兩校合作之項目，並跟進本校學生之成績表現。
2. 幼小銜接：訪校日，暑假適應課程及興趣班。
3. 中小銜接：荃葵青區中學觀校日及兩次升中家長講座晚會。
4. 小一開放教室，每年3月會開放小一課堂，家長親身感受上課情況。
5. 重視英語方面培育，提升學生的英語能力，課後開設英語拔尖班。
