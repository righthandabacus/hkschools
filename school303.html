辦學宗旨
傳道服務，愛心關懷。有教無類，全人教育。積極進取，勇於承擔。
我們相信：「教養孩童，使他走當行的路，就是到老，他也不偏離。」（箴言二十二章六節）每個人都是按着上帝的形象被造，各有獨特與尊貴的地方。藉着基督教教育能使人去惡從善，發揮天賦潛能，榮神益人。
學校特色
學校管理
學校管理架構 : 由法團校董會管理，校長領導學校發展及行政組、各功能小組及學科小組推動學校發展及日常校務。
法團校董會/學校管理委員會/校董會 : 成立於2008年8月30日
環保政策 : 推行現場派飯，減少浪費食物及使用飯盒。設園藝小組及一人一花活動，鼓勵學生綠化校園，愛護植物。
學校關注事項 : 1. 促進學生身心靈健康，肯定自我價值。從認識自我到完善自我，強化身心靈健康，發揮潛能，建立自信。2. 創設多元學習活動，經歷愉快學習旅程。透過多元化學習活動提升學習興趣，多參與，多分享，知識生活化。
教學規劃
學習和教學策略 : 課堂採用小班教學及小組學習，並運用分層課業，及/或按能力分組學習，照顧個別差異；課後提供多元活動，發展潛能；推行全方位學習，擴闊視野及學習經歷；透過電子媒介，在課堂上即時展示學生學習成果，增強師生及生生互動等，即時回饋學生，加強學生課堂的投入感，促進教與學的效能。
關鍵項目的發展 : 1. 閱讀計劃：設早讀、家長伴讀、閱讀策略訓練，推動配合主題的閱讀，課堂上教授英語拼讀技巧，鼓勵閱讀。
2. 資訊科技：融入各科課堂，並利用軟件及網上功課自學。
3. 專題研習：根據各級專題研習能力學習目標，培養共通能力。
4. 德育及公民教育：以聖經真理為基礎，推行品格教育課程及建立關愛文化；運用早禱、週會、品格成長課、節期活動及全方位輔導，培養個人良好品格，深化國民及公民教育，達致服務他們，建立關愛校園的文化。
共通能力的培養 : 透過專題研習、培養學生溝通、協作及研習等能力；運用色彩思維，培養學生創造力及批判能力。
學生支援
全校參與照顧學生個別差異 : 1. 品格教育：營建彼此欣賞和接納的校園文化。
2. 情意教學：培養學生樂觀、正向情緒及信念。
3. 資優課程：推行增潤課程及分組教學；提升高階思維能力：培訓體藝校隊及科探先鋒，發揮學生創意及潛能。
4. 支援措施：設立功課輔導班、特殊學習需要的訓練小組。
課程剪裁及調適措施 : 設增潤課程，如英語音樂劇、各級樂器培訓；視乎學生的特殊需要，訂立支援計劃，進行學習及評估調適。
家校合作及校風
家校合作 : 每年舉行家長接見日、親子運動會、旅行及家訪。透過家教會，成立親子閱讀組、家長義工隊，協助及支援校內外全方位學習活動；亦成立不同的監察小組，監察午膳、校服校車及書簿等供應商的服務。此外，與學校教會協辦家長班，教導家長如何在學業、品格及靈命上培養子女成長，亦讓家長間彼此分享及支持。
校風 : 1. 關愛滿校園：培養良好品格，家校親子活動，家長伴讀，實踐「一人一服務」，學長計劃支援小一新生適應，朋輩輔導，同學間彼此幫助，自勵計劃。經常表揚有良好品格、成績優異或學業有進步的學生，讓學生得到認同及肯定，同學間亦能互相激勵。
2. 學習無疆界：讓孩子經驗多元化及全方位的學習歷程，並透過不同的學習策略及方式，實踐主動學習，從而建立自律自信。
3. 服務獻關懷：培養領袖、樹立榜樣；匯聚資源，回饋社區；培育孩子，貢獻社會。
學校發展
學校發展計劃 : 一、促進學生身心靈健康，肯定自我價值︰
1. 認識自我，找出自己需要改進或提升的地方。2. 強化身心靈健康，為自己訂立未來的目標和方向。3. 讓學生發揮所長，盡展潛能。
二、創設多元學習活動，經歷愉快學習旅程︰
1. 不同能力的學生樂意參與課堂活動。2. 主動學習，樂於分享。3. 主題式學習。4. 具備高階思維能力。5. 電子化學習。
教師專業培訓及發展 : 強化團隊精神，優化課堂教學， 關注個別差異，培養健康心靈。
學費減免 : 不適用。
