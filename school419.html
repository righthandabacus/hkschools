辦學宗旨
著重全人的教育，培養學生的責任感，使成為對社會有用的公民及虔誠的基督徒。
學校特色
學校管理
學校管理架構 : 校長下有四個不同的行政小組：學校行政、課程、學生訓輔及學生活動及支援，負責管理不同範疇。
法團校董會/學校管理委員會/校董會 : 校董會及學校管理委員會。
環保政策 : 重視環保，愛護地球。致力培育學生將來成為綠色公民。透過多元化的環境教育課程與活動，使學生欣賞自然，關注環境保護。建立美麗綠色校園，讓學生在理想的環境下學習。
學校關注事項 : 1. 從做中學，研新求進；
2. 學生學習差異跟進。
教學規劃
學習和教學策略 : 課程靈活安排。除中文科外，數學科、音樂科、教理科、電腦科及常識科以英語學習，視藝科及體育科以廣東話為主，輔以英語教學。一至六年級活動教學。
關鍵項目的發展 : 1. 圖書課及早讀計劃，鼓勵學生多閱讀，增廣見聞，訓練思維；
2. 一至六年級遊學團結合專題研習，擴闊學生的視野，發展批判性思考；
3. 宗教教育、公民教育及輔導教育整合為生命教育；
4. 在行常課堂及宗教課加入"靜念"教育。
共通能力的培養 : 1. 跨學科學習；
2. 批判性思考；
3. 情緒智商。
學生支援
全校參與照顧學生個別差異 : 駐校學生輔導主任與老師共同照顧不同學習能力的學生，彼此包容，組織和諧校園。提供各種類型校內校外活動， 強化學生的認知和溝通能力。
課程剪裁及調適措施 : 優化中、英文科校本課程。
家校合作及校風
家校合作 : 家長義工隊到校協助學習活動。 家教會協助學校舉辦多項學術及文娛活動，家校溝通良好，合作愉快。
校風 : 1. 辦學團體道明會發源於西班牙；小學部的文化：根在香港，面向世界。
2. 校風純樸，學生自信、自律、自重。
3. 學生喜歡學校， 師生關係良好， 亦師亦友。
學校發展
學校發展計劃 : 1. 四語學習語境（中、英、普、西班牙語），為學習打好基礎；
2. 課程發展：品德及學業並重，跟進生活技能培訓，培養學生自重自律自學精神；
3. 家長教育：鼓勵家長參與校政，強化親子溝通，家校合作，互相支持。
教師專業培訓及發展 : 發展合作學習；跨境學習；教學策略；教學相長。
學費減免 : 不適用。
