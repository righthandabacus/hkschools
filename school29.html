學校特色
學校管理
學校管理架構 : 本校自1996年起實行校本管理，建立自評文化。法團校董會下設有家長教師會、校董教師諮議會、校務委員會、各科組及功能組，彼此互相配搭，共同管理學校。
法團校董會/學校管理委員會/校董會 : 法團校董會具代表性，成員包括各界專業人士、教師、家長及校友。
環保政策 : 本校以減少廢物、珍惜資源為切入點，提供各式各樣的學習機會，鼓勵學生更積極地肩負起保護珍貴自然資源的責任。
學校關注事項 : 1. 優化學習策略，提升學與教的效能；
2. 培養學生積極正面的價值觀，提升其自律自信自強的生命素質；
3. 引發正能量，讓師生家長以生命影響生命，共建和諧校園。
教學規劃
學習和教學策略 : 提供多元化的學習活動讓學生參與。學校推行全方位學習，學生透過各類比賽：運動、音樂、朗誦、藝術，以及聯課活動、社會服務、參觀、探訪和考察等，累積學習經驗，發展多元智能。此外，推行小一銜接課程、跨學科活動、英文識字計劃 (PLP-R/W)、STEM教育、電子學習、專題研習、及生命教育。
關鍵項目的發展 : 四個關鍵項目已融入各學習領域中，閱讀獎勵計劃及專題研習已成為恆常學習活動。現著重提升學生的資訊科技、高階思維、自學能力及推行生命教育。
共通能力的培養 : 本校以專題研習協助學生發展其共通能力，提供機會讓學生整合及應用各學習領域或跨學習領域的知識。
學生支援
全校參與照顧學生個別差異 : 為個別學生作情緒及行為輔導，推行自律達人獎勵計劃，鼓勵積極求學，自我完善；舉辦成長小組，推行大哥哥大姐姐計劃；舉辦家長親職講座，提升家長管教子女的技巧。此外，為學生提供多項課後延伸活動，讓學生發展潛能。
課程剪裁及調適措施 : 設功課調適指引及不同程度的工作紙，協助學生學習。
家校合作及校風
家校合作 : 本校設家長校董，參與學校管理，提升教育質素。家長教師會、家長日及各級家長會能增強家校溝通及合作。
校風 : 學校校風純樸，洋溢關愛文化，老師團隊精神佳，各持份者均愛護支持學校。本校連續四年獲頒關愛校園獎。
學校發展
學校發展計劃 : 三年發展計劃的重點 (2015-16, 2016-17, 2017-18)：
1. 優化學習策略，提升學與教的效能；
2. 培養學生積極正面的價值觀，提升其自律自信自強的生命素質；
3. 引發正能量，讓師生家長以生命影響生命，共建和諧校園。
教師專業培訓及發展 : 1. 延展課堂研習風氣，提升學與教效能；
2. 以集體備課、同儕觀課及協作教學，提升學與教質素。
學費減免 : 不適用。
其他 : 1. 資訊科技： 第四個資訊科技教育策略 - 加強學校的無線網絡基礎設施 (WiFi900)
2. 數學科： 率先推動電子學習
3. 體育科：「簡易運動計劃（網球）」
4. 聖經科：「老有所為活動計劃」
5. 視藝科：「創意學生獎勵計劃」
6. 加強小學常識科的科學及科技計劃（STEM教育—科技實驗教室）
