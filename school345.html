班級教學模式
配合教育局發展方向，全方位推動照顧學生個別差異。在教學上推行師生互動及生生互動的教學模式、著重學生的學習經歷，以培養學生自主學習及群性發展。
班級結構備註
設有早操、晨光閱讀、校園廣播、成長課及多元化課外活動。開放電腦室、閱讀室及自學活動區，提供課餘活動。
學習評估
全年全科測驗次數 : 2
全年全科考試次數 : 2
多元學習評估 : 除測考外，學生進行自評、同儕互評及家長評估。本校透過專題研習、觀察、生活體驗、資料搜集、口頭匯報、小組討論及訪問活動進行評估，重視學生均衡學習。
分班安排 : 三年級開始按學習能力需要。

體育：田徑隊、籃球、足球、泳隊、羽毛球、武術、小型網球。音樂：合唱團及手鈴隊。視藝：視藝創作室。另設校本服務團隊、公益少年團、領袖訓練、女童軍、幼童軍及小童軍訓練。
辦學宗旨
提供理想的學習環境，著重學生五育均衡發展，培育學生建立健康的自我形象和正確積極的人生觀。學校致力發展學生的潛能，培養學生自學的精神，讓學生成為樂於學習、善於溝通、勇於承擔、敢於創新、貢獻社會的良好市民。
學校特色
學校管理
學校管理架構 : 採取全校參與的科層管理模式，共同訂定學校發展計劃及關注事項，具備理想透明度及問責性。
法團校董會/學校管理委員會/校董會 : 一九九九年九月正式成立「學校管理委員會」，各成員均積極參與，致力校本教育改革。
環保政策 : 德育公民組帶領全校推行校園綠化計劃及環保教育，推行一人一花，慳電行動，提倡珍惜資源，愛護地球。
學校關注事項 : 推廣校本資優教育，發展學生尊重互信的正面價值促進健康愉快的校園生活。
教學規劃
學習和教學策略 : 1. 發展資優教育，照顧學習多樣性，豐富學生學習經歷，提供不同機會發揮潛能。
2. 中文科：一至三年級推行「喜閱寫意」強化學生語文基礎；三至六年級推行普教中課程。
3. 英文科：外籍英語教師協作教學，提升學生讀寫能力。
4. 數學科：重點發展學生解難及自學能力。
5. 常識科：通過深究活動和專題研習等，促進學生共通能力。
6. 德育及公民教育：透過生命教育，讓學生學習尊重和盡責。
關鍵項目的發展 : 1. 各科滲入資優教育元素。
2. 全方位戶外學習日，跨學科主題式學習活動，專題研習，擴闊學習經驗。
3. 透過校本輔導計劃培養學生良好品格。
共通能力的培養 : 發展校本課程，課堂活動著重幫助學生建立不同的共通能力；學校亦透過跨學科主題式學習活動、專題研習及境外交流學習活動等，為學生提供與人溝通、協作、解難及創作等機會，從而提升學生的共通能力。學校亦為學生提供服務學習機會，學習領導及協作。
學生支援
全校參與照顧學生個別差異 : 全校參與學生支援計劃：設立支援小組，推動扶平及補底教學，包括課餘增潤、啟發體藝潛能、全方位輔導服務等，有效照顧個別差異，提高學習能力及自信心。
課程剪裁及調適措施 : 透過有效的科研及課研工作，從集體備課中設計多元能力的教學活動及課業，調節教學策略，提升學生學習效能。
家校合作及校風
家校合作 : 家教會自九三年成立，為家校溝通的橋樑。家教會配合學校發展，推行多元化的校內外活動，與學校緊密合作，共育幼苗。
校風 : 通過正面的鼓勵，提升學生的自我形象，讓學生在充滿愛心和關懷的環境下，建立正確和正面的人生觀。
學校發展
學校發展計劃 : 為學生提供多元化學習經歷，發掘及發展他們的潛能。
教師專業培訓及發展 : 全校老師參與「資優教育」培訓，加強教師推動學生自主學習的專業發展。
學費減免 : 不適用。
其他 : 注重教研工作，配合學校發展需要，參與校本專業支援計劃，引入教育專業團體，優化學校課堂教學，追求卓越。
