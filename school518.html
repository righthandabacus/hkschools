辦學宗旨
本校矢志延續天主教會對教育的貢獻，維護五大核心價值：真理、義德、愛德、生命、家庭，將有關價值傳授給學生，為他們日後立身處世作準備。致力提供全人教育，讓孩童在關愛的環境下，愛主愛人，盡展所長，且能獨立自主，具備多元能力和國際視野，保育地球，學校成為力臻完善的學習社群。
學校特色
學校管理
學校管理架構 : 設有法團校董會，並在學校行政架構中，成立八大委員會，包括：學校發展委員會、學生發展委員會、學術發展委員會、教師專業發展委員會、財務及資源管理委員會、人事管理委員會、學生事務委員會及功能組別委員會，由校長帶領各委員會計劃和施行各項校政，並集體承擔，推動學校使命。
法團校董會/學校管理委員會/校董會 : 法團校董會成員包括辦學團體校董、教師校董、校友校董、家長校董。校政執行委員會成員包括校監、校長、副校長、家教會委員、校友會委員及教師代表。
環保政策 : 參加藍天行動，設綠化校園計劃及校園清潔計劃。設種植組及生態組教導學生種植和育蝶知識，以全方位學習模式培育學生保護環境的態度，並透過廢物回收及廢物分類，學習低碳生活，珍惜資源。
學校關注事項 : 關懷接納顯愛心．自主樂學展潛能。
教學規劃
學習和教學策略 : 包括資優教育普及化、自主學習、教學策略多元化、全方位學習、促進學習的評估等，另設延伸學習活動，包括早讀、專題研習、服務學習及語境教學等。
關鍵項目的發展 : 1. 培養學生從閱讀中學習的能力。
2. 培養學生資訊科技能力及素養。
3. 培養學生專題探究的能力。
4. 透過集會短講、戲劇、升旗禮及成長課等活動，培養學生德育及公民教育的意識。
共通能力的培養 : 透過各學科的課堂及課外的學習活動，如專題研習，全方位學習、辦論和討論等，培養學生九種共通能力。
學生支援
全校參與照顧學生個別差異 : 幼小銜接課程、小六與中一的銜接措施、功課輔導、區本計劃、輔導教學、陽光計劃、個別學習計劃、高飛計劃、超級高飛計劃、人才庫計劃、言語治療。
課程剪裁及調適措施 : 評估政策。
家校合作及校風
家校合作 : 本校家長教師會自1995年成立，擁有龐大的家長義工隊伍，協助學校推行各種活動。校董會更有家教會的代表，參與校政，增加學校校政透明度。
校風 : 1. 注重學生的品德培養，藉宗教氣氛的薰陶，透過不同的訓輔活動，培養學生正確的道德觀念。
2. 舉辦傑出學生、模範生選舉、班際清潔比賽等，培養學生遵守紀律及合作精神。
3. 鼓勵學生參與社會服務，如：探訪老人院和賣旗等活動。
學校發展
學校發展計劃 : 1. 學校電子學習試驗計劃。
2. 普通話教中文計劃。
3. 數學科導向學習評估。
4. 資優教育－人才庫先導計劃及領袖計劃。
5. 健康校園。
6. STEM（科學、科技、工程及數學）計劃。
教師專業培訓及發展 : 教師專業成長獎勵計劃、發展日、工作坊、研討會、課堂實踐、行動研究、協作教學、共同備課、同儕觀課、參觀、交流、分享及反思等。
學費減免 : 不適用。
其他 : 學校致力實踐天主教五大核心價值──真理、義德、愛德、生命和家庭，培養學生成為未來的領袖。學校十分重視發展學生的個人潛能，為他們安排「學藝薈萃耀油天」表演、 服務等多元化的課外活動。學生不獨積極參與校內外的活動和服務，也踴躍參加多項社區和校際的學術和體藝比賽。他們在各項校外活動及比賽中的表現優異，其中在朗誦、舞蹈、游泳、足球及音樂方面屢獲殊榮，表現尤為突出。
