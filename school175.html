班級教學模式
小一至小六推行小班教學，不設精英班。
班級結構備註
教學模式多元化，著重兩文三語的培育，並以培養學生的共通能力及發展全人教育為目標。

分班安排 : 各級均以能力混合模式分班，並為有需要的學生提供加強輔導教學。

健康校園生活 : 培養學生飲食均衡的習慣， 着重 「身心靈」的健康成長， 並推廣「勤做運動」計劃。 課前及小息設童樂角，讓學生進行閱讀、 跳繩、呼拉圈、跑步等活動。
備註 : 利用早會及成長課進行品德教育。設輔導班及拔尖補底課程，經常舉辦全方位活動，擴闊學生視野。營造校園關愛文化，設「每月壽星」。參與不同社區服務，如探訪老人院、賣旗、義工服務等。舉辦環保週，提高學生環保意識。
全方位學習活動
逢星期五舉辦課外活動，除戲劇、童軍、基督小先鋒、校園小記者等團體活動外，學生輪流參加不同活動組別，培養多方面興趣；放學後及週六設不同興趣小組（如舞蹈、鐘鈴、科學班、跆拳道、小結他、豎琴、非洲鼓、舞獅、話劇、動感聲藝及各種球類活動等）供學生自由參加。
辦學宗旨
本校秉承天主教辦學的理想，以「以樂事主」為校訓。在「愛」的薰陶下，致力培育學生的全人發展，使他們成為一個有責任感，熱愛公益，關心社會的好公民。
學校特色
學校管理
學校管理架構 : 天主教教育事務處、法團校董會、校長、學校發展小組、各行政及科務小組。
法團校董會/學校管理委員會/校董會 : 法團校董會成員共15人，包括校監（神職人員）、校長、校外人士、家長、校友及教師代表等。
環保政策 : 將環保知識融入各科課程，設環保大使推行環保活動，並設有綠色飯堂。
學校關注事項 : 1. 利用多元化的教學策略，培養學生的自主學習精神。
2. 發展天主教教育的五大核心價值，培養學生正確價值觀及良好品德。
教學規劃
學習和教學策略 : 透過協作伙伴、優質學校在香港計劃、提升小學英語計劃及「悅讀‧悅寫意」計劃等，推展校本課程。依據健康校園政策，各科課程配合培育學生「身、心、靈」健康發展。秉承天主教教育的理念，以「義德、愛德、真理、生命、家庭」為核心價值，透過一系列全校參與模式的活動，並配合「怡雅彩虹獎勵計劃」，培養學生良好品德及正確價值觀。
透過「怡雅校園電視台」，為學生提供多元化的學習平台，發展校本影像教材，增加學生學習興趣，提升學生兩文三語能力；推動德育及公民教育，發揮學生的多元智能；加強培育學生的各種共通能力，以提升學與教質素。
關鍵項目的發展 : 1. 推行多項計劃，提高學生閱讀興趣。
2. 以「義德、愛德、真理、生命、家庭」為核心，培養學生品德情意，推動關愛文化。
3. 課室設電子白板，善用資訊科技進行學習，逐步推行電子學習。
4. 藉專題習作，培養學生探究及自學精神。
共通能力的培養 : 透過推動校本課程、校本課業設計、專題研習、多元化的課堂內、外活動及參加跨境學術交流活動等，培養學生發展不同的共通能力。
學生支援
全校參與照顧學生個別差異 : 設學生學習支援小組，為有需要的學生提供適切的校本支援，如課前、課後功課輔導班、託管班，另有讀寫困難，言語治療，情緒培育等小組培訓課程，為學生、家長及教師作全方位支援。
課程剪裁及調適措施 : 各科組進行不同的計劃，專題研習以跨科協作模式推行。定期進行共同備課，並有多項協作計劃及校本課程。部分級別在中文及英文科進行同級分組教學。P.5，6英文科成績優異的學生，由外籍英語老師教授英文拔尖課程。本年度試行運用AQP的數據分析，以達至「評估促進學習」。
家校合作及校風
家校合作 : 與家長教師會緊密合作，舉辦不同的家長講座及親子活動，校長每月與家長舉行座談會，交流教養子女心得。
校風 : 樹立淳樸校風，重品德培育及兼容互愛的校園文化，設不同服務生，制服團體等，培養學生勇於為別人服務的品格。以郭怡雅精神為培育學生的目標：待人友善、關愛他人、言行得體、靈巧學習。
學校發展
學校發展計劃 : 1. 運用「怡雅校園電視台」推動德育與公民教育及提升學生兩文三語能力。
2. 推展「怡雅彩虹獎勵計劃」，以天主教五大核心價值培育學生正確價值觀及良好品格。
3. 延續參加教育局主辦的「悅讀‧悅寫意」支援計劃。
4. 延續參加教育局主辦的PLPR/W計劃。
5. 落實小班教學政策。
6. 增加語境教學機會。
7. 評估促進學習。
8. 逐步推行電子學習。
教師專業培訓及發展 : 1. 培訓老師有關照顧個別差異、自主學習、小班教學的教學能力和技巧。
2. 參加伙伴協作計劃，交流教學心得，推展校本課程。
3. 鼓勵老師進修課程，推動電子學習及專科發展。
學費減免 : 不適用。
其他 : 1. 與教育局、大學教育機構及教育團體合作推行學與教計劃，提升教學效能。
2. 跨境學術交流，增加學生多方面的識見。
3. 邀請不同機構到校為老師、學生及家長舉行講座。
4. 設獎學金制度：學業獎、進步獎、中英數學科獎，另為畢業生設積極進取獎。
