班級教學模式
以活動教學為學習模式，學生從探究活動中學習，提高學習的興趣，內化和鞏固所學，啟發學生的潛能。學生學習，以小班形式進行，以達「拔尖補底」的目標。
班級結構備註
課堂進行小組學習，營造高參與、多互動的學習環境，學生有效學習。
學習評估
全年全科測驗次數 : 2
全年全科考試次數 : 1
多元學習評估 : 多元化的評估模式，例如國際性考試，以及持續的進展性評估、學業獎勵計劃等，目的瞭解學生學習需要，鼓勵學生自主學習。而校內校外活動紀錄、義工服務紀錄等評估學生整體發展。小一全年沒有測考。
分班安排 : 一年級原班直升二年級，三至六年級學生按學業表現分班，並在高小的中、英、數課堂進行分組學習，加強照顧學生學習的需要。

辦學宗旨
本校五育並重，校訓為「博愛行仁，省善修德」。「藉賞識啟發學生潛能，以關愛諦造快樂人生」為理念。「著意為學生創造成功機會，培養學生自律，自尊，自信精神」為辦學目標。提供優質教育，達致全人發展。
學校特色
學校管理
學校管理架構 : 法團校董會以下，由陳志雄校長帶領，將學校管理架構劃分為四大範疇，分別為校本管理、學與教、校風與學生支援、課外活動，每組均設組長。
法團校董會/學校管理委員會/校董會 : 於2009年8月31日成立法團校董會。
環保政策 : 26℃以上開空調。所有文件雙面印刷。實施環保班房政策。舊衣、廢紙及膠樽回收，建設綠色校園。聯合國教科文組織可持續發展教育項目實驗學校。魚菜共生計劃。水耕種植計劃。
學校關注事項 : 藉賞識啟發學生潛能，以關愛締造快樂人生。
教學規劃
學習和教學策略 : 推行小班教學，運用小組學習，照顧學習差異。老師施行難點教學，共同備課、同儕觀課，強化學與教的成效。利用數據促進學習，瞭解學生學習的需要，評估教學成效。一至六年級設特色體育課程，如游泳、壁球等，聘請專業教練配合校外場地，發展學生的潛能。推行其他學習經歷，將學習經驗融入課堂內外，進行跨學科專題研習，全方位活動和境外學習。
關鍵項目的發展 : 推行全校早上閱讀活動和閱讀獎勵計劃，營造閱讀的風氣，培養學生閱讀的習慣。學生應用資訊科技進行自主學習。透過電腦科校本課程，學生運用平板電腦學習，提升運用資訊科技的能力。學生持續進行網上自主學習課程，培養自主學習的習慣。推行專題式研習活動，規劃每級學習的研習技能，培養學生探究精神及共通能力。德育及公民教育滲入各個學科課程，以及每星期進行「領袖培訓課程」、「ICAN全人教育」課程和相關的學習活動，培養學生正面思維，提升心理素質，達致積極樂觀的人生觀。
共通能力的培養 : 透過「領袖培訓課程」、級議會、專題研習和網上自主學習課程等，皆能有效強化學生的協作、創造、研習、資訊科技及自我管理能力。
學生支援
全校參與照顧學生個別差異 : 全班式和尖子抽離式資優培育課程，以及加強輔導班，照顧學生的學習差異，讓學生發展潛能。
課程剪裁及調適措施 : 利用分組及分層課業等，照顧學習差異。與區內的幼稚園及中學緊密溝通和協作，合辦幼小及中小銜接課程，讓學生盡早適應新學習階段。配合《幼稚園教育課程指引》，著重幼小銜接課程，調適功課、默書和評估的政策。配合六年級級議會的議決，設計中小銜接課程，為升中作好準備。
家校合作及校風
家校合作 : 以「開放校園」政策，鼓勵家長陪伴子女共渡校園生活。每年進行召募，龐大的家長義工人數已超過二百多人，更榮獲全港「有心學校」亞軍。家長協助水運會、旅行、聯歡會、參觀日等約30多項活動，共同發揮家校合作的精神。舉辦家長講座，提升家長心理素質、管教技巧及親子關係，與家長攜手培養協助子女成長。
校風 : 連續十二年榮膺「ICAN學校」及成為「模範ICAN學校」榮譽。此外連續10年榮獲關愛校園，又連續9年榮獲有心學校榮譽，老師秉持「啟發潛能教育」的信念，相信學生各有所長，並以尊重、信任、樂觀、關懷及透過5P的刻意安排，傳達關愛，發揮學生潛能，提升自信心。採用「ICAN」心理輔導課程，提升學生心理素質，達致成功、進步、快樂人生。本校更連續兩屆榮獲「啟發潛能學校成就大獎」。
學校發展
學校發展計劃 : 我們是一間愉快學習學校「Happy School」，「上午專心，下午探索」。在上課的下午時段，我們讓學生動手探究科學科技課程，有編程和機械、水火箭、魚菜共生等等，激發學習的興趣，照顧學習的多樣性，發展專才。我們有最長的午膳，學生午膳後，可以做功課，亦可以跟同學一齊玩遊戲，培養溝通和社交能力，享受校園生活。
教師專業培訓及發展 : 配合學校發展的需要，與教育局和校外機構協作，引入校外專業支援，推動教師專業發展活動，讓教師能與時並進，促進專業知識，優化教學效能。進一步以本校的成功經驗，與其他學校建立交流的平台，優化學習社群，提升教育團隊素質。本校著力發展教師備課和觀課的文化，建立互相觀摩文化，設科本教師分享時段，優化課堂教學，提升學與教的成效。
學費減免 : 不適用。
其他 : 本校推行啟發潛能教育和賞識教育，從不同途徑擴闊學生的視野，豐富學習經歷，發展潛能，增強自信：推薦參加教育局、香港資優學苑和大學的課程；鼓勵學生參加公開試和全港性比賽，學生多次取得代表香港的資格，到外地參加國際性比賽，數學科比賽有台灣、成都、馬來西亞、韓國等地，機械人比賽有維也納、羅馬尼亞、中國鞍山、韓國和美國；每年舉辦遊學團，曾到韓國、新加坡、澳洲和意大利遊學。
