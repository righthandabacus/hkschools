班級教學模式
學校利用多元教學策略，以照顧學生不同的學習模式。我們以學生為中心，透過小組學習形式以完成共同目標。我們鼓勵學生主動學習，使用探究方法，讓學生從做中學，從而建構知識。
班級結構備註
本校教學的理念是以人為本，以生命改變生命；全面照顧孩子，培育成材及優化課堂教學，提升學習效能。

2. 總結性評估︰測驗、考試。
分班安排 : 本校不按學生學業成績編班，每班水平相若。

健康校園生活 : 本校參加衞生署之「開心果日」、「陽光笑容滿校園」活動、香港中文大學之「滋味營養教室」、「新力軍培育」計劃，兒童脊科基金之「護脊校園計劃」。
備註 : 全體教職員為學生提供一個愉快的學習環境，靈活運用全日制學習時間，透過多元智能課增加師生溝通及相處機會。下午進行「學藝四重奏」，包括按能力分組進行拔尖補底課程、午間樂繽紛、學科新天地、資優培訓、STEM及天主教社會倫理課。
全方位學習活動
制服團隊：男女童軍、紅十字會、基督小先鋒、香港交通安全隊、香江茶道、少年警訊及升旗隊。 興趣班：球類、英語會話、流行鼓、管弦樂及敲擊樂團。 校隊資優培訓：球類、田徑、游泳、奧數、視覺藝術及歌詠團。
辦學宗旨
我們致力建設學校成為一個讓學生愉快及成功學習的園地，讓學生在德、智、體、群、美、靈方面得到均衡發展。我們接受學生的個別差異，願以因材施教的方式發展學生的潛能。
學校特色
學校管理
學校管理架構 : 以香港學校表現指標的四個範疇為行政架構的基礎，透過教師、科組及學校三個層面促進學校持續發展，在發展及成長中透過分析、計劃、監察及檢討的自評。
法團校董會/學校管理委員會/校董會 : 法團校董會15人，9位辦學團體校董，另設校長、2位教師、2位家長、1位校友及1位獨立校董。
環保政策 : 1. 本校獲「環境及自然保育基金」資助設置環保設施。
2. 在「環保校園，環保生活」的熏陶下，學生從小培育愛護環境的美德。
學校關注事項 : 自學自理自主求學問 和諧友愛精英顯效能
教學規劃
學習和教學策略 : 本校透過教育局「電子學習試驗計劃」、「電子學習學校支援WiFi900計劃」、優質教育基金「動感藝術教室」計劃進一步優化學校資訊科技教學環境，促進學生自主學習、創新合作的能力。為配合未來社會發展，本校設立STEM課程，結合科學、科技、工程和數學教育，透過應用程式編寫和「3D光雕」等課程中培養學生的創意、邏輯思考和解難能力。
本校已打造全新遊戲室，現存桌上遊戲已超過百款，包含了圖板、卡片、骰子、紙筆等類型。我們把桌上遊戲的元素融入日常課堂，在「學藝四重奏」時段加入桌上遊戲課堂，致力訓練學生的計算與解決問題能力、專注力、忍耐力、判斷力和社交能力。
關鍵項目的發展 : 1. 校本課程︰本校自1998年起推行校本課程，以學科為本，透過評估促進學習，深化優質課堂教學，加強學生自主學習，提升學習表現。
2. 關愛文化︰本校致力培養學生正確的價值觀和自律精神，建立正面和諧的校園文化。
3. 多元智能培育︰本校開展「一生一體藝」計劃，透過提供不同學習經歷的機會，拓寬學生的視野，培養學生個人的終身興趣。
共通能力的培養 : 本校進行的跨學科學習活動以宗教科天主教核心價值，透過專題研習、語境教學、講座、多元化的學習活動，建立學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 與大專院校合作進行課程設計、制訂照顧學生個別差異計劃、資優課程、支援小組、成長的天空、加強輔導教學計劃，照顧不同能力學生的需要。
課程剪裁及調適措施 : -
家校合作及校風
家校合作 : 定期舉行各項活動，並透過家長教師會成立愛心義工小組，增強家校合作的關係，提高家長及學生對學校的歸屬感。
校風 : 1. 本校校風淳樸，重視品德教育，採用「訓輔合一」的理念，積極透過班級經營策略推動「關愛文化」，培育學生健全人格。為小一學生推行多項「幼小銜接」活動。
2. 本校定期舉辦家長日、研討會及親子等活動，充分發揮家校合作成效。
學校發展
學校發展計劃 : 創建理想自學環境，提升學生自我管理能力，營造和諧共融校園，使學生從學習活動中有效學習。
1. 自學環境共創建，理想校園同愛護。
2. 持續增強學生自我管理能力，從而提升學習成效。
3. 和諧校園共融友愛。
4. 深化各校隊培訓活動，使學生從學習活動中獲得更有效學習。
教師專業培訓及發展 : 本校透過教師發展日、科本培訓、課堂研究、同儕觀課等作為發展學習型組織的平台，讓教師彼此共同學習，提升學教質素。
學費減免 : 不適用。
