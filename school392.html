班級教學模式
資優教育普及化；小一至小六推行小班教學；著重兩文三語：小一英文推行“DTS Programme” 、小二英文推行IELP課程、小一至小四普教中。
班級結構備註
除了正規課程外，設班主任課及「Go 高 Goal」拔尖補底課程；每週有週會、成長課及多元活動課。配合課程需要，亦進行全方位學習。
學習評估
全年全科測驗次數 : 2
全年全科考試次數 : 2
多元學習評估 : 設有進展性評估，以默書、小測、測驗及專題設計等形式進行；專題設計採用多元評估方式，包括學生自評及互評；中英文及普通話科設有口試及聆聽試；亦為有學習需要學生安排評估調適。
分班安排 : 各年級均以平均能力編班。

全方位學習活動

辦學宗旨
以基督為模範，致力傳揚福音，讓學生認識並履行天主誡命。秉承「勇、智、仁」校訓精神，培育學生道德勇氣、認知發展及仁愛品德。關注教職員專業成長，加強與家長聯繫，共同努力，協助學生認識祖國、關心全人類福祉。
學校特色
學校管理
學校管理架構 : 由法團校董會管理、並由校長帶領學校發展小組，發展及監察校務，由2位副校長及各中層行政人員負責不同範疇的行政工作。
法團校董會/學校管理委員會/校董會 : 校董會成員包括校監、校長、教育界人士、教師、家長、校友及其他界別人士。
環保政策 : 1. 榮獲全港第二屆綠色學校冠軍，實施「減少、回收、再用、替代」措施。
2. 設太陽能發電板，以作環境教育。
3. 進行有機耕種，以提升環保意識。
學校關注事項 : 持續義德、真理及生命元素，並致力培養學生效法基督的愛德，以仁愛待人，養成顧己及人的良好品格，與別人建立和諧的關係。
1. 培育學生仁愛品德，關愛家人及朋友
2. 公平相待學生，持續照顧學生的學習需要
教學規劃
學習和教學策略 : 1. 重視語境教學，推行小學英語教學計劃及普教中，提升學生兩文三語的能力。
2. 利用多元化的教學策略，如合作學習及啟發式教學等，激發學生的創造力及自主學習能力，並透過專題研習，培養學生共通能力。
3. 積極推行小班教學，強化學生的溝通能力，並於中、英、數、常的課堂進行課研，藉此提升學與教的效能。
關鍵項目的發展 : 1. 設中、英文閱讀課、閱讀獎勵計劃、好書推介活動，營造閱讀氣氛。
2. 把資訊科技融入各科，設網上學習平台，並發展電子學習。
3. 全方位學習，組織學生進行境外交流活動及專題研習，擴闊學生知識領域，發展共通能力。
4. 透過宗教培育及「靈訓輔」政策，培育學生正面價值觀。
共通能力的培養 : 實施資優教育普及化政策，透過課堂適切的學習活動設計、合作學習的應用、專題研習及科學探究的活動，發展學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 1.以「全校參與模式」照顧學生學習需要。2.設學生支援組，支援有學習需要的學生。3.為「尖子」舉辦抽離式資優培育課程。4.設課前學科輔導。5.設加強輔導班。6.設小一至小六課後功課輔導計劃。7.運用學習支援津貼舉辦多元學習課程。8.為有需要學生提供言語治療服務。9.推行「成長的天空」抗逆計劃。
課程剪裁及調適措施 : 1. 為有學習需要學生安排功課及評估調適。
2. 透過分層工作紙，協助學生學習。
家校合作及校風
家校合作 : 我們十分重視家校合作，除了舉辦家長日外，也舉辦家長教師交流會，並透過各年級的家長會、親職講座、家長學堂、家長教師會、家校活動，家長義工隊及家長觀課，增強家長與校方的溝通。
校風 : 校風淳樸，師生關係融洽，學生互愛共融，獲頒關愛校園獎勵計劃之「關愛校園」榮譽。
學校發展
學校發展計劃 : 本校實踐天主教教育的五大核心價值，並融入校訓精神，培育勇智仁兼備的「柴天人」。
教師專業培訓及發展 : 教師積極進修，持續與外界進行專業交流，定時進行共同備課、觀課及課研。
學費減免 : 不適用。
其他 : 1. 榮獲香港考評局頒發「優質評核管理認證計劃」證書（2011-2019）。
2. 著重英語培育，學生英語能力不斷提升，並在校際戲劇節（英語組）四年榮獲「評判推介演出獎」，成效可見。
3. 重視幼小銜接，為幼稚園學生設「小跳豆/ 小荳芽英文課程」。
4. 重視中小銜接，為小五、小六學生設週六增潤課程。
5. 「活力校園在柴天 開心學習每一天」是我們的目標。
