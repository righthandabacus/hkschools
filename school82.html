班級教學模式
小一至六以合作學習模式推行小班教學，達到優差並進，照顧學習差異。此外，配合多元化的教學活動，包括各科的校本課程、教授閱讀及思維策略、分組教學、跨學科學習、專題研習等，提升學生的學習效能及自主學習能力。
班級結構備註
按教育局學位分配需要，來年小一將會增至每班學生33人。本校仍會以合作學習模式推行小班教學為主，以照顧學習差異，達到優差並進，保持教學質素。

多元學習評估 : 除紙筆評估外，本校透過專題研習、課堂觀察、小組討論、口頭匯報、日常課業、自我評估、同儕互評等多元化評估，促進學生學習效能。每次考試之間進行不定期之進展性評估，並會善用相關數據，回饋學與教。
分班安排 : 按學業成績、操行等第及男女生人數平均分班。

辦學宗旨

學校特色
學校管理
學校管理架構 : 法團校董會監督，由校監、校長領導各功能組別推行學校發展工作。
法團校董會/學校管理委員會/校董會 : 由辦學團體成員、校長、教師及家長代表組成。
環保政策 : 全校參與減碳、各類回收、重用等環保實踐： 另有校園種植，讓學生親身接觸大自然。
學校關注事項 : 1. 改善教學範式，提升學業水平。
2. 加強學習技能訓練，學會學習。
3. 以愛維繫，建立關愛校園。
教學規劃
學習和教學策略 : 1. 小班教學，運用合作學習，為學生設計高動機、高參與、高展示及高回饋課堂，增加學生課堂投入感及成就感；教師定期進行共同備課、同儕及考績觀課、教學分享會及研討會。
2. 本校參加教育局支援和協作計劃，香港大學教育大學，香港中文大學和香港教育大學的支援計劃，以優化課堂教學設計。
3. 善用評估數據，審視學生的學習難點，制訂適切的教學策略及設計分層課業。
4. 運用多層次提問技巧，發展高階及多角度思維，讓學生享受探究事物的樂趣。
5. 加強學生自評及互評，使其監測自己的學習能力，反省學習方法。
關鍵項目的發展 : 1. 為資優學生安排拔尖課程；為成績稍遜學生提供適切的支援服務。
2. 舉行「課堂表現獎勵計劃」，培養學生積極的學習態度，強化其自我形象及提升自信心。
3. 推行「圖書閱讀獎勵計劃」，以個人閱讀歷程方式進行，由學校加以獎勵及表揚閱讀成果。
4. 推薦及鼓勵學生參加校外課程或比賽，提升自信。
5. 出版學生文集，以鼓勵學生創作及展示成果。
共通能力的培養 : 透過課內外活動、專題研習等訓練學生的各項共通能力。
學生支援
全校參與照顧學生個別差異 : 小一及早識別計劃，輔導成績稍遜者，為有特殊需要的學生作課程調適及學習支援。
課程剪裁及調適措施 : 評估回饋教學、課業設計、拔尖保底輔導、考試調適。
家校合作及校風
家校合作 : 設旅行、家長學堂、親子賣旗、義工家長服務、家長團契和為新生舉辦輔導日暨家長會。
校風 : 以愛凝聚，締造關愛校園，促進和諧、關愛的校園氣氛，提升師生對學校的歸屬感。
學校發展
學校發展計劃 : 1. 改善教學範式，提升學業水平，運用資訊科技提升教學效能。
2. 加強學習技能訓練，學會學習，以適應社會變遷及環境。
3. 以愛維繫，建立關愛校園，促成學生身心靈健康發展。
教師專業培訓及發展 : 配合關注事項及學校發展計劃，加強小班教學策略及照顧個別差異。
學費減免 : 不適用。
其他 : 設立獎學金：
1. 長江實業地產有限公司「長和獎學金」；
2. 聖公宗（香港）小學監理委員會「傑出學生獎」；
3. 聖公會奉基千禧小學家長教師會「進步獎獎學金」；
4. 聖公會頌主堂「頌主堂獎學金」。
