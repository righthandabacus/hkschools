班級教學模式
推行小班教學，採用全方位學習，如參觀、專題研習、合作學習、自主學習預習案、協作教學等學習模式。以新資助模式支援學習，設有中、英、數小組課後輔導學習。另獲新鴻基慈善基金贊助功課輔導計劃。
班級結構備註
本校由2012-13年度開始進行小班教學，以合作學習形式及普及資優教育照顧不同學習需要的學生。小二升小三及小四升小五分班。

多元學習評估 : 全年有四次考試，並輔以課堂觀察、日常課業批改及專題研習等方式作為評估工具，藉以了解學生學習情況。語文科特別注重於聽說讀寫能力的考查，務求達到「以評估促進學習」的目的。
分班安排 : 1. 依男女人數平均分班。

辦學宗旨
培養學生德、智、體、群、美五育均衡發展，使學生自強不息，充滿信心，從而能自律、自立、貢獻社會，服務人群。
學校特色
學校管理
學校管理架構 : 法團校董會之下，由校長及兩位副校長（行政、學術）領導十個行政小組（教務、總務、資訊科技、訓輔、活動康體、學生事務、學生支援、課程、圖書、德育及環境教育）及十一個學科小組（中、英、數、常、普通話、圖書、德公、電腦、視藝、音樂、體育），以推行校政及制定發展策略。
法團校董會/學校管理委員會/校董會 : 法團校董會於零七年九月成立，主席為校監，成員包括有五位辦學團體校董，一位獨立校董，一位當然校董（校長），兩位教師校董及兩位家長校董。另有四位榮譽校董及八位名譽校董。
環保政策 : 全校按既定的環保政策，實踐綠色生活。積極參與環保活動及綠色校園計劃等 。
學校關注事項 : 1. 透過多元的課程設計及教學策略，培育學生成為自主學習者。 2. 共同營造及鞏固MCYOS精神，展現「任安達人」的特質。
教學規劃
學習和教學策略 : 1. 中、英讀寫結合課程、英文電子寫作平台等校本課程，配合語境，並透過「雙語校園」活動，提升學生兩文三語的能力。數學科推行數學化教學，提升學生的解難能力，藉科探活動培養學生的探究精神。2. 善用多元化的教學策略及學習活動，加強課堂的互動性，培養學生自主學習的精神、溝通及協作能力。3. 推動多元化評估模式，針對學生的學習難點，提供有效的回饋。4. 透過主題學習週、全方位學習活動及境外交流，豐富學生的學習經歷。5. 推行普及資優教育，發掘學生潛能，照顧學生的多樣性。6. 以「翻轉教室」鞏固學習。
關鍵項目的發展 : 1. 本校加強推行資訊科技教學，促進課堂互動，擴闊學習的延伸性，培養學生自學能力，並灌輸正確運用資訊科技的素養。2. 深化校本價值教育，把MCYOS精神滲透於正規及潛在的德育及公民教育課程內，培養學生擁有「自律、自主、自強」的任安達人優良特質。3. 積極配合STEM的教育重點：發揮學生的創意潛能，籍此裝備自己以應對未來的挑戰，加強數學科與常識科在推展STEM的角色。4. 強化從閱讀中學習的效能：推行圖書課、早讀、親子閱讀、故事媽媽及閱讀獎勵等計劃，提升學生的閱讀興趣。
共通能力的培養 : 1. 透過全方位學習模式及引進合適的創意思維教學策略，如「逆向/擴散思維」、「腦圖」等，以激發學生的創造力。2. 運用高階思維教學策略，提升學生的批判性思考能力。3. 訂定全年學習主題，進行專題研習及跨學科學習活動，提升學生的溝通、協作、研習、解難及自理能力。
學生支援
全校參與照顧學生個別差異 : 為有不同學習需要的學生提供全校參與模式的支援輔導，包括課後輔導、朋輩支援及課外活動小組，並獲新鴻基慈善基金資助課後功課輔導及精英培訓計劃，致力照顧學生個別需要並發展專長。
課程剪裁及調適措施 : 透過課程調適、評估調適、學與教策略及精英培訓計劃，拔尖補底。
家校合作及校風
家校合作 : 家長義工隊全力支援學校事工，連續多年獲得全港性義工嘉許獎項；並設「任安家長學堂」、家長工作坊及聯誼活動，家校關係融洽。
校風 : 提倡關愛文化，學生守紀有禮，師生關係良好。
學校發展
學校發展計劃 : 1. 提升學與教的效能，培育學生成為自主學習者。
2. 展現「任安達人」的楷模，提升學生自律、自主、自強的能力。
3. 優化校園環境，啟發學生潛能。
4. 推行多元智能及學會制活動。
教師專業培訓及發展 : 透過教師專業培訓、教師學習圈、共同備課、同儕觀課及友校交流計劃，提升教師專業水平。
學費減免 : 不適用。
其他 : 引進外界支援：
1. 校本支援服務計劃：校本中文讀寫結合課程、數學語言教學法。
2. 教育局：Space Town（英文科）。
3. 姊妹學校：新加坡崇福學校、新加坡維新學校、北大附中深圳南山分校。
4. 與國際學校協作活動。
5. 成立「校長學習社群」、「常識科教師學習社群」及「外籍教師學習社群」，與友校進行專業交流。
