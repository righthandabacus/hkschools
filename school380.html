備註 : 本校星期五設課外活動課，課後設有學藝班、個別輔導計劃、課後學習及支援計劃和各校隊的培訓，另有每年一次的遊學活動。
全方位學習活動
本校設多項課外活動及學藝班，發展學生的興趣和潛能。學術方面有奧數、劍橋英語班等。體藝方面有球隊、合唱團、花式跳繩、步操樂團及中國舞班等。服務隊伍有幼童軍、交通安全隊、公益少年團及環保大使等。
辦學宗旨
秉承校訓，致力為學生提供一個愉快的學習環境，以培養學生在德、智、 體、群、美各方面的全面均衡發展，為社會培育富有自信、有責任感及有貢獻的良好公民；並以「學校家庭同攜手，開拓孩子好將來」作為辦學宗旨。
學校特色
學校管理
學校管理架構 : 學校管理委員會主席為教育局副秘書長，成員包括校長、教師代表兩名、家長代表兩名、社會人士兩名及校友會代表一名。
法團校董會/學校管理委員會/校董會 : 本校於1999年度成立學校管理委員會，推行校本管理，成員包括教育局官員1名、校長1名、教師2名、家長2名、社會人士2名校友1名。
環保政策 : 建設綠色校園，天台綠化及有機種植，為學生提供一個學習保護環境的地方。
學校關注事項 : 1. 推動自主學習，照顧學生的學習多樣性。
2. 深化價值教育，提升學生的品格素質。
教學規劃
學習和教學策略 : 舉行跨科學習活動，使學生掌握不同學習經歷，培養各項共通能力。透過科技探究，加強學生掌握批判思維能力。全校推行自主學習及合作學習，提升學與教的效能。
關鍵項目的發展 : 設早讀課及閱讀獎勵計劃以提升學生閱讀興趣。設主題學習及專題研習活動，為學生提供應用共通能力的機會。設生活教育課，加強德育及公民教育的培育。在不同科目結合資訊科技教學，並提供網上自學平台，促進學生互動學習，提升學與教效能。
共通能力的培養 : 透過合作學習的課堂活動及專題研習活動，發展學生溝通、協作、解難、研習、處理資訊等共通能力。
學生支援
全校參與照顧學生個別差異 : 按學習能力分組，促進課堂上的協作學習效果。低小推行朋輩輔導，協助學習能力稍遜的學生。設課後功課輔導，支援有特殊學習需要的學生。
課程剪裁及調適措施 : 安排多元化活動及課程，以提升資優學生能力：如奧數班及劍橋英語班、各項體藝訓練等。
家校合作及校風
家校合作 : 舉辦家長教育講座及親子活動，另設家長義工服務，以加強家校合作。
校風 : 1. 以全方位的活動，為學生營造富有關愛的校園文化，增進歸屬感。
2. 透過「成長的天空計劃」協助高小學生提升抗逆力。
3. 藉著多元而靈活的課程架構，促進學生健康成長。
學校發展
學校發展計劃 : 1. 推行全方位學生輔導計劃，積極培育學生品德修養和公民責任。
2. 推行小班教學，推動自主學習，深化合作學習，提高學與教的效能。
3. 建立完備的校本課程，加強照顧學生的學習多樣性，以提升學生的學業水平。
教師專業培訓及發展 : 支持教師參與專業培訓課程，透過校內專業培訓，提升教師對教育發展的關注，鼓勵共同備課、同儕觀課及於科務會議內分享教學心得。
學費減免 : 不適用。
