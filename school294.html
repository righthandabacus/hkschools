班級教學模式
以協作及小組輔導課的教學方式，採用教育局新資助模式，支援學生學習。
班級結構備註

學習評估
全年全科測驗次數 : 2
全年全科考試次數 : 2
多元學習評估 : 本校於各科推展不同類型的進展性及總結性評估，以檢視及改善教與學成效。
分班安排 : 小四至小六按成績編班，全年總積分較高的學生編入A、B兩班，其餘學生平均分配C、D、E班內。
小一至小三不按成績編班，採用混合編班制。

健康校園生活 : 本校提倡健康生活，透過健康飲食在校園計劃，推行健康飲食運動，並透個講座及活動，宣傳健康訊息。
備註 : 本校按衞生署「小學午膳營養指引」選擇合資格的供應商，培養學生良好的飲食習慣，並設專責委員會定期評估。
全方位學習活動
童軍、交通安全隊、田徑、羽毛球、壁球、獅藝、舞蹈、陶瓷班、合唱團、管弦樂團、銀樂隊、戲劇和英語班、數學精英培訓班、辯論隊等。此外，還有教育性參觀、旅行、運動會和生活教育營等，為學生提供均衡的學習機會。
辦學宗旨
讓兒童愉快及有效地學習，發展個人潛能，在德、智、體、群、美各方面作全人發展，能夠終身學習，日後對社會、國家和世界作出貢獻。
學校特色
學校管理
學校管理架構 : 校長領導下，共有2位副校長，以及校務主任、訓輔主任、學生輔導主任、學藝活動主任、行政管理系統及資訊科技主任、兩位課程主任、學務主任及學生支援主任共9位主任，其餘老師被編配協助以上各組別的主任以協助各類教學及行政工作。
法團校董會/學校管理委員會/校董會 : 本校於一九九九年成立學校管理委員會，由教育局首長級官員擔任主席，其他成員包括校長、教師、家長及社會人士，定期舉行會議討論校政，提升辦學質素。
環保政策 : 學校透過不同的渠道及舉辦不同的活動，向學生宣揚環保訊息，並定期評估環保措施的果效。
學校關注事項 : 1. 優化學與教策略，深化自主學習。
2. 培養學生的健康生活習慣，創設機會發展學生潛能，達致學生身心均衡的發展。
3. 加強價值教育，培養學生積極、承擔及關愛的態度。
教學規劃
學習和教學策略 : 透過多元化的參觀及考察，豐富學生的學習經歷；跨學科的主題學習活動，讓學生從探究中更全面獲取知識﹔推動電子教學，與時並進提升學生學習興趣及教學效能。
關鍵項目的發展 : 推行閱讀計劃，使學生從閱讀中學習；推行電子學習，為學生提供網上學習平台及以多媒體輔助教學； 進行專題研習活動，訓練學生研習能力；透過各類德育活動，培養學生良好的品格，加強公民意識。
共通能力的培養 : 於常識科推行專題研習及STEM活動，培訓學生共通能力，發揮學生自學和科學探究精神。
學生支援
全校參與照顧學生個別差異 : 運用及早識別策略，拔尖補底照顧學習多樣性，加強中、英、數學習效能。
課程剪裁及調適措施 : 舉辦富趣味性的學習活動，如小記者培訓班、奧數訓練、科學探究、體藝培訓等資優課程；校方亦開設輔導班，推行協作教學，聘請言語治療師等，為有需要的學生提供支援。
家校合作及校風
家校合作 : 家長教師會舉辦講座及親子活動，加強家長與學校之間的和諧伙伴關係，並支援學校推行學習活動。
校風 : 推行成長課，發展關愛文化；鼓勵學生參加各科活動及擔當服務生以提升歸屬感。
家長日及家長教師座談會加強家校溝通。
小一銜接課程和小老師計劃，安排高年級同學協助低年級同學學習。
學校發展
學校發展計劃 : 2015-2018關注事項：
1. 優化學與教策略，深化自主學習。
2. 培養學生的健康生活習慣，創設機會發展學生潛能，達致學生身心均衡的發展。
3. 加強價值教育，培養學生積極、承擔及關愛的態度。
教師專業培訓及發展 : 學校安排教師於固定時段進行共同備課和觀課，又與教育局合作進行共同備課，以提高老師的教學效能，並邀請學者為教師主講有關教學的議題及安排老師出外參觀和交流。
學費減免 : 不適用。
其他 : 設 「傑出學生選舉」獎勵在學行、服務、體藝方面有卓越成就的學生。
而 「自我挑戰獎勵計劃」、 「陸麗鳳校長獎學金」、 「謝麗華校長獎學金」、 「校友會獎學金」則獎勵學業成績有進步及服務方面表現卓越的學生。
