分班安排 : 設一班能力較強，一班普通話班，其餘兩班能力混合。

辦學宗旨
學校著重「愉快有效學習」，引導並培養孩子：認真學習，用心思考，快樂共處，愛己愛人。
學校特色
學校管理
學校管理架構 : 秉承保良局校董會指示，在校長領導下，以副校長及中層去領導各科組。
法團校董會/學校管理委員會/校董會 : 已成立法團校董會，共有13位成員，包括辦學團體校董、當然校董、教員校董、家長校董及獨立校董。
環保政策 : 有自我評估及自我完善的環保政策，逐步走向綠色環保學校。
學校關注事項 : 巧‧克‧力：巧於學習、克服困難、力求進步
教學規劃
學習和教學策略 : 有正規課程及全方位學習課節。以特定主題配合教學發展，透過資訊科技、戶外參觀及課外活動，培養共通能力。
關鍵項目的發展 : 設閱讀獎勵計劃，培養閱讀習慣。電腦網絡完善，加強電子學習教學，提升學生運用資訊科技的能力。注重跨學科發展，加強專題研習及議題探究能力。以德育及公民教育課程建立學生正面的價值觀，以及照顧學生全人成長需要。
共通能力的培養 : 每週設綜合課程，為學科知識提供綜合學習空間。透過不同活動，發展多元能力。
學生支援
全校參與照顧學生個別差異 : 透過拔尖保底進行分流教學以照顧學習差異，因應學習取向而進行調適。提供課後學習支援計劃、學校有駐校教育心理學家及校本言語治療師，為有需要的學生提供服務。又為能力卓越的學生提供課後資優課程。
課程剪裁及調適措施 : 中文科及英文科按學生需要完善課程，提高學生聽說讀寫的能力；數學科則進行課程調適，提升運算及解應用題的能力，並加入高思維訓練及運用電子學習教學，亦設有校本拔尖保底課程。
家校合作及校風
家校合作 : 家長積極參與家校活動，支援學校發展。家長義工隊得到教育局外評隊的讚賞。
校風 : 本校著重品德教育，校風純樸，學生品行優良，並透過德育及公民教育活動、服務學習及同儕互助計劃建立關愛文化。
學校發展
學校發展計劃 : 完善管理架構及職務，制定各項政策及程序手冊。課程以自學去提升學生聽、說、讀、寫及探究能力。著重教師發展及培訓，教師已通過有關語文能力評核，提供優質教學。學校已逐步建立自我問責文化，開展自我評估政策。
教師專業培訓及發展 : 全體教師於每年會因著關注主題及學科發展而進行不同的校本專業發展活動及培訓，並進行互相觀摩交流分享。
學費減免 : 不適用。
其他 : 每年進行學習交流計劃，與內地筆友通信並締結「姊妹學校」。同時在多項校外比賽中取得卓越成績。
