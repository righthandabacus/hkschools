學校特色
學校管理
學校管理架構 : 學校由馬鞍山循道衛理小學法團校董會管理。
法團校董會/學校管理委員會/校董會 : 校董會成員包括10位辦學團體校董、校長、2位教師、2位家長、1位校友及1位獨立人士。
環保政策 : 委派事務主任為資源監督，監督校內能源之使用，包括節省用紙、善用電源、減少用水及鼓勵減少廚餘等。
學校關注事項 : 學習愛．愛學習
教學規劃
學習和教學策略 : 在上課期間安排不同學習及訓練活動。在課後安排各項興趣班、校隊訓練及為學生提供拔尖補底活動。學生透過課堂教學、閱讀計劃、專題研習及課外活動，提升學生九種共通能力。以腦基礎學習(BBL)的教學策略來促進課堂中的互動以維持學習動機。
關鍵項目的發展 : 1. 透過閱讀獎勵計劃鼓勵學生自學。
2. 設計校本電腦課程，並鼓勵學生進行網上學習。
3. 設立校本專題研習課程，五、六年級學生均須到境外進行資料搜集。
4. 發展基督教生命教育課程，培養學生正面積極的人生。
共通能力的培養 : 透過學生協助帶領週會活動、主領午間廣播、專題研習及跨科目的活動，培養學生共通能力。
學生支援
全校參與照顧學生個別差異 : 設有輔導教學及資優課程，照顧學習差異，在課堂安排協作教學及小組學習。此外，另設拔尖小組，如：精靈一族、科學探究、奧數訓練、中英伴讀大使等啟發學生潛能。
課程剪裁及調適措施 : 為有特殊教育需要學生提供課程調適，並為資優學生提供小組訓練。
家校合作及校風
家校合作 : 透過不同活動加強家校合作。每年協助學校各種工作的家長義工達300人。家教會每年均舉辦家長講座、親子營、親子旅行，並選出兩位成員加入校董會參與校政。2011年度被頒發「卓越關愛校園」之「最關愛家校合作」獎及「最具創意服務計劃」義工行動獎。
校風 : 著重學生品格的培育，培養學生正確的價值觀。
1. 設計生命教育課程，培養學生正確的價值觀念，建立良好自尊感和良好的自我形象。
2. 安排多元化的課外活動及比賽訓練，提升學生對學校之歸屬感。
3. 透過建立互信的家校合作關係，發展學校工作，建立緊密的伙伴關係。
學校發展
學校發展計劃 : 提高學生品德素養及自學能力，關注學生、家長及教師的身心靈。
教師專業培訓及發展 : 每年按學校周年發展計劃制定校本教師發展計劃，提升學與教效能。
學費減免 : 不適用。
其他 : 1. 本校設有獎學金制度：
1.1 德行獎勵：紀念黃作牧師獎學金。
1.2 學業獎勵：家長教師會獎學金。
2. 每年均設境外學生交流3-4團。
3. 平均每學年獲校際比賽獎項（包括個人及隊際）約160項。
