全方位學習活動
乒乓球、足球、奧數班、魔術班等。學校每年均舉辦各種不同的活動讓學生參與，例如遊學團（校方會安排六年級學生參加），既可寓學習於娛樂，又可擴闊視野。另外，敬老會讓學生幫助無依低收入的長者，了解社會中的弱勢社群。
辦學宗旨
聖若望英文書院以教育出未來傑出人才為使命。以勤勉的態度、不懈的努力及追求和對學生及校友的承諾來實現這一使命。 在課堂上，我們追求出色的學業水平，快速的學習，通過轉換新的理念來促進學生形成創意的理解思考能力以及獲得卓越的教育經驗。
學校特色
學校管理
學校管理架構 : 由校長領導行政組，並由行政主任統籌及策劃。
法團校董會/學校管理委員會/校董會 : 法團校董會成員由辦學團體代表。
環保政策 : 讓每個成員，包括管理層、全體教職員、學生、家長攜手合作，採取各種善用資源的措施，避免浪費，並將環保信息融入生活中，推廣環保教育。
學校關注事項 : 除學習成績外，重點培養學生之品德，思考能力，批判精神等，實施全方位教育。
教學規劃
學習和教學策略 : 除日常課堂外，會為學生安排各種切合課程之參觀、講座、工作坊。
關鍵項目的發展 : 德育與公民教育融入各科中，適時施教，生活化活用教材，培養學生之正確價值觀和高崇的人格，讓學生成為一個有承擔的好公民。
共通能力的培養 : 透過課堂學習，活動（工作坊、講座成長課程等）的參與，培養學生之共通能力。
學生支援
全校參與照顧學生個別差異 : 聖若望英文書院通過持續觀察學生的課堂表現、課後功課、學業及日常行為評估等方式來跟進所有學生的進展程度。學校會將學生對知識的掌握情況以及需要改善的地方及時匯報給家長。
課程剪裁及調適措施 : 為外籍學生自訂中文教材。
家校合作及校風
家校合作 : 我們非常重視老師與家長的聯繫，因為家校之間的合作對兒童的學習成長非常重要。為促進教師與家長的交流，教師會定時與家長聯絡，匯報學生的學業進展和日常行為表現。學校也會定期組織家長日，家長可與班主任單獨面談，便於家長了解學生突出問題，解決方法和學生的總體表現。
校風 : 聖若望英文書院將繼續保持我們已故創始人梁博士的辦學理念，結合今日世界的需求標準，為年輕人提供教育，與時俱進。作為國際化社會中小小的一個團體，我們將繼續為每一個學生創辦高品質的教育，向學生灌輸良好的價值觀，技能和知識學科。
學校發展
學校發展計劃 : 我們的課程以三語教學為核心，重視語言、數學、常識、信息技術、體育、藝術及音樂多範疇的發展。我們的教師具有專業資格，教學水平優異，能夠幫助學生打下良好基礎，並為以後的發展做好準備。
教師專業培訓及發展 : 建立共同備課、同儕觀課文化，並且持續進行優化課堂教學及舉行教師專業發展活動，提升學與教效能。教師亦積極參與各種工作坊、交流活動及校外支援計劃，以促進專業成長。
學費減免 : 不適用。
其他 : 我們提供很多課外活動，例如訓練學生參加香港學校音樂及朗誦節，普通話比賽等。
