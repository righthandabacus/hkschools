辦學宗旨
秉承保良局「保赤安良」的宗旨，以「愛、敬、勤、誠」 為校訓。我們著力透過愉快有效的學習，為兩文三語奠下良好的基礎，以達致「德、智、 體、群、美」五育並重之全人發展。鼓勵學生關注社會，培養民族感情和責任。
學校特色
學校管理
學校管理架構 : 管理架構包括校董會、行政小組及各學科小組
法團校董會/學校管理委員會/校董會 : 保良局西區婦女福利會馮李佩瑤小學法團校董會。
環保政策 : 發展成為綠色學校、涵蓋教師及學生學習生活層面。並致力通過不同的課程及活動鼓勵學生推廣節約及循環再用。
學校關注事項 : 1. 優化課堂學習，提升學與教效能。
2. 培養學生積極學習態度及自學能力。
3. 培養學生良好品德，建立佩瑤風範。
教學規劃
學習和教學策略 : 中文科因應學生的需要，推行有效的校本課程；英文小一至小三推行PLP-R/W、小四至小六推行SURE及電子學習。數學滲入高階思維課程；常識推行專題研習、天文與科技。透過生活教育課推行健康教育、環保教育及德育與公民教育。透過不同類型的學習活動，多元智能活動及境內外交流活動讓學生盡展所長、拓寬視野。推行校本STEM課程，培養學生創造力和解決問題的能力。
關鍵項目的發展 : 從閱讀中學習：推行早讀、中、英文廣泛閱讀獎勵計劃、中、英文篇章閱讀及時事評論、並透過網上閱讀活動培養學生自主學習能力。專題研究：常識科每年均會進行專題研習，引導學生進行資料搜集、整理、歸納，完成報告後須向全校同學作匯報，以培養學生共通能力。此外，學校透過各學科、生活教育課、班級經營、境內外交流、講座、週會及服務學習活動推行德育及公民教育，以培養學生良好品德。資訊科技科更會教授學生編寫電腦程式。
共通能力的培養 : 透過各科課程、專題研習及多元化的學科活動培養學生共通能力。
學生支援
全校參與照顧學生個別差異 : 學校設有學生支援小組、言語治療師、教育心理學家及社工提供全面的支援服務。另外，學校亦設有朋輩輔導及課後輔導班以收學生補底之效；各科精英培訓計劃可進一步提升學生的潛能。本校又設「成長的天空」課程和各類成長小組，以推行情意教育。
課程剪裁及調適措施 : 推行校本英語閱讀及寫作課程、說話訓練及高階思維訓練課程等。將學生按能力分組，進行拔尖補底課程。
家校合作及校風
家校合作 : 本校設有家長教師會，定期舉辦親子活動、家長興趣班及家長專題講座，以提升學生及家長對學校的歸屬感。學校亦設有家長義工小組，定期到校協助學校推行不同類型的特別學習活動、參觀及比賽等以支援學校，促進家校合作。
校風 : 推行關愛校園，利用隱蔽課程及校本輔導活動，進行德育培訓，培養學生高尚的情操；發展多元化課外活動，增加學生自信及對學校歸屬感；通過組織家長教師會、家長日、功課輔導班及小一新生適應安排，加強家校合作，達致和諧關係。
學校發展
學校發展計劃 : 本校低年級以合作學習模式推行小班教學，推動學生積極交流及互相合作學習；而高年級會推行STEM教育及電子學習，培養學生的創新、自主學習精神。學科上，中文科以普通話教授中文；英文科於小一至小三推行PLP-R/W，小四及小六推行SUPPORTED READING；常識科發展校本天文課程，以訓練學生探究及自學精神。本校亦透過跨學科及多元化活動推行德育及公民教育，並引入攀石、劍擊、欖球及拉丁舞等活動，讓學生學習更多姿多采。
教師專業培訓及發展 : 為了優化學與教，老師會選讀合適的培訓課程，並進行教育研究。除每年設有三天校本教師培訓外，亦會舉辦科本培訓、境外考察交流及主題式教師發展項目。
學費減免 : 不適用。
其他 : 電腦機械人隊於國際賽事中屢獲殊榮。拉丁舞隊、中國舞隊於學校舞蹈節獲得優異成績，並曾被邀出席優勝者表演暨頒獎禮。而話劇組亦曾獲評判推介大獎，及被邀請作公開演出。田徑隊於區運會及全港運動會取得多項優異成績。辯論隊、合唱團、劍擊隊、欖球隊及游泳隊分別獲多項冠、亞、季軍。本校在「惜水學堂」節約用水教育計劃中，獲得全港至尊大獎，並在多項全港性環保比賽中，亦屢獲殊榮。
