辦學宗旨
本校以聖公宗的基督教精神辦學，着重全人教育，讓學生在德、智、體、 群、美、靈六育上得到均衡發展，活出「非以役人，乃役於人」的校訓。
學校特色
學校管理
學校管理架構 : 法團校董會-校長-行政人員-教師-職員-工友。
法團校董會/學校管理委員會/校董會 : 法團校董會。
環保政策 : 健康環保校園。
學校關注事項 : 1. 培養自學能力。
2. 提升學生自信。
教學規劃
學習和教學策略 : 1. 推展「普教中」計劃，一、二年級全級及三至六年級各兩班透過普通話教授中文，另一至三年級推行「PLPRW」計劃，提升學生兩文三語能力。
2. 透過全方位學習、專題研習，提升學生學習興趣和效能。
3. 推展校本資優教育課程，照顧學生學習的需要。
4. 透過境外學習活動，擴闊學生視野。
關鍵項目的發展 : 1. 透過提升學與教，讓學生認識及實踐自學策略，培養自學能力，以達致終身學習。
2. 透過不同的學習活動，建立學生正面的自我形象，加強責任感，培養承擔精神，發揮個人潛能，以提升學生自信。
共通能力的培養 : 九大共通能力，透過課堂及課堂以外，全面培養。
學生支援
全校參與照顧學生個別差異 : 統籌照顧資優或學習上有不同需要的學生，設立資優課程及加強輔導班，並調適功課量及評估形式，成立伴讀小組等。
課程剪裁及調適措施 : 為學生制定課程調適政策、課後小組輔導、課後功課輔導、英文增潤課程、精英閱讀獎勵計劃、資優教育計劃。
家校合作及校風
家校合作 : 家長教師會宗旨是效法耶穌「非以役人，乃役於人」的榜樣，加強家長與學校的聯繫，促進家長與教師之間的夥伴關係，培養家長之間的友好關係和增進學生的福利，讓他們在六年的小學生活更精彩和健康。本年度舉辦專題講座、周年旅行及舊校服回收等活動。
校風 : 1. 透過「生命教育」、「智慧語錄」、「成長的天空」等計劃及活動培養學生正確的價值觀及態度。
2. 推行「服務學習」，全校師生及家長積極參與義工服務。
3. 積極培養學生建立環保生活態度和習慣，提升學生對環境保護的注意，並身體力行地實踐綠色生活。
4. 定期舉辦中小學生及與內地學生交流活動。
學校發展
學校發展計劃 : 1. 培養自學能力。
2. 提升學生自信。
教師專業培訓及發展 : 舉辦教師專業發展日；專家支援；全校教師定期進行共同備課及同儕觀課。
學費減免 : 不適用。
