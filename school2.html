學校特色
學校管理
學校管理架構 : 聖保羅書院校董會
法團校董會/學校管理委員會/校董會 : 學校校董會按聖保羅書院校董會組成法例規定組成，共十六位成員。
環保政策 : - 電腦課本循環再用。
- 家長透過內聯網了解學校動向及資訊，當中九成以上的家長選擇透過內聯網簽收通告，節約用紙。
學校關注事項 : - 促進學校持續發展。
- 優化教學和學生學習。
- 強化學生對社區的關愛；照顧學習差異。
- 探索、開發和拓展學生在非學術領域的潛能。
教學規劃
學習和教學策略 : 學校以循序滲透方式，常識科以英語教學；然而，顧及學生的適應及能力，英語教學比重由低年級的25%逐年提升至高年級的75%。全方位學習活動於上課時間內進行，午膳後、課後及星期六均進行延伸學習。每週設有兩節聯課活動，以豐富孩子的學習經驗。
關鍵項目的發展 : 1. 閱讀計劃：每年舉辦多個閱讀計劃；家長教師會亦為低年級學生設午間「故事爸媽」時段。
2. 資訊科技：每星期一教節電腦認知課程，各科於課堂中廣泛使用無線上網系統教學，並鼓勵學生把科技應用於課業中。學校供給每位家長及學生內聯網戶口，加強家校溝通。
3. 專題研習：各科設有不同形式的專題研習。
4. 德育及公民教育：設有生命教育課及全年校本輔導活動計劃，鼓勵學生把愛和服務他人的精神從家裏帶到學校、社區、祖國，以及整個世界。
共通能力的培養 : 本校向來注重學生的全人發展，除課堂學習外，亦為學生提供多元化的學習途徑，發揮他們的潛能，讓他們從中學習協作、溝通、解難等共通能力，增強自信。
學生支援
全校參與照顧學生個別差異 : 訂立校本學生支援政策及學習支援小組，及早識別及幫助有特殊教育需要的學生。設有為「有特別學習需要」的學生訂定的評估措施。
課程剪裁及調適措施 : 各科在日常課堂著重訓練學生高層次思維；學校實施抽離式小組教學；學校又安排不同年級在中、英、數三科按程度分流進行小班強化教學；推行「自我提昇計劃」，讓具不同潛能的學生接受更專業和有系統訓練，課程包括科學探究、弦樂、聲樂、牧童笛、中國鼓隊、籃球、乒乓球、排球、辯論、戲劇及創意小荳芽。
家校合作及校風
家校合作 : 家教會於1995年成立，致力推動家、校合作，讓家長與老師之間，家長與家長之間能緊密聯繫，建立融洽和諧的關係，為孩子的成長共同努力。
校風 : 學校重視學生的品德培養，強調學生自律、自省，要學會「在適當的時間，適當的地點，做適當的事情」。本着基督愛人的精神，學校悉心培育學生熱心服務，關懷他人的良好態度。學校為學生規劃完整的德育課程，讓學生身體力行，從家中及學校做起，進而服務社區，拓展至關心國家和世界。學校重視每一位小朋友天賦才能，學生相處融洽，彼此互相愛護和欣賞，校園生活和諧愉快。
學校發展
學校發展計劃 : 1. 提升學生的學習機會。
2. 以四個關鍵項目來配合課程和教學法，並依從直接資助計劃要求。
3. 學校會持續地改善及發展以維持聖保羅書院的遠景和使命。
教師專業培訓及發展 : 為掌握及了解教與學、學科領域、資訊科技、管理技巧、教育政策等方面的最新趨向，老師會參與進修課程、工作坊、簡介會及教學經驗分享會。除了個人的專業進修課程，老師亦參與學校舉辦的培訓活動。
學費減免 : 本校設有學費全免或減免3/4、1/2、1/4學費助學金計劃（學習及校車車費資助）。有需要的學生(包括來自綜援家庭的學生及領取學生資助辦事處財政資助的學生)可提出申請學費減免。
- 新生可於開學前的2個月內遞交申請，其申請結果會在3個月內獲得通知。
- 本校學費減免計劃詳情包括家庭入息限額，請瀏覽網上版的超連結。
其他 : 獎學金：每年獲獎學生超過85名。
