辦學宗旨
本著聖公宗基督教全人教育的辦學宗旨，本校致力培育學生在德、智、 體、群、美、靈六育的發展。
1. 教導學生活出校訓「非以役人，乃役於人」的精神。
2. 推行優質教學，培養學生自學能力。
3. 啟發創意潛能，發揮學生多元智能。
4. 實踐品德教育，建立學生高尚情操。
學校特色
學校管理
學校管理架構 : 由聖公宗（香港）小學監理委員會屬下之法團校董會管理，校內設立行政、科目及各工作小組推展校務。
法團校董會/學校管理委員會/校董會 : 法團校董會已於2015年8月31日成立。
環保政策 : 本校積極參與環保工作，如各機構舉辦的環保活動、HK3030節能約章、舊衣回收比賽、清潔運動及售旗籌款。本校亦推動同學節約能源，每日自備手帕，並使用環保餐具，培養同學的環保意識。
學校關注事項 : 1. 以不同學習策略提升自主學習能力。
2. 配合課程及全方位學習，啟發創意及發揮多元智能。
3. 以獎勵計劃配合課程，發展「盡責承擔、尊重接納、以大我為重」的精神，成為「好學生、好兒女、好公民」。
教學規劃
學習和教學策略 : 1. 逢星期五下午為活動課或週會。
2. 每週三次閱讀課。
3. 專題研習以跨學科設計，旨在擴闊學生的學習經驗，提升學生的學習興趣，從而培養學生的探究能力。
4. 推行合作學習，加強照顧個別差異，以協助不同能力的學生提升學習效能。
關鍵項目的發展 : 1. 專題研習：旨在發揮學生自主學習精神，培養共通能力，探究精神及待人接物態度，幫助學生全人發展，提供多元化學習環境。
2. 閱讀計劃：培養學生的中、英文閱讀興趣；開展親子閱讀獎勵計劃。
3. 資訊科技科：全校裝置投影機及多媒體教學設備，並建立內聯網及校園電視台。
4. 全方位學生輔導：培養學生正確的價值觀及處事態度，並設有成長課，加強學生的品德教育。
5. 兩文三語：營造中、英及普的語言環境，讓學生多聽、多講、多讀及多寫。
共通能力的培養 : 透過正規課程及課外活動提升同學的共通能力，如解難能力、批判思考及正確的價值觀。
學生支援
全校參與照顧學生個別差異 : 學生支援小組統籌照顧資優及學習上不同需要之學生，為他們進行功課／評估調適，並設立學科增潤班、教學輔導、課後支援及部分級別之英、數科以平衡班（小組）上課等，以照顧學生不同的學習需要。
課程剪裁及調適措施 : 本校為有特殊教育需要的同學制定課程調適政策，如功課／評估調適。
家校合作及校風
家校合作 : 學校設有家長教師會，每年均為家長舉辦多元化的活動，與家長建立良好的伙伴關係，並設有家長義工，提升家長對學校的歸屬感。
校風 : 本校校風淳樸，以培養學生愛己愛人的品德，效法基督服務他人的精神為目標，深獲家長支持。
學校發展
學校發展計劃 : 1. 一至六年級英文科推行專科專教。
2. 一至三年級以普通話教授中文課。
3. 推行創造校園語言環境計劃，提升學生的英文、普通話能力。
4. 推行資優教育計劃。
5. 四至六年級推行教育局Keystage 2 Integration Programme，以提升學生英語閱讀及寫作水平。
6. 小一至小三推行教育局Space Town Literacy Programme。
7. 推行合作學習，提升學習效能。
8. 推行電子學習學校支援計劃(WiFi900)。
教師專業培訓及發展 : 學校積極推展教師發展及培訓活動，除每年三天的教師專業發展日，亦推動老師出外進修，並透過共同備課、同儕觀課、評課及專業分享會，以提升學與教的效能。
學費減免 : 不適用。
其他 : 學校舉辦境外交流活動，擴闊學生視野，增加學生課堂以外的學習經歷。
