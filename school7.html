辦學宗旨
本校秉承耶穌基督的精神，為學生提供優質的全人教育，使學生在靈、德、智、體、群、美方面有均衡的發展，並會致力營造充滿關愛的校園，使學生能發揮潛能、盡展所長、貢獻社會。
學校特色
學校管理
學校管理架構 : 法團校董會
法團校董會/學校管理委員會/校董會 : 於2014-2015年度成立。
環保政策 : 綠化校園；廢物分類回收。
學校關注事項 : 1. 提升學校團隊的身心靈健康。
2. 運用適切的學習方法，提升學生的學習效能。
3. 培養學生自理、自律、自覺、自學及自制的態度。
教學規劃
學習和教學策略 : 指導學生摘錄筆記和課前預習技巧，促進自主學習。利用分組活動及「合作學習」的教學策略以照顧學生的個別差異；編排雙教節，安排多元化的學習活動；導修及班主任課，照顧學生在學業和品德上的需要；全方位的學習和統整成長課和宗教教育課程，於不同學科發展資優課程及完善「人才庫」，讓老師能針對性地推薦尖子參加比賽及課程，使學生在不同範疇中能發展所長。
關鍵項目的發展 : 閱讀策略以螺旋式設計，貫穿各學習領域，推行不同的閱讀獎勵計劃，培養閱讀習慣。透過跨學科的專題研習，配合資訊科技進行資料搜集、分析及專題報告，讓學生更能妥善運用資源科技；德育及公民教育：建立關愛文化，服務他人的精神。安排「同根同心」國內交流活動以提升學生對國民身份的認同；組織海外學習交流團，擴闊學生眼界；科學週環繞科學探究主題，發展學生的探究能力。另外，更為資優學生提供適切的課程及學習支援，讓資優生更能發揮學習成果。
共通能力的培養 : 服務學習強調學生的參與，透過服務學習方案，有計劃的反思和討論，使學生發展批判性的分析和思考能力。發展學生的研習能力，運用資訊科技能力和自我管理能力。數學科有解難及非常規工作紙，語文科、常識科有思維訓練的工作紙或課堂分析、討論，藉此訓練學生的運算能力和解決問題能力。
學生支援
全校參與照顧學生個別差異 : 小二至小六課前或課後輔導班、小二、小四及小六英文科有加強輔導班、課後功課指導小組、週六小組補課。中、英、數、常拔尖課程和校本資優課程。
課程剪裁及調適措施 : 由教師及教育心理學家組成小組，研究照顧個別學習差異學生的方法，在課程和課業、課堂的分組活動中進行剪裁及調適，並配合評估，以提高教學的效度及提升學生的學習動機及自信。對於能力高的學生或個別學習領域中表現較強的學生，本校設拔尖及增潤課程，亦設有保底班以打好學生的基礎。
家校合作及校風
家校合作 : 本校設有家長教師會，定期舉行會議，加強家校合作及溝通。
校風 : 以「培養學生自理、自律、自覺、自學及自制的態度」為主題，營造純樸及守規的校風。
學校發展
學校發展計劃 : 1. 提升學校團隊的身心靈健康。
2. 運用適切的學習方法，提升學生的學習效能。
3. 培養學生自理、自律、自覺、自學及自制的態度。
教師專業培訓及發展 : 按教育局所推行的「學習的專業，專業的學習」進行。
學費減免 : 不適用。
