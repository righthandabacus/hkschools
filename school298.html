班級教學模式
本校有策略性對中、英、數三科學習有困難學生提供支援。一至六年級推行小班教學；三、四年級設中英文輔導班；五、六年級中、英、數三科按學生能力分組上課。
班級結構備註
一至六年級採取分班及合作學習模式。
學習評估
全年全科測驗次數 : 2
全年全科考試次數 : 2
多元學習評估 : 1. 各科以發展性評估與總結性評估考核學生能力及成績。
2. 學生課業加入家長評估及學生自評。
3. 提供網上學習資源，促進自學。
分班安排 : 按學科成績及操行表現。

健康校園生活 : 本校為健康促進學校，致力培養學生的健康飲食習慣及環保的意識，亦參與跳繩強心計劃及香港精英運動員協會活力學校計劃。
備註 : 每年參加衞生署「健康飲食在校園計劃」之生果日，並在校內舉辦素食日及清潔日，藉以提昇學生之健康及環保生活技能。本校亦組織「環保大使」小隊，在學習之餘推廣健康及環保訊息至家庭以及社區。
全方位學習活動
為拓展學生多元智能，學校開設不同的課外活動。周五下午及周六全校參與多元智能課；體藝培訓：足球隊、欖球隊、龍獅隊、泳隊、體操隊、中國舞蹈隊、小提琴隊、合唱團、非洲鼓班、小結他班；制服團隊：童軍、基督少年軍；興趣班：紙黏土、扭氣球及趣味英語。
辦學宗旨
1. 提供「德、智、體、群、美、靈」優質全人教育，使學生得到均衡發展。
2. 培養學生敦品力學的態度及勇於服務社群的精神。
3. 引導學生關心社會及祖國的發展，欣賞中華文化。
4. 營造愉快的學習環境，讓學生有多元化的學習經歷。
學校特色
學校管理
學校管理架構 : 基督教香港信義會
法團校董會/學校管理委員會/校董會 : 本校於2011年成立法團校董會，法團校董會成員共十四人，包括辦學團體代表、校長、教員、家長及獨立人士。
環保政策 : 1. 組織健康大使，推動環保教育，提倡節能，邁向可持續生活教育。
2. 參與一人一花活動，綠化校園，提高環保意識，實踐環保生活。
3. 培養學生建立環保生活方式，例如：舉辦環保聖誕派對及素食日。
學校關注事項 : 「學有所長，藝有所精」
1. 優化課堂教學，提升學習質素。
2. 傳揚福音，讓學生在生活中實踐基督精神。
教學規劃
學習和教學策略 : 以學生為本位，重視發展多元智能，八大學習領域和四個關鍵項目緊密配合，積極引入社會資源，並參與教育局及教育機構的學習計劃，引進新思維和最新的教學模式，與時並進，務求學生學會學習，以建構知識，發揮創意，展現潛能。
關鍵項目的發展 : 1. 專題研習：
小一至小四透過常識科作專題研習技能訓練；小五及小六由全校老師分組指導完成研習報告。
2. 從閱讀中學習：
編排圖書課及早讀時間；推行跨學科閱讀計劃及舉辦各項活動。
3. 運用資訊科技進行互動學習：
鋪設完善網絡，運用平板電腦進行學習活動；各級設資訊科技課；並成立「IT小助手」。
4. 德育及公民教育：
設早會、週會、班主任課和成長課，實施全方位輔導計劃，並與宗教科合作推行「宗德」校本課程。
共通能力的培養 : 通過八大學習領域、專題研習及全方位學習培育學生的共通能力，尤其重視溝通、協作及高階思維訓練。
學生支援
全校參與照顧學生個別差異 : 學習支援組，在低年級設主科輔導課，高年級設分組教學，照顧學生的個別差異。課餘有功輔班、朋輩輔導小組、劍橋英語訓練、奧林匹克數學培訓班等課程，提昇學生的能力。
課程剪裁及調適措施 : 按照學生的學習需要及能力，各科作出合適的增刪調適，課業及評估也有相應的配合；同時發展資優課程，為資優學生提供額外的學習機會。
家校合作及校風
家校合作 : 本校家教會設家長學堂及親子學堂，舉辦多元化興趣班及講座：舉行大型親子旅行：組織家長義工服務學校及社區。
校風 : 1. 提倡關愛文化，建立和諧共融的校園。
2. 以耶穌基督為榜樣，在生活中實踐基督精神，培養學生正確價值觀及良好態度。
3. 本校有完善的訓輔政策，讓學生在德、智、體、群、美、靈六育方面得到良好及均衡的發展。
4. 培養學生自尊自律的精神及積極樂觀的態度，提升學生面對成長的應變和自主能力。
5. 培養學生尊重他人和勇於承擔的態度。
6. 加強家校合作，提高家長管教技巧，以輔助學生成長。
學校發展
學校發展計劃 : 1. 優化課堂教學，提升學習質素，包括學與教效能、學生語文能力及自學能力。
2. 傳揚福音，讓學生以耶穌為榜樣，在生活中實踐基督精神，培養正確價值觀。
教師專業培訓及發展 : 1. 科組定期開會，分享研討會所獲及教學心得，推動研習文化。
2. 教師藉着共同備課、同儕觀課及協作教學，優化學與教的質素和效能。
3. 學校注重發展學生高階思維能力的策略及技巧，加強教師相關培訓及多安排教師參加校外研討會。
學費減免 : 不適用。
其他 : 1. 定期與國內姊妹學校互訪交流及舉辦境外考察活動以擴闊學生世界視野。2. 家長教師會設多項獎學金。
