校隊有數學、田徑、游泳、籃球、足球、排球、劍擊、羽毛球、乒乓球、敲擊樂團、管弦樂團、合唱團、舞蹈及辯論； 進修班有多元智能、口才訓練、 電視台訓練、KTS大使培訓、我係科學家、幼童軍、小女童軍、編程創客等，另設各語文學會。
辦學宗旨
以「作育英才」為辦學宗旨，為學生在五育方面得到均衡的發展，致力提供多元化的優質教育，使學生在愉快中學習及獲得發展潛能的機會。
學校特色
學校管理
學校管理架構 : 九龍塘校董會是學校最高決策組織，校董會成員皆為有豐富管理經驗的人士，帶領學校發展。學校內部管理由校長負責，並指導各科組發展的工作及加強團隊合作精神。學校設有多個部門和小組，負責教務、訓輔、課程統籌、課外活動、公民教育、環保教育、資訊科技、校務及家校合作等事宜。
法團校董會/學校管理委員會/校董會 : 學校的校董會是由十位獨立校董所組成。
環保政策 : 成立環保教育組，制訂師生環保節約守則；設學生環保大使；參與明智減廢計劃、綠化校園資助計劃、校園零廚餘及周一午膳素食；曾奪香港綠色學校獎、環保學校榮譽獎及「您想綠色校園」獎。
學校關注事項 : 提倡自信教育，促進學生自主學習。
教學規劃
學習和教學策略 : 各科制訂校本課程；老師進行共同備課，教學互觀活動，以提升教學效能；啟發學生潛能，發展資優教育及自信教育；語文科推行讀寫說結合；各科則加強運用知識的技巧及學習策略；並進行學科參觀、講座、各科組相關之課外活動、專題研習。
關鍵項目的發展 : 每周設閱讀及朗讀課；除閱讀獎勵計劃外，在語文科加入閱讀課，教授學生閱讀及賞析文章技巧。設校園電視台及電腦課，加強學生應用科技之涵養及自學技能，拉近學習差異。另設德育課、公民教育組及升旗隊。
共通能力的培養 : 推行學生資源庫及校本自學網上平台，鼓勵自主學習。
學生支援
全校參與照顧學生個別差異 : 各科設學生共同學習圈，促進自學文化；另設特別學習需要組、課後資優班及補課班；推行友輩學習計劃，照顧潛能未展之學生。此外，亦推行一人一職、全人發展、義工或領袖訓練；成立風紀、服務生、義工大使，讓學生發揮各種潛能；還設置各類資優培訓如辯論隊、音樂團隊、運動校隊、各科學會、KTS大使等。
課程剪裁及調適措施 : 各科加入資優或合作教學元素；為有特別學習需要之學生安排個別學習計劃，為其剪裁合適課程；或在評估中提供調適措施。
家校合作及校風
家校合作 : 設月訊、校長的話、班訊及校內電郵促進家校了解及溝通。設家教會促進家校合作，共同推展教育工作；定期進行家長專題講座及工作坊、家長校園當值和伴讀計劃，以協助家長進行親子教育；成立家長義工隊，關注及監察學生午膳及校車服務，以期善用各種資源，使學生在成長過程中得到各種需要和照顧。
校風 : 秉持傳統尊師重道、勤儉純樸之風，建立學生自信及承擔責任之精神，提倡關愛文化。
學校發展
學校發展計劃 : 完善各項校本課程改革，推動資優教育；建立健康校園，致力為學生建立健康的生活模式；提倡自信教育及自主學習，為學生創造機會，提供有意義的學習經歷，發揮學生潛能。
教師專業培訓及發展 : 建立老師專業學習社群：設教師海外交流及專業發展日；推行同儕互觀，共同備課及教學研究，優化教學質素；培訓領導團隊。
學費減免 : 不適用。
