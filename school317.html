班級教學模式
一至六年級推行小班教學，以靈活、互動的教學方法，照顧個別學習差異；二至六年級部分班別以普通話教中文，提升學生的中文及普通話水平；另以加強輔導教學計劃及過渡模式支援有特殊教育需要學童。
班級結構備註
初小 (P.1-P.3)推行教育局英語讀寫計劃 [Space Town]，高小(P.4-P.6)推行教育局綜合英語計劃 [KIP] 。

多元學習評估 : 多元化評估如課堂觀察、討論、訪問、口頭匯報等，以評估學生各方面的表現及學習進度。透過日常之持續性評估，如專題研習、進展性評估等，以鞏固所學。另為有特殊學習需要的同學作個別「評估調適」。
分班安排 : 本校不設精英班，不同能力的同學會平均分配於不同班級，以小班或小組形式學習，配合「合作學習」策略，以提升教學效能。

全方位學習活動
多元化的課外活動包括：話劇、英語拼音、奧數、田徑、游泳、籃球、足球、排球、跆拳道、舞蹈、繪畫、龍藝、陶笛、古箏、鼓樂、牧童笛、合唱團、童軍等；另設「多元智能課」，並舉辦不同類型的「全方位學習活動」，例如：德育講座、參觀、旅行、教育營、境外交流等等。
辦學宗旨
發揚校訓「博學力行」的精神，使學生在德、智、體、群、美五育方面得到均衡的發展，為社會培養良好公民，為國家培訓優秀人才。
學校特色
學校管理
學校管理架構 : 本校行政架構完善，分配職權明確，由法團校董會領導，任命校長統籌日常運作；校內共設有十四個行政組及十個學科組，另有多個事務委員會及專責小組。
法團校董會/學校管理委員會/校董會 : 共有十四位法團校董會成員，包括：校監、辦學團體校董、校長、教師校董、家長校董、校友校董及獨立校董，會由工商界、教育界及專業人士組成。
環保政策 : 學校重視環保教育，透過環境教育課程、專題研習、展覽、回收活動、環保健康午膳等措施，以提升學生的環保意識，並養成環保的生活習慣。
學校關注事項 : 1. 推動自主學習，以提升學生學業成績表現。
2. 完善評估機制，回饋教學，提升教學效能。
教學規劃
學習和教學策略 : 透過新穎的課程和教學方法，如「合作學習」、「全方位學習」、「電子學習」等，擴闊學生的知識層面，培養他們待人接物的態度，發掘潛能，讓學生能主動學習，與時並進。各級均推行「專題研習」，讓學生擁有不同的學習經歷，達致「愉快學習，健康成長」的目標。
關鍵項目的發展 : 專題研習：本校訂立了一套有系統的專題研習學習策略，能有效地幫助學生發展研習能力，並同時發展自主學習能力，讓小六學生更能順利銜接初中的通識課程。
從閱讀中學習：著重培養校內閱讀氣氛，推行各種閱讀活動及獎勵計劃，如早讀課、好書分享等，讓學生擴闊視野，增進知識，提高語文能力。
資訊科技教育：自編電腦課程，推動資訊科技教學，並積極發展電子學習，讓學生自主學習，務求知識與時並進。
德育及公民教育：培養學生的國民身分認同，讓學生充分認識祖國，從而關心國家，願意為香港及國家的發展作出貢獻。
共通能力的培養 : 根據不同的學習領域，發展學生的九種共通能力，包括創意、溝通、批判性思考、協作、運用資訊科技、運算、解決問題、自我管理及研習能力。
學生支援
全校參與照顧學生個別差異 : 本校設學習支援組，透過「三層支援模式」照顧學生的個別差異：第一層支援 - 優化課堂教學，協助有輕微學習困難的學生；第二層支援 - 為有特殊教育需要的學生，提供「增補」輔導，例如加強輔導教學計劃、小組學習、抽離輔導等；第三層支援 - 為有嚴重學習困難的學生，提供加強個別支援，例如設定「個別學習計劃」。
課程剪裁及調適措施 : 在課程及評估調適方面，教師可為被評估為有學習困難的學生在抄寫功課上作出調適，減少抄寫次數；並為不同能力的學生設計「分層工作紙」。教師亦可為這些學生的默書範圍作出調適，縮窄溫習範圍，或改變默書形式，讓學生更有信心地準備默書。此外，本校在考試時可因應個別需要，安排有學習困難的學生作抽離評估或准予加時。
家校合作及校風
家校合作 : 家長教師會定期舉辦活動和定期進行會議，促進家校合作；重視家長教育，經常舉辦家長講座及工作坊；定期舉辦分級家長會，讓家長與教師交流意見，加強家校溝通，共同關顧學生成長；組織不同的親子義工活動，使學生及家長能將愛與別人分享。
校風 : 參與成長的天空計劃，以提升同學的抗逆力；透過成長課、早會、週會等，培養學生正確價值觀、懂得熱愛生命及關心他人；舉辦校本輔導活動，培養學生良好品德；致力營造校園關愛文化，連續八年榮獲「關愛校園」榮譽。
學校發展
學校發展計劃 : 1. 透過課程統整及專題設計，提升學生的學習動機，達致學會學習的目標。
2. 創造良好語言學習環境，提升學生兩文三語能力。
3. 推動資訊科技教學，以及電子學習，務求知識與時並進。
4. 培養學生多元智能，使心智得到均衡發展。
教師專業培訓及發展 : 教師通過共同備課、互相觀課、協作教學及參加各種研討會和教育講座等，以協助教師發展，促進專業成長。
學費減免 : 不適用。
其他 : 1. 本校正推行「iPad電子學習計劃」以提升學生學習興趣和效能。
2. 本校正發展「非華語學生的中文學習課程」。
