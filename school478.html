辦學宗旨
秉承辦學團體基督化的辦學精神，以德、智、體、群、美、靈六育均衡發展為原則；透過靈性教育，培養學生「非以役人，乃役於人」的基督化人格，作為對人生、家庭、社區和國家應有的認知和態度，從而達至全人教育目的。
學校特色
學校管理
學校管理架構 : 學校由聖公宗（香港）小學監理委員會有限公司掌管，校董會及校長則管理學校的日常決策和事務，並設有多個行政、科目及功能組別，推動行政及教學。
法團校董會/學校管理委員會/校董會 : 由牧師、資深社工、家長、教師、校長及具豐富行政管理的專業人士組成的校董會。
環保政策 : 有廢紙回收設施。全校照明有光控節能設計。安裝節水器具。綠化校園，「一人一花」園藝活動。
學校關注事項 : 1. 加強教師專業發展，建立學習型組織，提升學生學習效能。
2. 推動科本閱讀。
3. 推動生命教育。
4. 推動資優教育，培養學生高階思維能力，發揮學生潛能。
5. 推動資訊科技學習。
教學規劃
學習和教學策略 : 深信每一個孩子都可以教。教師分析學生學習難點，從學生的難點起施教，重視學生的學習基礎，理解及應用，融會貫通。透過教師的觀課和評課，持續改進教學，使學生學習更有成效。採用活動教學、合作學習等策略，以變易理論施教，並以考試結果促進及回饋學生學習。
關鍵項目的發展 : 推行中英文廣泛閱讀計劃及圖書伴讀等。推動中英文篇章網上閱讀。電腦課教授運用資訊科技能力。透過專題研習，擴闊知識領域及學習技能。早會及週會以靈育帶領學生體驗教會學校的生活；德育課及成長課推動德育及公民教育；另設生命教育課，培養學生懂得珍惜生命及對生命有正確的價值觀。推行資優教育，發展高階思維能力。每年安排國內，台灣或新加坡交流活動，擴闊視野。
共通能力的培養 : 致力發展學生的共通能力，並浸透入各科課程及課外活動中，在課堂中以不同層次的提問及學習活動，提升學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 透過全校參與模式，以照顧不同學習需要的學生。亦為有特殊學習需要的學生提供訓練課程。推行全面的資優教育，選出尖子，加以栽培。設奧數班、科技探究班及STEM科技、英文進深班及高階思維訓練班等。
課程剪裁及調適措施 : 各科均實施課程剪裁，推行課業調適和評估調適政策。為有特殊學習需要的學生作適切的課程調適；為有讀寫困難之學生作加時及放大試卷等安排。設輔導組、拔尖組及各精英專訓小組。
家校合作及校風
家校合作 : 本校逢星期二有家長小組活動，並定期舉辦各級家長日、家長會、家長講座及家教會親子活動。舉辦小一新生適應班及指導日，並設小五、小六升中家長會。提供幼稚園升小一及小六與中一的銜接措施。家長到校參與家長義工服務及學校活動。
校風 : 學生友善有禮，尊敬師長。並設多項獎勵計劃，提升學生的自尊自信及發揮潛能。透過全方位及校本訓輔活動，培養學生積極的價值觀。
學校發展
學校發展計劃 : 1. 優化學與教，提升學生學習效能。
2. 推展中文及英文閱讀及寫作計劃，加強英文科戶外學習的機會。
3. 發展校本STEM課程，一年級起進行「編程」教學。
4. 推動科本閱讀。
5. 發展資優教育。
6. 推動資訊科技學習。
教師專業培訓及發展 : 每年舉辦多次教師專業發展日及教師進修分享會；教師每周進行共同備課，又設立同儕觀課，改善專業知識和技能，致力提升學生學業成績。
學費減免 : 不適用。
