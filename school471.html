健康校園生活 : 推行體適能計劃及形體操（本校的特色早操）。
備註 : 設早會、早晨朗讀、班級經營及生命教育。每週設英語日及普通話日。課程著重發展學生對語文的興趣及建立紮實的語文基礎，發展創意。
全方位學習活動
英語音樂劇、Musical Monkeys 英語唱遊小組、創意科藝工程計劃（LEAD，機械人及電腦程式編寫）、戲劇、陶藝、繪畫、欖球、手球、籃球、中國舞、爵士舞、非洲鼓、手鈴、歌詠隊、敲擊樂等，並設有資優培訓、多元智能課及展才活動。
辦學宗旨
營造優質的學習環境，建立有禮、參與、創意及關愛的學校文化，將資訊科技及多元學習融入於學與教中，培育品學卓越、重視健康人生及關心社會的明日棟樑。
學校特色
學校管理
學校管理架構 : 在法團校董會之下，校長領導課程及學務、學生及家長事務、延伸學習及支援三個行政組別推動學校工作。
法團校董會/學校管理委員會/校董會 : 成員包括大學教授、資深社會工作者、資深教育工作者、專業人士、校長、教師和家長代表。
環保政策 : 設校園綠化政策，環保午膳及採用環保照明系統。
學校關注事項 : 提供免費及全面發展的優質教育；全力關注學生的中英文及創意發展。
教學規劃
學習和教學策略 : 重視學生的基礎、理解和應用，採用活動教學及合作學習法，推動全方位學習。利用iPad、電子學習平台和多媒體教室輔助學習。重點提升學生的語文和創意能力。對有需要的學生進行資優培訓或個別學習計劃。
有系統地發展學生的多元智能。
關鍵項目的發展 : 1. 以關愛為起點，培養學生自律、關心社會和認識國家。
2. 重視朗讀，設「閱讀課」及每年二十多項閱讀活動。
3. 校本學習策略發展學生高階思維及進行專題研習。
4. 利用iPad、網上學習平台、網上閱讀和寫作等輔助學生自主學習。
共通能力的培養 : 透過課堂、多元智能及課外活動發展學生九大共通能力，其中以溝通、創意和批判性思維為重點。
學生支援
全校參與照顧學生個別差異 : 以合作學習等策略照顧學生差異，設資優培訓小組和照顧能力稍遜學生的增進班，有完善轉介機制安排專家支援。
課程剪裁及調適措施 : 設有課程及評估調適。
家校合作及校風
家校合作 : 設家教會、家長學堂、家長讀書會及家長義工隊。
校風 : 提倡有禮、參與、創意、關愛文化；建立開明、積極、關懷、信任的環境。本校是多屆「關愛校園榮譽」得獎學校。
學校發展
學校發展計劃 : 著重促進學生學習成效，培育學生健康成長，建立積極的人生態度和加強家校合作。英文科發展完整的校本閱讀課程；中文科深化讀寫結合和普教中。以合作學習、電子學習等途徑提升學生學習水平，發展創意。
教師專業培訓及發展 : 參與研討會、工作坊，與大學及教育局合作進行優化教學行動研究等計劃。為教師建立個人專業發展計劃。
學費減免 : 不適用。
其他 : 支援學生的主要措施超過30項，為學生提供英語訓練的外籍義工達20多人。
