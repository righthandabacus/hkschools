班級教學模式
本校各班乃雙班主任制的小班。課堂以小組合作學習為主，老師有限度講學，學生學習大部分透過探究、討論及老師和同儕回饋學習。另對應不同學科要求還設有小組抽離式學習，高年級分組學習，全級共同課等不同安排。
班級結構備註
校方安排協作老師於中、英、數學課堂中作支援小組學習及照顧個別差異。學生輔導老師會按各級同學之學習需要及發展提供恰切的支援。家長義工團隊亦會提供與學習相關的支援、家長朋輩輔導及安排課堂以外的學習活動。

多元學習評估 : 本校利用課堂參與、課業、實作報告、學習前後測、口頭匯報、成果展示等種種方式來回饋學與教，填補總結性評估不足及減輕測考帶來的負面壓力；各項評估具清晰指引及達標要求，老師會不時跟進，以保學習質素。
分班安排 : 五、六年級中、英、數科目會按學生學習需要分小組上課，以提升學習成效。

全方位學習活動
本校設有學生必修的單元課程來豐富學生的學習經歷，整體內容包括文化學習（如家政、書法、話劇）、運動、科普、服務學習及世界公民教育； 100多項課外活動供學生選擇，計有：運動、音樂及表演藝術、視覺藝術及棋藝類等。另有增潤學習週讓學生從有趣的探究中學習。
辦學宗旨
我們相信每個孩子都需要被尊重，每個孩子都可以教導，都可以有成就。本著基督的愛，提供優質全人教育，強化家長角色及功能，為孩子建立一個積極、健康及開朗的學習氛圍，共同體驗愛的教育。
學校特色
學校管理
學校管理架構 : 本校由辦學團體委任專業人士擔任校董，組成校董會管理學校。
法團校董會/學校管理委員會/校董會 : 辦學團體委任專業人士擔任校董，成員包括專業人士、家長及校友代表等。
環保政策 : 全校參與節能減排活動，設多部動能單車自行創造電能；把物件循環再用，建立環保習慣。用環保飯盒清洗後循環再用；定立環保政策，如紙張循環再用，設回收箱收集廢紙，關掉不必要的電燈，以走路代替乘升降機，加強環保意識等。
學校關注事項 : 以學習愛、愛學習為念，尊重及克盡己責為訓，推行基督教教育、正向教育、校本課程、科技普及教育及處理學生個別差異等。
教學規劃
學習和教學策略 : 課程著重提升學生學習興趣，自學及探究能力。學習模式主要透過探究、合作學習、實驗、考察等方式，配合完備資訊科技配套，讓同學在實踐中學會比較、分析、綜合和創造，從中建構知識。語文課程因應個別同學的需要剪裁，照顧個別差異，並靈活運用助教、協教和分組策略，提升整體學習效能。本校延伸學習活動讓同學得以全人及均衡的發展，各項活動內容可見本文「全方位學習活動」。學校亦透過Responsive Classroom® Approach及Essentials 聖經課程培育同學品德、社交和學習能力。
關鍵項目的發展 : 學習中心藏書三萬五千多本，每班有中英文圖書逾500本，課程大力推動閱讀風氣、興趣及提升閱讀技巧。課室均設多媒體教學設備，並設有手提電腦100台，平板電腦70台，四至六年級推行一人一平版電腦，配合全校無線上網，以便資訊科技輔助學習。學校設有定期學生週會，以進行宗教、德育及公民教育。各科推行校本課程，以專題研習模式，豐富學生學習。
共通能力的培養 : 透過豐富的校園生活及課程配合，提升學生的自我管理、社交、探究、溝通、解難和創造等能力，啟發多元智能。
學生支援
全校參與照顧學生個別差異 : 自編校本課程。務求在學習內容和過程提供選擇，以提升學生的參與及表現。校方亦聘請教學助理及邀請家長入班支援，以助提升不同能力同學之學習效能。此外，教育心理學家及駐校社工為有需要同學提供小組學習活動。
課程剪裁及調適措施 : 各科按學生興趣、能力和需要設計校本課程及學習活動，配合時局、針對知識、技能和態度三方面。課業以開放式題目為主，鼓勵思考和知識運用。設小組抽離式學習，提升整體學習效能，並設有評估考試調適安排及設跨科主題的增潤學習週。
家校合作及校風
家校合作 : 透過家校晚會、網頁、校長來信等途徑，促進家校的溝通；推動家長教育，學校與救恩堂每星期日合辦家長班，家長老師聚首一堂，一同分享教孩子之道；又鼓勵家長參與學校活動協作，甚至在課堂裡輔助同學學習。
校風 : 本校校風純樸，鼓勵學生愛己愛人、關心社區，藉不同群體活動實踐基督教教育。5、6年級設服務團隊，為學生安排培訓及服務，以實踐服侍別人；安排高班學生扶助幼小，並促進社交能力。積極推動家校合作，師生家長關係和諧。每年設新生適應週及舉辦多類型親子家校活動，務使學生及其家庭得到適切關顧，校園充滿歡樂正向風氣。
學校發展
學校發展計劃 : 致力處理學生差異，培養學生具備世界公民的素質；推動「正向教育」，培育學生有正面積極的人生觀；鼓勵學生多運動，多吃水果，建立健康校園；加強藝術教育；強化生命教育，師生實踐扶貧及捐助活動，建立關愛文化，推動學生使用資訊科技學習，提升學生探究協作能力，建構知識。
教師專業培訓及發展 : 學校重視教師專業發展，並定期為教師舉辦切合學校需要的培訓，如教師培訓工作坊、教育理念分享研討會、小組教學等；另又安排不同範疇的工作坊，如合作學習、自主學習、探究式學習、照顧個別差異、資優教育及正向教育等。甚至安排國內、外互訪交流活動以提升老師素質。最後，配合集體備課，校長、科主任觀課，以強化所學所得。
學費減免 : 不適用。
