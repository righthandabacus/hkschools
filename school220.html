辦學宗旨
以基督教之精神為基礎，全力推行全人教育，以達致完美之全人教育為目標。
學校特色
學校管理
學校管理架構 : 學校發展委員會負責向校長及行政會提供校務建議，行政會共有17位成員，包括校長、副校長等。
法團校董會/學校管理委員會/校董會 : 包括校長、六名辦學團體校董；及一名教員校董及一名替代教員校董；及家長校董、替代家長校董各一名。
環保政策 : 培養學生對環保生活的重視。
學校關注事項 : 照顧學習差異、發展電子學習平台、建立健康校園。
教學規劃
學習和教學策略 : 每節45分鐘，方便進行合班或分班教學。每年，校方都會按各級需要，進行跨科整合課程，提升學生的共通能力。
關鍵項目的發展 : 1. 「從閱讀中學習」透過各項獎勵計劃及閱讀課，培養學生閱讀興趣及自學精神。
2. 「專題研習」以班制形式進行。
3. 「資訊科技」方面，校方上載家課日誌及學習資料庫等。
4. 建構校本「德育與公民教育」課程。
共通能力的培養 : 透過校本課程、學習領域、主題學習及多元化活動，建立學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 推行拔尖補底計劃，設資優課程及資優課外活動，設有導修堂，照顧學生成長及學習需要，並有學科輔導、功課輔導班及託管班。
課程剪裁及調適措施 : 製作不同程度的課業，並有略教、精教及增潤課程，並設有評估及考試調適。
家校合作及校風
家校合作 : 舉辦講座、興趣班、家長會、親子活動；定期出版家校通訊。
校風 : 1. 推行班風計劃，以建立良好的校風。
2. 推行伴學計劃，讓高班學生照顧低班同學，發揮關愛精神。
3. 舉辦小一適應課程，協助小一新生適應新的學習環境。
4. 高小全面推行成長的天空計劃。
5. 提供課程及評核調適，以協助學習有困難的學生。
6. 完善訓輔體系，提升學生自尊感。
學校發展
學校發展計劃 : 1. 關顧學習差異，善用小班優勢，提升學習效能。
2. 推行「資優」及「建立基本能力」課程，讓學生發展潛能。
3. 增強學生運用資訊科技能力，發展網上自學平台。
4. 分階段落實推行「普教中」計劃。
教師專業培訓及發展 : 推動教師專業發展，包括合作設計教材、經驗分享和觀摩等。
學費減免 : 不適用。
其他 : 第六十六屆香港學校校際朗誦節比賽英詩集誦初小男女混合組冠軍、P6英詩獨誦男子組季軍、P6英詩獨誦女子組亞軍、五、六年級普通話集誦季軍、P6粵語散文季軍；西貢區德育演講比賽亞軍；第六屆「中華挑戰盃」人才大賽粵語講故事季軍；第二十一屆聖經朗誦節小六組冠軍；第58屆體育節陸地冰球錦標賽新秀少年組男子組及女子組個人冠軍及進階青年組隊際冠軍等。
