教學規劃
學習和教學策略 : 教學的信念：
1. 知識是建構得來的。
2. 經歷為本是一種有效的學習模式（互動、探究、合作）。
3. 教師在教學上的主要角色是促導者。
關鍵項目的發展 : 1. 設計「校本課程」，以圖書教學，引發學生學習動機，強調因材施教，靈活教學模式，活化課堂。
2. 與教會合作，設立基督教生命教育，從小培育學生有正向積極、珍惜生命的價值觀。
3. 運用Microsoft Office365教育版雲端學習平台，學生免費透過此雲端學習平台進行自主式學習。
共通能力的培養 : 設有服務學習，讓孩子也學會服務他人。
參考日本、台灣等地區經驗，設立「服務學習」，透過服務中學習自理、自立、自動自覺，強調「主動不怕吃虧」、「不依賴父母」及「助人為快樂之本」的人生態度，幫助孩子建立未來工作觀。
學生支援
全校參與照顧學生個別差異 : 下午設有功課輔導課，在老師的指導下完成大部份的功課，讓孩子輕輕鬆鬆回家去，務求為孩子創造餘暇空間，發展多元智能，並鼓勵家長每天有一小時親子時間，共建優質和諧的家庭生活。
課程剪裁及調適措施 : 1. 設計「校本課程」，靈活剪材，並因應學生學習情況，彈性調適課業和評估方法。
2. 多元化課堂設計和課外活動。
家校合作及校風
家校合作 : 1. 放學後舉辦多元化活動，包括：英語改進班、興趣班、運動訓練、音樂藝術發展。
2. 開放學校圖書館，讓家長和孩子一起入內享受共讀時光。
3. 與教會合作，開辦多項親子活動課程（例如親子烹飪班、手工藝班、親子敬拜隊等）。
4. 設有服務學習，讓孩子也學會服務他人。
校風 : 本校持守基督教優質教育宗旨，重視愛與堅持，關懷和管教並行。
學校發展
學校發展計劃 : 我們相信每個孩子都值得擁有一個正向、健康、愉快的童年。所以，我期望宣基（坪石）的孩子每天都能懷著興奮愉快的心情上學，在廣闊的學習天地中馳騁，掌握知識，發揮個人的潛能；更重要是培養出尊重自己和他人的品格，在生活中懂得照顧自己和他人，珍惜光陰實踐所學，最重要是學會愛人和被愛。
為此，我們會打破一些學校固有的教學模式，嘗試在「考試制度」和「功課量」中大刀闊斧，作出改變，為孩子創造一個廣闊的學習空間，還孩子一個健康的童年，讓他們茁壯成長。
教師專業培訓及發展 : 我們帥領一群有異象、有使命承擔的教師和傳道牧人，矢志為莘莘學子提供優質的教學和培育美善的品格，同時本著基督的捨己精神，向學生、家庭和社區傳遞基督的愛。
