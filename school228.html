班級教學模式
本校一直實施小班制，設有跳班制，照顧學生的個別差異；本校亦設有才華班、資優小組及個人資優計劃，提供專業化課程給資優生，以提昇資優生的潛能。

學習評估
全年全科測驗次數 : 2
全年全科考試次數 : 2
多元學習評估 : 除一般科目，視藝、音樂、體育、IT及才華班亦對學生作學期評估報告，發掘其潛質作日後發展；本校亦為小一學生進行多元智能測試、小三學生進行中階多元智能測試及小六學生進行升中多元智能測試。
分班安排 : 分班主要以平均學生能力及男女比例為標準。另外，亦以多元智能的評估作參考。

健康校園生活 : 綠色學校金獎、素食綠行計劃、有機種植、魚菜共生、保源密令、健樂計劃，注重學生的BMI指標，每年舉辦陸運會及水運會。
備註 : 課程注重兩文三語的掌握，鼓勵學生的創意思維。藉以培養學生的探究精神及發掘學生的九種共同能力。在高小階段，學生會在不同年級參加校外考試，包括英語，普通話及數學。
全方位學習活動
本校課後活動內容豐富，包括語文類、藝術類及體育類，如級際的午間活動比賽。亦有課後資優培訓。學生每年都進行參觀活動，每年級有兩次戶外學習日及四、五、六年級的境外交流、Green Leader及環保大使。 另外，學校亦設有童軍、幼童軍和小童軍。
辦學宗旨
我們重視多元智能，讓每個學生都在愉快的環境中健康成長，發展才華。我們的抱負是為社會培訓具備有良好的品德和教養的優秀人才，擁有專業知識和思辨能力和有遠大的目光。創意、愛心、勤奮，是我們栽培孩子的重點。
學校特色
學校管理
學校管理架構 : 分四大範疇，十個行政小組，分別是「管理與組織」、「學與教」、「學生成長」和」才華發展」。每個範疇都設統籌，每個小組均設組長。校長與範疇統籌組成校政委員會，制定各項政策。
法團校董會/學校管理委員會/校董會 : 本校按照新修訂的校本條例成立校董會，成員包括辨學團體代表、家長代表、教師代表、校長、獨立人士等。校董會已按香港公司法例正式註冊及運作。
環保政策 : 培養學生關注環境，保護環境，熱愛大自然的生活態度，希望能坐言起行，實踐綠色生活，包括養成簡約、節儉的價值觀，多菜少肉的飲食習慣，和減少不必要對地球資源的破壞和浪費。
學校關注事項 : 才華發展。資優教育–資優五層架構、因材施教。
生命教育–以品帶學。綠色教育及綠色校園。STEM。
教學規劃
學習和教學策略 : 除一般課程外，另設有科學及創意科、中國歷史及文化科、課後才華班、資優班、努力小組及個人資優計劃。
關鍵項目的發展 : 1. 綠色教育。2. 德育課、生命教育、心靈教育、領袖訓練。3. 創意教育、戲劇教育。4. 科學專題、時事專題、中國歷史科及國學（高小）。5. 閱讀計劃。6. STEM。
共通能力的培養 : 各種共通能力分別滲透在各學科中推展。另外，校本課程「科學與創意」則特別針對創造力和批判性思考能力而設，「成長課」則強調溝通能力，協作能力和自我管理能力。
學生支援
全校參與照顧學生個別差異 : 1. 資優教育組設有五層資優教育培訓架構。 2. 小班教學給予學生更多的關顧。採取彈性升留級制度，容許資優學生跳班。 3. 學進組統籌中、英、數三科的支援。 4. 努力小組協助成績稍遜學生。 5. 駐校臨床心理學家協助。
課程剪裁及調適措施 : 普教中、校本中史科、科學及創意科、課後才華班、數學加速、小六升中科銜接課程、外籍英語老師、個別資優計劃。
家校合作及校風
家校合作 : 1. 透過家教會充分發揮家校溝通、校政諮詢，以致支援學校活動等團體合作精神。2. 學校定期舉辦家長日、親子活動、家長晚會等活動，促進家長及學校雙方的了解及合作。3. 家長參與義工團隊，協助學校活動，體驗孩子校園生活。
校風 : 1. 校方推動關懷文化，重視孩子的品德，「做最好的自己」。2. 家校合作的文化，互相信任合作，推動多項親子活動。3. 駐校專業臨床心理學家，提供學業評估及跟進。4. 班長、組長及領袖生訓練，培訓領袖。5. 資優教育包括課後才華班、課後資優班及個別資優計劃。
學校發展
學校發展計劃 : 1. 注重班級的經營。強化班主任的角色、有系統培訓班長及組長，發展每班圖書角。
2. 積極推動科技教育，協調科技、科創及視藝各科。
3. 繼續推動綠色管理及環保教育，培訓綠色領袖及舉辦各項綠色活動。
教師專業培訓及發展 : 資優教育、良師計劃、音樂欣賞、教師運動會、籃球、羽毛球、綠色教育、中層管理、減壓及養生知識、校本培訓及出外進修等。
學費減免 : 本校設有學費全免或減免3/4、1/2、1/4學費。有需要的學生(包括來自綜援家庭的學生及領取學生資助辦事處財政資助的學生)可提出申請學費減免。
- 新生可於開學前的3個月內遞交申請，其申請結果會在2個月內獲得通知。
- 本校學費減免計劃詳情包括家庭入息限額，請瀏覽網上版的超連結。
其他 : 1. 西貢區校際全年總錦標男女子金獎學校。
2. 校際音樂節多項冠軍、亞軍及季軍。
3. 校際朗誦節多項冠軍、亞軍及季軍。
4. 小四、小五及小六境外交流計劃。
5. 綠色學校金獎。
6. 環境運動委員會環境卓越獎銅獎。
