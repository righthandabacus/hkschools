健康校園生活 : 曾獲香港中文大學及世界衞生組織香港健康促進學校金獎多年，推行整全健康教育。另外也獲衞生署嘉許為「至『營』學校」，推行健康飲食文化。
備註 : 為本港優質教育基金健康促進學校計劃的資源學校，推廣整全的健康校園文化。學校加強衞生防疫、護齒和性教育，設家長午膳監察，推行健康飲食文化。午膳後增益活動增進聯繫，建設輕鬆愉快氣氛，推動和諧關愛校園。
全方位學習活動
學術體藝並重：校園電視台、戲劇、朗誦、合唱團、辯論、科學探究、手鈴、敲擊樂、手工藝、童軍、英語面試、奧數、公益少年團、交通安全隊、足球、籃及田徑類等。 另聘導師開辦中國舞蹈、武術、3D 打印等。設「仁、愛、勤、誠」四社，培養學生自理自治文化。
辦學宗旨
培養學生身體力行，以德、智、體、群、美五育為課程綱領，塑造完美人格，開創豐盛人生，並著重發展個別潛能，鼓勵自學、體驗和實踐，建立積極樂觀，熱心服務的態度和貢獻社會國家的豁達胸懷。
學校特色
學校管理
學校管理架構 : 設平衡協作行政管理架構，六大範疇統領學校發展，科組緊密聯繫。
法團校董會/學校管理委員會/校董會 : 成立法團校董會，加入校友、家長、老師及獨立人士為校董，施行現代化協作管理。
環保政策 : 學生「地球保衞隊」，推廣及監察減碳節能、惜物減廢和綠化校園工作。
學校關注事項 : 建立優質教師團隊、提升學與教效能、為學生、教職員、家長營造關愛文化。
教學規劃
學習和教學策略 : 在正規課程、全方位學習和各項比賽的配合下，推行小班教學，透過主題學習、課程統整、科本探究、戲劇教育、合作學習、美感教育、及專題研習等策略，發展創意學習及培養學生共通能力，並進行促進學習評估。
關鍵項目的發展 : 「閱讀」︰推廣「師生共讀萬卷書」計劃、好書流、周會閱讀及家長義工；
「資訊科技」︰「聰明教室」(SMART Classroom)適時透過I-PAD學習、遠程教室的視像學習、電腦課、各科自學平台等，掌握相關知識；
「專題研習」︰生活化專題研習及歷史探究，配以研習技能，掌握相關知識；
「公民教育」︰透過周會、戶外學習、成長課、跨境考察、培養學生正確價值觀和對祖國的認識。
共通能力的培養 : 合作學習、主持活動、朗誦、演講，增溝通能力；辯論、兒童論壇，培養批判思考能力；專題設計、美感教育，提升創意思維；專題研習、數理活動，培養解難能力；「成就學堂」培養自我管理能力。
學生支援
全校參與照顧學生個別差異 : 全校推行小班教學計劃，實施合作學習法。設加強輔導班，小三至六主科分組學習課。學校參與大專院校相關計劃作支援，並設設後輔導、拔尖（延展）、補底課；學校設學生支援組，有為家課、測考方面調適，中、英文學長伴幼及言語治療；電子英語跨級自學平台；語文老師接受照顧讀寫困難學生培訓。
課程剪裁及調適措施 : 按學生需要實施課程調適及剪裁，對有特殊學習需要的同學設考試調適卷，另設恆常功課輔導；因應學生興趣和能力參加各種拔尖培訓。另設幼小銜接班，升中面試班等。
家校合作及校風
家校合作 : 奉行「家校合作，子女得益」政策，家長參與學校管理和發展（健康教育及關愛校園事務）；持續推行家長教育課程及朋輩輔導，曾獲家校合會頒發「最多姿多采」獎。設每周「家長加油站」以加強溝通。
校風 : 持守啟發潛能教育理念，在地方、人物、政策、計劃和過程中推行愛的教育，重視正面的家校和師生關係，學校與社區的融合，建立守禮互重的和諧校園。
學校訓輔合一，注重學生身心發展，兩度獲頒香港健康促進學校金獎（中大）、三度榮獲國際啟發潛能教育成就大獎(IAIE)及連續9年獲頒「關愛校園」獎（香港基督教服務處）。
學校發展
學校發展計劃 : 1. 持續和優化在「啟發潛能教育」和「健康促進」兩方面的發展；
2. 參與教育局或大專院校、提升教學效能的計劃，發展高階思維，培養學習自主學習的能力和態度；
3. 優化「聰明教室」，推行電子學習，加強視像學習，擴闊學生的視野；
4. 英文科實踐「FUＮ LEGO PROGRAM」，以提升學生的學習興趣和能力；
5. 推行「生命教育」，培養學生正確價值觀。
教師專業培訓及發展 : 1. 設教師專業發展計劃、共同備課及同儕觀課。
2. 參與教育局或大專院校支援計劃。
3. 持續培訓、提升教師的教學技巧，例如︰照顧有特殊學習需要學生的技巧、小班教學認知及電子教學技能。
學費減免 : 不適用。
其他 : 辦學團體、校友、家教會各設獎學金，獎勵同學敦品勵行，力學向上。
