辦學宗旨
以耶穌基督仁愛精神，「愛主愛人」為校訓，為學生提供良好的學習環境，使他們學會學習，與家長建立良好的夥伴關係，培養孩子們的優良品德與健全人格，致力啟發孩子智慧，發掘潛能，重視靈、德、智、體、群、美六育以至全人發展，從而達到終身學習，將來成為貢獻國家、服務人群的良好國民。
學校特色
學校管理
學校管理架構 : 本校成立法團校董會，由不同界別的校董共同管理學校，推展「校風及學生支援」、「管理與組織」、「學與教」、「校風及學生支援」和「學生表現」各範疇工作。
法團校董會/學校管理委員會/校董會 : 獻主會小學法團校董會於2012年4月2日成立，由十五位校董組成，成員包括七位辦學團體校董，替代辦學團體校董，獨立校董，當然校董，家長校董及替代家長校董，教員校董及替代教員校董，校友校董。
環保政策 : 透過環保活動和計劃，宣傳環保訊息，包括節能減碳活動、廚餘回收、環保栽種活動、設置分類回收箱及製作綠色微電影。
學校關注事項 : 學習主題為：「普及資優教育 建構自主學習 　達至全人發展」
教學規劃
學習和教學策略 : 推行資優教育，透過高層次提問及資優十三招學習策略，以及分層課業，包括基礎實踐、進階實踐及多想一步，提升學生高階思維能力。設立人才庫，甄選學生參加抽離式課程，發展潛能。加強兩文三語教學，推行普通話教中國語文，推行跨學科學習活動及舉行主題學習日，並善用社區資源，進行全方位活動以擴闊學生其他學習經歷，如海外考察。推行電子學習，課室已裝設無線上網設備，配合互動電子白板及平板電腦，利用學習平台及軟件進行互動學習。
關鍵項目的發展 : 推廣閱讀計劃，如親子閱讀、書叢悅閱俱樂部、網上閱讀。持續發展專題研習、配合資訊科技挑戰獎勵計劃，推廣互動學習。配合靈育、德育、公民教育及全方位學生輔導設計校本課程。
共通能力的培養 : 透過資優教育，提升學生高層次思維、創造力和個人及社交能力。
學生支援
全校參與照顧學生個別差異 : 學校透過「學習支援」、「加強言語治療」等津貼，增聘教學及專業人員，以「全校參與」模式，為學生提供輔導教學、成長小組、測考調適等措施，並設校本教育心理服務、手語雙語共融計劃，照顧學生的需要，融入校園生活。學校重視家校合作及教師培訓，透過家長日、講座、茶聚等渠道，與家長及教師溝通，共同培育學生之成長。
課程剪裁及調適措施 : 選拔人才參加抽離式課程，發展學生潛能，包括中、數、常、視藝、領袖訓練及基礎課程。透過「三層支援模式」照顧學生的多樣性，支援課程包括校本輔導課和加強輔導班。
家校合作及校風
家校合作 : 重視家校溝通，攜手合力推動資優教育，培育人才。提供家長教育，輔導子女學習及成長。組織家長義工，為學校服務，如參與活動及故事爸媽，籌募學校發展及推廣基金，並協助推行學校各種多元學習活動。
校風 : 建構關愛文化，設有風紀隊及各範疇的服務大使，為學校服務，提升領導能力；透過非常規課程，如早會、週訓，推行靈育、德育、及國民教育活動，培養學生「愛主愛人」的精神。
學校發展
學校發展計劃 : 學習主題為：「資優教育啟潛能 多元學習展菁英」，學校全面推行資優教育三層架構，培育人才，並以「策劃—推行—評鑑」為自我評估模式。
教師專業培訓及發展 : 學校推展資優教育，所有教師接受資優教育基礎課程；中層教師完成深造課程；科主任亦已完成進階課程。教師積極參與電子學習及STEM課程，以提升學與教的效能。
學費減免 : 不適用。
其他 : 推行健康校園。
