健康校園生活 : 為使學生身心靈健康，推行全人教育，重視品德培養；安排學生參加體藝訓練，培育心靈成長；推行健康午膳、健康工作坊及護脊校園計劃。

3. 安排境外交流，擴闊學生視野。
全方位學習活動
本校以一生一體一藝一服務為目標、策劃多種體育、藝術及制服團體活動，積極參與校外比賽，讓學生認識自己、欣賞別人，獲取成功經驗。各級安排不同形式參觀活動、講座及大型演出，藉以提升學生的多元智能及展示學習成果。
辦學宗旨
本校秉承聖公會的辦學宗旨，肩負「非以役人，乃役於人」的使命，培育學生具基督化人格，認識基督教信仰。以發展全人教育為目標，使兒童在德、智、體、群、美和靈育各方面都得到均衡發展，希望他們在接受我們的教導後，能不斷自學、思考、探索、創新和應變。
學校特色
學校管理
學校管理架構 : 按管理與組織、學與教、學生支援服務及學生表現設立專責組別發展校務。
法團校董會/學校管理委員會/校董會 : 法團校董會舉行例會，就校政作出決策及監察。
環保政策 : 1. 校舍位於啟德發展區，全校採用區域水冷系統，即利用海水製冷，有效減低電力消耗。
2. 鼓勵學生節約及循環再用，為保護環境盡力；全面推行綠化，建設健康環境。
學校關注事項 : 1. 持續優化教學策略，提升學與教的成效；
2. 培養學生良好的生活習慣，以提升學生的自我管理能力；
3. 發展可持續發展的綠色教育，培養學生成為地球管家。
教學規劃
學習和教學策略 : 多元教學策略包括以腦科學為基礎的教學法、照顧個別差異、強化教學效能。靈活課時、全方位跨科學習活動，培育全人發展、STEM教育。
關鍵項目的發展 : 1. 培育學生七種首要的價值觀和態度，即堅毅、尊重他人、責任感、國民身份認同、承擔精神、誠信和關愛。
2. 從學會閱讀到從閱讀中學習，以增強學生的學習能力，達致終生學習。
3. 透過專題研習掌握溝通、協作、研習等共通能力。
4. 運用資訊科技進行互動學習，鼓勵學生自學。
5. 發展生命教育，讓學生懂得愛惜生命及規劃人生。
共通能力的培養 : 透過正規課程及全方位學習提升溝通能力、協作能力、創造力、批判性思考能力及解難能力。
學生支援
全校參與照顧學生個別差異 : 1. 推行新資助模式，靈活運用資源支援學生。
2. 學生支援委員會評估學生學習需要及進行調適。
3. 設輔導班，鞏固基礎；設拔尖課程，提升能力。
課程剪裁及調適措施 : 老師編製校本中、英、數教材，增潤學生的學習。按能力分組教學，提升學習效能。老師透過共同備課，剪裁教學內容，照顧學生個別差異。
家校合作及校風
家校合作 : 1. 家長義工協助校務及加強家校溝通，攜手培育兒童茁壯地成長。
2. 設家長學堂及福音茶聚。
校風 : 1. 以基督的心為心，建立關愛校園。
2. 透過獎勵計劃，激發學生在德、智、體、群、美、靈的潛能。
學校發展
學校發展計劃 : 1. 透過學科學習、活動、服務獎勵，提升學生自信心。
2. 強化小班教學效能，善用以腦科學為基礎的教學。
3. 建立校園閱讀文化，提升閱讀興趣及能力。
4. 師生共同建立關愛文化。
教師專業培訓及發展 : 發展專業學習社群，建立學習型教師團隊；施行以腦科學及思維為本教學，提升學與教效能。
學費減免 : 不適用。
