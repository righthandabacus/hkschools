辦學宗旨
以基督的精神作為教育方針。致力提供多元化的全人教育，讓學生在關愛的環境中愉快學習、健康成長，發展個人潛能及終生學習的能力，建立自信與積極的人生觀，培養正確的公民意識，回饋社會，貢獻國家。
學校特色
學校管理
學校管理架構 : 法團校董會下，由校監/校長領導各行政組及科務組推行校政。
法團校董會/學校管理委員會/校董會 : 由資深教育界人士及獨立人士組成：包括家長，教師及校友校董。
環保政策 : 課程滲入環保，綠化校園，節約能源，廢紙回收；環保小組及環保大使推動環保教育，推行再生能源綠化計劃。
學校關注事項 : 提升學與教效能，培養學生正面的價值觀和態度。
教學規劃
學習和教學策略 : 推行小班教學，運用合作學習和語境教學，設宗教德育課，培養學生多角度思考，鼓勵自學，特設不同資優學習小組，讓學生發揮潛能。
關鍵項目的發展 : 1. 從閱讀中學習：重點培訓中英文閱讀策略，推行多項閱讀獎勵計劃，親子伴讀及故事爸媽計劃，安排學生及老師定期分享圖書；
2. 專題研習：綜合課程設計和全方位學習活動，提升研習技巧及探究思維能力；
3. 運用資訊科技：透過網上平台(iLearner)，提升自學能力；利用平版電腦和電子白板進行互動學習；
4. 德育及公民教育：以天主教核心價值為本的「生活教育」「生命教育」課程建立積極生活態度，「班級經營」活動培養自理，自律，互助合作精神。
共通能力的培養 : 重點提升創造力，批判思考能力，高層次思維能力，社交能力，自學能力及自我管理能力。
學生支援
全校參與照顧學生個別差異 : 課前輔導，加強輔導班，英語班，拔尖班，增潤課，分能課，學習及生活技巧訓練班，新生指導日；小一適應課程，升中選校經驗分享：協助中小幼銜接。
課程剪裁及調適措施 : 為有需要學生調適課程，測考，功課；提供言語治療服務。
家校合作及校風
家校合作 : 以家長會，家長日，校訊，家教會會訊，網頁保持家校溝通；家長義工協助學校推行活動；舉辦家長講座及工作坊提供管教子女資訊。
校風 : 全方位獎勵計劃鼓勵積極進取；基本組織及服務小組培養承擔，勇於服務的態度和責任感；「小義工」進行朋輩輔導；訓輔活動培養關愛文化。
學校發展
學校發展計劃 : 1. 以天主教教育核心價值為本，培養學生正面的價值觀和態度。
2. 持續優化校園設施。
3. 優化課堂教學及評估，提升教學質素。
4. 建立書香校園，培養學生愛閱讀的習慣。
5. 發展資優教育，啟發學生潛能。
6. 繼續各項體藝培訓。
教師專業培訓及發展 : 境內外交流考察，專題講座及工作坊，觀課評課；與教育局／大專院校協作。
學費減免 : 不適用。
