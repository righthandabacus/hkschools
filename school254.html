班級教學模式
為學習有困難及成績落後的學生提供「加強輔導教學」，透過調適課程和教學策略，幫助學生鞏固學科的基礎知識，掌握學習方法，增強信心，改善學習效能。

全方位學習活動
本校為學生提供多元化課外活動。服務團隊有基督小先鋒、童軍、公益少年團、小園丁、環保大使、英語大使、伴讀大使等。另有各項球類訓練隊、泳隊、田徑隊、中國舞隊、管弦樂團、歌詠團、集誦隊、奧數隊等。興趣班有國術班、魔術班、空手道班、水墨畫班、拼音班等。
辦學宗旨
本著基督福音精神，培養學生良好品德，獨立思考及明辨是非的能力。營造良好學習氣氛，培養學生公民意識，關心社會，愛護祖國。
學校特色
學校管理
學校管理架構 : 法團校董會 → 校長 → 副校長 → 主任 → 全體老師。
法團校董會/學校管理委員會/校董會 : 本校設有法團校董會，校董會內有教師、家長及校友代表。定期舉行校董教師諮議會。
環保政策 : 本校已制定一套環保政策，內容包括環境管理及環境教育等。
學校關注事項 : 提升學生自主學習的能力，實踐天主教學校的五大核心價值，培育學生正確的人生觀及價值觀。
教學規劃
學習和教學策略 : 本校推行自主學習，透過不同學科及跨學科課程，以及多元化教學策略，發展學生潛能，策略計有：應用思維工具、跨課程閱讀、STEM課程、電子學習、翻轉課堂、合作學習、全方位學習、專題研習、資優教育；教師運用AQP平台，分析學生評估的強弱點、並於回饋周跟進；設計「三高」（高參與、高互動、高趣味）教學策略以提高學生的學習興趣。
關鍵項目的發展 : 1.從閱讀中學習，設有圖書協作課、閱讀獎勵計劃；推行中英文廣泛閱讀、初小跨課程閱讀；一年級設故事姨姨/叔叔、伴讀大使等計劃
2.資訊科技：全校設有電子白板及多媒體電腦輔助教學；三至五年級數學課設定互動電子學習課
3.五、六年級進行專題研習；一至六年級進行跨學科全方位學習以訓練不同的專題研習技能
4.德育及公民教育：以宗教培育為根基，輔以全校參與的全方位訓輔工作，讓學生認識天主教教育五大核心價值；推行全人教育，透過成長課、具「天主教社會倫理」特色的國情專題教學及非正規課程培養學生良好的品格和積極正面的人生觀
共通能力的培養 : 透過正規及全方位學習活動發展學生多種共通能力。教師在課堂上利用不同的教學策略及學習活動發展學生溝通、合作、明辨性思考及創造力等共通能力。透過合作學習訓練學生協作能力，利用自學套件、獎勵計劃、網上學習平台等，訓練學生自學能力。
學生支援
全校參與照顧學生個別差異 : 由特殊教育需要統籌主任(SENCO)領導學生支援組，為有特殊學習需要及有經濟困難的學生提供學習支援，如教育心理服務、言語治療服務、社交小組、課後輔導及功課輔導等，以提升學習成效。
課程剪裁及調適措施 : 為有特殊學習需要學生作家課及評估特別安排及調適。
家校合作及校風
家校合作 : 透過家長座談會、家長聯絡網，使家校更能緊密聯繫。本校家教會發展成熟，家長義工除協助學生午膳外，亦推動閱讀及課程統整的活動。
校風 : 透過正規、非正規及隱蔽課程，配合不同類型的活動及全入教育，協助學生建立正面和積極的生活態度，並配合天主教教育五大核心價值，同時推行具「天主教社會倫理」特色的國情教育，以啟迪學生的身、心、靈，並通過自我賦權，發揮心理力量，戰勝逆境和考驗，追求個人的人生目標。
我們期望能為學生的全人發展及終身學習奠定基礎，以邁向豐盛卓越的人生。
學校發展
學校發展計劃 : 在學與教層面方面，本校透過運用不同的學與教策略，如應用思維工具、制訂個人學習計劃、預習、筆記、有趣的閱讀報告形式、探索活動等，以提升學生學習興趣與動機及自主學習能力。此外，教師運用評估數據，回饋學生，促進學習。
本校亦透過家長教育、不同獎勵計劃等來提升學生的力學精神。此外在各學習領域和活動中滲入天主教教育五大核心價值的元素，同時推行具「天主教社會倫理」特色的國情教育，讓各持份者認識天主教的核心價值觀，並在生活中實踐。
教師專業培訓及發展 : 教師積極進修。此外，學校為了讓教師能在專業上有所發展及配合學校發展，每年均為教師進行適切的校本培訓。
學費減免 : 不適用。
