班級教學模式
小一至小六實施小班教學。小一至小六年級各有兩普教中班。另開設兩個加強輔導班，進行課程剪裁及課業調適，引入合作學習教學策略在課堂教學中，提高學生自信心及學習動機。
班級結構備註
運用不同教學策略，如：合作學習，自主學習等進行教學，以提升教學效能。

全年全科考試次數 : 3
多元學習評估 : 評估多元化，包括持續性評估及單元評估，透過專題研習、閱讀報告、概念圖製作、角色扮演和口頭報告，評估學生學習效能。在考試時，有學習困難的學生將會獲得照顧。
分班安排 : 小一至小三不以成績分班，小四至小六以學習態度和成績分中線，Ａ、Ｂ班為躍進班，Ｃ、Ｄ班為普通班。

舉辦陸運會，綜藝滙演，遊學團。課外活動多元化，包括：獅藝 、舞蹈 、合唱團、非洲鼓、管樂、敲擊樂、跆拳道、田徑訓練、球類、畫班、小小科學家、奧數、劍橋英語班、英語話劇、英文閱讀寫作班、普通話班、航空課程。
辦學宗旨
本校以成為優質英文小學為辦學方向，幫助學生升讀以英文為主要教學語言的中學是本校最主要的教育任務。本校首重「智育」，使學生在小學階段已汲取豐富的知識，繼而提倡「德育」，使學生接受中國傳統文化薰陶，盡公民的義務。
學校特色
學校管理
學校管理架構 : 法團校董會委任校長處理校務，推行校本管理。
法團校董會/學校管理委員會/校董會 : 已於2007年8月1日成立法團校董會。
環保政策 : 全校參與，實踐綠色生活。
學校關注事項 : 我們致力塑造學生成為一位滿有教養，善於溝通，思維創新，靈活多變，終身學習，迎戰未來，可敬可愛的世界公民。
教學規劃
學習和教學策略 : 自編中文科教材，注重寫作訓練，並運用社區設施進行全方位學習，增強學生的生活體驗，設英語和普通話日，培育學生語文能力，英文科推行幼聯英語計劃，以故事教學，並將英語話劇元素融入課程內，籍此進一步提升學生閱讀及寫作能力，掌握更多詞彙。此外亦進行校本STEM課程，設計多元智能體驗學習活動，培育學生邏輯推理及解難分析能力，並以抽離式推行優資尖子課程，啟發尖子學生潛能，因材施教。
關鍵項目的發展 : 設閱讀課及各科都設有閱讀活動。早上設英普故事角及「早讀時間」。學生於電腦課學習資訊科技技能及利用考評局「基本能力評估」進行網上自學。設德育課，加強學生德育及公民意識。「綜合活動課」一、二、三級各設主題，培養專題研習技巧 。
共通能力的培養 : 透過校本課程在聯課時段以綜合課形式有規劃地培養學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 主要科目課程分核心和增潤兩部分，課程及功課按學生的程度調適。採用合作學習及多感官教學法，照顧不同能力學生的需要。設中、英、數各科的課後輔導班、升中英文數學班等，務求拔尖補底。
課程剪裁及調適措施 : 中、英、數、常均已設計學習能力層遞工作紙，老師在集體備課時訂出核心課程及增潤課程，提升學生學習成效。
家校合作及校風
家校合作 : 家校緊密合作，透過「家長學堂」及「家長茶聚」鼓勵家長參加工作坊及研討會並加強與校溝通：每年均舉行全校親子大旅行及水果日，家長熱心參與義務工作，協助學生舉辦各項活動如陸運會等。在學期初九月時到校協助照顧小一生午膳，並透露心聲：「沒有選錯學校」。
校風 : 管教認真，校風淳樸，學生守規有禮。師生關係良好，學生對學校有歸屬感。
學校發展
學校發展計劃 : 2015-18 學校發展計劃：
1. 推展校本價值教育，提升學生品德素養。
2. 建立學生自主學習模式，強化學生學習技巧，提升學教效能。
教師專業培訓及發展 : 聚焦發展提升老師提問技巧及教學策略，優化共同備課及同儕觀課，有計劃地編排教師參加科任及所屬行政組別的校外機構舉辦的工作坊及研討會，全校教師均完成網上資優課程及百分之五十的教師已接受殊教育培訓課程。
學費減免 : 不適用。
其他 : 15-16年度開展電子學習計劃，透過互動學習軟件，培養學生自學精神。
成立何小學生會及校友會，增強學生及校友的歸屬感。
