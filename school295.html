班級教學模式
全校推行小班教學；小二至小六每級設一班普教中班；全校採用合作學習模式及高階思維策略，以提升學與教的效能；鼓勵學生多參與課堂活動，亦多展示學生學習成果。
班級結構備註
1. 按學生專長設不同小組以培訓資優學生。
2. 以抽離式小組進行學業輔導。
3. 外籍英語教師以協作教學模式進行部分英語課。

多元學習評估 : 小一入學初期不設考試，全學年三次總結性評估；小二至小六每學年四次總結性評估，並設有進展性評估。評估方法多元化，如非紙筆評估、口頭報告、學生自評、互評。
分班安排 : 小二至小六每級設一班普教中班，其餘各班均為混合能力班。

健康校園生活 : 本校關注學生身心健康，並重視學生的體適能發展，除設立生活教育課外，亦安排水果月鼓勵學生多吃水果，並進行活力操。
備註 : 本校關注學生的全人發展，重視兩文三語。設早讀課培養學生的閱讀習慣，並開設各類培訓班，讓學生的多元智能得以發展。設「一年一學」普及體藝課程，拓展學生興趣，發展潛能。推動「STEM」教育，培養創造力、協作和解難能力。
全方位學習活動
本校重視學生全人發展，課外活動共分五個範疇： 1. 服務：幼童軍、環保大使、義工隊等；2. 藝術：英語話劇、中國舞、管樂團等；3. 學術：中英普朗誦、數理思維班等；4. 體育：乒乓球、游泳、足球等；5. 興趣：校園電視台、機械人創作、STEM無人機、農耕樂等
辦學宗旨
為社會提供多元化教育服務，使學童成長後能盡展所長，回饋社會。為學童進行全人教育，提供一個優良學習環境，使能發揮個人潛能，成為具備知識技能、有獨立思考力、勇於承擔責任和關注社會事務的良好公民。培育學童有正確價值觀和積極人生觀，並以校訓「勤儉忠信」勉勵他們拓展豐盛人生。
學校特色
學校管理
學校管理架構 : 學校由法團校董會管理，並分成多個行政組別，包括：課程、教務、學術、學生支援、訓輔、課外活動及總務等不同範疇。學校已建立自評文化，透過各組進行的計劃、實施及檢討，並善用持份者問卷數據，不斷完善教與學。
法團校董會/學校管理委員會/校董會 : 於2006年成立法團校董會。成員包括辦學團體代表、校長、教師、家長、校友及獨立人士，共同釐訂學校發展項目。
環保政策 : 參與環境保護大使計劃、一人一花活動、三色回收桶計劃，校園內進行有機種植，並實行惜物減廢。
學校關注事項 : 1. 促進學習型組織的發展，加強團隊成員間的專業交流與溝通。
2. 強化照顧學生多樣性的校本課程及教學策略，持續提升學與教的效能。
3. 培養師生健康身心，共建健康校園。
教學規劃
學習和教學策略 : 本校設有：綜合課、閱讀課、生活教育及專題研習等。著重學生學習過程，從中學習，並推行合作學習法及以生為本的教學理念。
關鍵項目的發展 : 圖書課培養學生的閱讀興趣及能力；綜合課讓學生進行專題研習及服務學習；有完善的電腦室及全校多媒體教室的設置；每週進行德育及公民教育活動；規劃「STEM」課程，並設立跨學科學習體驗活動。
共通能力的培養 : 一至六年級均設有綜合課及STEM課程，提昇學生的共通能力和思考能力。課程中加入「東華三院德育及公民教育」元素，讓學生明白東華精神及價值觀。
學生支援
全校參與照顧學生個別差異 : 推行全校參與學生支援模式，按學生需要開設主科輔導課；推薦學生參加資優課程；實施配屬計劃，使有特殊學習需要學生得到全面照顧。
課程剪裁及調適措施 : 教師透過集體備課調整課程，如中文科識字及文學單元學習、閱讀及寫作教學，英文科「iRead」課程，閱讀策略教學，各科運用多元學習策略及思考方法。
家校合作及校風
家校合作 : 家長義工隊以支援教師教學，家長學堂能強化家長教育，家長教師會定期舉辦親子活動。
校風 : 學校重視學生的品德及價值觀培養，以訓輔合一方式建立學生正確價值觀；設立全校摘星獎勵計劃，鼓勵學生積極進取，肯定他們的成就；建立關愛互助的校園文化，培養學生正向積極的人生觀。
學校發展
學校發展計劃 : 2015-2018學年關注事項：
1. 學校團隊發展
2. 學校整體課程規劃
3. 學生身心均衡發展
教師專業培訓及發展 : 學校重視教師專業團隊的發展，除設立教師發展日外，亦安排教職員專業發展研討會，透過組織教師學習圈、同儕備課、觀課交流及教學分享，提昇教學效能。
學費減免 : 不適用。
其他 : 學校設立多項獎學金以表揚學生的不同成就，包括：學業成績獎、學業進步獎、傑出學生獎、課外活動獎等；每年均進行跨境交流活動，擴闊學生視野；本校學生曾在公開校際比賽中獲得多個獎項。
