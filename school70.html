班級教學模式
小一至小四進行小班教學，部份科目按學生能力作分組分層教學。於小五、小六課程加入自主教學元素，培養學生自學、自我完善的能力及態度。為非華語學生安排銜接課程及功課輔導。
班級結構備註
常識科加強科學探究及生物科技教育元素，並於小五、小六加入主題式探究課程。

全年全科測驗次數 : 2
全年全科考試次數 : 2
多元學習評估 : 持續性評估：著重學生的學習進程，學生日常的課堂表現、家課、專題研習等均為評估的一部份，既避免過份操練考試，亦讓老師能及早回饋學生學習。
分班安排 : 按學生的能力及特質平均分配，英語按能力分組授課。

健康校園生活 : 學校參加「健康飲食在校園」計劃，定期進行早操及健眼操，並增設發電單車，讓同學養成良好飲食習慣及鍛練強健的體魄。
備註 : 在一條龍學校，師生有長時間的相處，能及早發掘學生的潛能。中小同學一起學習，能提昇溝通技巧。
全方位學習活動
除中、英、數等學科興趣小組外，特色活動包括天文小組、英語戲劇、步操管樂、射箭、高爾夫球、攀石、園藝及農耕、科學團隊等。
辦學宗旨
秉承黃大仙師「普濟勸善」的寶訓，透過儒、釋、道三教要義，教導學生修養品德，發揚樂善好施、服務社會的精神。重視學生五育的發展，培養學生獨立思考的能力，並具自學的習慣及終身學習的態度。
學校特色
學校管理
學校管理架構 : 學校以十二年中、小一條龍模式管理，有利專科專教與及中、小連貫課程的推行；更讓小學同學有機會在設備更完善的中學實驗室及其他特別室上課。
法團校董會/學校管理委員會/校董會 : 本校法團校董會成員包括辦學團體校董、校長、教師及家長代表組成。
環保政策 : 推動校園綠化，設有機農耕地，培養學生珍惜食物的習慣。校內環保設施包括太陽能及風力發電系統、各式節省能源電器、分類回收癈物、舊書本舊校服轉贈計劃等。
學校關注事項 : 學生的兩文三語、自我學習、與及科學探究的能力。同時亦著重培養學生的品德情意。
教學規劃
學習和教學策略 : 施行四十五分鐘課節，有利互動課堂的展開。
在母語教學以外，透過各學科加強學生的英文學習元素。
於高小年級推動自主教學，加強學生自學及小組學習的能力。
關鍵項目的發展 : 1.透過「篇積智慧」自編閱讀冊子及其他活動，推動海量閱讀計劃，以提昇語文能力。2.善用平板電腦及無線網絡，推行「反轉課堂」，加強互動教學。3.加強英語能力：推動小組分層教學，推動英語閱讀及寫作計劃，讓學生按個人能力不斷求進；透過多元活動，以營造「英語處處」學習環境。4.善用優越的生物科技流動實驗室及生物科技教研室，加強各級的科學學習元素，引入微生物、DNA等課程。5.可譽一直致力推動科學及科技教育，學校應教育局邀請，將參與專業發展學校計劃，為友校提供支援，協助學校發展生物科技課程，提昇學生學習動機及參與。
共通能力的培養 : 透過學科學習及活動、多元智能課、專題研習週等，培養學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 1. 透過分流教學、課後提昇班、補習課及功課輔導班等，照顧學習差異。
2. 為有需要的同學作課程調適，亦為學習能力較高的同學安排進階課程。
課程剪裁及調適措施 : 分班學英語、分層課業、統整課程、發展高階思維。
家校合作及校風
家校合作 : 家校合作：家教會、家長義工積極投入學生活動，親身帶領親子興趣班及學習活動。
校風 : 發展嚴而有愛、融洽有序的校園關愛文化，在「家與校」、「師與生」、
「長與幼」的和諧共處當中，讓學生健康地成長。
學校發展
學校發展計劃 : 1. 發展學生自主學習的態度與能力
2. 按不同學階的需要，對學生的個人成長提供支援
教師專業培訓及發展 : 鼓勵教師專業發展，設同儕觀課、新入職老師輔導、友校教學經驗交流等。
學費減免 : 不適用。
其他 : 本學年，學生在多項比賽中獲獎，計有中文普通話朗誦、英文朗誦、校際田徑比賽、全港數學挑戰賽、資優解難比賽、學校舞蹈節比賽等。
