辦學宗旨
秉承基督精神，俾學生能在基督的愛中學習與成長，建立其人格，陶冶其性靈，實踐「非以役人，乃役於人」精神，作社會的棟樑。
學校特色
學校管理
學校管理架構 : 本校是聖公宗（香港）小學監理委員會屬下小學，由校董會管理。
法團校董會/學校管理委員會/校董會 : 校董會之成員包括牧師、中學校長、工程師及商界人士，能從不同角度為學校提供意見。
環保政策 : 成立環保小組，推行環保教育，裝置再生能源設備，建設綠色校園，實踐綠色生活。
學校關注事項 : 本校致力提升學與教的素質，推行小班教學和合作學習。為照顧學生的需要，本校把發展自主學習、照顧學生學習多樣性及發展生命教育訂為學校重點關注事項，各科組均會實施相關策略以配合學校發展工作。
教學規劃
學習和教學策略 : 除日常課堂外，課程組會為學生安排各種切合課程之參觀、研習、工作坊、全方位學習雙週等活動；此外各級還會進行專題研習、STEM科探活動、五年級推行服務學習、一年級推行「小一100日」等，以提升學生共通能力。
關鍵項目的發展 : 繼續配合教育改革，發展四大關鍵項目，包括：
資訊科技教育：全面採用學校內聯網及網上學習平台，課堂推展平板電腦教學，運用網絡進行教學和溝通，提升學習效率。
專題研習：初小推行 「自己動手執書包」、 「小一100日」 及「生活技能」等課程，從小培養自理能力。五年級進行服務學習，讓學生切身體現籌劃及實踐服務活動的過程。
從閱讀中學習：恆常進行好書推介、閱讀網站推介、親子伴讀、大哥姐伴讀、家長閱讀工作坊及參觀圖書館等活動。
德育及公民教育：全面配合學校關注事項，發展生命教育。
共通能力的培養 : 透過日常教學及各類活動培養學生的九種共通能力，尤其著重溝通、協作、創意及研習能力。
學生支援
全校參與照顧學生個別差異 : 1. 為成績稍遜的學生安排輔導課。
2. 參加融合計劃，為個別學生調適課程及評估模式。
3. 設小一「及早識別」計劃、英語拼音課程及「幼小銜接」活動。
4. 設免費課後功課輔導班。
5. 推行新來港學童適應課程及課後校本支援計劃。
6. 言語治療師及教育心理學家到校提供專業服務。
課程剪裁及調適措施 : 設小組輔導班以協助成績稍遜的學生；此外中英數三科亦設有小組課後輔導班，以協助學生提升學習表現；部份有需要學生按個別情況調適課程。
家校合作及校風
家校合作 : 定期舉行會議及舉辦活動，如家長講座、親子課程等，另外家長亦積極參與校內各活動及工作，如早上協助學生量度體溫、運動會家長義工及協助圖書館工作等，以增強家校合作的關係，提高家長及學生對學校的歸屬感。
校風 : 本校積極推動基督化全人教育，推展「生命教育」及「閞愛文化」，培養學生的良好品德，發展學生體藝潛能，讓學生盡展所長。
學校發展
學校發展計劃 : 1. 深化學生自主學習的能力，培養自主學習的習慣，提升學習的自信。
2. 照顧學生學習多樣性。
3. 推展生命教育，以培育學生正面人生觀。
4. 中文科開設「普教中」班。
教師專業培訓及發展 : 建立共同備課、同儕觀課文化，並且持續進行學習圈等教師專業發展活動，提升學與教效能。教師亦積極參與各類工作坊、交流活動及校外支援計劃，以促進專業成長。
學費減免 : 不適用。
其他 : 學校重視關愛文化的建立，連續五年在基督教服務處主辦的「關愛校園獎勵計劃」中榮獲「關愛校園」榮譽。
本校為響應全球減少碳排放運動，過往數年積極在校裝置不同的節能設備，又舉辦活動鼓勵同學養成節能減排的習慣，成效顯著。
