班級教學模式
1. 小一至小四按學生的需要，把三班學生分成四組於四個課室中上課，加強師生比例，尤勝小班教學。2. 設課時內或課時外外籍老師協作教學課程，提升英語水平。3. 小二至小三開設以普通話教授中文的班級。

學習評估
全年全科測驗次數 : 0
全年全科考試次數 : 3
多元學習評估 : 本校採用進展性（日常習作及學習活動）及總結性評估（每年三次考試），促進學習和改善學與教效能，加強學習信心。
分班安排 : 由小一至小四進行「分科分流」小組教學，提升學習效益，設立8人基礎組，以同質分組策略設計合適該組學生需要的課程，鞏固學科水平，亦讓尖子學生進階學習資優課程。

健康校園生活 : 曾參與香港健康學校獎勵計劃、一人一花計劃、綠色學校獎、足脊健康操計劃、全港熄燈大行動、廚餘堆肥實驗計劃、 ICAN學校等，屢獲採訪及獎項。
備註 : 1. 每週60餘組免費多元智能及小息活動，另外40項專科班及校隊課程，讓學生熱愛上學，充實校園生活。
2. 由小一至小六推行「一人一職」計劃，提升學生自信及服務精神。
3. 90%小六畢業生勇闖高峰成就，成功升讀心儀中學。
全方位學習活動
1. 舉辦北京、台灣、新加坡、日本及澳洲語言文化交流團，增強學生語文能力及擴闊視野。 2. 每週一至六舉行中西樂器、視覺藝術、體育、中外舞蹈、學術研習和資優網上學習計劃，發展每個人的潛能，學生屢獲殊榮。
辦學宗旨
以耶穌基督信、望、愛的精神為經，活動教學與訓輔合一的模式為緯，全人發展為目標，透過各項領袖訓練及制服團體計劃，使學生能愉快地成長，成為自信、自治和自強的領袖人才。
學校特色
學校管理
學校管理架構 : 真鐸學校校董會是以關愛為使命的團隊。
下設教務、課程統籌、訓育及輔導等不同行政小組，以推行學校發展計劃及執行各項工作。
法團校董會/學校管理委員會/校董會 : 真鐸學校校董會（由教育家、藝術家、律師、會計師、牧師和校長組成，以關愛為使命的團隊）。
環保政策 : 1. 推廣環保意識。
2. 推動生命教育。
3. 培育學生成為地球的好管家。
學校關注事項 : 發揮學生潛能，推進全人教育。
教學規劃
學習和教學策略 : 參與教育局資優教育組，資優教育學院、馮漢柱資優教育中心及各大專院校發展學生於中、英、數、常、電腦、普通話、音樂、體育和視藝科才能，促進卓越學習表現。
關鍵項目的發展 : 1. 每天設早晨閱讀課及全方位閱讀計劃，讓學生熱愛閱讀，提升語文能力。
2. 德育、公民及國民教育：每週成長課、頭條新聞啟示及國情教育週。
3. 專題研習：自小一起推行至小六，跑出課室戶外實地學習，曾獲明報專訪及教育局獎項。
4. 資訊科技學習﹕My IT School、E-class、古生物學及天文學網上學習課程。
共通能力的培養 : 1. 思維能力：以各科課程，課業與評估為本，發展學生創造力、批判思考能力、解決問題能力和研習能力。
2. 社群適應力﹕以課堂中的互動活動提升學生協作能力、溝通能力和自我管理能力。
學生支援
全校參與照顧學生個別差異 : 1. 為全校每位學生設立關顧老師進行個別學生計劃。
2. 設放學後托管計劃，協助約100名雙職家長看管子女至6:30。
3. 每週32組課後學業及加強輔導，提升學生書寫、識字、專注力、人際溝通、語言表達、和中、英、數科的溫習及考試策略。
課程剪裁及調適措施 : 1. 設立小一至小三分層課業及評估，照顧學習差異。
2. 設立小五、六英文專詞培訓計劃，為升中的同學作好準備。
家校合作及校風
家校合作 : 1. 家長教師會每年舉辦德育、康樂活動及興趣小組，增進家長與教師間聯繫，透過家校緊密合作，攜手培育孩子的全人發展。
2. 每年平均60次家長教育講座，每週2天家長小組活動，逾200位家長義工大使，攜手推動學與教項目。
校風 : 1. 貫徹基督教「一個也不能放棄」精神。
2. 學生品性純良、有禮，喜愛學校，服從師長。
3. 學校重視關愛文化和家校一心精神。
學校發展
學校發展計劃 : 多功能活動樓層加建工程約於2018年落成。
教師專業培訓及發展 : 1. 進行校本教師專業培訓，凝聚一群具專業教學策略和關愛文化的教師團隊。
2. 本校教師曾獲邀出席教育局、會議展覽中心、科學館、浸會大學、香港教育學院等大專院校分享及主持教育講座，為德育及公民教育、資優教育、學生支援、資訊科技教育、英文科、常識科、小一適應及家長教育等範疇作出分享。
學費減免 : 不適用。
其他 : 1. 校園特色﹕防風林約120棵樹、有機果樹園約60棵樹、空中花園、攀石牆、天文望遠鏡。
2. 為家長設小一新生適應課及學前評估。
3. 以及開學適應週。
4. 引述教育局給予本校的外評報告評語：學校歷史悠久，在新校長和全體教職員的努力下，在各個範疇的表現均屬良好。學校在教學資源、學習評估、外間聯繫及學校文化等各方面均有優良表現。學生中、英、數學科表現令人滿意。
