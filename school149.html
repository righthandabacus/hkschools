辦學宗旨

學校特色
學校管理
學校管理架構 : 由法團校董會管理，日常事務由校長及行政管理小組負責。
法團校董會/學校管理委員會/校董會 : 本校已成立法團校董會，校董會成員包括辦學團體代表、教員校董、家長校董、校外人士校董及校友校董。
環保政策 : 本校重視環境教育，透過課程、專題研習、參觀、講座等，從小培育學生環保意識，希望新一代能愛護大自然，善用資源。
學校關注事項 : 三項關注事項：
1. 提升學與教成效，優化校本課程。
2. 加強教師專業發展，提升學與教效能。
3. 認識祖國。
教學規劃
學習和教學策略 : 著重發展學生的共通能力及多元化的學與教策略，提供不同的學習經驗，以提升學習成效。
關鍵項目的發展 : 1. 從閱讀中學習：透過多元化的學習活動教授閱讀策略，推行閱讀計劃、早會圖書推介、早讀等活動，培養學生的閱讀習慣，增強學生閱讀能力。
2. 德育及公民教育：透過周會、早會、主題活動及成長課，培養學生正確的價值觀和態度。
3. 專題研習：推行專題研習，讓學生提升研習技巧。
4. 資訊科技教學：於各科廣泛應用資訊科技，促進學生學習，並增強自學能力。
5. 藝術教育：培養學生對藝術的興趣及欣賞藝術的能力。
6. STEM：培養學生對科學及電腦知識的探求及邏輯思索，高年級學生更學習編程知識。
共通能力的培養 : 透過合作學習、專題研習、解難活動和全方位學習活動，提升學生共通能力。
學生支援
全校參與照顧學生個別差異 : 本校設有學生學習支援組，利用各項津貼支援有需要的學童，提供不同的支援策略及多元學習的經歷。拔尖補底課程支援不同能力的學生，提升他們的學習成就。此外，為有特殊教育需要的學生提供校本支援服務，包括個別學習計劃、小一及早適別、升中轉介、功課及測考調適、各項小組治療及課後班，以照顧學生的學習需要。
課程剪裁及調適措施 : SENCO計劃設立特殊教育需要統籌主任，為學生策劃個人化學習計劃，根據學生的需要，訂立學習目標及提供適合的支援項目。根據學生的能力為學生進行調適。另外，老師為學生設計高階思維訓練之課業，以啟發他們的潛能。
家校合作及校風
家校合作 : 設有家長教師會及家長義工隊，透過家校活動及義工服務，增強家校溝通和合作。此外，家長教師教會舉辦親子旅行、華服日、親子歷奇活動等活動加強家校合作。本校致力推動家長教育，鼓勵家長參加講座，協助提升他們與子女之間的溝通和瞭解，支援家長處理子女成長過程中所遇到的困難。
校風 : 本校重視品德教育，營造互相欣賞、關懷的氣氛及提升學生的成就感，從而建立關愛文化和提升學生的自信。學校推行生命教育，幫助學生建立正面的價值觀及積極的人生態度。本校亦推行「好學生」獎勵計劃，鼓勵學生努力學習，積極參與活動，關懷身邊的人，建立正確的價值觀。
學校發展
學校發展計劃 : ‧優化中英數常的校本課程
‧推行品德教育
‧發展STEM教育課程
‧為學生提供多元化的課外活動及學習經歷
‧為學生提供適切的學習支援
教師專業培訓及發展 : 透過聯校教師專業發展日及校本專業培訓，重點發展教師照顧學生學習的多樣性。學校更鼓勵教師參加教育局的進修課程及學習圈，以提升他們的專業素質。另外，本校亦安排教師參觀不同的學校，讓他們進行考察及交流。教師透過共同備課及課研計劃提升教學效能。
學費減免 : 不適用。
其他 : 本校積極參與各活動及比賽，學生成績優異，取得多個獎項：第53屆學校舞蹈節甲等獎；第68屆學校朗誦節獨誦亞軍、季軍；九東遊戲比賽季軍；沙田武術錦標賽拳術亞軍；南拳冠軍、武術季軍；健康生活填色比賽冠軍等。
