班級教學模式
推行活動教學及校本資優課程，設校本生活智慧課，推行「合作學習」模式，提升學生的溝通及互動學習能力。設加強輔導班及課後輔導課。二至六年級推行普通話教中文模式。
班級結構備註
向教育局申請現金津貼，為家境清貧學生安排課後學習及支援計劃。增聘兩名外籍英語教師，推行校本英語拼音及閱讀課程。

分班安排 : 二年級全級按成績平均分班。三、四年級設一班精英班，其餘按學業成績平均分班；五、六年級設一班精英班，一班次精英班，其餘按學業成績平均分班。

健康校園生活 : 推行健康校園政策，提供現場派飯，並推行健康飲食在校園、生果日等。小息設有跳繩區及跑步區，推行體適能計劃及舉辦衞生講座。
備註 : 本校著重學生五育的均衡發展，致力提供以學生為本的學習生活，積極培訓學生溝通能力、創意思維及自學能力。並於小息、午膳時段，安排各級學生到圖書館、電腦室、各自學中心及其他場地進行活動。定期帶領學生參加境外交流活動，放眼世界。
全方位學習活動
管弦樂團、中國鼓、古箏、合唱團、英文民歌、朗誦、故事演講、辯論、書法、奧數、多媒體製作、智能機械人、科學專題探究、舞蹈、戲劇、視藝、田徑、籃球、足球、排球、羽毛球、乒乓球、游泳、花式跳繩、童軍、公益少年團、少年警訊、義工服務等。
辦學宗旨
本校以發揚愛、敬、勤、誠的精神，提供優質的小學教育服務。學校傳授知識、訓練技能及鍛鍊體格外，更培養學生優良的品格、正確的人生觀。透過學校生活，讓學生懂得珍惜自己，且能關心別人，實踐所學以回饋社會。
學校特色
學校管理
學校管理架構 : 法團校董會下設學校發展、校政諮詢及改善委員會，另設多個行政及學科組別，各負權責，以制訂及推行學校發展計劃。
法團校董會/學校管理委員會/校董會 : 本校於2007年6月註冊成立法團校董會，成員包括校監、辦學團體校董、校長校董、家長校董、教員校董及獨立校董。
環保政策 : 推行綠化校園、廢物分類回收、智慧用電、環保大使、惜食計劃等。建設再生能源設施，生態魚池、種植園地、蝴蝶長廊、地質研習角，讓學生實地學習保護環境。
學校關注事項 : 1. 深化學生自主學習和建構知識的能力，培養主動學習的精神。
2. 培養學生良好品格及正面價值觀，共建關愛文化。
3. 優化學校行政電子化應用，加強知識管理效能。
教學規劃
學習和教學策略 : 1. 加強學生自主學習培訓。
2. 優化課堂教學，推行合作學習，促進同儕之間的互動學習。
3. 推行全方位學習模式及課程統整；著重學生閱讀興趣的培養。
4. 舉辦不同的專題學習活動及聯課活動。
5. 推行系統性創意思維課程，加強學科創意教學，舉辦解難活動。
6. 設生活智慧課，推行環境、品德及生命教育，促進學生多元智能的培育。
關鍵項目的發展 : 1. 初小中文科推行「喜閱寫意：小學校本分層支援課程」，英文科推行PLPR及Read Write Inc.閱讀計劃，高小推行KIP計劃。
2. 每週設英語日、普通話日及閱讀日。推行廣泛閱讀、閱讀獎勵計劃，教授閱讀策略。
3. 推行資訊科技輔助教學。
4. 推行公民教育、環保教育及德育成長課等。
共通能力的培養 : 於各科融入口語溝通、公開演說、分組討論及探索研究，發展學生批判性思考和創意等共通能力。
學生支援
全校參與照顧學生個別差異 : 1. 設學生輔導主任提供學生輔導及培訓。
2. 提供校本言語治療及教育心理服務。
3. 一年級設及早識別計劃。
4. 設學生支援小組、加強輔導教學計劃、小組訓練、課後輔導等。
5. 設校本資優課程，發展學生潛能。
課程剪裁及調適措施 : 為有特殊學習需要學生設家課及考試調適。校本資優課程為學生進行高階思維及能力訓練，發展學生潛能。
家校合作及校風
家校合作 : 重視溝通，設世德家書、家長會及家長日等。家長教師會擁有一隊優秀的義工隊，舉辦親子旅行、攤位遊戲、講座、親子比賽、興趣班等。設「家長閱讀大使」，協助組織學生「讀書會」。
校風 : 校風優良，重視學生公民意識及道德觀念的培養。學生均自律、守規、有禮貌，融洽相處，營造和諧關愛的校園氣氛。
學校發展
學校發展計劃 : 1. 深化學生自主學習和培養學生主動學習的精神。
2. 加強照顧學生個別差異，發展資優教育課程。
3. 提升學生的個人素質及良好的生活習慣。
4. 加強境外的遊學交流，擴闊學生視野及實際學習經驗。
5. 繼續推行環保計劃，建設綠色學校。
6. 繼續發展全班式及抽離式資優課程計劃。
7. 鋪設全校WiFi網絡，透過電子學習增加互動學習機會。
8. 透過互動式的學校手機應用程式，加強家校溝通和合作。
教師專業培訓及發展 : 藉教師發展日、工作坊、共同備課、同儕觀課、教學行動研究、教學分享及教學觀摩，以達致專業交流。
學費減免 : 不適用。
其他 : 1. 2001-2008為教育局資訊科技卓越中心。
2. 曾獲保良局傑出學校獎、香港藝術發展獎「藝術教育獎」銅獎、香港環保卓越計劃銅獎、香港綠色學校金獎。
3. 每年舉辦世德音樂會、綜藝匯演、六年級畢業宿營等。
