班級教學模式
小一至小六推行小班教學；小三開始分流教學；推行資優課程；小三、小四設有德育戲劇課；小五及小六常識科增加批判性思維課程；各級設有功課輔導時間。
班級結構備註
本校一直擁抱「一個也不能少」的信念，以「全校參與」的模式推展關愛及共融文化，致力把青松侯寶垣小學建立成一個關愛校園。每年學校會按學生的需要編訂多達數十項的計劃支援學生成長。

辦學宗旨
追求卓越學術表現，發揚校訓「尊道貴德」精神，因材施教，五育並重，培養學生關心及服務社會。
學校特色
學校管理
學校管理架構 : 行政小組管理校務、課程發展、支援學習、學務、課程、外務、資訊科技、訓導、輔導、學生健康和活動事宜。科組主任和學校改善小組協助發展學校。
法團校董會/學校管理委員會/校董會 : 法團校董會於二零零八年一月三十一日成立，成員包括校監、辦學團體校董、教師和家長校董。
環保政策 : 環保課程和活動，建造綠色校園，並獲外界嘉許。
學校關注事項 : 1. 透過健康校園計劃促進學校發展。
2. 提升學生高階思維能力，促進學與教。
3. 藝術教育。
教學規劃
學習和教學策略 : 設有功課輔導節，協助學生解決功課問題，而週五下午設有課外活動節。早上設有閱讀時間。全方位學習、科技活動、外地學術交流、專題研習、校園電視台、多媒體製作、自學獎勵計劃、社會服務。老師會備課，調適及設計教材，並進行主題式學習。
關鍵項目的發展 : 推行校本德育課程及自我挑戰獎勵計劃、STEM、藝術教育、自主學習、全方位學習、閱讀課程、健康教育。
共通能力的培養 : 本校各科發展共通能力，例如常識科種籽計劃（透過探究式活動，發展學生批判性思考及創意能力）、小記者活動（加強學生溝通、合作及解難能力）。
學生支援
全校參與照顧學生個別差異 : 本校設有「耀星」資優培訓計劃，以發掘及培訓資優學生。老師調適課程以照顧不同程度的學生。本校設有學生學習支援組，統籌支援學生學習工作。學習支援小組協助有學習需要的學生。如為有需要同學安排功課輔導，進行協作教學和抽離班等。
課程剪裁及調適措施 : 教師共同備課拔尖補底，設立校本課程，如﹕英文詩篇寫作及奧數課程，強化自覺學習能力，回應小班教學策略，資優課程及社區支援服務。
家校合作及校風
家校合作 : 本校家教會著重家校合作，舉辦不同活動，例如親子活動多元化的家長興趣班，務求達致學校、家長以及學生融為一體。
校風 : 本校秉承「興學育才」辦學宗旨，培養學生高尚情操。學生有禮、守規，懂得尊重他人，校園氣氛和諧。我們舉辦小一及升中銜接課程，提供多元化的課後輔導。
學校發展
學校發展計劃 : 多年獲「關愛校園」榮譽，以正向心理學提昇學生抗逆力，強調生命教育，成就學生。
教師專業培訓及發展 : 教師發展日、備課會、觀課、講座及進修。
學費減免 : 不適用。
