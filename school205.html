辦學宗旨
秉承鳳溪公立學校一貫辦學宗旨，以「謙、信、勤、敏」為訓，實施德、智、體、群、美全人教育；培養學生獨立思考能力，協助他們建立正確及積極的人生觀，提供優良學習環境及均衡課程，讓他們愉快地學習及發展潛能。
學校特色
學校管理
學校管理架構 : 鳳溪公立學校設董事局，統籌及管理屬下之教育及慈善事務。下設教育小組推動及監察教育事務發展。
法團校董會轄下由校監、校長、辦學團體、家長、教員等代表組成。校董會下設行政會議，分設校務、課程、教務、訓育、輔導、資訊科技等小組，以推動學校發展計劃及執行各項工作。
法團校董會/學校管理委員會/校董會 : 於2005年成立，現屆校董共20人，分別由校監、校長、9名辦學團體代表、2名名譽校董、2名獨立人士代表、1名校友代表、2名家長代表、2名教員代表組成。
環保政策 : 1. 參與康樂及文化事務署校園綠化資助計劃。
2. 鼓勵循環再造及回收物資，校內設循環分類回收桶。
3. 課室窗戶鋪設節能隔熱膜。
4. 鼓勵健康用膳及珍惜食物。
5. 設有學生環保清潔大使。
學校關注事項 : 創意與傳統並重，體藝伴學術齊飛。
教學規劃
學習和教學策略 : 校本課程著重啟發學生高階思維，包括英文閱讀及寫作、童詩創作、數學解難及速算、網上科學探究教室、專題研習、演說課程、太極、高爾夫球及射箭等，通過網上學習活動、跨科組活動及全方位學習拓闊學生之學習經歷。
關鍵項目的發展 : 閱讀：廣泛閱讀計劃、閱讀獎勵計劃、圖書伴讀等。
專題研習：校本基本技能課程。
德育：成長教育、價值教育、國情教育、班級經營及服務學習。
資訊科技：協作式互動學習及網上學習等。
共通能力的培養 : 於校園電視台、課堂學習及聯課活動中，以滲透方式發展學生的創意、溝通、協作、分析、明辨和解難等能力。
學生支援
全校參與照顧學生個別差異 : 設加強輔導班、學生支援小組及言語治療服務；並於二至六年級設功課輔導；資優生設小小科學家、小小演說家、奧數及英語話劇等課程。
課程剪裁及調適措施 : 為特殊學習需要學生作教學、功課及考試調適：為讀寫困難學生作抽離式加時及放大試卷安排。
家校合作及校風
家校合作 : 家長愛護學校，每天到校參與義務服務。 家教會每年舉辦講座、工作坊及領袖訓練班等，並津貼學校舉辦學習活動。 本會主席為法團校董會校董，在會議中反映家長意見。 家長認同及支持學校，稱讚教師關心學生，及讓不同能力的學生得到適切照顧。
校風 : 1. 校風純樸，學生率真活潑，有禮守規，尊敬師長，對學校有歸屬感。
2. 朋輩間相處融洽，校園體現互助互愛精神。
3. 學生對學習感興趣，樂於參與課堂活動，愉快學習。
4. 學生遵從師長指導，師生關係和諧融洽。
5. 學生愉快地投入學校生活，喜愛閱讀，積極參與各項活動。
學校發展
學校發展計劃 : 銳意發展為學習型組織、學術型機構。
1. 推展教學策略優化計劃。
2. 優化電子互動平板電腦教學計劃。
3. 發展專業學習社群。
4. 優化學校自評。
5. 發展多元體藝課程。
教師專業培訓及發展 : 同儕觀課、同研共備、分享學習圈。與香港大學、香港中文大學、香港教育大學、香港城市大學、香港教師中心、教育局等合作不同計劃。
學費減免 : 不適用。
其他 : 1.教學特色：電子學習、教學研習、自主學習及多元學習組成「鳳小四習」的教學特色。各科透過課堂研究及電子模式等推展自主學習。2.電子學習概況：WiFi100電子校園，設BYOB班及全校教室設互動白板或大電視；全新SteamLab推展STEAM及CoolThink編程課程；是全港首12間編程教學中心。3.其他：與港大、中大、教大、城大、香港教師中心及教育局等合作不同計劃。
