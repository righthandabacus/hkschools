全方位學習活動
同學於星期五下午課外活動節可參加不同的多元化活動，包括話劇、花式跳繩、合唱團、英語木偶劇場、手工藝、羽毛球、田徑等。 此外，本校於平日放學後及星期六設有跆拳道、少林功夫、中樂、西樂、民族舞班等十多項課餘興趣班供同學自由參加。
辦學宗旨
本校致力推行全人教育，為學生提供優質卓越的教育服務。不但著重發展多元化課程，而且建立多角度探索思維能力，引導學生兼容不同文化，培養藝術的審美眼光，致力培養成為「德才兼備、善於溝通、樂於服務、敢於創新」的孩子。
學校特色
學校管理
學校管理架構 : 學校管理委員會、校長、副校長、行政小組。
法團校董會/學校管理委員會/校董會 : 學校管理委員會主席、校長、教師、家長及社會人士。
環保政策 : 推行環境教育，知識與實踐並重。
學校關注事項 : 1. 優化校本課程，強化學生學習能力，建立自主學習的文化。
2. 啟發個人潛能，強化學生正面的價值觀及身心健康。
3. 發展電子學習計劃，促進學生有效學習。
教學規劃
學習和教學策略 : 1. 生活教育課與個人成長課結合，與社區組織協辦講座和活動。
2. 運用合作學習及電子學習以訓練及培育學生積極互賴的學習精神及自主學習的態度，提升教學效能。
3. 進行跨學科專題研習，以常識科為平台，中、英、數、視藝、電腦科加以配合。學生以小組形式進行研習，再進行匯報及分享。
4. 與荔灣區沙面小學結成姊妹學校，互相交流。
關鍵項目的發展 : 1. 圖書課教授閱讀技巧和策略，讓學生學會從閱讀中學習。
2. 進行資訊科技教學，電腦課提供基礎知識，為跨學科專題研習提供支援，協助學生搜尋資料及編輯報告。
3. 進行跨學科專題研習。
4. 在德育及公民教育方面，透過成長課及健康校園講座，培養學生建立健康的生活態度。
共通能力的培養 : 共通能力是學習的基礎，本校透過跨學科專題研習，發展學生的九種共通能力（協作能力、溝通能力、創造力、思辨能力、運用資訊科技能力、運算能力、解決問題能力、自我管理及研習技能），讓學生學懂把這套共通能力轉移至其他學習領域及情境。
學生支援
全校參與照顧學生個別差異 : 透過及早識別和輔導，推行課程和評估調適，透過增益班、課後支援小組及小組學習，照顧不同學習能力的學生。
課程剪裁及調適措施 : 為照顧學生的學習多樣性，於不同級別開辨拔尖增益班，如：課前增益、課後拔尖，課堂小組學習等，課程上作出調適及增潤，以配合不同能力學生的需要；課堂上以合作學習及電子學習策略，讓學生積極互賴地學習，培養自主學習的態度；此外，本校推薦學生報讀校外資優課程，與校外專業機構合辦延伸課程，擴闊學生的學習經驗
家校合作及校風
家校合作 : 家長擔任義工，協助學校推行各學習活動，如親子工作坊、環保校服收發活動、聖誕聯歡會、畢業禮等，促進交流合作。
校風 : 1. 透過學科大使、十大傑出學生選舉、模範風紀選舉等活動，提高學生對學校的歸屬感。
2. 成立風紀隊協助學生養成自律守規的精神。
學校發展
學校發展計劃 : 1. 透過有系統教師持續專業發展、共同備課及同儕觀課，以提升學與教效能。
2. 加強《學會學習》的四個關鍵項目。
3. 銳意藉着各科課堂教學，鞏固學生的學習態度及成績。
教師專業培訓及發展 : 學校配合每年校務計畫的關注事項，選定教師專業發展日及教師培訓活動的主題，並邀請相關的專業團體或人士協辦工作坊，從而建立一個專業及凝聚專業知識的實踐社羣。
學費減免 : 不適用。
