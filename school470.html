團隊組織：童軍、公益少年團、校園電台小記者。各科培訓：奧林匹克數學組、機械人操作組、合唱團、田徑組、球類訓練、游泳隊。興趣班：樂器班、書法班、國畫班、跆拳道班。六年級戶外教育營。
辦學宗旨
本校以「篤行力學」為校訓，重視修養學問及品德，鼓勵學生在日常生活中實踐所學，貫徹知行合一的精神。透過學業、訓育、輔導及活動四方面的培育，令學生在德、智、體、群、美五育得到均衡及全面的發展。
學校特色
學校管理
學校管理架構 : 辦學團體是香港潮陽同鄉會，學校由香港潮陽小學法團校董會管理。
法團校董會/學校管理委員會/校董會 : 成立法團校董會，成員包括辦學團體校董、獨立校董、家長校董、校友校董及教師校董。
環保政策 : 設廢物分類，鼓勵循環再用。設有廚餘處理機處理廚餘。
學校關注事項 : 1. 自主學習。
2. 跨學科學習。
3. 教師專業發展及教師評核。
教學規劃
學習和教學策略 : 1. 以課程發展議會建議的中央課程為基礎，設計校本課程，加入多元化學習活動。
2. 小一至小六小班教學。
3. 小一全級「普教中」，小二及小三分班「普教中」。
4. 小一開展Primary Literacy Programme - Reading 。
關鍵項目的發展 : 1. 舉辦閱讀獎勵計劃，推行中英文廣泛閱讀計劃及Primary Literacy Programme - Reading。
2. 鼓勵學生於網上平台進行互動學習。
3. 各科專題研習活動。
4. 各科滲透德育及公民教育元素；舉辦主題式校本輔導活動。
共通能力的培養 : 透過課堂及課外活動訓練培養學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 1. 二至三年級設「小學加強輔導計劃」。
2. 一至六年級「默書調適」計劃。
3. 二至六年級課後「補習班」。
課程剪裁及調適措施 : 1. 中、英、數、常四科進行課程調適。
2. 各科推行拔尖計劃。
家校合作及校風
家校合作 : 1. 家長教師會每月定期舉行會議，籌備親子活動及為家長安排講座、研習班；會上由家長委員反映家長意見，校方作出回應及檢討。
2. 成立家長義工隊，協助籌備學校大型活動、照顧小一新生、日常當值及推廣閱讀風氣等。
校風 : 透過學業、訓育、輔導及活動四方面培養學生在德、智、體、群、美五育的發展。
學校發展
學校發展計劃 : 1. 培養學生的主動學習態度。
2. 優化課堂教學，提升學與教的效能。
3. 讓學生認識國家的歷史及現況，學習及欣賞中華民族之文化及藝術，培養他們對國家之情懷。加強學生對國民身份之認同，並激勵他們立志將來為國家作出貢獻及承擔。
教師專業培訓及發展 : 1. 舉辦校本教師教學經驗分享會、講座及工作坊。
2. 與國內學校及本地學校作教學交流及經驗分享。
學費減免 : 不適用。
其他 : 1. 設立獎學金獎勵學術成績優異、校外比賽成績優異及品行優異的學生。
2. 本校合唱團及舞蹈組在校際比賽中屢獲冠軍及優等獎優異成績。
