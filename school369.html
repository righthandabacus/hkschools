辦學宗旨

學校特色
學校管理
學校管理架構 : 法團校董會訂定學校的教育方針，校長帶領各行政小組計劃及統籌各項校務進行。
法團校董會/學校管理委員會/校董會 : 本校於二零零五年成立法團校董會，成員包括辦學團體校董、教師校董、校友校董、家長校董及獨立校董。
環保政策 : 以體驗式的環保課程及活動培育學生保護環境的態度和習慣。
學校關注事項 : 1. 透過優化課堂設計及教學策略，引發學生的好奇心及探究精神。
2. 創設各種學習體驗，讓學生敢於面對困難，培育學生積極樂觀的態度。
教學規劃
學習和教學策略 : 學習策略以學生為本，注重閱讀、探究、協作、明辨性思維等自主學習能力的培養。學校設有主題學習日、課統週、STEAM課程、通識課等，讓學生掌握知識和技能，培養正確的價值觀。
關鍵項目的發展 : 1. 透過閱讀統整各學科課程，培育學生享受閱讀，並能透過廣泛及有深度的閱讀成為有品格、具內涵及自主能力的終身學習者。
2. 推行電子學習計劃，發展電子教材、學材，配合創新的教學模式，培養學生自主學習。
3. 在各級推行「多元價值教育文化藝術特色課程」，以提升學生的人文關懷及藝術情操。
4. 透過不同的活動，培養學生的德育及公民教育的意識、正確的道德觀及價值觀。
5. 有系統地發展「專題研習」及「課程統整週」，教導學生運用思維策略，學習整理及分析資料，並讓學生進行自評及他評，提升自省能力及批判性思維。
共通能力的培養 : 1. 在各級推行「多元價值教育文化藝術特色課程」，透過優質的學習活動，讓學生體驗不同的學習經歷，以提升學生的人文關懷及藝術情操，為學生建構美好的心靈世界，培養學生的共通能力。
2. 推行多元化體驗式STEAM 課程，將學科的知識有機地整合，培養學生整全的人文價值和具備科學科技素養的創新思考能力。
學生支援
全校參與照顧學生個別差異 : 資優課程包括：數學精英培訓班、Reading Wonderland、English Activity Club、小編輯、奧林匹克機械訓練班等四十多項，讓學生發展潛能；又設有輔導課、家長及區本功課輔導、成長飛躍輔導計劃等，輔助成績稍遜的同學。
課程剪裁及調適措施 : 本校根據新資助模式下的學習支援計劃，開設輔導教學組，協助同學學習。
本校為小一及小六提供升小及升中適應課程、工作坊等，另有大哥哥大姐姐計劃，協助同學盡早適應。
家校合作及校風
家校合作 : 設有家長教師會，家長網絡等，廣納家長意見，完善學校政策。我們也重視家長教育，經常舉辦家長學堂、工作坊、親子活動等以加強家長對學校的歸屬感。
校風 : 1. 透過創設的情景，讓學生從經歷以積極樂觀的態度尋求解決問題的方法。2. 透過不同的活動，深化學生積極樂觀的價值觀，持續培養「舊墟寶湖人」的特有內涵。
學校發展
學校發展計劃 : 1. 透過多元化體驗式校本STEAM課程，將學科的知識有機性地整合，使學習與生活扣連，讓學生通過活動體會學習的樂趣和意義，從而培養學生整全的人文價值和具備科學科技素養的創新思考能力。
2. 持續以閱讀統整各學科課程發展，以閱讀為學習核心，讓學生享受閱讀，並能透過廣泛及有深度的閱讀獲取知識，感受世界，成為有品格、具內涵及自主能力的終身學習者。
教師專業培訓及發展 : 有系統的週年專業發展計劃，每年安排培訓及交流，提升教師的專業。校長及教師更常被各政府或各教育機構邀請作教學分享。
學費減免 : 不適用。
其他 : 每年均舉辦海外交流計劃，增強學生對各地文化的認識及體驗，部分學生更被邀請到海外參加國際性的比賽，增廣見聞。學生曾到上海、南京、韶關、廣州、大連、重慶、青島、煙台、北京、天津、東莞、英國、澳洲、新加坡、新西蘭、美國、蘇格蘭、俄羅斯、台灣、韓國、日本等地進行交流活動，有的更寄住當地家庭，與當地學童一同上課及生活。
