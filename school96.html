全方位學習活動
本校十分重視學生的課餘生活，並透過《常規性活動》、《多元智能課》、《學科聯課活動》、《午間活動》、《課餘活動》、《試後活動》及《暑期活動》七大範疇。讓學生有機會走出課室學習、鍛煉身心，為生活增添色彩。
辦學宗旨
本校以基督精神為本，輔以校訓「榮神愛人」之使命，為學生提供優質教育，使學生能在德、智、體、群、美、靈六育方面有均衡發展，致力提供愉快的學校環境；啟發潛能，建立良好的人際關係和正確人生觀。
學校特色
學校管理
學校管理架構 : 領導人員分別在管理與組織、學與教、校風及學生支援這三個範疇推展校務。全體教師分配入上述組別，透過自評機制，自我提升專業水平。
法團校董會/學校管理委員會/校董會 : 於2011年成立法團校董會。成員包括辦學團體代表、校長、教師、家長、校友及獨立人士，共同釐訂學校發展項目，審批財政預算、人事安排及檢討工作成效，提高透明度及問責性。
環保政策 : 實踐5R環保理念，如使用雙面紙、塑膠、鋁罐及廢紙回收、以電子郵件溝通及減少列印文件，以課室現場分飯模式安排學生午膳，減少浪費飯盒。推行環保教育，加強學生環保意識。
學校關注事項 : 1. 運用適切的教學策略及資訊科技工具，以提升學與教之效能。
2. 培養學生良好的品德及正面的價值觀。
教學規劃
學習和教學策略 : 學校按學生的需要發展校本課程，為學生提供豐富的學習經歷，鞏固學生的基礎知識，發展共通能力。透過小班教學，制定清晰教學目標，運用合適的教學策略和資源，促進課堂互動，以協助學生建構知識，並培養學習能力，建立正確的態度和價值觀。
本校獲優質教育基金撥款近一百五十萬元，於2017年至2018年期間，進行「一個也不能少的學與教取向─應用課堂學習研究進行優化課堂教學以照顧學生的學習差異」計劃。2016/17年度，於六年級中、英及數三科各選一個課題，透過校外專業顧問主持專業培訓、工作坊、共同備課等，培訓種子老師，實踐課堂學習研究，促進同儕交流教學經驗及專業成長。
關鍵項目的發展 : 本校積極提倡「Waiting time is Reading time」，設有午間閱讀時間，培養學生閱讀習慣；中、英、數三科均設多元化閱讀活動。除電腦課外，不同學科均利用資訊科技進行學習。透過舉辦不同講座和獎勵計劃，培養學生的品德價值。此外，透過「專題研習」活動，使學生把學習與生活連繫，有效掌握概念和運用知識。透過校園電視廣播，為學生提供多元化的學習平台及分享機會。
共通能力的培養 : 本校著重發展學生的協作能力、溝通能力、創造力及批判性思考能力（4c），培養學生的自學能力，增強學生的獨立思考及自律自理的習慣。
學生支援
全校參與照顧學生個別差異 : 設有學生支援組，及早識別有學習困難的學生，由駐校社工、言語治療師及教育心理學家提供輔導服務；同時，為學生提供適切的支援服務，如課後功課輔導、讀寫樂課程、想法解讀課程、寫作協作、功課及評估調適等。此外，本校推行資優教育，開辦不同的資優課程，以發展學生潛能。
課程剪裁及調適措施 : 本校提供資優課程、評估調適、功課調適及讀寫訓練。
家校合作及校風
家校合作 : 家長教師會定期舉辦家長教育課程、旅行及興趣班等。家教會積極鼓勵家長參與學校活動，如故事爸媽及圖書館義工，以加強家校協作，促進親子和家校關係。
校風 : 本校校風淳樸，學生有禮守規，同學間彼此接納，和諧共處。學校透過不同的學生成長支援措施，促進學生的個人及群性發展。
學校發展
學校發展計劃 : 1. 加強生命教育—品德培育。
2. 資訊科技教育及自主學習—強化學習能力。
教師專業培訓及發展 : 本校積極鼓勵教師參與教師發展日、研討會及工作坊。同時，舉辦聯校教師專業發展日及交流活動，參與教育局及大專院校的校本支援計劃，藉此提升教師專業知識。在校內，透過教師間之同儕備課、觀課交流及教學分享，提升教學效能。
學費減免 : 不適用。
其他 : 校友會及校友設立獎學金，以表揚學生在學業成積（品學兼優和成績進步）、音樂、體育、藝術或制服團隊等有出色表現的學生。
