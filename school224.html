辦學宗旨
本校旨在為學生提供全人教育，培育學生在德、智、體、群、美方面得到均衡的發展。
學校特色
學校管理
學校管理架構 : 設有法團校董會，為學校重要事情決策。校內日常運作則由學校行政會管理 ，由校長領導，成員包括副校長及主任，屬下設有多個學科及功能組別，各負權責。
法團校董會/學校管理委員會/校董會 : 法團校董會於2008年成立，成員包括辦學團體校董、獨立校董、校長校董、教師校董及家長校董。
環保政策 : 1. 置有太陽能及風能發電設施。
2. 節約用電：氣溫達攝氏25度才開啟空調。
3. 節約用紙：如行政電子化，使用電子通告；用「環保紙」或雙面印刷；校園設有廢物分類收集箱。
4. 節約用水：在洗手間裝置自動感應水龍頭，免浪費用水。
5. 午膳採用環保餐具。
學校關注事項 : 學校管理、教與學、教師發展、訓輔合一。
教學規劃
學習和教學策略 : 1. 強調課堂互動學習。
2. 實施多向學習模式，照顧學習差異。
3. 將校本特色課程如戲劇教育、專題研習、服務學習、多元智能課等編入上課時間表，透過整學年的學習活動，培養學生各種共通能力。
4. 安排雙連教節，以便多元的互動學習活動。
5. 靈活運用每週多元活動節，為學生安排各類型的講座或參觀、社群服務、文藝欣賞等學習活動，配合課程改革及豐富學生的學習經歷。
6. 帶領學生參加生活營及境外交流活動，擴闊學生視野。
關鍵項目的發展 : 1. 學校廣泛應用資訊科技在各科的學與教中，利用完善無線網絡設施及電子器材，發展各科電子教材及試行教學，為落實電子學習作準備。
2. 從閱讀中學習：推行多項中英文閱讀計劃（包括網上閱讀），安排每天早讀及午讀時間。
3. 專題研習：在小四至小六推行。
4. 德育及公民教育：透過週會、德育課程、常規訓練及訓輔活動，培養學生正面的價值觀及公民意識。
共通能力的培養 : 本校在各主要學習領域及校本特色課程如戲劇教育、專題研習、服務學習、多元智能課中滲入溝通、分析、創意、解難等共通能力的培養，以配合課程改革。
學生支援
全校參與照顧學生個別差異 : 1. 實行多向學習，透過各種課堂活動，照顧學習差異。
2. 提供學習支援計劃，輔導助理進入課堂照顧個別學習有需要的學生。
3. 設有「個別學習計劃小組」，為個別特殊學生訂立不同的目標和學習方法。
4. 有教育心理學家及言語治療師駐校，為有需要的學生提供服務。
課程剪裁及調適措施 : 1. 在中英數科，設有不同程度的學習課程，因材施教。
2. 為資優班學生設定中英數增潤課程。
3. 為有需要的學生安排課後小組輔導。
家校合作及校風
家校合作 : 家長教師會與校方建立了良好的伙伴關係，每年舉辦各種親子活動如家長也敬師、親子旅行、家長興趣班及陸運會親子活動等。
校風 : 本校強調訓輔合一，重視紀律，每年透過不同的活動培育學生盡責守紀，並積極推動「關愛校園文化」，建立健康校園。學校於2015年9月成立了社。社是一個學生互相學習的群體，過程中，可孕育學生間的「活力」、「動力」、「給力」和「凝聚力」。學生間有著共同目標，彼此共同付出與承擔，一起分享努力的成果。
學校發展
學校發展計劃 : 1. 繼續推展多向學習，照顧學習差異。
2. 試行電子學習。
3. 因應各科的需要改善課程。
4. 鞏固多元化學習及評估。
5. 保持良好校風。
6. 促進教師專業發展，為推行電子學習作好準備。
教師專業培訓及發展 : 1. 每年三天教師專業發展日，提供專業培訓。
2. 持續公開評課、集體備課、同儕觀課、教學分享。
3. 組織教育交流團。
學費減免 : 不適用。
其他 : 培訓學生參加國際賽事，如參加國際小學數學及自然科學奧林匹克賽。
