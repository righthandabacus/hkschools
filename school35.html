全方位學習活動
舉辦劍擊班、跆拳道班、滾軸溜冰班、獨輪、小提琴班、話劇、西洋畫班、歌詠隊、舞蹈班等；制服隊伍包括交安隊、男、女童軍、少年警訊及公益少年團。校外活動計有旅行、運動會、教育營、開放日及參觀，並培訓學生參加各項校際比賽。
辦學宗旨
發揚「禮、義、廉、恥」四維精神，尊親崇孝，教育學生「立好志、讀好書、學好人、做好事」。培育品學兼優、奉公守法、以貢獻社會、服務人群為己任的良好公民。推動全人教育，讓學生在五育各方面有均衡的發展。
學校特色
學校管理
學校管理架構 : 校董會為本校最高權力組織，下設行政事務委員會及學校發展組，負責學校發展工作。校董會主席及校董與校長和教師間關係密切，促進校本管理工作。
法團校董會/學校管理委員會/校董會 : 校董會負責學校管理、行政及運作，全力支援學校發展，已於2008年成立法團校董會。
環保政策 : 學校重視環保教育，透過課程活動及生活體驗，培養學生環保習慣。
學校關注事項 : 學生為本、能力主導、全人發展。
教學規劃
學習和教學策略 : 推行小班教學及探究教學，以學生為主，老師引導，重視學習過程培養學生探索自學精神。設立演藝時段，推行立體圖書計劃，讓學生在活動中愉快學習發展潛能。
關鍵項目的發展 : 1. 推行專題研習，提升學生自學及共通能力；
2. 將劍擊、手鐘、西洋畫訓練融入校本課程之中；
3. 強化從閱讀中學習策略，舉行「立體圖書日」；
4. 學生運用資訊科技互動學習，提升學習效能；
5. 課程融入德育元素，設德育課，舉行德育活動，培養學生品德情意。
共通能力的培養 : 設計多元化課程，發展學生溝通、創造及批判性思維能力。
學生支援
全校參與照顧學生個別差異 : 制訂支援策略，設「拔尖補底」及加強學習輔導班，調適課程，提升學生學習興趣。
課程剪裁及調適措施 : 配合校本課程，各科以學生實際需要作出恰當的調適。
家校合作及校風
家校合作 : 家長教師會匯聚家長和教師的力量，推動家校合作，支援學校發展，舉辦親子活動，定期印發會訊。
校風 : 校風純樸，五育並重，學生具優良品格及個性，勤奮向學，熱心公益，師生、校友及家長關係親切，同學互相幫助。
學校發展
學校發展計劃 : 1. 增強多元互動教學策略；
2. 優化課業策略，照顧學習差異；
3. 提升學生溝通及分享技巧，建立彼此接納文化。
教師專業培訓及發展 : 舉行不少於三日教師專業發展日，參加教育局校本課程發展及優質教育基金課堂研究計劃，協作規劃校本非華語及中文課程，提升教師專業水平與效能。參觀境內或境外學校，建立學習社群，優化教師專業素質與效能。
學費減免 : 不適用。
