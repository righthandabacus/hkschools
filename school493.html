班級教學模式
本校實施小班教學，並採取多元化的策略學習。本校參與教育局新資助模式，成立學生支援小組，透過小組課內輔導課，個別學習計劃或與外間專業支援團隊協辦課程（如言語治療、學習策略），以配合學生多樣性的學習需要。

多元學習評估 : 依照最新課程指引推展評估策略，各科於單元教學後設進展性評估，診斷學習情況，並設總結性評估，檢視學習成效，試後備有學習能力計劃跟進差異，採用多元化評估模式，如學習歷程檔案、實作評估等紀錄學習歷程。
分班安排 : 按學生全年的學業成績分配班別。高年級英文科設分流學習，分層提昇學生的英語能力。

辦學宗旨
本基督精神，發展全人教育；藉宣講福音，培育豐盛生命。
學校特色
學校管理
學校管理架構 : 學校管理架構主要按學校表現指標分為四大範疇，不同層面的領導者均具備專業領導能力，營造不斷學習，精益求精的文化，積極推行各項發展，竭力達成學校的教育使命。
法團校董會/學校管理委員會/校董會 : 於二零零六年度成立法團校董會，加入成員如教師、家長及獨立人士，共商校政，自我完善，以增加校董會的透明度和問責性。
環保政策 : 1. 設置不同的環保設施，如持續再生能源教室、天台花園及節能電燈等，讓學生從生活中建立環保生活。
2. 舉辦廢紙回收及廚餘回收計劃，加強循環再造的意識。
學校關注事項 : 1. 發展學生的自主學習能力。
2. 促進學生身、心、靈全面發展，協助學生建立正向的人生觀。
3. 全面提昇評估素養，增強學習效能。
教學規劃
學習和教學策略 : 1. 學校實行專科專教，亦於各學科開展多元化的校本單元教學，啟發學生思考。小一至小六全面推行小班教學，課堂著重學生經歷學習過程，透過師生互動，達致學習成效。
2. 小二至小六推行普教中課程。
3. 小一至小三推行RWI英語拼音課程，鞏固發展學生兩文三語的能力。
4. 學校於總結性評估後推行學業能力提升計劃，以確保學生的學習成效。
5. 於各科引入自主學習的策略，激發學生的學習動機，從而讓學生掌握學習方法，鞏固知識。
關鍵項目的發展 : 1. 進行早讀課、午間伴讀、圖書課、網上閱讀423世界閱讀活動等。
2. 讓中、小學生能夠欣賞及傳承中華民族的卓越精神與文明，加強學生對國民身份之認同。
3. 推行全方位戶外學習活動及境外遊學專題研習，培訓學生研習及思考能力。
4. 推展電子學習進行學與教及研習活動。
共通能力的培養 : 全校推行全班式資優教學，著力培養學生創造力，高階思維能力及社交協作能力。學校內各學科藉校本特色課程，提升學生自主學習的能力。透過專題研習，跨科學習，境外遊學及主題學習，發展學生綜合的共通能力。
學生支援
全校參與照顧學生個別差異 : 與專業支援人員包括言語治療師、教育心理學家，共同訂定全校參與模式，支援學生學習需要，並透過資優培訓小組，發展學生潛能。
課程剪裁及調適措施 : 1. 各科每週進行同級備課會議，制訂級本的梯階學習材料及課業，並進行單元式進展性評估。
2. 課內時段設立導修課，讓科任老師能適時跟進學生學習情況。
3. 學校推展抽離式資優教育進行拔尖的培訓，如奧數、中文寫作、科技探研、發展學生不同的專項的能力。
4. 小四至小六進行英文科分流教學，因材施教。
家校合作及校風
家校合作 : 1. 組織家長義工隊，協助推展校內活動。
2. 舉辦「家長學堂」培訓，提高家長教養孩童的認知和技巧。
3. 舉辦不同類型家長講座、親子活動以加強家校聯繫。
校風 : 以靈訓輔合一的方針，致力發展全人教育。舉辦主題性校本輔導活動，提升積極自信，建立關愛校園文化，提升學生之領導才能。
學校發展
學校發展計劃 : 1. 提升學習成效，培養學生主動表達，發展學生的自主學習能力。
2. 促進學生身、心、靈全面發展，協助學生建立正向的人生觀。
3. 全面提昇評估素養，增強學習效能。
教師專業培訓及發展 : 1. 系統規劃全校、科組及個人教師發展計劃，透過集體備課、評課、交流及分享活動，互相觀摩，促進團隊邁向專業。
2. 組織專業學習社，總結課程發展成果，作對外交流及分享，回饋教育界。
學費減免 : 不適用。
其他 : 1. 設立學業及非學業成就獎學金，鼓勵學生參與校際及國際性比賽及各項獎勵計劃；
2. 手鈴隊曾獲邀前往新加坡、台灣及韓國進行跨國際音樂交流；
3. 本校教師榮獲行政長官卓越教學獎、優秀教師獎；並多次獲教育局及大專院校進行示範課、工作坊及講座，教學成效備受肯定；
4. 本校恆常舉辦多元化幼稚園聯繫課程及跨校交流活動，如科探活動、環保遊踪、交通安全城活動及體藝課程等。
