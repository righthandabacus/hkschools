學校特色
學校管理
學校管理架構 : 本校直屬天主教香港教區，由校監領導天主教聖安德肋小學法團校董會管理，並授權校長領導學校改善小組、各政策小組、功能小組及學科小組制定學校政策及發展計劃，執行各項校務工作及落實天主教學校的全人教育。家長教師會及校友會支援學校的發展及日常運作，為學校發展提供建議。
法團校董會/學校管理委員會/校董會 : 天主教聖安德肋小學法團校董會於2014年4月1日成立，成員共14人，包括9位由辦學團體委任的校董、1位獨立人士校董、教員校董及家長校董各2位。校監為法團校董會的主席。
環保政策 : 教導學生珍惜天恩，做個地球好管家。
學校關注事項 : 核心價值： 努力學習，欣賞別人，關愛他人，恆常祈禱，分享喜樂。
教學規劃
學習和教學策略 : 1. 推展「校本電子課程」，增加課堂互動，提升學習興趣，增進知識，鞏固學習，並增強評估及回饋效能。（本校為電子學習學校支援計劃之先導學校）。 2. 全面推行「語境教學」，營造真實語言環境，提升學生語文能力。 3. 推展「資優教育普及化」，設思維訓練課、資優課程及資優小組，發展潛能。（本校為教育局資優教育夥伴學校成員）。 4. 下午課堂以連堂為主，同級編組上中、英、數VIP課，以照顧學生多樣性。 5. 英文科發展「C3計劃」及安排英語活動日，並獲教育局支援一、二年級的英語圖書教學計劃。 6. 中文科以普教話為教學語言，並安排中國文化週。
關鍵項目的發展 : 1. 從閱讀中學習：推展全校閱讀獎勵計劃，包括早讀、網上閱讀、中英(PLP-R)圖書教學、世界閱讀日（伴讀大使計劃、故事姨姨、好書推介及電子圖書館）。 2. 資訊科技：運用資訊科技進行互動學習，全校課室設電子白板、電腦、投影機及實物投影機。中、英、數、常各科已採用電子學習，培養學生資訊科技素養及自主學習能力。 3. 專題研習：小一至小六設計系統式專題研習，配合戶外學習，培養學生共通能力。 4. 德育及公民教育：建立正面文化，透過宗教教育、週會、早會、成長課的課程設計進行高質素的品德教育。
共通能力的培養 : 各級進行戶外學習及專題研習，加上系統性思維訓練課程及境外學習，讓學生從多樣化的環境和社區資源中體驗，培養共通能力。
學生支援
全校參與照顧學生個別差異 : 1. 設學生培育小組，釐定照顧學生多樣性的策略，並以全校參與模式， 作課程調適。
2. 下午設「學藝樂繽紛」時段，按學生能力分組，照顧學生學習需要。
3. 利用校本電子學習平台，按學生能力設計不同程度課業。
課程剪裁及調適措施 : 學習經歷 ：營造真實語言環境，全面推行「語境教學」。課堂內外師生均以英文或普通話交談，提升學生語文能力。
家校合作及校風
家校合作 : 設家長義工隊及家長班代表，促進家校合作及溝通。
家長教師會舉辦多元化活動，如家長興趣班、親子旅行等，家長都十分支持，參與活動人數與年俱增。
校風 : 透過不同的校本活動，貫徹正面文化的推行，如：「以你為榮」、「每天一讚」等。
推展班級經營，鼓勵班內一人一職，彼此服務。
設有不同的服務團隊：午膳大使、語境大使、活動推廣大使、小老師計劃，讓高年級同學照顧低年級同學。
透過宗教培育、幼小銜接課程、成長課、德育及公民教育、各項校本輔導活動，協助學生全人發展。
學校發展
學校發展計劃 : 1. 以基督精神為本，培養學生良好的人格素質。2. 透過資訊科技教學，促進學生的自主學習。3. 優化語境教學計劃，進一步提升學生英文及普通話的溝通能力。4. 優化校本資優教育計劃，進一步發展學生的潛能。5. 發展校本促進學習的評估，協助學生作全人發展。
教師專業培訓及發展 : 1. 重視教師培訓及專業發展，中、英、數、常各科均安排共同備課，並安排同儕觀課及公開課，透過專業交流提升教學質素。2. 有系統地提升整體教師專業能力，安排資優教育、電子學習、學生成績分析及評核認知等專業培訓。
學費減免 : 不適用。
其他 : 1. 學生能主動用英語及普通話交談。 2. 課外活動多元化，以舞蹈、視覺藝術、音樂、資訊科技、游泳、朗誦及空手道的表現最為優異。舞蹈組更連續數年奪得校際舞蹈節西方舞全港總冠軍。
