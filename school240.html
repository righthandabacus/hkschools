班級教學模式
設有兩班加強輔導班、精英班及小班教學。課前設拔尖班及補底班；課後設英語會話小組、STEM學習小組、由本校老師教導的功輔小組、「區本計劃」功輔小組。另有專題研習及多元化戶外學習。
班級結構備註
實施小班教學，老師更能個別指導同學功課，關心同學的身心發展。
老師充滿愛心，同學學得愉快。

透過中英文創意默書，超越100分上限，以提昇同學學習動機。常識科設專題研習。
分班安排 : 以成績分班，並會考慮同學的行為品德表現。

健康校園生活 : 設健康校園小組，推廣健康飲食文化，如舉辦水果月、「良好的健康飲食習慣」等計劃；推廣生命教育及健康教育；另有監察小組監察飯盒及小食部的食物。
備註 : 推行多元化閱讀計劃、課外及戶外學習、培訓學生成為領袖及樂於表演，以發揮學生潛能；課後設由本校老師教導的功輔小組、「課後學習支援伙伴先導計劃」功輔小組、「區本計劃」功輔小組及託管班及成長的天空，以助學生成長。
全方位學習活動
有課後訓練：舞蹈、舞獅、節奏樂等； 服務團隊：公益少年團、童軍、英語大使等； 課外活動：金紫荊廣場升旗禮、電台英語錄音、朗誦比賽等； 社會事務活動：探訪老人院、十大港聞選舉、賣旗籌款等； 表演：鴨脷洲社團新春團拜嘉年華、繽紛舞台展才華等。
辦學宗旨
本校以「誠、信、勤、敏」為校訓，教導同學學習知識和技能，並建立正確價值觀。致力推動五育均衡發展，提供優質教育，使學生在融洽的氣氛中學習。樹立優良校風，以培養學生優良品格和樂於服務他人的美德。
學校特色
學校管理
學校管理架構 : 成立法團校董會、行政小組及各科小組。
法團校董會/學校管理委員會/校董會 : 辦學團體6人，獨立人士3人，校長1人，老師2人，家長2人。
環保政策 : 環保小組推行環保教育及活動，例如舉行環保講座、廚餘回收、有機耕種、校園「敢」動行、智「惜」用電計劃等。
學校關注事項 : 培養學生的品德修養；運用多元化教學策略；善用社區資源。
教學規劃
學習和教學策略 : 1. 老師共同備課及觀課。
2. 參與教育局校本課程協作計劃。
3. 進行全方位戶外學習。
4. 專題研習。
5. 親子閱讀。
6. 加強與幼稚園學習的聯繫。
關鍵項目的發展 : 閱讀計劃︰推行多元化閱讀計劃，如圖書活動、中英經典詩文背誦計劃、廣泛閱讀及中文每日一篇。
資訊科技︰學校設有電腦科及發展電子學習。
專題研習︰常識科設有專題研習，科組舉行跨課程計劃及展覽，圖書館配合科組活動及計劃，舉辦專題圖書展覽。
德育及公民教育︰設有德育短講、成長課及周訓。
共通能力的培養 : 透過專題研習、戶外學習及潛在課程學習。
學生支援
全校參與照顧學生個別差異 : 老師關愛同學，為有特殊需要學生進行課程調適；課前設拔尖補底班、課後設本校老師教導的功輔小組、「區本計劃」功輔小組、讀寫小組、情緒訓練小組及社交小組；給與同學表演機會，讓同學有成功感；另有新來港學童支援計劃、同儕伴寫計劃、輔導計劃、心理學家支援及語言治療計劃；學校經常獲教育局讚賞。
課程剪裁及調適措施 : 學校有校本課程，如「悅讀．悅寫意」讀寫課程、識字樂、趣味默書、單元教學及課程調適等。
家校合作及校風
家校合作 : 學校與家長保持緊密聯繫，老師經常致電、寫信、直接與家長交談等方式聯繫家長。並經常舉行家教會活動，家長更成為法團校董之一。
校風 : 老師關愛同學，培養同學優良品德；同學間互助互愛，共同建立和諧友愛的校園文化。
學校發展
學校發展計劃 : 學校繼續在教學上施行多元化的教學策略，提升閱讀文化，增加課外活動，擴闊學生的視野；一方面做好拔尖，一方面做好補底工作；繼續加強德育教育，並引入外界資源，支援學生學習和成長，給同學一個家的感覺。
教師專業培訓及發展 : 透過社區資源及分組討論，積極培養教師專業成長。
學費減免 : 不適用。
其他 : 積極引進校外資源，如與兩個扶輪社結盟支援學校，及經常邀請合適的機構到校舉行活動，以擴闊學生視野。
