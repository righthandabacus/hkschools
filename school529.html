辦學宗旨
堅持「五育並舉」的方針，使學生得到全方位、多元化的教育，從而有效地發掘及培養每個學生不同的潛能和專長。
學校特色
學校管理
學校管理架構 : 追求卓越，並推動學校成為學習型組織。
法團校董會/學校管理委員會/校董會 : 本校已成立法團校董會。
環保政策 : 本校重視透過不同綠化計劃等培育學生愛護環境，珍惜資源。如建立有機認證的園圃。
學校關注事項 : 「全球能力」- 多角度思考、全面觀察和有效與人溝通的能力。
教學規劃
學習和教學策略 : 本校主要以英語為教學語言，著重「多元文化」；有濃厚的英語學習環境。學生需學習英文、中文及普通話，及第三語言（可選讀法文、西班牙文或日文）。本校重視藝術教育及學生體育的訓練，聘請專業導師負責發展學生各方面的潛能。另設跨科專題研習，提供多元學習經歷。
關鍵項目的發展 : 1. 英文、中文科各有閱讀獎勵計劃。
2. 推行資訊科技教學。
3. 跨科及各科的專題研習。
4. 設成長課，成立訓育輔導組及公民教育小組，推動公民教育。
共通能力的培養 : 透過課堂活動及各種學習經歷，培養學生的全球能力。
學生支援
全校參與照顧學生個別差異 : 1. 利用分班及分組教學，照顧能力或語言不同的學生。
2. 課後開辦精英小組及輔導小組。
課程剪裁及調適措施 : 本校提供多元化課程及豐富課外活動，務求全方位地教育學生。配合本校的理念及使命，本校希望能在多元文化的環境下全面發展及培養學生的世界觀。本校不時加強語言學習環境，改進學與教的方法，融合學校活動與多元課程，以提升學校的教育質素。
家校合作及校風
家校合作 : 本校十分重視家校合作，家長積極參與學校政策的制定及推行。他們亦非常熱衷協助學校舉辦各類活動。
校風 : 1. 本校提倡多元文化共融的價值觀，推崇和諧及充滿關懷的學校生活及人際關係。
2. 設立家長教師會和舊生會，加強對學生的支援。設立課後「精英小組」及「輔導小組」，照顧學生的個別學習差異。
學校發展
學校發展計劃 : 繼續加強學生在語言學習、藝術及體育方面的發展，並將資訊科技結合學習，為學生提供更多學習機會。
教師專業培訓及發展 : 推行校本課程改革，鼓勵教師繼續進修，提升學與教質素。
學費減免 : 本校設有學費全免或減免1/2學費。有需要的學生(包括來自綜援家庭的學生及領取學生資助辦事處財政資助的學生)可提出申請學費減免。
- 新生可於開學後的1個星期內遞交申請，其申請結果會在4個月內獲得通知。
- 本校學費減免計劃詳情包括家庭入息限額，請瀏覽網上版的超連結。
其他 : 1. 科目獎：此獎學金會頒發給中、英、法、普通話、數、常、音、體、美科各頭五名學生，校方根據學期末各科之總成績擬定得獎學生名單。
2. 名次獎：此獎學金會頒發給一至五年級全級頭五名學生。
3. 傑出學生獎學金（非學術範疇）。
4. 遊學團獎學金。
