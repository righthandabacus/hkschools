班級教學模式
全校以活動教學為主，強調單元教學和合作學習。初小為標準小班；小五、小六設校本小班，主科分由兩位老師任教；加強輔導教學計劃以抽離方式上課。
班級結構備註
設雙班主任制，雙英語老師制，初小雙中文老師制，初小課後中文基礎班，高小中英數補習班，課後功課輔導班。

備註 : 注重身體和心靈的培養：每天早讀，推廣天天做運動文化，設免費興趣班，實行一生一體藝。 課後開放學校圖書館及電腦室予學生使用，培養自我管理能力。 設清潔服務隊，午膳後清潔課室，培養責任心。
全方位學習活動
興趣小組包括︰牧童笛、籃球組、兒童舞、彩虹大使、男女童軍、朗誦團、英語興趣組、英語話劇、跆拳道、小結他組、棋藝組、閱讀會及繪畫班。隔週進行德育課或活動課。
辦學宗旨
提供優質教育，德、智、體、群、美並重，發揚校訓「忠、信、勤、 樸」的精神，教導學生以服務人群為己任。我們深信人人能學，人人皆可成才，故致力於照顧不同學習差異，使學習成為永續不斷的過程。
學校特色
學校管理
學校管理架構 : 法團校董會、校長、學校改善小組。
法團校董會/學校管理委員會/校董會 : 法團校董會成立於2008年，由14位校董組成，包括辦學團體校董、獨立校董、當然校董及家長校董。
環保政策 : 回收廢紙、校服、舊書、綠化校園。
學校關注事項 : 1. 我們能學，力求上進。
2. 加強照顧個別差異。
教學規劃
學習和教學策略 : 1. 優化英語教學。
2. 探究式學習，着重實踐。
3. 高階思維、批判能力的培養。
4. 通過義工訓練，發揮推己及人的精神。
5. 透過參觀和跨境交流，擴闊視野。
關鍵項目的發展 : 1. 閱讀︰設早讀；英文書層級閱讀；中、英廣泛閱讀計劃；圖書課。
2. 德公︰培養自我管理能力，發揚關愛精神，使學生愛己、律己、愛人、愛國。
3. 專題研習：五、六年級進行專題研習；一至四年級進行跨學科主題學習。
4. 資訊科技：設電腦科，i-Learner網上學習平台，從實踐中學習。
共通能力的培養 : 首階段培養溝通能力，批判性思維能力和創造力。
學生支援
全校參與照顧學生個別差異 : 設加強輔導班、課後功課輔導班、大小同學計劃、雙英語教師，進行拔尖補底。 設小六升中輔導課，小一適應課等，加強銜接。
課程剪裁及調適措施 : 為有特殊學習需要的學生作課程調適，例如個別抽離輔導、功課調適、加長考試時間等。
家校合作及校風
家校合作 : 透過活動（小一迎新，家長義工隊，家長學堂：新來港學童輔導班及賽馬會全方位學習基金等）、電話或家訪，使家、校保持緊密的聯繫與合作。
校風 : 1. 「小學校．大家庭」：提倡關愛文化，注重師生、生生的支持與關顧。
2. 無種族界限，學生尊重彼此的文化。
3. 學生能以禮待人。
學校發展
學校發展計劃 : 1. 優化單元教學，提升合作學習和提問技巧的效能，發展學生的高階思維和九大共通能力。
2. 優化單元設計及發展學生的創意思維。
3. 加強德育訓練，使學生愛己、律己及愛人。
教師專業培訓及發展 : 英語教學，小班教學，非華語學童教學，照顧讀寫障礙，高階思維訓練等。
學費減免 : 不適用。
