全方位學習活動
合唱團、中樂團、非洲鼓、口琴、羽毛球、乒乓球、足球、田徑、醒獅隊、中國舞、健體中國舞、扯鈴、水墨畫、童軍、機械人工程師、科技探究等。除此之外，還有各種各樣教育性講座及參觀。
辦學宗旨
致力創造愉快的學習環境、提供均衡的課程，誘發學生的潛能，培養他們的自律精神及公民責任感，使成為明辨是非、尊重別人、自愛、自主、勇於克服困難的好學生。
學校特色
學校管理
學校管理架構 : 校長帶領以副校長及主任組成的行政小組，在其他教職員的協助和校管會的監察下，分工合作，推動學校發展。
法團校董會/學校管理委員會/校董會 : 學校管理委員會由主席、老師代表、家長代表及社會人士代表組成。
環保政策 : 透過參與社區相關活動，重點推動學生實踐低碳生活。
學校關注事項 : 1. 推動自主學習，提升學生學習效能。
2. 協助學生「挑戰自我，自愛自重」，建立良好品德。
3. 加強學生對古今中國文化的認識，讓學生實踐中國民族的美德。
教學規劃
學習和教學策略 : 1. 積極推動課堂研究，中、英、數及科技教育透過各大學及教育局專家進行專業交流。
2. 學校按學生的需要發展校本課程，為學生提供豐富的學習經歷，包括外籍英語教師學習活動時段、'English Fun Day'、「普通話日」，另有各科科本活動、科研活動及專題研習等，鞏固學生的基礎知識，發展共通能力。
3. 透過小班教學，制定清晰教學目標，運用合適的教學策略和資源，促進課堂互動，以協助學生建構知識，並培養學習能力，建立正確的態度和價值觀。
關鍵項目的發展 : 1. 推廣閱讀風氣：每逢星期二、三、五早上設「閱讀課」，營造有利閱讀的環境，並提倡親子閱讀。
2. 資訊科技：推行 WIFI900，在電腦科、常識科及數學科推動電子學習，提升學習成效。
3. 專題研習：透過「專題研習」活動，使學生把各科的學習與生活連繫，有效掌握概念和運用知識。
4. 德育及公民教育：配合生活教育課及德育主題講座，推行國情教育、品德及價值教育。
共通能力的培養 : 本校積極發展學生溝通、創造和批判性思考等能力，以助他們建構知識及解決問題。
學生支援
全校參與照顧學生個別差異 : 運用校本學習支援津貼推行融合教育，按優次為校內有特殊學習需要的學生提供適切支援。
課程剪裁及調適措施 : 1. 針對教學難點及學生能力設計教材。
2. 為照顧學習差異，四至六年級按學生表現實施分組教學。
3. 提供多元化優質課程，照顧不同能力學生之需要。
4. 致力培養學生建立良好生活習慣，活出豐盛健康人生；建立健康積極的生活態度。
家校合作及校風
家校合作 : 定期舉辦家長教育活動，並舉行親子活動、家長興趣班，與家長教師會合作無間。
校風 : 本校為區內歷史悠久的優良小學，校風淳樸，學生知禮守規，師生關係良好。我們舉辦幼小及中小銜接活動，協助學生及家長適應轉變；積極推動校園關愛文化；重視德育及公民教育，培養良好品德及歸屬感；推行朋輩伴讀及家長伴讀計劃。
學校發展
學校發展計劃 : 1. 推動自主學習，提升學生學習效能。
2. 協助學生「挑戰自我，自愛自重」，建立良好品德。
3. 加強學生對古今中國文化的認識，讓學生實踐中國民族的美德。
教師專業培訓及發展 : 為全體老師舉辦教師專業發展日，重點協助老師優化教學策略，以提升教學效能。每周設教研日，各科積極推動教研。透過共同備課、同儕觀課，建立學習型社群，提升學與教的質素。安排教師積極參與專業進修，並與友校交流，以配合學校每年關注事項及科本發展，並掌握教育新趨勢。
學費減免 : 不適用。
其他 : 自2016年開始，本校與香港婦女基金會何郭佩珍耆康中心合辦長者學苑，推動區內長幼共融，名為「李陞小學香港婦女基金會長者學苑」；鼓勵學生進行境外學習，培育學生宏闊的世界觀。
