學校特色
學校管理
學校管理架構 : 本校07年成立法團校董會，為學校最高決策機構，成員包括辦學團體代表、校長、教員代表、家長代表、校友代表及獨立人士代表。轄下由校長領導之四大範疇行政組別，包括學校發展與行政管理、學與教、學生成長支援及對外聯繫。各組別設不同行政小組，共同努力推動學校發展。
法團校董會/學校管理委員會/校董會 : 校監、校長、辦學團體校董、教師校董、家長校董、獨立校董、校友校董共同管理學校的運作。
環保政策 : 1. 每學年的學期初，訂定全學年的環保活動，並培訓多班之環保大使。
2. 透過早會、周會或多元學習課，向學生推廣環保訊息。
3. 邀請外界專業機構舉辦適切的講座或工作坊，讓學生接受及更新他們的環保概念，付諸於行為上。
學校關注事項 : 1. 優化教學策略；
2. 加強照顧學生學習差異；
3. 培養學生基華五大核心價值觀。
教學規劃
學習和教學策略 : 1. 英文科以校本教材教學。
2. 中文科以「普教中」。
3. 數學科設校本解難課程。
4. 常識科設校本探究課程。
5. 中、英、數、常四科之網上學習平台。
6. 三年級開始推行資優教育課程。
7. 生命教育課內容乃環繞校本五大核心價值：以禮待人、積極自信、愛己愛人、樂於服務、堅毅不屈作螺旋式設計，並於不同年級舉辦聯課活動令學生的情感記憶加強。
8. 一至四年級設課程統整，五、六年級設跨學科專題研習。
9. 透過各科教學策略提升學生自學能力。
關鍵項目的發展 : 1. 跨學科專題研習培養學生共通能力。
2. 把閱讀策略融入中、英文科。
3. 以生命教育課培養學生正確價值觀和良好生活態度。
4. 資訊科技融入各科，學生利用網上學習平台自學。
共通能力的培養 : 透過各學科專題研習、跨學科活動、多元智能課及資優教育課程來培訓。
學生支援
全校參與照顧學生個別差異 : 1. 教育心理學家、言語治療師和社工支援學生及家長。
2. 透過小一及早識別計劃評估及支援有特殊學習需要的同學，調適課程照顧不同能力的學生。
3. 設立人才庫。
4. 提供按能力導向的資優教育課程。
5. 為能力稍遜學生提供課後支援小組及活動。
6. 調適六年級課程以配合中學學習模式。
課程剪裁及調適措施 : 四至六年級中英數設學科分組教學。
家校合作及校風
家校合作 : 1. 小一新生適應日和各級家長座談會。
2. 家長觀課日。
3. 親子旅行。
4. 家長教育講座和工作坊。
5. 家長協助學校推行各項活動。
校風 : 1. 全校參與訓輔合一。
2. 社制培養學生團隊奮鬥精神。
3. 「一人一職」培養學生責任感。
4. 「一團隊一服務」關心社會計劃。
5. 建立關愛校園文化。
學校發展
學校發展計劃 : 1. 透過多元化的學習模式，發展學生高階思維能力。
2. 推展校本生命教育課程，培養學生正確的價值觀及人生觀。
教師專業培訓及發展 : 優化教學策略，加強教師資優教育、電子教學及支援策略的培訓。
學費減免 : 不適用。
其他 : -
