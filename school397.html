全方位學習活動
學校設有會社活動，培養學生領袖才能。周五設多元智能課，周六設自費興趣班，供全校學生參與，讓學生全面接觸不同類型的活動，豐富學生的學習經歷。學會及校隊包括有基督小先鋒、公益少年團、小／幼童軍、少年警訊、舞蹈、跆拳道、球類、田徑、歌詠、敲擊樂等。
辦學宗旨
本校為一所天主教學校，以基督福音為本，彰顯淳樸、愛心和服務，以樹立優良校風；為培養學生的良好品格及學業成績，致力提供一個理想的學習環境，使學生在喜樂和融洽的氣氛中，積極學習，努力進取，期能滿全基督精神，為社會培育優秀的人才及良好公民。
學校特色
學校管理
學校管理架構 : 本校由辦學團體天主教香港教區於1969年創立，由荃灣天主教小學法團校董會管理。校監帶領法團校董會為學校持續發展制定政策。校長領導行政小組及三個功能小組策劃及推行校務。
法團校董會/學校管理委員會/校董會 : 法團校董會由校監、校長及十三名校董組成。
環保政策 : 本校已參與環境運動委員會「學校廢物分類及回收計劃」，定時回收學校可循環再造的物資；另外，學校多年來參與康文署「綠色校園資助計劃」及「百佳綠色學校計劃」，積極在校內推行各項環保、節能及綠化活動。
學校關注事項 : 敬主愛群、力學求真；校風淳樸、以愛育人；多元教學、盡展潛能；精英團隊、家校合作；語境教學、生態校園；菁英輩出、成績斐然。
教學規劃
學習和教學策略 : 本年度的學習主題是追求真理，目的是讓學生認識天主教學校的核心價值—真理¸我們認識天主的智慧，培養獨立思考和分析能力。在生活當中要言行正直，誠實坦率。此外，本校透過跨學科課程及多元化教學策略，發展學生潛能，包括：資優教育、閱讀、電子學習、公民及環境教育、專題研習、語境教學、思維教學等。
關鍵項目的發展 : 1. 以宗教培育為根基，輔以全校參與的全方位訓輔工作，讓學生認識天主教教育五個核心價值；培養學生良好的品格和積極正面的人生觀。
2. 透過舉辦中、英文閱讀獎勵計劃、早讀、同輩伴讀、每週新書推介、閱讀講座、圖書館參觀、書展等不同的閱讀活動，培養學生建立廣泛閱讀的良好習慣。
3. 在課室內增設電子白板；老師製作校本網上電子教材及學習資料庫、師生透過網上電郵及討論區系統互相學習及交流。
4. 學生以小組合作形式進行專題研習，老師因應不同學習階段，指導學生學習不同的專題研習技能。
共通能力的培養 : 透過正規及全方位習活動發展學生共通能力，例如：多元化活動、會社比賽、戶外參觀等。課堂上老師利用不同的教學策略及學習活動發展學生溝通、合作及創造力等共通能力。透過合作學者訓練學生協作能力，利用自學獎勵計劃、網上自學平台及學習日誌，訓練學生自學能力。
學生支援
全校參與照顧學生個別差異 : 學校設訓輔組、學生支援組及資優教育組、照顧學生不同需要。透過讀寫小組、專業訓練課程、朋輩支援等，增強孩子們的自信心，並透過三個層次的資優教育模式，發掘和培育資賦優異的學生，為他們提供適切的學習支援。
課程剪裁及調適措施 : 為有特殊學習需要學生作家課及評估特別安排及調適。
家校合作及校風
家校合作 : 家長教師會為家校合作奠定了良好基礎，不單擔當學校與家長間的橋樑、協助學校舉辦家長或親子活動，參與學校發展並提供寶貴意見。家長義工協助學校推行各類活動，如圖書管理、校服義賣等。
校風 : 我們相信每位學生都是獨特的，而學校教育可以激發無限潛能，促進人格發展。本校以天主教教育為根基，透過全方位訓輔、專業而有承擔的教師團隊、家校無間的合作、培養學生正面積極價值觀，營造關愛文化及淳樸校風，讓學生在一個理想環境中，奠定終身學習基礎。本年度參加及舉辦了教育局「好心情@學校」計劃、明愛「愛與誠」綜合性教育劃、「成長的天空計劃」、基督生活團「陶怡計劃」等。
學校發展
學校發展計劃 : 2015-2018 學校發展計劃關注事項：
1. 認識天主教教育核心價值。
2. 學生主動投入學校生活。
教師專業培訓及發展 : 因應學校發展及教師個人發展需要，教師團隊透過公開課、共同備課、同儕觀課及校外專業交流等專業發展活動提升教學效能。本年度學校為老師安排了不同範疇的專業發展，包括有「新入職教師伙伴計劃」、「教區小學提升學與教效能支援計劃」、「中文大學教育學院優質教育基金小學英國語文網絡計劃」等。
學費減免 : 不適用。
其他 : 學生在公開及校際比賽獲取多個獎項：學校朗誦節–粵語、普通話、英語獨誦亞、季軍；普通話集誦冠軍、英語集誦亞軍；香港奧林匹克數學比賽一等、二等獎；荃灣、葵青區舞蹈大賽金獎；學校舞蹈節甲級獎；校際籃球比賽冠軍；香港國際學生創新發明大賽銀獎等。
