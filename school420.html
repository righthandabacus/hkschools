辦學宗旨
以基督化全人教育為目標，除傳授知識外，更幫助學生明辨是非，培養良好品格，服務社會，實踐校訓「爾乃世之光」的精神。
學校特色
學校管理
學校管理架構 : 校務委員會：中、小、幼校長及副校長。
行政委員會：校長、主任、教師代表。
教務組：科主任、圖書館主任、資訊科技。
法團校董會/學校管理委員會/校董會 : 校董會由教育、社福、建築、法律、商業界別、校友代表組成。
環保政策 : 1. 魚菜共生計劃。
2. 種植花卉植物。
3. 參與康文署「綠化校園」、「一人一花」及團體舉辦的活動。
4. 設置廚餘機，將廢棄物轉為有用的種植肥料。
5. 申請環保基金「節能減排計劃」—更換節能光管、天台油上隔熱塗層、玻璃窗貼太陽薄膜。
學校關注事項 : 提升學生高階思維技巧、表達能力與態度、正面價值觀及「標準真光兒
女」的內涵。
教學規劃
學習和教學策略 : 電子教學（高年級）
透過學生會、童軍及團契、朋輩輔導、野外營會、專題研習、文化活動學習、境外遊學團等，讓學生得到均衡的德、智、體、群、美、靈育發展。
關鍵項目的發展 : 國情教育，國慶舉行升國旗儀式及展示國家現時發展資料。學生參與歷奇營、境外學習普通話、共融日營、領袖訓練，以增加科與科間的聯繫及溝通。
共通能力的培養 : 跨學科課程、網上學習、專題研習、合作教學及閱讀計劃。
學生支援
全校參與照顧學生個別差異 : 1. 閱讀計劃。
2. 專題研習。
3. 分組教學。
4. 課後輔導。
課程剪裁及調適措施 : 1. 數學精英培訓。
2. 體育精英培訓：田徑、羽毛球、排球、跳繩及舞蹈。
3. 領袖訓練：童軍、學生會、風紀。
4. 中英數輔導班。
家校合作及校風
家校合作 : 設有家教會，舉辦家長講座、親職教育工作坊。家長義工：故事大使、午息當值、新聞報導、功課輔導、陸運會遊戲、寫作教學導師。
校風 : 1. 暑期活動安排：游泳班，英語閱讀計劃，英語話劇，樂器班，小四至小六英文，數學進深班。
2. 午飯安排：家長可選擇供應商的飯盒、學生自備或家人送膳。
3. 家長心聲：家長對學校有強烈的歸屬感及讚揚教師教學熱誠，對學生愛護關懷。
學校發展
學校發展計劃 : 1. 發揚「標準的真光兒女」的內涵與特質
2. 電子教學
3. 生命教育與正面價值觀
4. 主動學習
教師專業培訓及發展 : 三年內進修時數不少於一百五十小時，課程包括本科教學、訓輔導項目及專業培訓。
學費減免 : 不適用。
其他 : 培育「一人一藝」、領袖及體能訓練。
