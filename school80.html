班級教學模式
本校由2010年小一起採用「普通話教中文」，以學生為本教學，透過合作學習，戲劇教學，於課堂滲入資優及電子學習元素等促進學生學習；並制定政策及課程，以照顧學生個別差異。
班級結構備註
學生支援小組負責統整資源，推動及早識別，家校合作等策略。並於課前、課後設培訓小組，加強輔導工作。

多元學習評估 : 設有進展性、總結性評估及引入多方參與評估方式，包括專題研習、課業設計、觀察、 學生自評及互評、家長評估等，並特別為有特殊需要的學生延長測考時間或安排專人讀卷。
分班安排 : 小一至小六平均分班。

辦學宗旨
實踐基督完美教育，灌輸學生聖經的教誨，五育均衡發展，達致全人教育的目標。同時致力訓練學生成為國家良好公民，肩負服務社會的使命。
學校特色
學校管理
學校管理架構 : 實施校本管理，通過集體協商，確立權責分明的行政架構，由委員會及專責小組統籌校政，建構「點、線、面」的自評文化。
法團校董會/學校管理委員會/校董會 : 校董會由教育界及專業人士組成，積極參與校務發展，建立良好的夥伴關係。
環保政策 : 全校參與環保政策，在校廣植植物，把環保教育融入課程內。
學校關注事項 : 1. 實踐生命教育，培養學生全人素質。
2. 提高學生兩文三語的能力。
3. 發展校本資優教育，提升學生的情意發展及啟發學生的潛能。
教學規劃
學習和教學策略 : 本校主要運用合作學習、戲劇教學、於課堂滲入資優教育及電子學習元素以優化課堂，配合不同的學習需要，達成學習目標。我們讓學生在智能、德育及公民教育、社會服務及體藝上有足夠的發展。不同學科的學習相互聯繫，推行聯課活動，提供多元化的學習經歷。
關鍵項目的發展 : 1. 專題研習：提高學生自學精神，培養共通能力及正確價值觀。
2. 從閱讀中學習：設圖書課、閱讀時間。各科加強閱讀元素，培養校園閱讀文化。
3. 運用資訊科技：設資訊科技課程，並在課餘開放多媒體教室。大部份課室設置電子白板，促進課堂互動。教師亦利用不同電子學習資源，檢視學生的學習情況，並提供即時回饋。
4. 德育及公民教育：透過正規及非常規課程，協助學生建立正確的價值觀和人生觀。
共通能力的培養 : 透過正規課程、校本課程及課外活動鞏固學生共通能力。
學生支援
全校參與照顧學生個別差異 : 有特殊需要學生參加加強輔導班及針對性課程等，安排個別學習計畫，為小一及小六學生舉辦工作坊，讓他們及早適應新環境。
課程剪裁及調適措施 : 為有特殊需要學生於教學、課業及評估上作調適，以及舉辦拔尖課程，引入資優教育三層支援架構，培育精英。
家校合作及校風
家校合作 : 學校採取開明態度，增強家校合作。特設「陽光電話」，老師與家長保持聯絡，加強溝通及意見交流；家教會舉辦不同的活動，如：親子旅行，家長講座，家長義工茶聚，食物製作班等，共同培育學生成長。
校風 : 學校重視「愛」的教育和領袖培訓，透過「讓我閃亮計劃」、「Pilot Buddies Scheme」及「一人一職」等活動，讓同學由「愛自己」到「愛他人」，培養互相欣賞、積極承擔的品格，建立關愛校園及發揮領袖才能。
學校發展
學校發展計劃 : 配合關注事項，全力發展校本課程及活動，提供優質的教學。
教師專業培訓及發展 : 設教師協作及專業發展時段，進行交流，分享觀摩，共同備課及課後檢討。
學費減免 : 不適用。
