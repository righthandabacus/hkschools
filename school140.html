辦學宗旨
以天主教教育五大核心價值：真理、生命、愛德、義德和家庭為基礎，秉承主耶穌愛主愛人的精神及「容孩近我」的愛心，提供全人教育，致力發展學生潛能；以「公、誠、勤、毅」為校訓，使學生在德、智、體、群、美、靈各方面均獲得均衡的發展，將來成為社會的好公民。
學校特色
學校管理
學校管理架構 : 實施校本管理問責考績制度。重經驗亦勇於創新，發揮個人潛能亦營造團隊文化，管理制度完善，具高透明度。法團校董會由辦學團體成員、家長、教師、校友、教育界及專業人士等各界別的持份者組成。學校並設立行政、教務、輔導、訓導、學務和課外活動組，鼓勵更多老師參與校政。
法團校董會/學校管理委員會/校董會 : 李笑容修女、李好珍修女、吳美珠修女、李淑芳女士、羅偉國校長、羅家欣女士、何樹清先生、林復娟女士、鄧守正先生、蔡美玉校長、李偉鋒副校長、鍾志霞老師、李嘉碧女士、張樹強先生和彭炎飛女士。
環保政策 : 本校致力培育學生環保生活態度，歷年均獲「綠色學校」獎，曾獲香港環保卓越計劃—界別卓越獎（學校：小學）銅獎。
學校關注事項 : 1. 天主教教育五大核心價值：真理、生命、愛德、義德和家庭。
2. 持續發展電子學習，促進學生有效學習。
教學規劃
學習和教學策略 : 周五課外活動，周三多元化活動。正規課進行小組探究式學習、合作學習等。全方位學習如健康學校、綠色學校、服務學習、專題研習、網上平台等，讓學生獲得多元學習經驗，並訓練多角度思考。
關鍵項目的發展 : 1. 閱讀：多元化活動推動閱讀文化。
2. 資訊科技：內聯網學習平台促進自學文化，善用平板電腦提升教學效能，運用教學軟件促進教學效能。
3. 專題研習：積極推行分科及跨科專題研習。
4. 德育及公民教育：深化國民教育活動、成長課及信仰氛圍，幫助學生建立正面價值觀。
共通能力的培養 : 學習策略課程、專題研習及習作、成長課程、歷奇學習。
學生支援
全校參與照顧學生個別差異 : 課後學習支援計劃、資優課程、學科輔導、小班教學、成長的天空活動。設計拔尖補底課程，資優班加強照顧學生需要。
課程剪裁及調適措施 : 按學生能力調適課程及評估、拔尖推中補底措施、升留班政策等全面照顧學生能力差異。
家校合作及校風
家校合作 : 定期會訊和活動、協辦課後及暑期活動和功輔班、安當家長學堂、義工服務等。
校風 : 校風淳樸，透過宗教、德育、訓育及公民教育活動，使學生領悟待人和處事的正確態度。學生支援組及訓育組照顧學業情緒或行為有偏差學生。家校在「關愛」的訓輔理念下，致力營造充滿關懷、尊重、互愛的學習環境。
學校發展
學校發展計劃 : 電子教學、科技教育、普及化資優教育，高階思維訓練。
教師專業培訓及發展 : 專業發展日、教學啟導、開放教室、共同備課、同儕觀課等均提升教師效能，邁向更專業的境界。
學費減免 : 不適用。
其他 : 升中派位理想，統一派位獲派首三志願的學生超越88%，而獲派往全英文班中學的則有超越40%。
