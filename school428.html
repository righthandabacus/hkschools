備註 : 除正規課堂外，學校亦在放學後或假期時為學生提供豐富且多元化的課外活動，讓學生發揮潛能，盡展所長。本校亦設有輔導小組，協助學生有效地學習，提升自信自尊，完善資優教育的模式及理念。為配合各項活動安排，本校將為有需要的學生提供第二輪校車接送服務。
全方位學習活動
以學術（朗誦、各科大使、英文校訊）、音樂（多種中西樂器班、中樂團、手鐘隊及合唱團）、體育（籃、足、排、泳、羽、手、田及體操等）、藝術（劇團、繪畫類、棋藝、舞蹈及武術)、服務（童軍、CYC、JPC、風紀等）五大範疇。每年均獲獎無數，成績卓越。
辦學宗旨
以普濟勸善為校訓，培育學生勤勉向學，樂善好施，服務社會的精神，並注重五育的均衡發展。
學校特色
學校管理
學校管理架構 : 實施校本管理，把學校成員按專長及學校發展需要劃分不同的行政組別。架構精簡、專業問責，為學校提供全面及優質的管理。
法團校董會/學校管理委員會/校董會 : 已成立法團校董會。成員除辦學團體、校長及教師代表外，更包括家長代表及校外獨立人士，以高效、具透明度及專業的態度管理學校。
環保政策 : 一向致力推動環保，鼓勵減廢措施，並組織健康學校發展組，推動學校的健康及環保政策。
學校關注事項 : 「自助助人‧自重重人」。
教學規劃
學習和教學策略 : 根據每年不同的課程發展目標，靈活運用課時，彈性安排上課時間，為學生設計多元化的學習經歷及活動，讓學生從親身體驗中，獲得五育的均衡發展。整體課程設計，以全方位學習及跨課程活動，建立學生廣闊的知識基礎，並舉辦課程綜合週及設立全人教育課，為學生提供不同的學習經歷，認識生活，放眼世界。學校更積極發掘校外資源，與不同的教育機構及大專院校合作研究，支持學校發展課程，與時並進，為學生向「學會學習」的目標進發。
關鍵項目的發展 : 1. 以多元化活動提升學生閱讀興趣。
2. 在數碼校園盡顯學生多方面的資訊科技潛能。
3. 跨學科的研習活動，讓學生聯繫概念，融會貫通。
4. 德育成長課，有效培養學生的良好品格和價值觀。
共通能力的培養 : 根據各學習領域的特質和重點，再配合學生實際的學習需要，培養他們各項共通能力，讓他們充分發揮潛能。
學生支援
全校參與照顧學生個別差異 : 除於課堂上作出相關照顧外，並設多個學生自學平台。請參閱：
http://itid.holap-p.edu.hk重視中小幼銜接。為小一同學安排入學前迎新、小一適應課程及加強營造幼稚園氣氛的樓層佈置。亦為小六同學安排參觀中學、面試工作坊及不同主題的升中講座和活動，協助學生及早適應不同階段的學校生活。
課程剪裁及調適措施 : 以三層資優教育模式作為拔尖政策基礎。有系統、有方向地發掘和培育資賦優異的學生，為他們提供適切的教育機會，使他們能夠在富彈性的教學方法和環境下，充分發揮個別潛能。
家校合作及校風
家校合作 : 積極進行家長教育，提升家長對學校的歸屬感。家長義工隊伍龐大，為學校提供人力資源，協助學校運作。
校風 : 以自助助人、自重重人的可立精神為基礎，配合不同主題的德育活動，為學生建立正面的道德價值觀。
學校發展
學校發展計劃 : 1. 培養學生自律精神，學會自重、自愛。
2. 啟發學生潛能，推動自主學習，提升多元能力。
教師專業培訓及發展 : 教師積極參與持續專業發展。按個人、科組及學校發展的需要，參與不同類型的在職培訓活動及課程。透過共同備課、同儕觀課及科會內的經驗分享，提升教學成效。本校更積極安排教師與本港及海外的學校進行交流，以提高教學專業效能。
學費減免 : 不適用。
其他 : 本校獲辦學團體大力支持，鼓勵學習，為學業成績優異的同學設立多項獎學金，並設有不同的品行優異獎、進步獎、服務及課外活動優秀獎項，鼓勵學生自我提升。除此之外，本校每年均推薦學生參加社區傑出學生選舉，屢獲殊榮。每年在區內選出的10名傑出學生中，本校均佔一定數目的席位。學校更舉辦頒獎禮及榮譽榜，褒揚優秀學生，為同學建立良好的朋輩學習榜樣。
