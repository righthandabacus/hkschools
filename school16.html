班級結構備註
全校推行「小班教學」，並以「專題研習」及「戶外學習」作跨學科研習基礎。一年級推行「以普通話教中文」預備班。

分班安排 : 小一至小四平均分班，小五至小六按學業成績分班。

全方位學習活動
設校園小記者、幼童軍、小女童軍、基督少年軍、公益少年團及少年警訊等團隊。成立視藝、電腦、環保、創意思維、中英數科等學會。另有歌詠、朗誦、樂器、武術、劍擊、球類、機關王、光雕、花式跳繩、辯論隊等專項培訓。
辦學宗旨
培養兒童基督化人格，發展德、智、體、群、美、靈六育，以配合廿一世紀需要。
學校特色
學校管理
學校管理架構 : 除法團校董會外，學校還設立不同行政部門負責校內各樣事務。
法團校董會/學校管理委員會/校董會 : 法團校董會的成員包括辦學團體代表、校長、獨立人士及由選舉產生的教師及家長校董組成。
環保政策 : 本校是一所環保學校，透過環保守則教導學生有關環保的知識。每年更會舉辦全校性的環保活動，鼓勵學生及家長參與，培育環保生活習慣。
學校關注事項 : 開開心心上學來，快快樂樂回家去，認認真真求學問，健健康康美人生。
教學規劃
學習和教學策略 : 靈活運用多元活動課，彈性安排參觀活動。活動教學發展智能：以德育及公民教育、體藝活動、社會服務、跨學科戶外學習、專題研習及提供不同服務學習等充實學生五種基要的學習經驗。
關鍵項目的發展 : 1. 閱讀計劃：推行親子閱讀、閱讀百寶袋、早讀課、主題閱讀月、閱讀獎勵計劃等。
2. 專題研習：每級開展跨學科專題研習。
3. 資訊科技：自編校本教材，並以不斷更新的器材配合教學。
4. 德育及公民教育：推行基督教全人教育及校園關愛文化。
共通能力的培養 : 滲透全校課程，並重點以跨學科活動及專題研習培養學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 默書及測考調適、功課調適、課後輔導班、資優培訓等「全校參與」模式，更設有中文、英文及數學科的輔導班及拔尖課程，照顧不同學習能力的學生。
課程剪裁及調適措施 : 中文、英文、數學、電腦、視藝等科目均設有校本設計課程，以配合時代及學生能力的發展，提升學習的水平。
家校合作及校風
家校合作 : 家長積極參與學校舉辦的活動如講座、親子旅行等，並踴躍擔任義工，發揮家校合作精神。家校密切聯繫，藉家長日、家長座談會、通告、電郵、刊物及陽光電話等建立有效的溝通途徑。
校風 : 1. 校風淳樸，師生關係融洽，校園洋溢愛與關懷。
2. 著重宗教培育，以基督為榜樣，廣傳福音，認識真理。
3. 訓輔與宗教結合，培育基督化人格。
4. 以「一個也不能少」的精神，幫助每一位學生及家庭。
學校發展
學校發展計劃 : 1. 持續提升教師教學及學生學習，培育學生成為自主學習者。
2. 優化健康校園政策，提升學生的自律、自信的個人素質。
教師專業培訓及發展 : 每年安排校本專業交流活動，如友校交流、教師發展日、教師分享會、共同備課會等，讓教師交流教學專業心得，又鼓勵教師參與不同專業範疇的持續進修活動。
學費減免 : 不適用。
其他 : 1. 積極拓闊學生視野：境外學習遊、服務學習、參觀。
2. 積極鼓勵學生參加校外活動及比賽：校際中文、普通話、英文、聖經朗誦、音樂、田徑、視藝及資優比賽等，成績優異，獲獎無數。
