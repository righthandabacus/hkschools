班級教學模式
全校推行「活動教學」，以「合作學習」模式上課，課堂設計滲入資優教育元素，資訊科技互動學習，綜合解難能力，照顧學習差異，著力培養學生高層次思維能力。
班級結構備註
合作學習：互補、雙贏及共同目標。
思維訓練：多角度思考，解難。
資優教育：高階思維、創造力和個人社化。

分班安排 : 四至六年級按學業成績分班。

健康校園生活 : 本校為大埔倡健計劃榮譽學校，健康教育組積極以專題及恆常活動建立健康與安全的學校環境；促進學生身心健康發展，遠離毒禍及營造和諧的校園。
備註 : 本校設早讀及全校廣播操及護脊操，午膳後設有午間活動。每級均設游泳課鍛鍊學生體魄；上課前亦設「活力角」，如短跑及跳繩等。清潔糾察協助監察校園整潔情況；各級推行禁毒課程，啟發潛能教育小組推行「別取笑我」(DLAM)校本課程。
全方位學習活動
本校教師帶領的活動超過65組，如小作家種籽計劃、英語智叻星、奧數、科學探究及男女童軍等，超過3500人次參加。家教會興趣班約50項，超過2000人次參加：包括管弦樂團、中樂團、劍擊等，戶外參觀超過3000人次。
辦學宗旨
本校致力提供優質的全人教育，培養學生德、智、體、群、美、情、靈， 七育均衡發展。鼓勵學生主動探索世界，具國際視野，啟發學生的潛能。
學校特色
學校管理
學校管理架構 : 學校領導強，行政效能高，行政組別結構清晰，透明度高，家長對學校支持及信任，學校成員清楚學校的理想及使命。學校積極援引各種資源，發展校務。有校本管理經驗，發展自我完善精神。行政組提高師生互動溝通及學習氣氛；提高教師專業能力，持續自我完善；提高家校溝通及家長教育。
法團校董會/學校管理委員會/校董會 : 已成立法團校董會，管理本校及促進本校教育。校監校董關懷支持學校發展，經常出席本校的活動及典禮，提供寶貴意見，持續全面優質教育。
環保政策 : 重視實踐四用原則，培養學生珍惜資源和愛護環境的價值觀和態度。校本環保活動如：圖書、衣物、綠田園、觀察中華白海豚、戶外考察；到幼稚園推廣環保訊息，設立環保資訊網頁。
學校關注事項 : 誘發學生好奇心、探索精神，發展學生創新、科技和生活技能，優化教師專業能力、持續培養學生世界公民的素質。
教學規劃
學習和教學策略 : 重視培養學生兩文三語的能力；培養學生閱讀習慣、閱讀技巧及營造閱讀氣氛；讓學生掌握數學概念，應用於生活；科普探究學習；進行專題研習，發展學生自學能力；德、智、體、群、美、情、靈七育並進、培育學生素質；每學期進行課程統整校本課程；設計主題多元探究學習活動，提昇學生學習經驗；推行資訊科技互動教學；照顧學生個別差異，實施資優教育及輔導教學；課堂進行有系統思維訓練全面推行啟發潛能教育，透過5P元素：政策、地方、人物、計劃及過程，啟發師生潛能。
關鍵項目的發展 : 1.從閱讀中學習：電腦化圖書館藏書超過五萬冊。學校重視閱讀，早上設閱讀課，推展多元化閱讀活動，進行閱讀策略訓練。2.資訊科技互動學習：全港首十間資訊科技先導學校，已重修兩間電腦室，增購平板電腦。全校課室安裝互動式電子白板，購置教學及學習平台，學習環境高於標準，提昇教師及學生的資訊素養。3.專題研習：有系統訂定各級研習技能，學生學習不同模式的研習方法，培養共通能力。4.德育及國民教育：校本課程如小領袖訓練及一至六年級生活教育課，自我增值課等。全級學生到內地交流學習，認識中國文化；又放眼世界，到不同地區考察。
共通能力的培養 : 援引專業設計課堂，語文科透過讀寫聽說和有系統的思維訓練培養學生的溝通、創造、批判性思考等能力。數學、常識科及專題研習發展學生的自我管理、研習、解難、批判性思考和運算能力。
學生支援
全校參與照顧學生個別差異 : 開設拔尖補底小組，與教育局、大學協作提高課堂效率。課外輔導教學由專職老師設計增益性教學活動及駐校心理學家、言語治療師、輔導計劃，讀寫樂等等，支援不同需要的學生。提早識別能力稍遜，有不同困難程度的學生，提供適切關注和支援。
課程剪裁及調適措施 : 著意推動校本資優培育課程，學術及情意教育並進，促進學生全人發展。為全港首八間資優教育網絡學校的核心學校，與教育局資優教育組協作多項計劃，舉辦多元拔尖課程。又將資優教育元素融入課堂－高層次思維能力、創造力和個人及社化能力。
家校合作及校風
家校合作 : 家教會成立二十五年，組織完善，透過加入校本管理、各班家長網絡、家長講座、面談、電子通告，讓家長瞭解學校發展。家長信任支持學校老師，家長到校協助人數達300人超過5000人次。
校風 : 校風優良，設校風及學生成長支援組，統籌及協調訓育、輔導、公民教育、人生滿希望（危機小組）、環境教育、健康教育及啟發潛能教育等活動，推行全方位輔導，共同營造校園關愛文化，重視師生與家長的關係及社區聯繫。全面推行啟發潛能教育，將尊重、信任、樂觀、關愛、刻意安排，與課程融合，更兩次獲頒美國「國際啟發潛能教育傑出學校大獎」。
學校發展
學校發展計劃 : 優化教育，與時並進。擴展師生國際視野，致力發展兩文三語、課堂研究、藝術發展、資優教育、校本課程，讓師生共同提昇自我。聯繫教育局、大學、國內外教育機構進行協作計劃，援引專業知識。培養學生具創新、有實踐能力和理想、七育並進的新一代。
教師專業培訓及發展 : 鼓勵教師進修，參加教學研討會、講座及往海外、內地考察，按需要及興趣進修，自我增值。設經驗分享時段互相交流。與大專院校及教育團體進行協作計劃及交流，本校更獲各方邀請分享學校的經驗和成果，培養中層領導人才，領導梯隊。
學費減免 : 不適用。
