全方位學習活動
校方鼓勵學生多元化發展。課外活動分學術、體育、藝術、服務及興趣五大類共30多組，如：朗誦、數學、電腦、游泳、舞蹈、田徑、球類、中、西樂器、歌詠、步操管樂團、話劇、男、女童軍、基督少年軍及公益少年團，發展學生潛能。
辦學宗旨
本校秉承路德會辦學精神，推行基督化全人教育，使學生在和諧氣氛下學習，建立良好品格與積極的人生觀。使學生在靈、德、智、體、群、美六育獲得均衡的發展。
學校特色
學校管理
學校管理架構 : 行政管理架構清晰，各行政小組有效地推行各項校政。
法團校董會/學校管理委員會/校董會 : 2007年8月成立法團校董會，加入家長校董及教師校董，增加學校政策的透明度。
環保政策 : 透過學校節能政策及每年的教育活動，推動環保。
學校關注事項 : 1. 學與教的持續及深化發展。
2. 持續及深化生命教育，推廣正向校園文化。
教學規劃
學習和教學策略 : 各科設校本課程，更以小班教學、拔尖補底等模式配合不同學生的需要，提升學與教效能。
透過「課程統整」加強學習成效；重視自學能力的培養，鼓勵學生自主學習；發展資訊科技電子教學，提升學生的學習興趣。
關鍵項目的發展 : 1. 推行生命教育，發展校本生命教育課程，讓本校每一位學生都能靠著主愛，抱有積極的人生態度，勇敢面對人生中的各種困難和挑戰。
2. 各科推展電子學習，如透過eClass系統提供網上學習、校方提供I-Pad輔助學習等。
共通能力的培養 : 透過校本跨學科專題研習，引發學生的學習動機，從而發展各種共通能力。
學生支援
全校參與照顧學生個別差異 : 加強拔尖補底教學，開辦劍橋英語課程，奧數培訓和中文創意寫作；設學生支援小組，為有特殊學習需要的學生提供照顧。
課程剪裁及調適措施 : 因應學生程度剪裁課程，為有需要的學生作考試調適安排。設加強輔導教學班。
家校合作及校風
家校合作 : 定期與家長聚會及舉辦講座、工作坊及親子活動等，加強彼此溝通。每年家長教師會均舉辦多元化活動如：親子旅行，聚餐，座談會等。
校風 : 以學生為中心，運用多元化教學，透過獎勵計劃或活動，讓學生能均衡發展。校風嚴謹淳樸，透過基督全人教育，培養學生良好的品格，如自愛自律、尊重及關懷他人等。按學生的個別需要，作全力支援，如小一及新生訓練日，小老師計劃等。
學校發展
學校發展計劃 : 為提升學與教效能，優化校本課程；推行生命教育，建立學生正面的價值觀及積極的人生態度。
教師專業培訓及發展 : 設教師發展日、同儕觀課、專題講座及工作坊，全校教師均須出席及參與，並鼓勵教師參加與教學相關的課程。
學費減免 : 不適用。
其他 : 國際文化校園及假期聖經學校：每年邀請美國短宣隊到校展開一系列英語及宗教活動，提升學生英語水平；另學校參加「促進香港與內地姊妹學校交流計劃」及舉行境外學習，以擴闊學生國際視野。
