辦學宗旨
著重五育均衡發展，提供多元學習機會，讓學生發揮潛能。致力提升學生兩文三語水平，使其具備良好溝通能力，自信地表達觀點。發展學生批判性思維，懂得自律，待人以誠。與家長攜手合作，培養學生成為良好公民。
學校特色
學校管理
學校管理架構 : 本校由仁濟醫院辦學，並由法團校董會管理。校務行政由曾嘉麗校長會同行政主任多人分別主理，全面地照顧學生的需要和發展。
法團校董會/學校管理委員會/校董會 : 本校已成立法團校董會，校董包括辦學團體代表、校長、教師、專業人士、家長及校友。
環保政策 : 夏天時氣溫高於25.5度才開啟空調，垃圾分類，廢紙回收。
學校關注事項 : 培養學生自主學習能力，配合個別發展需要；提升學與教效能，以面對教育的新挑戰；推動生命教育，建立正確的價值觀。
教學規劃
學習和教學策略 : 每年均在各級進行戶外學習日、多元學習課、遊學團及專題研習等，讓學生發展共通能力。課堂設計強調高思維技巧訓練、照顧學生差異，並推行合作學習。
關鍵項目的發展 : 1. 推行「資優學生支援計劃」，發展資優學生潛能。
2. 校本中英文廣泛閱讀計劃，以鼓勵學生閱讀。
3. 所有課室均設有電腦、實物投影器及投射器，輔以電腦室及一個多媒體教學及自學中心。
4. 推行電子學習、STEM課程、校本電腦課程，讓學習能配合教育新趨勢。
5. 注重學生品德培養，透過校本輔導計劃及環保回收活動等營造健康生活的學習環境。
共通能力的培養 : 透過「專題研習」、「晨光舞台」、「校園電視台」、「閱讀分享」、
「英語日」及「普通話日」等跨學科活動，啟發學生潛能，提昇他們的
共通能力。
學生支援
全校參與照顧學生個別差異 : 1. 推行新資助模式輔導，通過全校參與模式，以照顧學生學習差異。
2. 推行「校本資優學生支援」計劃，發展能力高學生的潛能，亦透過全校參與式輔導計劃，為不同學生設計出不同的輔導課程。
課程剪裁及調適措施 : 全校推行小班教學，令互動的教學方法更見成效。
家校合作及校風
家校合作 : 家長教師會促進家長積極參與及支援各項學校活動，並透過不同家長講座、家長日及陽光電話保持密切聯繫。
校風 : 採用「訓輔合一」方式推行全方位校本輔導，聘請專業輔導老師處理及籌劃各項工作及活動。亦設功課輔導小組及精英領袖訓練小組，照顧個別差異。於學期初設小一適應課程，讓學生盡快融入學校生活。
學校發展
學校發展計劃 : 發展兩文三語目標，並實行普通話教授中文。培養學生自主學習能力，配合科技、科學的發展。推行課堂研究，以促進教師專業發展。
教師專業培訓及發展 : 以提升學與教效能為發展重點，透過共同備課，中港姊妹學校交流、同儕觀課及科研促進教師專業交流，提升教學質素。
學費減免 : 不適用。
