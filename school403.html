多元學習評估 : 全年有兩學期，每學期均各設兩次考試。安排不同類型的評估方法，如專題研習、小組討論及匯報、習作表現、課堂觀察等，評估學生的學習表現及成果。

全方位學習活動
除了參觀、教育營、學科活動和聯課活動外，還有管樂團、歌詠、朗誦、奧數訓練、環保小組、書法、基督小先鋒、紅十字會、武術、游泳、體操、舞蹈、童軍、公益少年團、交通安全隊、少年警訊、資訊科技訓練和球類等。
辦學宗旨
本校秉承「敬主愛人」的基督精神，以「犧牲與修和」為宗旨，藉天主的慈愛、基督的恩寵、聖神的光照為信念，傳揚福音為主作實踐，達至全人的教育。
學校特色
學校管理
學校管理架構 : 法團校董會為管理核心，授權校長帶領共十九個行政組別，如課程發展組、學校改善小組、教師專業發展及考績組、教務組、網上校管系統組、資訊科技組、學業輔導組、校園電視台製作組、支援有特殊教育需要學生組、學務組、活動組、小班教學發展組、危機應變小組、訓輔組等等。
法團校董會/學校管理委員會/校董會 : 學校由法團校董會管理，成員共13人，包括校監、五名辦學團體代表、校長、兩名家長代表、一名校友代表、兩名教師代表及一名獨立人士。
環保政策 : 本校積極推動環境教育，加強學生環保意識，鼓勵學生參與環保工作，故設有環保小組，定期為學生舉行環保活動。
學校關注事項 : 1. 提升家校的環保意識。
2. 培養學生正面積極的人生觀。
3. 透過電子學習提升學與教的效能。
教學規劃
學習和教學策略 : 實施循環周制，另有連堂的編排，方便各類學習活動。活用小班教學策略，採用異質分組學習。進行動感校園活動，作全方位學習。
關鍵項目的發展 : 1. 推介書叢榜圖書，舉辦晨曦閱讀台及故事小舞台，鼓勵學生閱讀。
2. 本年度中、英、數、常四科進行電子教學，部分科目更引入STEM元素。
3. 德育及公民教育︰為學生建立正確的價值觀，以「生命教育」為主題。
共通能力的培養 : 透過課堂學習、專題研習及跨學科活動等培養學生共通能力。
學生支援
全校參與照顧學生個別差異 : 1. 設有輔導小組，為中英數成績較弱同學在課前進行輔導。
2. 設有功課研習小組，為學習有困難的同學提供援助。
3. 設學習支援計劃，輔助學生學習。
4. 設有拔尖班，以發揮學生的潛能。
5. 運用「新資助模式」及「加強言語治療津貼」撥款，幫助有需要的同學。
課程剪裁及調適措施 : 1. 藉共同備課調適課程，提升課程質量。
2. 在07-08年度起已設有「普教中」課程。
3. 教育局小學校本課程課程發展組到校支援，在小五常識科進行高階思維、科學探究、專題，以培養學生的思維及學習技能。
家校合作及校風
家校合作 : 與家長攜手合作舉辦多項活動，如親子遊、競技日、親子攤位遊戲等，同時家長又積極參與學校舉辦的活動。
校風 : 校風淳樸，師生關係融洽，推行班級經營計劃，建立關愛校園。
學校發展
學校發展計劃 : 1. 培養學生正面積極人生觀。
2. 提升家校環保意識。
3. 透過電子學習提升學與教的效能。
教師專業培訓及發展 : 每年均訂立教職員的專業發展計劃，鼓勵教師進修，終身學習。
學費減免 : 不適用。
其他 : 1. 每年凡考獲全級第一、二、三名，操行獎及優異生獎的學生均可取得獎學金。
2. 2016年日本大阪歷史及文化學習活動。
