班級教學模式
全校課室已裝置互動白板及無線網絡配合活動教學，一至六年級推行普通話教中文，二年級至六年級推行電子課程，學生自攜電腦上學，開展自學模式。
班級結構備註
中、英文科推行愉快默書，隔周進行。英文科推行牛津大學出版社之牛津英語原音拼音課程。數學科以提升解難、自學能力為主。常識科則增設天文課程，擴闊學生的思維及視野。

健康校園生活 : 1. 每周設全民運動日。2. 參與開心果月活動。3. 體適能測試。4. 每周三天素食午膳及小食部不設肉類及高糖、鹽小食。5. 參加至「營」學校認證計劃。
備註 : 早會安排升國旗、唱國歌及校歌，另設有閱讀分享及新聞廣播節目，豐盛學生的精神及力量。每周一天收聽電台新聞報導，並加討論，以達致全面的通識教育。每天早上設導修課，由班主任督導學生朗讀英文電子課本。家長教師會協辦課後功課輔導班。
全方位學習活動
課外活動約有三十多項專項活動，包括英語及普通話話劇、科學尖子班、管弦樂團、廣播劇、口才班、3D打印班、機械人班、羽毛球班、高爾夫球班、射箭班、幼童軍、基督少年軍、流行樂隊、劍擊、獅藝、毛筆書法、繩藝及穿珠、武術、合唱團、拉丁舞、爵士舞及跆拳道班等。
辦學宗旨
本校秉承「鳳溪公立學校」辦學宗旨，以「謙信勤敏」為訓，推動學生全人教育，為學生營造優良的學習環境，培養學生獨立思考能力，並建立正確及積極的人生觀，讓他們愉快地學習，為日後回饋社會作好準備。
學校特色
學校管理
學校管理架構 : 由法團校董會負責監管學校運作，並以校監、校長及兩位副校長帶領、監察及推動學校發展，學校成立不同行政小組及課程發展組以規劃學校的發展和運作，共同討論、決策及執行，並透過自評機制以自我完善。
法團校董會/學校管理委員會/校董會 : 二零零五年八月三十一日成立法團校董會。
環保政策 : 1. 鼓勵循環再造及回收物資。
2. 設有學生環保大使。
3. 自設飯堂每周三天素食，現場分飯及使用環保午膳餐具。小食部不售含內肉類食物及含咖啡因之飲料。
4. 已設有太陽能發電系統、太陽能隔熱膜及生態園。
5. 設有有機耕種農地及開展水耕耕種。
學校關注事項 : 1. 促進學先自主學習，培養終身學習的能力。
2. 持續發展「以生為本」之校本電子課程，促進自主學習。
教學規劃
學習和教學策略 : 二至六年級推行校本電子課程；學生自攜電腦為電子書包，並採用Sharepoint為學習平台。以培育學生自主學習及高階思維為教學理念。
關鍵項目的發展 : 1. 透過水耕耕種，建立跨學科的STEAM及生命教育的學習模式。
2. 持續發展「以生為本」校本電子課程及優化設備。於2015年成功申請優質教育基金發展校本電腦科編程課程，而本年度更成為全港首批參與「賽馬會運算思維教育」小學之一，全面推行校本編程課程。
3. 推行身心靈全人發展：於2015年榮獲「關愛校園之推動生命教育」優異獎，有系統地推行價值教育，培養學生知禮守規、自律自重及關愛他人的文化。
4. 提升學生多元智能的發展：管弦樂團已成立4年；於體育科課程內加入瑜伽、射箭及高爾夫球等活動，提升學生專注力。
共通能力的培養 : 透過學習活動及經歷，重點培養學生之協作、溝通、解難及創造力，並設計高階思維活動。
學生支援
全校參與照顧學生個別差異 : 1. 校本電子課程。
2. 開設拔尖資優課程、發揮學生潛能。
3. 英文科分組上課。
4. 設立小學加強輔導班。
5. 提供課後支援班。
6. 為小一學生設英語基礎班、讀默寫小組及自理小組等。
7. 提供言語治療服務。
8. 設幼小及升中銜接措施。
課程剪裁及調適措施 : 根據學生能力評估後之報告，安排功課及考試調適。
家校合作及校風
家校合作 : 本校於一九九四年成立家長教師會，定期出版校訊、舉辦家長日、家長大學工作坊及親子旅行等活動，並成立家長義工隊。
校風 : 1. 歷史悠久，校風淳樸，學生知禮守規。
2. 致力啟發學生個人潛能，裝備學生有敏銳的思考、遠大的視野、廣闊的國際網絡，並以提升學生的解難能力、批判性思考能力、運算思維能力，協作能力及溝通能力為主，為學生裝備所需以面對二十一世紀的資訊發達及充滿創意的年代。
3. 建立關愛文化、和諧校園，讓學生在優質、健康的環境中成長，建立身心靈全人發展。
學校發展
學校發展計劃 : 1. 促進學生自主學習，培養其終身學習的能力。
2. 持續發展「以生為本」之校本電子課程及優化教學資源與設備。
3. 發展校本電腦科編程課程，並推展科學、科技及數學教育，以提升學生運算思維、創新及解難能力等共通能力。
教師專業培訓及發展 : 不定期與課程核心小組老師到訪外地小學交流，每月也舉辦一次公開課予其他學校老師到校參觀。
學費減免 : 不適用。
其他 : 本校曾5度榮獲「行政長官卓越教學獎」，包括課程領導、資訊科技教育、英文及數學（2度）。在學科方面，中文及英文科於本年度的校際朗誦節表現出色，共獲獎56項： 此外獲第十三屇全港小學普通話話劇比賽亞軍，第十九屇普通話演講比賽（語常委）傑出參與學校獎：數學科曾獲全港十八區（北區）小學數學比賽全場亞軍。截至4月20日止，本校已獲400多項的校外獎項。
