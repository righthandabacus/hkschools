全方位學習活動
提供每級不同的多元智能活動，培養九種共通能力的培養。課外活動組別超過三十項，包括校園電視台、舞蹈、管弦樂團、國術、跆拳道、中英文話劇、辯論組、籃球、足球、羽毛球、排球、飛碟組、藝術體操、花式跳繩、古箏組、手鐘隊、合唱團及國畫組。
辦學宗旨
以「仁、愛、勤、誠」為校訓，培養學生身體力行；以德、智、體、群、美五育為課程綱領，塑造完美人格，開創豐盛人生。
學校特色
學校管理
學校管理架構 : 分為：行政、財務、社會關係、庶務、發展。而行政事務分校務、教務、訓育及輔導、課外活動、學生事務、總務及資訊科技組。
法團校董會/學校管理委員會/校董會 : 定期舉行會議，各校董共同就校務進行討論決策，貫徹校本管理精神。
環保政策 : 參與廢紙回收計劃、琚校自然教室（隔熱塗層）、綠化校園、環境及自然保育基金之綠色節能力能及節水器具改裝用水設備。
學校關注事項 : 持續提升學與教的質素、發展資訊科技教育及落實健康校園政策。
教學規劃
學習和教學策略 : 一至六年級均進行合作學習法，培養學生人際溝通技能，並鼓勵朋輩學習，提升學習效能。每週進行週會、成長課及課外活動。本校一至六年級推行小班教學，設有多元智能課，培養共通能力。各主科均設有不同活動，更帶學生進行境外考察及與內地學生進行交流，豐富學生學習經驗。每年均進行全方位跨學科學習課程，提供多元化的學習活動。
關鍵項目的發展 : 閱讀計劃：早讀課、英文閱讀及伴讀計劃。
資訊科技：課室設有電腦、投影機及實物投影機，推行電子化教學。
專題研習：小一至小六進行專題研習。
德育、公民教育：早會、週會、班主任課、成長課、專題講座。
共通能力的培養 : 課程中着重培養學生的共通能力，協作、溝通、解難等。兩星期一次多元智能課，以不同形式的學習活動來啟發學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 設有輔導班，為能力稍遜的學生補底及進行「校園乾媽爹」計劃，支援有特殊教育需要的學生；默書及測考調適；設有校本資優課程，分三層架構培訓有潛質的學生，如奧數、英語話劇組、英文大使、校園小記者等。
課程剪裁及調適措施 : 同級備課，制定不同程度課業。
家校合作及校風
家校合作 : 親子工作坊、閒聚星期二、家教會旅行，家長、學生、教師相互間增進溝通。家長義工服務隊，發揮家校合作精神。
校風 : 校風純樸，重視個人氣質與品德的培養；老師合作無間，師生關係融洽；學生品行端正、斯文有禮、好學進取，樂於參與課外活動。
學校發展
學校發展計劃 : 提升學生的學習能力，加強教師的教學效能，加入多元化教學策略及評估形式。培養學生自學習慣，實踐終身學習的目標。另外，致力建構學校為學習型組織，營造健康校園，讓學生健康成長。
此外，加強學生中、英語文能力，延續英文閱讀計劃及普教中計劃，為學生打好語文基礎；更致力為學生拓展國際視野，幫助學生掌握終身學習的技能，培養正面的價值觀，得到全人的發展。
教師專業培訓及發展 : 協作教學、共同備課、教研、觀課、講座、工作坊、境外考察及校本支援計劃，初小中文分層支援模式計劃。
學費減免 : 不適用。
其他 : 本校積極推動教師發展及關愛文化，兩度榮獲「卓越關愛校園--主題大獎，國際啟發潛能教育聯盟「2015國際啟發潛能學校成就大獎」。並連續七年獲得關愛校園獎；
同時，學生表現亦出色，獲以下大獎：
香港國際青少年表演藝術節2016獲香港總決賽鋼琴比賽亞軍；香港女子跆拳道比賽獲女童色帶組亞軍；全港小學微型小說續寫大賽冠軍；新閱會一頁書創作全港冠軍及雷霆傘兵升空火箭全港亞軍。
