班級教學模式
小一至小六實行小班教學；又設兩組小學加強輔導教學計劃，服務主要對象為小三至小六有需要的學生。小二至小六的中文、英文及數學科會進行同級分組，減低學生之間的個別差異，令課堂教學更有效率。
班級結構備註
小一採取助教入班模式。小二至小六採取同級分組模式（以小五為例，16/17年度本校共有兩班小五，於中文、英文及數學堂時，會將學生分為三組，每組由一位老師授課，師生比例會減至1:8至1:20。

多元學習評估 : 透過觀察、日常的課業、專題研習及持續評估等，全面評定各學生的學習進度。學校重視持續評估，以多元評量方式評定學生的整體表現。
分班安排 : 小一採取助教入班模式；
小二至小六中文、英文及數學則採用同級分組模式。

全方位學習活動
本校的興趣小組有：拉丁舞、保齡球、閃避球、跳繩、小結他、幼童軍、女童軍、趣味數學、奧林匹克數學、多元智能、棋藝、魔術、紙黏土、摺紙、朗誦、電腦、乒乓球、籃球、英語活動小組（由外籍老師教授）、義工服務組及資訊先鋒。
辦學宗旨
致力營造良好的學習環境，並在各學習領域上實踐人本教育理念，讓學生自主學習及盡展潛能。同時，致力培養學生獨立思考、解決問題的能力，又讓學生參與社區活動，培養學生服務的精神。
學校特色
學校管理
學校管理架構 : 法團校董會下設校政委員會以監察學校運作。
法團校董會/學校管理委員會/校董會 : 法團校董會各成員熱心教育。
環保政策 : 1. 實行環保4R政策。
2. 參與綠色學校計劃，推行多項綠色校園活動。
學校關注事項 : 2015/16至2017/18學年關注事項
1. 聚焦發展學教評，培養自學自習能力；
2. 深化正向思維，照顧學習多樣性；
3. 持續推行家長教育，構建綠色校園。
三年發展主題「學自學 信用愛」
教學規劃
學習和教學策略 : 1. 全學年上課日為193日。
2. 實行循環上課日，並藉著特定上課日來安排全方位學習、主題及跨科學習等活動。強調學習者為主的學習模式，重視體驗式的學習。透過校本課程聯繫四個關鍵項目，促進學生的思維訓練、提升學生的技能及共通能力。
關鍵項目的發展 : 1. 設閱讀時段，並安排老師及家長義工講故事，又舉辦閱讀日，以推動閱讀風氣。
2. 藉電子學習平台及電子資料庫促進學生學習。
3. 透過科目或全方位學習活動來進行專題研習。
4. 施行校本德育及公民教育課程，培育學生成為良好公民。
共通能力的培養 : 以科本及跨科模式，培養學生的共通力。
學生支援
全校參與照顧學生個別差異 : 1. 以全校參與模式為有特殊學習需要的學生提供適切的個別化教學計劃與輔導服務。
2. 聘請教學助理，以照顧及指導有特殊學習需要的學生。
課程剪裁及調適措施 : 1. 各科已實行校本課程。
2. 增聘額外教師，並在部分年級進行中文、英文及數學三科的同級編組上課，使學生盡展所長。
家校合作及校風
家校合作 : 1. 家長認同本校的辦學理念。
2. 本校的關愛文化獲家長的肯定。
3. 家教會統籌學生午膳、校服、校車服務及訂購書簿等工作。
4. 成功向優質教育基金申請約47萬元的資助，推行為期兩年的家長教育。
校風 : 1. 重視學生全人發展，提供不同類型的支援服務。
2. 提供小一銜接及多元智能課程。
3. 設功課輔導時間及小老師計劃。
4. 午膳時段設李炳廣播電台，提供多元化節目。
學校發展
學校發展計劃 : 1. 堅持人本教育的理念，樂於接納不同需要的學生，致力造就每個學生。
2. 學校為學習型組織，重視教師的專業素質。
3. 已成功爭取外界資源，以推動各學習領域的校本課程。
教師專業培訓及發展 : 1. 強調學與教的質量，鼓勵教師持續進修。
2. 與教育局及多間大學合作，於不同科目進行校本支援課程，提升教師教學技巧。
學費減免 : 不適用。
其他 : 1. 第十一屆香港盃外交知識競賽（小學組網上問答比賽）（冠軍）及最佳表現學校獎。
2. 獅子盃科學科技比賽（兩個銅獎，最巧妙設計獎及最受歡迎獎）。
3. 華夏盃全國數學奧林匹克邀請賽華南賽區晉級賽（一等獎，二等獎及三等獎）。
4. 香港公園綠色大搜查 2016/2017（亞軍、優異及最佳帽子設計獎）。
5. 2016-17全港閃避球新秀賽（小學組亞軍）。
