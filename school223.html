班級教學模式
本校由小一至小六均以活動教學及小班教學模式進行。全校的英閱課則由外籍英語教師及科任老師共同施教。一至三年級的英文科參加英閱PLPR/W 計劃。
班級結構備註
開學前提供小一學習適應週及小六暑期補課班。

全年全科測驗次數 : 2
全年全科考試次數 : 2
多元學習評估 : 評估包括2項，每年2次測驗和2次考試及各科的進展性評估。各科皆設有專題研習，以跨學科進行：更引入學生自評及他評機制。
分班安排 : 四至六年級按學生的學業成績分班。

午膳安排 : 由指定供應商提供及由家長安排。
健康校園生活 : 午膳以飯盒形式進行，班主任與學生在課室內共同進膳，藉以照顧學生用膳及學會珍惜食物。飯商會定期供應水果或果汁，午餐的款式多元化。
備註 : 在星期一的公民訓練課講解「週訓」及舉行德育活動，並不定時播放校園電視台節目。
本校學生每天更將吃剩的飯菜分類放進回收筒，再經廚餘機製成有機肥料，為學校及鄰近社區提供有機耕作的養分。
全方位學習活動
按學術、音樂、體育、藝術、宗教及服務六個範疇設立三十多個興趣小組以發展學生多元智能，並定期舉辦參觀及教育性活動如節慶嘉年華及校運會。為提高學生對祖國的認識，每年籌辦各種與國內外交流的活動如台灣交流團等。
辦學宗旨
本著基督精神，培育兒童在德、智、體、群、美及靈育上有均衡發展，從而達致基督化的完美人生。我們致力成為學習型組織，全面實施優質教育，培養均衡人才，讓學生在多元化的學習環境中成長。
學校特色
學校管理
學校管理架構 : 辦學團體（聖公宗（香港）小學監理委員會）委託校董會監察學校的運作。
法團校董會/學校管理委員會/校董會 : 辦學團體委任5位人士及1位獨立人士和民選家長校董及教員校董各兩名組成法團校董會，負責帶領及督導學校的日常運作。
環保政策 : 進行安裝太陽能光復供電及再生能源閣。參加無冷氣日、一人一花苗計劃。建立綠色校園及垃圾分類箱。太陽能廚餘機投入生產有機肥料。進行基德小農夫—有機耕種計劃。
學校關注事項 : 1. 小班教學。
2. 學與教。
3. 生命教育。
教學規劃
學習和教學策略 : 以兩文三語進行早禱。每年各科舉辦活動日。透過全方位學習，應用九種共通能力。舉行學習交流以擴闊學生的視野。
關鍵項目的發展 : 1. 透過建立「圖書館網上系統」及配合各科學習的專題書展，培養學生的自學能力。
2. 設資訊科技課及電腦輔助學習課，加入教學平台及個人學習檔案系統。
3. 各科均進行專題研習課，培養學生的自我學習態度。
4. 每週皆編排公民訓練及成長課。
共通能力的培養 : 幫助學生學習的基礎，本校舉辦不同的學科活動日，如中普週、英文日、數學日、科技學習日等，培養學生的各種共通能力。
學生支援
全校參與照顧學生個別差異 : 1. 制定校本課程調適政策。
2. 開設校本資優課程。
課程剪裁及調適措施 : 為資優學生提供跳級機制。自設校本工作紙，課程剪裁，默書及測考調適，課堂抽離及協作。
家校合作及校風
家校合作 : 成立家教會，舉行親子活動，如旅行、參觀、培訓班、比賽等，並經常協助學校舉辦各項活動。家教會更設立獎學金。
校風 : 推行校本訓輔工作，設有成長課、講座及公民教育活動以培訓學生，更設立輔導課程。
學校發展
學校發展計劃 : 營造兩文三語學習環境：
1. 校本英閱計劃。
2. 聘請多名外籍英語老師，每級推行英閱計劃。
3. 在六年級數學科提供銜接中學課程。
4. 常識科推行科學探究學習，並設有常識科資優課程-小小科學家和培養學生的生活技能。
教師專業培訓及發展 : 全校大部分的中文科教師通過普通話基準。數學科教師進修專科專教課程，安排教師接受小班教學及多種教學策略的培訓。
學費減免 : 不適用。
其他 : 1. 每年頒發每級八項獎學金。
2. 舉辦學生遊學團及交流團如日本、新加坡、台灣及韓國等。
3. 在課外活動中，屢獲公開及校際比賽獎項如乒乓球、游泳、田徑、中英文朗誦及管弦樂團等。
