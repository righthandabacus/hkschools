健康校園生活 : 學校去年參與「至『營』學校認證計劃」，本年度繼續推行健康午膳餐單，全面監察膳食衞生情況；鼓勵學生培養做運動的良好習慣。
備註 : 學校採用循環日制，上午以核心課程為主，下午安排主題式聯課活動，為學生提供均衡課程。
全方位學習活動
除了教育性參觀及講座外，週五課外活動項目包括：田徑、樂器訓練、幼童軍、棋藝活動等。而課後及週六的學藝班有小司儀培訓、英語話劇、中國舞、爵士舞、足球、籃球、乒乓球、欖球、游泳、書法、跆拳道、管弦樂團、各類樂器訓練等。
辦學宗旨
本校秉承「敬業樂群」校訓，致力啟發學生潛能，提升自學能力，為終身學習奠下穩固基礎。學校亦透過全方位學習活動培養學生的多元興趣及共通能力，為社會培育一群富創意、具使命感及關心社會的良好公民。
學校特色
學校管理
學校管理架構 : 校長、副校長、主任4名、圖書館主任、教師16名、助理教師3名、學生輔導教師及外籍英語老師2名。
法團校董會/學校管理委員會/校董會 : 主席由香港特區首席助理秘書長出任，校長為當然委員，其餘包括教師、家長及社會人士代表各兩名。
環保政策 : 推行環保大使計劃，協助美化校園及推動環保回收活動。
學校關注事項 : 推動自主學習（包括電子學習）及培養學生情緒智能。
教學規劃
學習和教學策略 : 以「六日循環週」運作，上午為「核心課程」。下午為主題式聯課活動，為學生提供全面而均衡的課程。每年以不同專題為主題，進行跨學科專題研習活動，加強學生視野。
關鍵項目的發展 : 按學生的需要制定四個關鍵項目（閱讀計劃、資訊科技教育、專題研習及成長教育課程）的發展計劃。此外，透過推行不同的獎勵計劃和組織服務隊伍，強化學生的正面行為。
共通能力的培養 : 學校致力於發展學生的創造及批判性思維，通過專題研習、訓練活動及不同服務崗位的培訓，以提升學生的能力。
學生支援
全校參與照顧學生個別差異 : 本校實施「雙班主任」制，加強對學生的關顧。大部分課堂採用「協作教學」，增強師生互動。部份年級和科目進行「同級分組教學」，實踐「拔尖補底」、「因材施教」的精神。
課程剪裁及調適措施 : 實施同級分組教學、協作教學、增益/進益課程、小組輔導等。
家校合作及校風
家校合作 : 家長教師會致力於推動及鼓勵家長積極參與學校活動，攜手共建一個家校同心的校園。
校風 : 本校校舍古雅，校風純樸，師生關係融洽。校方運用不同教學模式及活動讓學生積極投入學校生活。設「功課輔導課」指導學生，亦會即日上載家課供家長查閱。家長積極支持，配合學校，家長教師會亦肩負橋樑及促進學校發展的角色。
學校發展
學校發展計劃 : 學校銳意提升學生的學習效能，加強學生在學業及學業以外的表現。訂定有系統教師培訓計劃，鼓勵教師透過持續進修及經驗交流。學校已建立完善自評機制，運用多元評估工具提升工作成效，促進學校整體效能。
教師專業培訓及發展 : 每年舉行3次教師專業發展日，並鼓勵教師持續進修。
學費減免 : 不適用。
其他 : 曾月英校長、麥惠英校長、易嘉怡校長、家教會獎學基金；香港朗誦節英文亞軍及季軍、中文季軍、普通話亞軍；全港小學數學挑戰賽金獎；香港仔海豚小學非撞式欖球邀請賽2016冠軍；港島西區小學校際田徑比賽女甲100米亞軍；中西區分齡游泳比賽2016男青50米蛙泳季軍；中西區滅罪宣傳創作比賽（小學海報設計）冠軍；香港學校舞蹈節高級中國舞及初級兒童舞分別獲得甲等獎。
