辦學宗旨
以佛化教育為本，注重學生品學兼修，德、智、體、群、美、靈六育並重，且培養學生「諸惡莫作，眾善奉行」之慈悲精神，建立正確之人生觀，樂善好施，將來成為服務社群的好公民。
學校特色
學校管理
學校管理架構 : 法團校董會管理學校，學校行政組及發展組就學校的規劃發展和運作，共同討論、決策和執行，並透過自評機制自我完善。
法團校董會/學校管理委員會/校董會 : 法團校董會於2007年6月成立，現由辦學團體成員、校長、教師、家長、校友及校外獨立人士共16位不同界別的持份者組成。
環保政策 : 環保組監察校園環保設施，節約資源，綠化環境，推行環保教育活動及參與環保獎勵計劃。
學校關注事項 : 1. 培養學生自學習慣和自律精神；
2. 照顧學生多元學習需要，推展校本資優教育；
3. 建立「正覺人」的良好品格素質和健康生活模式。
教學規劃
學習和教學策略 : 1. 小班教學，推行小組協作學習，運用多元化教學策略及以評估促進學習。
2. 各科重視課前預習，以培養學生自學能力和態度。
3. 推行圖書教學、讀寫教學、拼音教學、「英語樂」增潤課程、思維技巧訓練及跨學科專題研習。
4. 無線網絡覆蓋所有課室，讓學生使用流動電腦裝置進行電子學習。
5. 靈活運用課時，安排全方位學習及服務學習活動。
關鍵項目的發展 : 1. 設圖書課、推行不同閱讀計劃及活動，配合課程進行課後延伸閱讀，推展跨課程閱讀。
2. 內聯網自學堂計劃、電子課件、網上閱讀及自學平台。
3. 系統地發展學生的研習能力及共通能力，包括溝通協作、明辨思維、創意解難和自我管理學習能力。
4. 班主任課推行班級經營、生活反思，公民課培養正確價值觀及進行成長課和生命教育，德公組統籌品德教育活動。
共通能力的培養 : 透過多種跨學科及本科的課堂教學和課堂外多元化的學習活動，例如專題研習、戶外學習、境外教育交流等，培養學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 小一適應措施、小一及早識別計劃、小三中英數學習小組、各級加強輔導教學小組、課後功輔班、專注力提升班、讀寫訓練、社交樂、自理小組、言語治療、成長的天空等。
課程剪裁及調適措施 : 1. 推行校本資優教育計劃，增設資優小組，提供資優課程。
2. 按學生不同能力進行拔尖、補底、扶平措施。
3. 中、英、數、常四科透過共同備課調適課程、教學內容及設計不同程度的課業。
家校合作及校風
家校合作 : 分級家長會、全校家長日、關愛電話和家書等。與家長教師會合辦活動、興趣班、講座及工作坊。透過學校刊物和網頁讓家長更了解學校。家長義工支持和協助學校，家校關係良好。
校風 : 校風淳樸，訓輔合一，學生自律互愛、敬師守規、有禮愛校、師生融洽。校園洋溢關愛和諧文化，屢獲關愛校園獎。
學校發展
學校發展計劃 : 1. 提升學生學習的動機、態度及自學的能力。
2. 透過課堂內、外照顧學生學習的多樣性，加強校本資優教育培訓。
3. 培養學生自律、守規、誠信、關愛的人格素質，樹立「正覺人」的形象。
教師專業培訓及發展 : 按學校發展、科組發展和教師個人專業發展三個層面推動。學校除參與專業教學支援計劃外，會因應每年的關注事項，組織相關的培訓活動，又經常舉行備課、觀課、評課及教學分享會，以促進教學交流文化。此外，教師積極參與持續專業進修課程，緊貼教育趨勢，提升教學專業能量。
學費減免 : 不適用。
其他 : 設覺光長老紀念獎學金、佛教正覺蓮社學校獎學金、何德心居士飛躍進步獎學金、家長教師會獎學金。
