學校特色
學校管理
學校管理架構 : 除校董會外，學校還設立不同行政部門負責制定，監察、執行及評估校本發展方向及政策。
法團校董會/學校管理委員會/校董會 : 法團校董會
環保政策 : 製定及推行環保政策，透過各項活動，加強學生的環保意識和實踐環保行動，培育學生愛護地球的品德。
學校關注事項 : 1. 效法基督，陶造愛主愛人的使命。
2. 透過專業發展，提升教師領導能力。
3. 透過多元學習，提升學生主動學習的積極性及發掘學生潛能。
教學規劃
學習和教學策略 : 教師因應學生的學習興趣及能力，調適課程和教學策略，提供切合學生興趣、能力和需要的學習活動和經驗。
關鍵項目的發展 : 1. 從閱讀中學習—推行中英文閱讀課、閱讀獎勵計劃、故事爸媽、大姐姐說故事、好書推介活動及主題閱讀月。
2. 專題研習—以循序漸進的方式於各級教授專題研習的技巧，並連繫各科，就有關主題進行專題研習。
3. 德育及公民教育—透過宗教課、生命教育課、月會、訓輔及公民教育及早會分享，培養學生正確的價值觀及積極的人生觀。
4. 運用資訊科技—各科透過電子學習，加強學生使用資訊科技輔助學習的能力，讓學生發揮創意，並促進自學能力。
共通能力的培養 : 教師致力提升學生的共通能力及融入各科課堂教學之中。
學生支援
全校參與照顧學生個別差異 : 1. 本校為小一新生及插班生提供入學支援，好讓她們盡快融入本校校園生活。
2. 本校會為小六畢業同學在學習及升中方面安排銜接活動。
3. 設有輔導課，支援中、英、數有學習差異的學生。
4. 本校會為有特殊教育需要的學生提供適切的支援。此外，學校亦為有需要學生提供評估調適。
課程剪裁及調適措施 : 中、英、數三科切合學生能力和需要，進行課程調適及發展校本課程。
家校合作及校風
家校合作 : 學校重視家校合作，透過不同的渠道如家教會、家長講座、工作坊及親子活動等，與家長保持緊密聯繫，共同促進學生在學業和身心健康的發展。
校風 : 1. 透過「新學年學生適應家長會」，讓家長明白本校的運作與教學方針。
2. 透過不同的活動、工作坊及講座，讓學生能培養出正確的價值觀。
3. 推行「守護天使」、「出色小義工」和「一人一職」計劃，建立關愛的文化。
學校發展
學校發展計劃 : 1. 老師積極參與校內及校外研討會或工作坊，提升教師的專業發展。
2. 統整各科課程，並透過運用進展性評估，強化教學效能。
3. 各學科透過溝通和協作，參與不同的校本專業發展活動，提升學與教質素。
4. 進行各種多元化活動，讓學生掌握有效的學習策略和技能，促進學生自主學習，積極面對人生。
教師專業培訓及發展 : 透過參與相關的教學研討會及學科分享會，提升老師的專業發展，加強老師的教學效能，優化學與教素質。
學費減免 : 不適用。
其他 : 由於跑馬地校址需進行「全日制擴建及改建工程」，本校於2016年度起遷往柴灣興華（二）邨暫借校舍。
