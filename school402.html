健康校園生活 : 營造關愛文化，安排成長課、早會、週會、多元化「健康校園」活動及逢星期二、五進行全校健體操等。
備註 : 1. 星期一及星期四為宗教早會，星期二、五早上設普通話及英文學習活動，星期三為好書推介。
2. 本校乃一所學習型校舍，全校各樓層、梯間及課室都充滿豐滿的學習教材，好讓學生時時刻刻都有感官性及啟發式的學習。
全方位學習活動
本校由校內老師組織不同小組活動及培訓項目讓學生參加：如童軍、基督少年軍、集誦、獨誦、廣播劇、田徑、各種球類、話劇、國內交流、講座及工作坊等；與校外機構合辦興趣小組：珠心算、跆拳道、奧數、英語拼音、舞蹈、繪畫、木管樂器、科研及武術等。
辦學宗旨
1. 提供全人教育，著重靈性培養。
2. 學生能自我增值，終身學習，成為社會上的好公民。
3. 為老師提供完善的教學環境，建立合一的團隊精神，增強學校的歸屬感和使命感。
4. 學校每一位成員都能成為出色的典範，以基督的愛服務社群。
學校特色
學校管理
學校管理架構 : 本校設有法團校董會監察學校運作。本校亦設立行政、學科、財政、危機等小組管理不同範疇工作；家長教師會定期舉行會議，收集各方意見，改善校政，為學生提供最優良的教與學。
法團校董會/學校管理委員會/校董會 : 已於2014年8月30日成立法團校董會。
環保政策 : 設環保小組，制定環保指引，藉各類活動培養學生的環保意識及鼓勵他們實踐環保生活。
學校關注事項 : 1. 提升學生的自信心；
2. 提升學生的自學能力。
教學規劃
學習和教學策略 : 課程調適；英文協作教學；抽離教學；協作學習；應用資訊科技進行互動學習；專題研習；課前預習及課後延伸以培養學生自學能力；照顧學習差異。
關鍵項目的發展 : 閱讀計劃：設閱讀課、圖書課及英語閱讀計劃，鼓勵學生參加閱讀獎勵計劃。
資訊科技：教學上應用資訊科技，並於內聯網設自學區。
專題研習：以常識科為主軸，設定主題，以跨學科模式及內容進行探究式學習，並提供教育性參觀，積極培養學生的九大共通能力。
德育及公民教育：透過各科教學、早會、週會、成長課及學生活動滲入有關德育元素。
共通能力的培養 : 於課內外學習及活動，積極培養學生的九大共通能力。
學生支援
全校參與照顧學生個別差異 : 設優學組，為有需要的學生釐訂輔導、課業及評估調適計劃，並設有大哥姐計劃、手肌及執筆技巧訓練班、社交技巧訓練班，學習技巧訓練班、讀寫技巧班、專注力訓練班、言語治療班、抽離式小組教學及課後功課輔導班；另有資優培訓和拔尖班，如中文毛筆書法班、奧數班、英語寫作班及體藝培訓等。
課程剪裁及調適措施 : 評估及課業調適。
家校合作及校風
家校合作 : 藉不同類別的家長會及家長講座，增強家校合作；家教會協辦校內外各類型的學習活動，積極發揮支援作用，如擔任故事姨姨及義工等。小一更設立「親子英語班」，支援家長教導學生英語。另設有家長團契及不同的家長學習小組。
校風 : 營造關愛互助文化，鼓勵學生互愛互諒，培養正面、積極進取的態度；培養學生對學校的歸屬感。
學校發展
學校發展計劃 : 1. 提升學生的自信心；
2. 提升學生的自學能力。
教師專業培訓及發展 : 教師積極參與課堂改善計劃、培養共同備課及同儕互評文化，藉此提高學與教之效能；教師更參加各類型有關學科講座、研討會或工作坊；教師發展日；校本培訓工作坊；教師分享會；各科均推行專科專教及與內地教師積極交流，互相學習等。
學費減免 : 不適用。
其他 : 家教會設立各項獎學金以鼓勵學生積極求進步的態度，並鼓勵學生作多方面發展。獎學金分為：中、英、數優異獎，中、英、數進步獎，科技獎，文學獎，數學獎，體育獎，音樂獎及視藝獎。
積極爭取校外資源，與外間機構合作，為學生提供最優質的學習環境與發展空間。
