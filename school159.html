健康校園生活 : 學校積極推廣健康生活文化，包括制定並實施運動健康政策、健康飲食政策，落實食物監察計劃和參與衞生署「校園至『營』特工」及認證計劃等。
備註 : 校方利用午息時間安排文娛活動，增加學生的展示及學習空間；此外，校方逢週三下午舉行多元學習課，為學生提供更多跨級及跨科的活動，亦讓科組安排更多增潤正規課堂及聯課學習的機會，以能擴闊學生的視野及學習領域。課外活動方面，分循環組及固定組兩大類別。
全方位學習活動
每週設有活動課，項目多達30多項，另設課餘興趣班，包括舞蹈、樂器、視藝、球類、劍橋英語等。學生經常應學校及社區團體邀請參與各項表演及比賽，成績卓越。此外，校方亦積極推展資優教育，在六級進行不同知識的專項培訓，以發掘及延展學生的潛能。
辦學宗旨
秉承聖公會小學校訓 「非以役人 乃役於人」的精神，提供基督化的全人教育；培養學童以 德、智、體、群、美、靈六育作為均衡全人發展，應以「教養孩童，使他走當行的道，就是到老他也不偏離。」（箴言22章6節），為培育孩子的使命。
學校特色
學校管理
學校管理架構 : 法團校董會下由校長領導及副校長、中層行政人員協助，以管理與組織、學與教、校風及學生支援、學生表現四方面設立不同行政架構以有效管理學校及推展學校關注事項涵蓋的工作，法團校董會內亦設立不同小組，以支援學校日常行政運作，校董教師諮議會則是校董與教師間溝通的重要橋樑。
法團校董會/學校管理委員會/校董會 : 法團校董會由以下人士組成：
1.校監（校董會主席）2.校長（當然校董）3.辦學團體校董 4.替代辦學團體校董 5.獨立校董 6.教員校董 7.替代教員校董 8.家長校董 9.替代家長校董
環保政策 : 節約能源：使用節能電器，設立可再生能源發電系統，推行無紙校園，養成良好資源運用習慣。
減少廢物：實行環保4R原則及綠色碳審計政策。
簡約生活：多接近大自然，無污染旅行，少崇尚物質，心靈富足。
學校關注事項 : 1. 培養學生的自信及堅毅精神。
2. 提升學生的學習動機，培養良好的學習態度。
3. 推動教研及善用分享平台，以促進教師的專業成長。
教學規劃
學習和教學策略 : 透過多元、互動的教學模式，提升學生的課堂參與和引發學生的高階思維，亦透過專題研習、網上學習等，培養學生的自學能力；並為學生提供多元的學習活動，擴闊學生的學習空間，讓學生得到廣寬而均衡的學習經歷。
關鍵項目的發展 : 學生已養成了閱讀的興趣，學校於各級、各科推動延伸閱讀、多媒體及網上學習、專題研習等，提升學生的自學能力。本校致力推動生命教育，設立生活教育組，以基督的真理引領學生建立正確的價值觀，讓學生學會敬神愛人。
共通能力的培養 : 課程、教學及課業設計均以培養學生的創意、溝通、解難及批判性思考能力為目標。
學生支援
全校參與照顧學生個別差異 : 1. 設學習支援輔導班及輔導教學班照顧成績稍遜學生；並有駐校校本教育心理學家及言語治療師、學生輔導主任及人員支援學生的個別學習及情緒需要。2. 設校本資優課程，逐步實施全班式資優教育，並為各級學生提供抽離式校本資優教育，設多項專項培訓，培育不同才能的學生。3. 為不同學習需要的學生提供能力提升課程。
課程剪裁及調適措施 : 1. 透過共同備課設計照顧學生學習差異的課堂教學策略和課業。
2. 課程統整及舉辦大型學習活動。
3. 精英班設增潤課程。
4. 英文科設抽離拔尖班。
5. 按需要設中英數抽離課堂教學。
家校合作及校風
家校合作 : 本校設家長教師會和家長義工隊，建立家長與學校的良好夥伴關係。
校風 : 以基督的關愛精神為根基，推動和諧、關愛的校園文化，嚴而有愛地帶領學生走當行的路。
學校發展
學校發展計劃 : 1. 持續深化學生認識基督真理，實踐基督服務精神，並協助學生訂立個人目標及夢想，對個人、家庭、學校及社會作出承擔及貢獻。
2. 透過全方位學習活動及多元化教學模式，持續提升學生的學習動機及效能，並透過資訊科技及廣泛閱讀，進一步促進學生的自學文化。
3. 持續深化教師的授課及評課能力，透過引入不同的教學法，持續促進教師的專業發展及領導能力，成為高協作及高效能的教學團隊。
教師專業培訓及發展 : 配合學校的發展，安排教師專業發展及培訓，建立專業學習社群。
學費減免 : 不適用。
