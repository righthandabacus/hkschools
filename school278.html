全方位學習活動
發揮全方位學習精神，激發學生潛能，建立自信，使心智與情意得以平衡發展，活動分四大類：智能訓練、學術學會、興趣小組及運動學會。受學生歡迎的有歌詠團、舞蹈班、話劇班、手鐘鈴隊、野外定向隊、學校運動代表隊及電腦興趣小組。
辦學宗旨
本著融合多元民族文化，發展多元智能的教育方針，開拓學生的世界觀，建構學生獨立思考、善於溝通及學以致用的能力，培養學生熱愛生活、樂觀積極、敢於創新、勇於承擔及參與服務的精神。
學校特色
學校管理
學校管理架構 : 屬地利亞教育機構管轄，具明確循理的組織架構，實行高效益的管理。
法團校董會/學校管理委員會/校董會 : 1. 監察學校行政。
2. 檢視各既定政策。
環保政策 : 一人一花、綠化校園、減少廚餘、自備餐具。
學校關注事項 : 和而不同。多元智能。種族共融。
教學規劃
學習和教學策略 : 1. 英語為教學語言，兼重法文、普通話及粵語。
2. 互動教學。
3. 多元文化學習課程。
4. 全方位學習，進行跨科專題研習，如「多元文化融合」及「社區探索」等。
5. 廣泛閱讀計劃。圖書教學。
關鍵項目的發展 : 推展中、英文「悅目動聽讀書樂」計劃，結合自選閱讀與講故事，提升閱讀興趣。推行網上學習，訓練資訊科技能力。透過專題研習，學生掌握分析、組織及整理數據的能力。營造關愛的學習環境，鼓勵學生參與服務社區活動及「環保」活動。
共通能力的培養 : 學習三文四語：英文、中文、普通話及法語。
學生支援
全校參與照顧學生個別差異 : 1. 為不同需要學生安排增潤課程及拔尖補底。
2. 安排中一學前適應工作坊及活動。
照顧學生心理及學習需要：
1. 學生輔導主任、家長輔導主任及教育心理學家提供適當輔導。
課程剪裁及調適措施 : 為非華語學生提供中文調適課程，可參加香港中學文憑考試及英國GCSE考試。
家校合作及校風
家校合作 : 定期家長教師會面及互通電話聯絡。
校風 : 1. 多元民族，共融文化。
2. 校本課程。
3. 因材施教，發展學生專長。
4. 開辦課餘功課輔導班。
5. 暑假為學生舉辦多元化活動。
6. 提供新生適應班。
7. 定期出版校報，邀請家長撰文。
學校發展
學校發展計劃 : 1. 為學生開展「一人一藝術」計劃。
2. 訓練學生自學、解難、創造力及運用資訊科技能力。
3. 延展「環保小先鋒」計劃。
4. 與教育局繼續開辦予不同國籍學童的「新來港學童啟動課程」。
教師專業培訓及發展 : 境外交流，針對性講座及工作坊。
學費減免 : 不適用。
