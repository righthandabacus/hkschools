班級教學模式
揉合國際與本港教育模式，採用IB國際文憑配合本地課程，引進腦基礎教學，著重思考及價值教育，低至高年級逐步加強英文教學，讓學生順利銜接中學。
班級結構備註
為資優及有其他需要的學生提供支援。英文科定期為學生進行閱讀能力評估，提高閱讀能力。

多元學習評估 : 學校注重質方面的評估發展。除了測考筆試、口試，亦著重學生平時課堂、家課及專題研習等表現。某些科目加入學生自評及家長評。多元化評估。
分班安排 : 課堂中會按需要進行能力分組教學。

健康校園生活 : 學校致力全人教育，特別注重學生的身心發展。透過各種政策及活動，鼓勵學生堅持運動，參與體適能。學校廚房為學生提供新鮮健康午膳，並推廣健康小食。
備註 : 啟思小學是一所有朝氣、有前瞻力的學府；學生在充滿關愛的環境下愉快學習與成長。教師除教學外，更視自己為學生的成長教練，以正面方法引導學生主動學習、追求卓越；自愛自律，全面發揮個人潛能。
全方位學習活動
學校非常注重學生全面發展，除一般課外活動外，更與其他機構合辦高質素多元智能發展課程，每週安排的課外及課後活動多達百項，包括：各種校隊訓練、管弦樂團、各種樂器班、童軍、舞蹈、武術、體育訓練及學習營。
辦學宗旨
為孩子提供卓越而健全的教育，揉合中外教育精粹，制定一套獨特而具高效能的教育模式，以學生為中心，實行全人教育，在孩子成長的關鍵時刻，奠下日後成功的基礎。
學校特色
學校管理
學校管理架構 : 校董會負責監察學校運作；校長、副校長負責管理及發展事宜；校內設各科目及行政小組，統籌及推展學校各項工作。
法團校董會/學校管理委員會/校董會 : 校董會
環保政策 : 著力培養環保意識，鼓勵以行動實踐。設環保大使推動環保工作，並將環保訊息，滲透於各科學習及活動當中。
學校關注事項 : 學校學術發展與個人成長並重，尤其著重思維發展，學習能力和個人素質以及潛能的培養，與世界觀的建立，讓學生在小學的關鍵時刻，奠下日後成功的基礎。
教學規劃
學習和教學策略 : 1. 採用各種具效能的教學法，全方位引發學生學習興趣，教導學生學以致用。
2. 實行全校跨學科「思考教育」學習，各級設探究主題，讓學生有系統地掌握有效的學習方法。
3. 特別著重語文學習，增加外籍教師及英語授課教時，造就良好英語環境；學生可選擇以普通話或粵語學中文。
關鍵項目的發展 : 1. 閱讀－每天設閱讀時間、推行閱讀獎勵計劃、網上中英文閱讀，建立閱讀習慣。
2. 科技－結合教與學各個環節，透過刻意安排，務使資訊科技成為教與學的好工具。
3. 專題研習－每年設六個探究單元，各科教師有系統地在課堂建構學生能力。
4. 德育－透過重視價值教育的IB 課程、EQ及思考課程、班級經營、訓輔導組舉辦的活動，進行高質素的品德教育。
共通能力的培養 : IB 課程對共通能力非常重挸，要求教師把有關元素結合在各科教學中，並需要學生定期對自己的態度與共通能力作反思，教師亦需觀察學生的表現，向家長匯報。腦基礎教學鼓勵合作學習與高思維訓練，是加強學生共通能力的最佳平台。
學生支援
全校參與照顧學生個別差異 : 教師視自己為學生的成長教練，照顧學生全人發展，特別留意學生個別在發展及成長的需要，給予禦防性的引導及支援，透過課程發展及訓輔導組統籌各項措施及活動，協助每一位學生品學並進，健康成長。
課程剪裁及調適措施 : 學校結合IB及本地課程，並運用腦基礎策略，推動思考、探究及價值教育，提升教學效能。特別注著語文教學，中文科可選擇以普通話或粵語學習，循序漸進，讓學生能掌握兩文三語，並能為日後以英文作為主要學習語言作好準備。
家校合作及校風
家校合作 : 家校關係緊密，有多元溝通渠道，家教會全力協助學校與學生發展。
校風 : 學校兼承「啟發潛能教育」理念，刻意營造關愛、尊重、信任、正面的校園文化，並透過課內課外的各種體驗建立學生的十個素質，培養他們成為積極探究、勤於思考，善於交流、勇於嘗試，智識淵博，堅持原則，胸襟廣闊，懂得關愛，全面發展及及時反思的人。
學校發展
學校發展計劃 : 1. 持續優美IB與本地課程的結合。
2. 繼續善用腦基礎教育發展更多元化教學策略，進行高思維教學及培養良好的思維習性。
3. 與啟思中學進一步聯繫，持續全面提升學生英文能力。
教師專業培訓及發展 : 1. 教師不斷參與IB 、BBL 、IE 等專業培訓，包括海外考察及國際會議，擴闊視野。
2. 教師培訓涵蓋學術及個人成長及發展兩方面。
3. 學校鼓勵每位老師鑽研一個他／她有興趣的專項，成為該範疇的專家，協助學校及學生發展。
學費減免 : 不適用。
其他 : 啟思小學竭力為孩子提供
1. 一個充滿愛與關懷的環境。2. 全面的照顧。3. 一個富啟發性的學習文化。4. 有效率，多元化而配合二十一世紀社會發展的教育模式。5. 充份的機會，讓學生發展興趣及所長，啟發潛能，培養素質在孩子成長關鍵時刻，奠定未來成功的基礎。
