學校特色
學校管理
學校管理架構 : 本校由瑪利亞方濟各傳教修會創立，並於二零一五年八月成立法團校董會，制訂學校的行政及發展方向。校長以下，分別有兩位副校長及學校發展委員會管理日常學校事務。
法團校董會/學校管理委員會/校董會 : 法團校董會有15位成員，共同管理學校、推動教育；確保本校的教育願景、辦學使命與核心價值得以實行。
環保政策 : 學校透過綠化校園及校園清潔計劃，讓學生實踐環保的低碳生活。
學校關注事項 : 1. 優化學與教。
2. 培養正確價值觀，活出聖羅撒人忠、誠、仁、愛素質。
教學規劃
學習和教學策略 : 以6天循環週上課，設早讀課及課外活動課。不同的學習活動豐富學生的學習經歷。每年根據關注事項訂定跨學科學習活動。
關鍵項目的發展 : 從閱讀中學習：以不同的閱讀活動及策略來提高學生的閱讀興趣和能力。
運用資訊科技進行互動學習：透過校本電子自學平台進行學習。
專題研習：學生根據專題研習內的任務，開展探究性的活動。
德育與公民教育：透過校本德育課程、成長及學習歷程檔案培養學生正確的價值觀。
共通能力的培養 : 不同學習領域的學習活動培養學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 為有特殊學習需要的學生給予適切的支援。
1. 為小一學生提供「小一適應」課程。
2. 為小二學生設有「展翅班」。
3. 設中、英、數拔尖課及輔導課。
4. 設有功課輔導班。
5. 設有言語治療訓練課。
課程剪裁及調適措施 : 為有特殊學習需要的學生在課程及評估方面作出調適。
家校合作及校風
家校合作 : 家長教師會作為學校與家長的橋樑，「愛心玫瑰」義工團，可讓家長從參與中了解學校。
校風 : 校風淳樸，致力培育學生六育發展。校園宗教氣息濃厚，師生相處融洽，互相關懷，對學校有強烈的歸屬感。
學校發展
學校發展計劃 : 推動全人教育，加強校本課程規劃，豐富學生的學習經歷及提升其思考能力；另方面致力培育工作，透過多元化活動，期望培育一群關心他人、自律、開朗的學生。
教師專業培訓及發展 : 配合學校的關注及各科的發展，透過校外支援及教研活動，促進老師的專業交流及提升其教學能力。
學費減免 : 不適用。
