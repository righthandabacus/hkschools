全方位學習活動
重視學生多元智能發展，課外活動涵蓋學術、體育、藝術、制服團隊等範疇。學校積極參加校際朗誦節、音樂節、舞蹈節、創藝展、學界體育聯會各項比賽，包括田徑、球類、劍擊，獲獎無數。又為學生安排戶外學習、旅行、參觀、教育營及境外交流等活動，豐富他們的學習經歷。
辦學宗旨
秉承聖公會辦學宗旨，著重學生在德、智、體、群、美及靈性方面的均衡發展，並以基督的愛去愛己愛人、服務社會，貢獻己力，實踐「非以役人，乃役於人」的真理，達至全人教育。
學校特色
學校管理
學校管理架構 : 在法團校董會以下，學校設有多個行政部門及功能小組推行校務工作。
法團校董會/學校管理委員會/校董會 : 法團校董會成員包括7名辦學團體校董，1名辦學團體替代校董，1名獨立校董，1名校長校董，1名教員校董，1名教員替代校董，1名家長校董，1名家長替代校董，1名校友校董。
環保政策 : 透過多元活動，培養學生愛護環境及珍惜擁有，實踐「低碳開步走，童心愛地球」的生活態度。
學校關注事項 : 貴自醒，重自律。
教學規劃
學習和教學策略 : 1. 重視閱讀，設閱讀課，推行閱讀策略、識字教學，增強學生閱讀能力。
2. 發展學生自主及獨立學習能力，透過專題研習、摘錄筆記、課前預習，培養自學習慣。
3. 教學策略著重從活動、探究、參與、實踐中學習，幫助學生建構知識、刺激思維和啟發學習興趣。
4. 以探究式教學和合作學習模式促進學習效能。
關鍵項目的發展 : 1. 啟用電子通告及電子繳費。
2. 提供多元活動，啟發學生潛能。
3. 加強資優及支援的工作及發展電子學習，照顧學生的學習差異。
4. 透過不同的培育計劃及宗教教育，教導學生正向價值觀。
共通能力的培養 : 1. 小組學習培養協作能力。
2. 著重說寫訓練，建構溝通能力。
3. 創新思維，培養創作力。
4. 多角度思考，啟發批判性思考能力。
5. 提高資訊素養，加強運用資訊科技能力。
6. 掌握基本運算能力。
7. 刺激思維，提升解難能力。
8. 學習蒐集和處理資料，建構研習能力。
學生支援
全校參與照顧學生個別差異 : 「學生支援組」協助培育資優學生，並支援學習上有困難的學生。
課程剪裁及調適措施 : 以「學生為中心」理念剪裁課程，致力擴闊學生的學習經驗，提升他們的學習動機和學習效能。中文科調適課本教材，增設識字教學、閱讀策略及中華文化校本課程；英文科採用校本拼音、寫作及閱讀課程；數學科採用校本調適課程。
家校合作及校風
家校合作 : 1. 透過家教會，建立家校合作夥伴關係，並促進家校溝通聯繫。
2. 家校合力舉辦活動，提升家長和學生對學校的歸屬感，並促進學生在學業和身心各方面健康成長。
3. 家長義工協助推動學校工作和學生活動。
4. 透過家長會、家長日、家長聊天室、家長通告、學校網頁、電子通訊平台及校訊，與家長保持緊密溝通。
校風 : 校風淳樸，學生有禮守規，積極參與校內活動及社區服務。教師、家長與學生對學校歸屬感強。
學校發展
學校發展計劃 : 1. 簡化學校行政，促進工作效能。
2. 照顧學生的學習多樣性。
3. 推展生命教育。
教師專業培訓及發展 : 1. 著力提升課堂設計、觀課和評課的質素，加強學與教效能
2. 老師持續進修，提升專業能力
學費減免 : 不適用。
其他 : 設學業優異獎學金；體藝傑出表現獎學金。
