辦學宗旨
秉承天主教耶穌寶血女修會辦學宗旨，藉教育事業傳揚福音，務使上主賜予人類真、善、美、聖的潛質得以發揮。以兒童為中心，創設一個優良學習環境，培育學生在德、智、體、群、美、靈各方面的均衡發展，以達全人教育。
學校特色
學校管理
學校管理架構 : 本校由天主教耶穌寶血女修會創辦，法團校董會制訂及監管一切校政運作。
法團校董會/學校管理委員會/校董會 : 由辦學團體代表、校長、獨立校外專業人士、教師代表、家長代表等組成。
環保政策 : 推行不同類型的環保活動，如綠化校園、舊校服及廢紙回收、惜飲惜食計畫、綠色聖誕活動，學校也更換了省電光管，以節省用電量。
學校關注事項 : 1. 優化課堂教學 促進自主學習；
2. 培養正向價值觀，以「海寶人」為己任（具有自理能力、盡責堅毅）。
教學規劃
學習和教學策略 : 每節課35分鐘，有15分鐘閱讀課；星期三有一小時課外活動，課後及星期六有不同課外活動。小一開始推行普通話教中文和小班教學，利用合作學習策略和高階思維的訓練，加強師生互動，促進自主學習；又安排全方位學習、科學探究、專題研習、戶外教育營等課程，讓學生在高參與、多展示、多欣賞、受尊重的環境中經歷學習，得到全人發展。
關鍵項目的發展 : 1. 德育及公民教育：透過正規校本課程及專題講座，培養學生正面價值和態度（堅毅、自理）。
2. 從閱讀中學習：加設早讀時間，增加學生閱讀機會；進行跨學科活動及協作教學，協調各科的發展。
3. 專題研習：推展各科的專題研習，深化學生的研習技巧，培養自學精神。
4. 資訊科技進行學習：進行各科的電子學習試驗計畫，透過教師設計的學習活動，提升學生的參與及互動，並推行翻轉教室的學習，促進自主學習。
共通能力的培養 : 透過各學科教學，培養學生九種共通能力。
學生支援
全校參與照顧學生個別差異 : 為小三至小六資優生開設不同資優小組。提升小一、小二及小四的中、英、數能力而設增值組。
課程剪裁及調適措施 : 為個別學生制訂教學及輔導策略調適。
家校合作及校風
家校合作 : 家教會有良好機制，與校方密切聯繫，攜手合作為學生創設理想的學習環境。
校風 : 本校以福音的價值觀為經，訓輔合一的方法為緯，配以不同活動如宗教禮儀、聖善之光、「你的紅封包」捐贈及服務學習等，以培養學生愛主愛人，委身侍人的精神。
學校發展
學校發展計劃 : 1. 積極推動電子學習，以增加學生的參與度及提升學習能力。
2. 著重學生品德培養，使成為愛主愛人的人。
3. 積極發展STEM教育，推行校本編程計劃，促進學生運算思維及解難能力。
教師專業培訓及發展 : 設有啟導教師計畫，培育新入職教師。本校教師有應邀出席本地或內地分享或交流。
學費減免 : 不適用。
其他 : 本校設有多個獎學金以鼓勵學生在學術及非學術的成就，包括：家長教師會獎學金、學與教基金、體藝基金、永恆教育基金等。此外每年均為學生舉辦海外交流遊學團以擴闊學生的學習領域及視野。
