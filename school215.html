辦學宗旨
以「博文愛德」為校訓，培育「博愛」精神；支持專業領導及教學，營造優良學習氛圍，與持份者共建尚學社群；促進學生全人發展，建立高尚品格及正面價值觀，終身學習，追求卓越；培養學生成為才德兼備、獨立思考、善於溝通、具國際視野、重視文化承傳、關愛社會及國家、具領導能力的人才。
學校特色
學校管理
學校管理架構 : 法團校董會/校監/校長。
法團校董會/學校管理委員會/校董會 : 法團校董會已於二零零六年成立。
環保政策 : 透過德育及公民教育課程，教授學生環保知識。設置分類回收設施，安排學生參與環保活動，提升環保意識。
學校關注事項 : 愛己愛人，主動學習。
教學規劃
學習和教學策略 : 利用小班教學的資源，鼓勵教師採用合作學習模式，加強照顧學生的個別差異。並採用螺旋式教學，為學生打穩學習基礎。
中文科一至六年級均以普通話授課。
英文科採用大圖書教學法，透過有趣的故事、活潑的語文學習活動和多元化的教材，讓學生能在愉快和輕鬆的環境下學習英語。
數學及常識科推行探究式學習模式，通過探究，學生可在較真實及有趣的方式下掌握抽象的概念及知識。
關鍵項目的發展 : 1. 閱讀計劃：設早讀課及圖書課，參加校外閱讀計劃，幫助學生建立閱讀技巧，養成閱讀習慣，提升語文能力。
2. 資訊科技：一至六年級設有電腦課程，另於高年級開始按需要滲入電子學習元素，提升學生的學習興趣，並幫助學生建立運用資訊科技的能力。
3. STEM教學： 於一至六年級推行STEM教育，透過強化綜合和應用跨學科知識與技能的能力，提升學生的學習興趣，培養創造力和解決問題的能力，讓學生發揮創意潛能。
4. 德育及公民教育：編排適切的活動及課程，資助學生進行境外交流，擴闊視野。
共通能力的培養 : 將培養學生共通能力的工作滲透入日常學科教學之中。
學生支援
全校參與照顧學生個別差異 : 1. 推行合作學習，有效照顧個別差異。
2. 設加強輔導教學組，為能力稍遜的學生提供小組輔導。
3. 全校推行小班教學，加強照顧學生的個別差異。
4. 進行課程調適，兼顧資優及能力稍遜的學生。
5. 設拔尖班，進一步提升學習能力較強的學生。
6. 設功課輔導及補習班，支援學生不同需要。
課程剪裁及調適措施 : 為有特殊教育需要的學生提供課程調適、考試調適等。
家校合作及校風
家校合作 : 已成立家長教師會，全體家長均為會員。家教會的委員積極與學校合作，推進會務及校務。家長可隨時與校長及教職員聯絡，表達對學校工作的意見。
校風 : 1. 幫助學生養成博學於文，愛人以德的良好品格。
2. 指導學生建立環保節儉的生活態度，戒除浪費享樂之風。
3. 教導學生學會感恩惜福，尊敬長輩，愛護弱小。
4. 設立強帶弱，長扶幼的機制，加強朋輩互助。
5. 加強家校合作。
學校發展
學校發展計劃 : 本校15—18年發展重點：
1. 優化學與教，提升學生的學習表現，強化學生主動學習的態度。
2. 讓學生學會「愛己愛人」：讓學生學會欣賞自己和別人，主動關心身邊有需要的人，強化學生關心及服務社群的態度。
教師專業培訓及發展 : 定期進行教師培訓，鼓勵教師參與校外進修。
學費減免 : 不適用。
其他 : 學校設有獎學金獎勵學業表現優異的學生，每年都鼓勵及推薦有潛質的學生參加校際比賽。
