班級教學模式

全方位學習活動
透過課外活動，課後學藝班以及由校外導師帶領的興趣小組，提供多元化的學習活動，讓學生發展潛能，本校並有校運會、聖誕聯歡會、戶外教育營、旅行及境外學習等大型活動，讓學生投入愉快的校園生活。
辦學宗旨
致力提供優良的學習環境及高質素的教學活動，培養學生德、智、體、群、美的均衡發展，奠定其終身學習的基礎，使他們享有愉快的童年，邁向積極人生。
學校特色
學校管理
學校管理架構 : 校長、副校長、主任、課程主任、學生輔導主任、圖書館主任、教師、外籍英語教師共54位。
法團校董會/學校管理委員會/校董會 : 由教育局代表、校長、教師、家長、校友及社會人士協同管理學校。
環保政策 : 老師帶領「環保大使」，推動環保工作，發放環保資訊，舉辦教育活動、講座及比賽。為推廣再生能源，提升環保意識，學校成功申請環保及自然教育基金，安裝再生能源發電系統：太陽能光伏板發電，微型風車發電及動能單車產電。另外，設有廚餘處理系統，製造有機肥田料，美化校園。
學校關注事項 : 1. 自主樂學好「南」兒。
2. 重德樂學好「南」兒。
教學規劃
學習和教學策略 : 以學生為中心設計教學計劃及課堂活動。主要科目編排連堂進行聯課活動。透過參觀探訪、語文活動、數學遊蹤、科技探究、綠化校園、體藝比賽及全方位學習，豐富學生學習經驗及發展多元智能。
關鍵項目的發展 : 1. 閱讀文化：閱讀計劃、親子伴讀及早讀課。
2. 資訊科技：電腦課、電腦早晨及、校內、外比賽及STEM教育。
3. 專題研習：小五至小六推行配合常識課程的專題研習。
4. 德育及公民教育：成長課、主題講座及週會，培育學生德育及公民教育七大關鍵項目。
共通能力的培養 : 透過課堂活動、專題研習、基本邏輯思考訓練及多元化學習活動發展九大共通能力。
學生支援
全校參與照顧學生個別差異 : 設學習支援小組，統籌照顧學生個別差異的策略。
課程剪裁及調適措施 : 因應學生能力作課業調適，並提供拔尖補底課程，包括輔導小組、言語治療、至「專」小特工、劍橋英語、校園小記者、中文寫作拔尖課程及奧數班等，以提升學習興趣及社交技能。
家校合作及校風
家校合作 : 成立家教會，舉辦家校活動：小一迎新、親子活動、分級家長會及家長講座。設家長義工隊，協助早讀、課外活動及大型活動。
校風 : 校風淳樸，訓輔並重。推行南元小之星獎勵計劃、小師長計劃、小五歷奇教育營、風紀培訓、禮貌大使計劃、環保綠色特攻隊及小義工訓練計劃，著重培養良好習慣，啟發關愛服務精神。
學校發展
學校發展計劃 : 2015-2018學校發展計劃
關注事項(一)：
建立「自主學習」學與教文化，發展自主學習學與教文化，透過教師專業發展，建立教研學習圈，推行自主學習教學模式及教學設計，培育學生的自主解難能力及自學信心與動機及建立學生自主學習的態度、能力與策略，培育學生成為「自主樂學好南兒」。
關注事項(二)：
推動全方位價值教育，將現時品學兼優及良好校風持續優化，透過全方位價值教育的校本課程與教育活動設計，加強培育學生道德觀及責任觀（包括七種首要價值觀），培育學生成為「重德樂學好南兒」。
教師專業培訓及發展 : 專科專教、專業進修、共同備課、同儕觀課及專業支援。
學費減免 : 不適用。
