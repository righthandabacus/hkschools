班級教學模式
課堂設計以小組討論、合作學習、電子學習為主，讓同學間能互動及愉快學習；各級均有跨學科專題研習、戶外學習等，提升學生學習的興趣及視野。
班級結構備註
小一推行課程統整，銜接幼兒課程，讓學生愉快學習、自主學習；各級均為小班教學，讓我們的團隊能照顧學生的個別差異；各級設加輔班，因材施教。

全方位學習活動
於星期五下午設中華國萃班，以培養學生對中國文化的興趣，課後亦設不同的發展潛能班及興趣班，以提升學生的多元發展，本校亦經常舉辦不同的戶外學習及境外學習活動，如日本、韓國、新加坡、馬來西亞、台灣、北京、上海等地，讓學生擴闊視野，增廣見聞。
辦學宗旨
以孔聖「有教無類」的精神培育人才，又以「因材施教」方式發掘兒童的潛質，培育學生「在家做個好孩子，在校做個好學生」。
學校特色
學校管理
學校管理架構 : 設法團校董會，確立學校發展方向及制定政策；設行政、教務及課程發展委員會及學生事務委員會，協助推行校政。
法團校董會/學校管理委員會/校董會 : 本校自2008-2009年度起成立孔教學院大成小學法團校董會，成員共13名，包括辦學團體校董6名，校長校董、獨立校董、校友校董、2名家長校董、2名教員校董。
環保政策 : 本校致力推動環保教育，以專題探究、環保講座、回收比賽及實地考察等，提升同學的環保意識；制定一系列環保措施，如開關空調守則、節約用水措施及供應環保午餐，成為環保校園。
學校關注事項 : 1. 優化學與教，提升學習效能。
2. 照顧個別差異，盡顯學生潛能。
3. 營造互助互愛健康校園的關愛文化。
教學規劃
學習和教學策略 : 1. 推行合作學習，以學生為本教學法，促進同儕互動，建立積極的學習態度，發展人際溝通協作及社交能力。
2. 重視師生交流與回饋，提升學習效能。
3. 跨學科課程統整，豐富學生學習經驗。
關鍵項目的發展 : 1. 培養閱讀風氣，設立多元化閱讀獎勵計劃。
2. 推行電子學習，培養學生自學精神。
3. 自行編訂儒家德育課程，以多元化活動培養學生的品德。
4. 每年進行跨學科專題研習，並提供戶外學習，培養學生共通能力。
共通能力的培養 : 各級課程均滲透專題研習技巧，如資料分類、篩選、腦圖、統計圖、訪問技巧及問卷設計等，使學生能有系統地掌握各種共通能力。
學生支援
全校參與照顧學生個別差異 : 小老師、加輔教學計劃、校本心理學家、拔尖補底班、精英班、資優班、小一、二功課輔導、小班教學、言語治療及學習支援小組以協助能力稍遜及較高的學生提高學習效能。
課程剪裁及調適措施 : 中、英、數課程剪裁及調適，協助能力稍遜及較高的同學提升學業成績；定期補課，鞏固同學所學。
家校合作及校風
家校合作 : 本校家長教師會在一九九四年成立，主要為加強家長與學校之間之聯繫與溝通，以促進學生學業及品德之培育。每年均舉辦多項家長教育活動，使家長能互相交流管教子女之心得，並經常舉辦親子活動，增進親子關係及提昇學生及家長對學校的歸屬感。
校風 : 推行訓輔合一校本活動，營造學校關愛文化。定期舉行家長會，並增設電子溝通渠道，以增強家校合作的果效。藉升旗禮、成長課及德育分享會提高學生的德育及公民意識。
學校發展
學校發展計劃 : 1. 英文科：參加小學校本支援服務（英國語文），得到教育局專才來校與老師設計課堂活動；推行教育局初小讀寫計劃、網上閱讀計劃及增聘多位外籍英語老師，提高學生英語能力。
2. 常識科：參加教育局專業發展學校計劃（在常識科中有效運用學習歷程檔案，提升學生學習科學與科技範疇的效能–服務由聖文德天主教小學提供）。
3. 推動電子學習，照顧個別差異，促進自主學習。
4. 建立關愛校園文化，加強學生的歸屬感。
5. 提升學生自信心去面對挑戰。
教師專業培訓及發展 : 按學校發展及本校教師專業發展的需要，安排校本專業發展日及培訓，以發展學習型組織，進一步提升學與教的效能。
學費減免 : 不適用。
其他 : 1. 課節包括周會、儒家德育課、國民教育課、成長課、早讀課及課外活動等。多位駐校外籍英語教師教授學生英文，並定期舉辦英語活動。
2. 提供托管服務至晚上七時。
3. 開設英文補習班。
4. 開設小一、二功課輔導班。
5. 設立多項學業及德行獎學金。
