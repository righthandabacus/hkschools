健康校園生活 : 本校獲認證為香港衞生署舉辦之至「營」學校認證計劃的成員學校之一，提倡健康校園，注重學生健康成長，定期設水果月、健康午膳等。
備註 : 學校除著重學生學習外，亦推廣體育活動，其中排球隊屢獲優秀成績，成員更獲選為區代表。另照顧學生心靈成長，定期舉辦不同主題的周會、德育及公民教育課，小一及升中生銜接課程，安排不同的校外體驗活動（如全方位學習活動、境外交流），讓學生擴闊視野。

辦學宗旨
注重德、智、體、群及美育的均衡發展。教導學生成為有禮貌，肯負責，勤學，自愛自律的好學生；使成為心地善良，氣度平和，自強不息的好少年。
學校特色
學校管理
學校管理架構 : 校政執行委員會下分校風及學生支援、學教、發展組。
法團校董會/學校管理委員會/校董會 : 法團校董會已於2007年8月31成立，名為柏立基教育學院校友會何壽基學校法團校董會，現由11位校董組成，包括5位辦學團體校董；獨立校董、當然校董、家長校董、替代家長校董、教員校董及替代教員校董各1位。
環保政策 : 全校參與綠化社區、校園，環保午餐，節能減廢。
學校關注事項 : 1. 提昇學生自我管理能力，成為一個自律守規的好學生。
2. 建立優質課堂教學，發展學生自主學習能力。
3. 凝聚家長團隊，共建和諧校園。
教學規劃
學習和教學策略 : 貫徹辦學的教育理念---「讓每一個孩子都有成功的機會」，課堂內利用多元化的學習策略和活動，提升學生學習動機、自主學習能力及成功的學習體驗。參考小班教學學習元素，設計優質的課堂教學，展現學生不同的潛能。本校為教育局Wifi 100計劃先導學校，同時亦是「屯門區第一光纖小學」（即全校網絡皆用光纖連接），積極發展學生電子學習的知識和技巧。
關鍵項目的發展 : 1. 致力提昇學生兩文三語學習能力。
2. 設計「多參與，高展示」的優質課堂，讓學生強化學習的效能及成功感。
3. 各科建立預習制度及發展電子學習課程，培養學生自主學習能力及技巧。
4. 籌劃及優化全方位學習活動，擴闊學生視野及學習經歷。
5. 學校設英語班（數學、常識均主要以英語授課）、中文科設粵教中、普教中班，亦著重語文學習的讀寫聽說能力，積極照顧不同需要的學生學習語文。
共通能力的培養 : 協作能力、溝通能力、創造力、批判性思考能力、運用資訊科技能力、運算能力、解決問題能力、自我管理能力及研習能力，並滲透內外課程。
學生支援
全校參與照顧學生個別差異 : 學習差異；課堂研究，照顧不同能力學生、個別學習計劃、課後功課支援班、小班同級分組教學、教師助理入班、校本言語治療服務、發展資優教育。
課程剪裁及調適措施 : 學教及評估調適、分層工作紙，功課及評估調適。
家校合作及校風
家校合作 : 家校合作緊密，校長茶座，家訪，熱辣辣速遞，家教會，家長義工，家長網絡、家長課程、家長教育等活動，家校共同栽培孩子，創出燦爛人生。
校風 : 教育局外評報告稱讚學生淳樸，學校關注培養學生良好品格，提供充滿尊重、體諒、和諧的共融校園，並重視家校合作，積極支援不同能力學生，且成效良佳，獲校外專業肯定。且貫徹「有教無類」精神，能以「每一個孩子都有成功的機會」信念貫徹於學校政策內，成功建立關愛共融的校園，並於2015年榮獲卓越關愛校園「牽引共融」主題大獎 — 全港小學組得主。2016年更再次獲得卓越關愛校園「締造希望」主題優異獎。
學校發展
學校發展計劃 : 1. 提昇學生自我管理的能力，成為一個自律守規的學生。
2. 建立優質課堂教學，發展學生自主學習能力。
3. 凝聚家長團隊，共建和諧校園。
教師專業培訓及發展 : 持續提昇教師專業，強化學教表現，分享、交流成果。
學費減免 : 不適用。
