班級教學模式
本校一至六年級均實施小班教學，提高教學效能，啟發學生學習動機。學校推行新資助模式，並於三至六年級舉辦輔導教學，照顧不同能力學生之需要。
班級結構備註
在16/17年度，二至三及五至六年級各兩班施行普教中，四年級有三班施行普教中，一至六年級設校本科學與科技課，及電腦編碼課程。

健康校園生活 : 著重學生身心發展，參與運動獎勵計劃、健康校園及關愛校園計劃、及連續三年獲至『營』學校認證計劃嘉許，又設運動區，培養學生每天晨跑及運動習慣。
備註 : 著重學生身心得到均衡發展，並養成良好健康習慣，故採取一系列跨學科活動及課程，提升學生體適能。推行之措施包括：1. 充足運動課時。2. 主題運動區。3. 個別學童體重管理。4. 多元化簡易運動課程（如劍擊，棒球，創作舞等）。
全方位學習活動
課外活動在課後及星期六進行，活動多樣化，包括：舞蹈、童軍、劍擊、公益少年團、朗誦訓練、 圍棋、棒球隊、機械人創作、科學學會、水墨畫、跆拳道、足球、武術、舞龍隊、田徑、游泳、乒乓球、戲劇、美術學會、反斗IT兵團、劍橋英語、敲擊樂、合唱團、校園電視台等。
辦學宗旨
本校秉承香港青少年德育勵進會辦學的精神，以德育為首，為學生提供優質全人教育，使學生在「德、智、體、群、美」五育各方面得到均衡發展，並致力培養學生積極的人生觀，讓他們發展潛能，成為自立自主的終身學習者。
學校特色
學校管理
學校管理架構 : 設立學校管理與組織、學與教、校風及學生支援、學生表現等工作小組，與各持份者共同訂定學校發展計劃，跟進檢討及改善。
法團校董會/學校管理委員會/校董會 : 本校已於2011年成立法團校董會。
環保政策 : 成立可持續發展教育組，檢視校內環境政策，並透過環境建設，如自然教育廊、環保發電單車、課室及樓層回收箱等，配合課程及各不同類型的活動推行可持續環境教育。本校亦設學生環保大使，培養學生環保意識，鼓勵學生身體力行，把訊息帶到社區。
學校關注事項 : 以德育為首，致力為學生提供多元化和優質的全人教育。
教學規劃
學習和教學策略 : 1. 著重德育培養，發展學生多元智能。
2. 推行高階思維教學，優化思維訓練。
3. 透過系統的探究及體驗，培養自主學習能力。
4. 配合學與教新趨勢，利用電子學習提升學與教效能。
關鍵項目的發展 : 1. 推展價值教育，培養整全的價值判斷能力，讓學生建構自立、自主、自力的學習模式。
2. 深化賞識教育，透過整體氣氛提高賞識的元素，培育學生正向價值觀。
3. 藉建立校本教學電子平台及課程調適，促進學生自主學習及照顧學習差異。
共通能力的培養 : 透過合作學習、專題研習、小組協作、多元智能課及藝術活動，培養批判思考、創造力等共通能力。
學生支援
全校參與照顧學生個別差異 : 本校訂立整體照顧有特殊教育需要學生的政策，有系統地規劃和安排教師接受有關培訓課程。學校會按學生的需要，設計個別學習計劃，因應能力差異調適課程，採用多元化的教學方法。為進一步讓家長了解學校的支援措施，學校會透過恆常溝通機制，積極加強與家長的溝通和合作，例如邀請家長出席個案會議等。
課程剪裁及調適措施 : 課室內加設支援助理，協助有學習需要的學生。全校老師參與支援在學習上有需要之學生，另設分能力課業配合學習需要。
家校合作及校風
家校合作 : 家教會舉辦積極參與校政諮詢及籌辦活動，家長義工為教與學創造空間，舉辦家長教育工作坊等，促進家校合作。
校風 : 校風淳樸，著重品德培育，透過推行德育課、全方位輔導計劃，提升學生自尊感及建立正確價值觀。以學生為本，舉辦具預防性、發展性及補救性活動，輔以社區資源，為學生成長提供全面支援。
學校發展
學校發展計劃 : 重點關注事項：
1. 推展自主學習目標：提升學生自我管理能力和動機，提供學生自主學習的方法。
2. 提升教師專業發展：提升教師領導科組的能力，藉著境外交流，提高發展課研的能力。
3. 深化賞識教育：提高整體氛圍的賞識元素，並於學習層面正面鼓勵和讚揚孩子。
教師專業培訓及發展 : 校長和教師經常參與與校外及校內的專業交流，分享教研成果，達致專業自主。
學費減免 : 不適用。
其他 : 本校去年中一派位成績優良，獲派第一至第三志願人數達97人（91%），獲派直資中學人數達7人（6.6%）。至於課外活動方面成績也相當驕人，取獲冠軍共34項，亞軍40項，合共成績達245項。本校更曾取得行政長官卓越教學獎（體育學習領域）殊榮，致力以運動養德，透過運動禮儀，學習尊重別人和團體合作等。
學校地圖

  • 本校樓高七層，設有25個課室、1個操場、1個籃球場，多個特別室及其他設施，例如：藝術花園、攀石牆、棋品坊、自然教育廊、寵物角、心靈守護場、科學探知館、閱讀花園、童夢間、香草園、校園電視台及詩人茶座等。每一個校園的設置都包含不同的學習元素，讓學生在跨領域、跨學科和跨感官的學習上增益。

  • 本校著重學生身心得到均衡發展，並養成良好健康習慣，故採取一系列跨學科活動及課程，提升學生體適能，達致運動養德、棒球育群、攀石啟智、龍舞創美的課程目標。

  • 藉建立校本教學電子平台及課程調適，促進學生自主學習及照顧學習差異。透過移動學習0000、電子教室、自學電子平台、電子書等學與教的模式，讓學生掌握資訊科技的能力，優化個人學習的歷程。
