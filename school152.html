辦學宗旨
本校秉承校訓「非以役人，乃役於人」的精神；致力提供學生優質及愉快的學習環境，提供均衡教育，發展學生個人潛能；培養學生日後成為有獨立思考能力的人，建立終身學習的態度；提升家長對學校的歸屬感，達致家校合作；並提升教師的專業水平，建立優秀的教師團隊。
學校特色
學校管理
學校管理架構 : 法團校董會以下由校長及主任組成學校發展組為全校發展作策略性設計：設不同行政小組推行學校發展計劃及執行各項工作。
法團校董會/學校管理委員會/校董會 : 聖公會主恩小學法團校董會
環保政策 : 1. 鼓勵循環再用及再造：
a. 設回收箱
b. 舊衣回收
c. 舊校服回收及義賣
2. 減少廢物或浪費：
a. 舉行「清淡一餐」等活動，提醒學生珍惜食物。
b. 參加「地球一小時」等活動，加強學生節省能源的意識。
學校關注事項 : 1. 提升教師的專業知識以優化課堂教學。
2. 提升學生學習能力，建立自信，發揮潛能。
3. 讓學生珍惜生命。
教學規劃
學習和教學策略 : 1. 優化閱讀教學，提升閱讀能力。
2. 設計多元化的預習及延伸學習活動，發展自主學習。
3. 推動STEM教育，加強實作學習及電子學習，以促進學生綜合和應用知識與技能及解難的能力。
關鍵項目的發展 : 1. 透過宗教教育、生命教育及健康教育，培育學生正面的價值觀和態度。
2. 設早讀及圖書課，推行中、英文閱讀計劃、抱抱爸媽（家長講故事）及Morning Reading（外籍英語老師講故事）活動。
3. 各級推行專題研習。
4. 運用資訊科技進行學習
4.1 各科設網上學習平台，延展課堂學習。
4.2 校本電腦課程，協助學生掌握提升學習效果的資訊科技技能及應用的正確態度。
4.3 推行電子學習，學生在課堂上使用平板電腦，以提升學習動機及加強互動和回饋。
共通能力的培養 : 透過各科策劃的學習活動，培養相關的共通能力，拓展學生的潛能。
學生支援
全校參與照顧學生個別差異 : 1. 全方位照顧學生個別差異：小老師計劃、大哥哥大姐姐計劃、考試關懷服務、駐校言語治療師、校本教育心理服務、課前輔導班、中、英抽離小組。
2. 拔尖培訓：校隊訓練、英語學會、藝術新領域等。
課程剪裁及調適措施 : 為有特殊學習需要的學生作功課調適。多元化評估，照顧學生不同的能力。
家校合作及校風
家校合作 : 1. 家長教育：邀請專家舉行工作坊及聯校家長講座。
2. 聯誼活動：小一親子旅行、親子運動會、家教會親子旅行。
3. 家長義工：設家長義工隊、家長講故事之抱抱爸媽、參與親子賣旗、校服回收及義賣等活動。
4. 透過電話、面談、每月「與校長斟一斟」、分級家長會及家長日等途徑，促進家校合作。
校風 : 學生品性純良，有禮受教，樂於服務，關愛氣氛充滿校園。設「卓越服務之星」計劃、班本服務崗位計劃、風紀、車長及小義工等服務生制度，提倡服務精神。定期探訪長者及到幼稚園服務，共建關愛文化。
學校發展
學校發展計劃 : 1. 建立穩健的專業領導。
2. 照顧學生的多樣化，提升學與教效能。
3. 讓愛在校園生根立基。
教師專業培訓及發展 : 教師發展日、共同備課及同儕觀課，聯校專業觀摩與交流等。
學費減免 : 不適用。
其他 : 1. 中文科持續推行「普教中」。
2. 四至六年級學生參與境外學習交流。
