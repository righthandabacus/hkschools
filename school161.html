全方位學習活動
週五下午全校進行學生綜合活動課，週六為學生安排不同體藝興趣班，包括繪畫、舞蹈、球類等。平日放學後，由教師挑選學生進行十多項專項訓練，包括英語話劇、劍橋英語、奧數、創意寫作、獅藝等，使學生有不同發展機會。
辦學宗旨
本校肩負起百年樹人的使命，為學生提供優質的小學教育，並期望通過教育培養出愛國愛港、敢於承擔的高尚情操的孩子，為未來作出貢獻。
學校特色
學校管理
學校管理架構 : 學校行政架構分六組，分別處理學校事務及總務、課程發展、學校支援、學生訓輔、學生活動及資訊科技教育。校長連同各科組組長組成行政小組，定期進行會議，跟進學校各範疇的發展。眾教師參與一至兩組工作，協助不同科組的運作，加強校政的透明度。
法團校董會/學校管理委員會/校董會 : 成員包括校監及辦學團體校董8人，校長，教師校董2人，家長校董2人， 校友校董1人及獨立人士校董1人，共16人。法團校董會每兩個月召開會議一次，跟進學校運作並積極提供學校發展的意見。
環保政策 : 學校每年為各級安排環保專題學習活動，並重視校園生活的實踐。目前校內採用T5光管及一級能源效益的冷氣機，訂定節能指引，以節約能源。節約用水，並鼓勵學生使用手帕，減少用紙巾。設備更替重環保回收及再用，如紙張雙面使用，採用環保墨水、再造紙等。
學校關注事項 : 締造關愛校園；齊建共融社區。
教學規劃
學習和教學策略 : 1. 小班教學：鼓勵小組合作學習，加強同儕互賴互助。
2. 資訊科技教育：運用資訊科技整合教材，輔助學與教。
3. 全方位學習：參觀、校外比賽、戶外學習、社會服務等。
4. 體藝發展：畫班、舞蹈、話劇、合唱團、小提琴班、田徑、球類等。
5. 自主學習：網上學習、閱讀計劃、積極進取計劃等。
6. 以長帶幼：小老師、伴讀計劃、領袖培訓與服務等。
關鍵項目的發展 : 1. 德育及公民教育：結合健康成長計劃及訓輔活動，培養學生正面的價值觀、行為和態度，於日常生活中實踐。
2. 從閱讀中學習：校園廣置圖書，為各班提供報章，配合閱讀計劃與分享活動，培養學生閱讀習慣，從閱讀中吸收課外新知，時事動態。
3. 專題研習：全校推行，輔以科學探究活動，發展學生的共通能力。
4. 資訊科技互動學習：利用網上學習資源輔助教學，同時引領同學利用多媒體學習，自主學習。
共通能力的培養 : 透過課堂協作及互動學習，各科多元化的學習活動，讓學生從具體的學習經歷中逐步發展溝通、協作、解難、創意、批判思維、研習等九大共通能力。
學生支援
全校參與照顧學生個別差異 : 1. 課程調適：按學生能力增減課業的廣度及深度；加強輔導小組進一步支援成績稍遜的學生。 2. 合作學習：不同能力學生共同完成學習任務，互賴互助。3. 朋輩支援：以長帶幼，以強助弱，發揮同儕互助精神。4. 功課輔導：引入校外支援，於課後支援學生功課上困難。
課程剪裁及調適措施 : 拔尖：課業中預留挑戰題目或個人發揮空間，鼓勵追求卓越，包括推薦參加資優課程。補底：按學生需要調適功課、默書乃至測驗卷的範圍及難度。如有需要，安排編入加強輔導小組。
家校合作及校風
家校合作 : 學校提供中英文學校資訊，如通告、校訊等，輔以日常直接對話，於面談提供語言翻譯等措施，確保能與不同國籍家長保持緊密聯繫。學校常舉辦家長講座、工作坊等進行家長教育。家教會不時舉行活動，如親子旅行等，以增進家校情誼和合作，為下一代的教育而努力。
校風 : 校內不同國籍的學生形成一個共同學習的社群。校園尊重及接納不同文化，包容人與人之間的差異，同時提供支援讓學生健康成長。學生能知禮守規，師生關係良好。各級學生一同活動，多民族學生能融洽相處。由2010年開始，連續七年獲頒關愛校園獎，且在2011年榮獲年度卓越關愛校園之「最積極推動多元文化共融」主題大獎及在2015年獲卓越關愛校園之「牽引共融」優異獎。
學校發展
學校發展計劃 : 1. 優化多元化校本課程，加強跨科課程聯繫：透過資訊科技教學支援，配合促進學習評估策略，優化各主要科目課程的發展與統整；2. 加強照顧學習差異，支援有特殊需要學生：包括拔尖補底、非華語學童學習及促進多民族共融協作的發展；3. 深化教師專業發展，提升學與教的成效：深化教師對小班教學、課程設計與照顧個別差異所需要專業知識與技能。
教師專業培訓及發展 : 每年進行科本及校本教師發展活動，亦鼓勵教師按個人發展需要，參加校外進修課程，讓校內教師專業發展能同時兼顧學校發展與教師個人專業發展的需要。發展項目包括：1. 小班教學策略。2. 照顧個別差異。
學費減免 : 不適用。
其他 : 校本課程及學生支援同時關顧本地生及非華語學童的需要，讓多民族學童能取長補短，共融學習及成長。學校結合不同資源設立一系列獎學/助學計劃，讓不同潛能學生均有發揮及獲肯定的機會。
