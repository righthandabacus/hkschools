提倡「一生一體藝」。提供以下課外及聯課活動：田徑、球類、游泳、戲劇、手鐘、合唱等各類音樂及視藝活動 、制服團體、基督小先鋒 、英語話劇 、舞蹈、奧數、朗誦、機械人製作、趣味數學、粵劇等，以發展其體藝才能。
辦學宗旨
本校秉承會祖鮑思高神父的教育理念，以校訓：誠、信、儉、樸，培養學生良好的品德，充分發揮個人潛能，迎接未來的新挑戰，務求達致「全人發展」的目的。
學校特色
學校管理
學校管理架構 : 根據學校表現指標，設四個管理範疇：管理與組織、學與教、校風及學生支援及學生表現。各範疇均設行政手冊及指引，以策劃、推行及評估學校的各項工作。
法團校董會/學校管理委員會/校董會 : 設有法團校董會，成員包括慈幼會神父、修士及合作者。
環保政策 : 舉辦「一人一花」、有機耕種等環保活動。
學校關注事項 : 1. 培養學生自學態度及能力。
2. 培育學生良好品格及自律精神。
3. 營造宗教氛圍，培養學生向天、向人的感恩心。
教學規劃
學習和教學策略 : 1. 全校進行小班教學。
2. 小二至六每級兩班以普通話教中文。
3. 小五、六設通識導修課。
4. 初小設中文科「喜閱寫意」讀寫支援計畫。
5. 初小設英文科「PLP-R/W」計畫。
6. 高小延續發展英文科校本閱讀及寫作計畫。
7. 全面推行電子學習及貞潔文化。
8. 運用合作學習策略教學。
9. 全校推行STEM專題研習課程。
10. 小一至六全方位學習活動。
關鍵項目的發展 : 閱讀獎勵計畫、圖書館課、英文伙伴閱讀計畫、網上學習平台及時事討論區、校園電視台、3D列印及立體繪畫課程、專題研習、德育及公民教育課。
共通能力的培養 : 學校透過各學科，培養學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 陽光學長；言語大使；加強輔導教學計畫；讀寫障礙輔導；校本支援計畫；資優生訓練計畫；中、英、數加油站；駐校社工、駐校心理學家及言語治療師。
課程剪裁及調適措施 : 為有特殊教育需要學生提供評估調適（包括試場、時間及作答的特別安排）。
家校合作及校風
家校合作 : 透過家長證書課程、親子旅行、課餘活動班及家長義工組等活動，加強家校合作。
校風 : 透過一人一服務計畫、小一適應課程、成長課、均衡教育獎勵計畫、陽光學長、歷奇營、「成長的天空」計畫、境外省內文化交流等活動。
連續四年榮獲「關愛校園」榮譽學校殊榮，為學生建立關愛校園文化。
學校發展
學校發展計劃 : 1. 培養學生自學態度及能力。
2. 培育學生良好品格及自律精神。
3. 營造宗教氛圍，培養學生向天、向人的感恩心。
4. 深化均衡教育發展計畫。
5. 推行「小班教學」，各級實踐「合作學習」教學模式。
6. 推行「融合教育」。
教師專業培訓及發展 : 本校集中於教師學科的專業發展及教學策略的提昇；此外，「電子學習」及「小班教學」技巧亦為本校老師培訓的重點。
學費減免 : 不適用。
其他 : 設有和富慈善基金李吳劍鳴獎學金、謝家賢神父獎學金、譚鳳珍獎學金及家長教師會獎學金。
