3. 鼓勵學生以水果為小息小食。
全方位學習活動
宗教及制服團隊：團契、公益少年團、小女童軍、童軍； 體育：籃球、乒乓球、羽毛球、北派少林拳、跆拳道、游泳、滾軸溜冰； 語言：普通話話劇、英語戲劇、劍橋英語班、閱讀組； 音樂及藝術：歌詠團、樂器班、美術精英組、舞蹈； 其他：校園電視台等等。
辦學宗旨
本著基督愛的精神，配合靈活的教學法，提供優質教育，讓孩子在愛中成長及在愉快的環境中學習，培養正確的人生觀。使孩子由明白愛、感受愛，進而付出愛，達至校訓「能仁能勇、愛主愛人」的理想。
學校特色
學校管理
學校管理架構 : 校董會制定政策及發展方針，校監監察工作的推行。校長執行校董會決策，設不同委員會推行各種工作。
法團校董會/學校管理委員會/校董會 : 聖三一堂小學校董會。
環保政策 : 1. 雙面用紙、設置光碟回收箱、參與廢紙回收計劃。
2. 設太陽能發電，廚餘機及動能單車。
3. 使用可再用餐具。
學校關注事項 : 用愛心培育學生；
用智慧啟迪學生；
用人格引領學生。
教學規劃
學習和教學策略 : 1. 週五下午為多元智能課，擴闊學生的生活體驗。
2. 透過生活體驗營，鍛鍊學生的獨立能力。到國內外小學交流，觀課及體驗當地學習情形。
3. 透過遠程教學，促進交流協作。
關鍵項目的發展 : 1. 中英文網上閱讀計劃及校內閱讀計劃。小一至二設圖書堂。
2. 學生可在多媒體室上網學習，製作圖像及報告。
3. 每年兩次專題研習。
4. 藉週會、早會、成長課、各種輔導小組及講座推行公民教育。
共通能力的培養 : 1. 藉着小組討論及課堂活動，學生學會表達自己的意見及尊重他人的見解。
2. 透過專題研習學懂搜尋資料，啟發學生對學習的興趣。
學生支援
全校參與照顧學生個別差異 : 1. 為學生作課後個別功課輔導及設立小組輔導。
2. 升中輔導。
課程剪裁及調適措施 : 1. 為有特殊需要學生的功課，作出調適。
2. 兩科主科不及格，需要留班重讀，以便打好基礎。
家校合作及校風
家校合作 : 1. 家長可於早上8:25前進入校園，與子女一起小息，與老師分享面談。
2. 每星期舉行家長學堂，加強家長之間的支持。
3. 舉行家長講座，提供教養子女的技巧及方法。
4. 邀請家長義工參與學校工作。
校風 : 1. 小一新生輔導。
2. 早讀課。
3. 校本計劃，鼓勵學生積極上進。
4. 融洽師生關係。
學校發展
學校發展計劃 : 1. 加強英語教育，在英語課程中加入戲劇。
2. 推行綜合藝術教育課程。
3. 發展遠程教室及校園電視台。
4. 推動創意教育，訓練學生利用資訊科技學習的能力。
教師專業培訓及發展 : 1. 舉行教師培訓，教學觀摩及交流。
2. 推行教師自評及自我完善機制。
學費減免 : 不適用。
