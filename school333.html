本校開設一至六年級各四班，採分組式活動教學。每課節均有兩名教師；另每節英文課其中一名教師為外籍教師，加強英語學習。

分班安排 : 按學業成績平均分配於四班中。

全方位學習活動
本校因應學生的需要及潛能發展，開設不同類型的興趣小組，如中英文朗誦、戲劇組、合唱團、視藝、球類活動、游泳、舞蹈、棋藝、制服隊伍等。
辦學宗旨
本校重視培育學生的德育及全人發展。透過多元化的教學模式，培養學生有高尚的情操及發展各方面的潛能，使他們既會獨立思考、又善於與人溝通；既願意努力學習知識，更富於創造與革新的精神。
學校特色
學校管理
學校管理架構 : 學校為保良局屬校，由法團校董會管理。
法團校董會/學校管理委員會/校董會 : 法團校董會成員包括校監、校長、家長、教師、獨立人士及辦學團體代表。
環保政策 : 本校致力推行環保教育，減少不必要的能源浪費，同時在作出採購決定時，會考慮環保因素，採購對環境影響較少的貨物。此外，本校已全面採用節能光管。
學校關注事項 : 建構關愛校園；關注特殊學習需要學生；提升協作教學效能。
教學規劃
學習和教學策略 : 我們為學生設計廣闊、平衡、綜合的課程，並靈活運用課時，為他們提供有趣的、具挑戰性的學習經歷。本校利用校園以外的資源教學，並以跨學科專題研習及聯課活動配合學習，發展學生潛能及創意思維能力。
關鍵項目的發展 : 學生透過閱讀來改善資料搜集及探究的能力。教師利用生活化教材、小組討論、話劇表演和專題研習等活動，幫助學生掌握與閱讀理解有關的高層次認知策略。
共通能力的培養 : 本校課程主要透過活動教學模式讓學生主動學習。教師針對學生的學習需要設計多元化教學活動及課業，藉以培養學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 我們深信每個學生在認知及情意發展，興趣及潛能上均有不同，我們會積極了解學生不同的需要及潛能，並設法協助他們學得更好。
課程剪裁及調適措施 : 學校會因應學生的程度及需要，制定校本課程及裁剪課程。
家校合作及校風
家校合作 : 學校的家校合作政策是採取全校參與形式。學校因應不同的需要通過發放全校通告、聚會或與家教會合辦活動等，與家長保持緊密聯繫，希望藉此與家長建立良好的伙伴關係，以便更好的教育我們的孩子以及得到他們的有力支持。
校風 : 朋輩支援，關愛共融。
學校發展
學校發展計劃 : 學校強調「兩文三語」的教育，致力為學生奠定良好的語文基礎：在正規外，我們亦將組織活動，致力營造一個真實的語言環境，讓學生在一個以英文及普通話作為主要溝通工具的地方進行學習。
教師專業培訓及發展 : 學校積極提升教師教學效能，進行同儕觀課及協作教學。
學費減免 : 本校設有學費全免或減免1/2學費。有需要的學生(包括來自綜援家庭的學生及領取學生資助辦事處財政資助的學生)可提出申請學費減免。
- 新生可於開學後的6個星期內遞交申請，其申請結果會在6個星期內獲得通知。
- 本校學費減免計劃詳情包括家庭入息限額，請瀏覽網上版的超連結。
其他 : 傑出學生獎、每級前三名獎學金、傑出學生獎勵計劃、學習之旅。
