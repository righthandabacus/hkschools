課外活動多樣化，涵蓋學術、體育、藝術和制服團隊。 制服團隊包括：基督少年軍、基督女少年軍、幼童軍、紅十字會少年團等。 校隊包括：游泳、籃球、足球、排球、歌詠、節奏樂、牧童笛、樂團、朗誦等。 興趣班包括：校園電視台、珠心算、樂器、舞蹈、書法、話劇等。
辦學宗旨
培養學生具備優良品德、高雅情操、豐富學識、健康的人生態度、正確的價值觀，以及強烈的社會意識。
學校特色
學校管理
學校管理架構 : 本校乃中華基督教會香港區會直屬小學之一。
法團校董會/學校管理委員會/校董會 : 於2006年成立法團校董會。
環保政策 : 培育學生建立環保生活方式。
學校關注事項 : 優化課堂教學，提昇教學素質；重視學生的學行表現；建立優質團隊。
教學規劃
學習和教學策略 : 重視均衡教育、全人發展；照顧學生個別差異；推行小班教學；在專題研習課教授專題研習技巧及實踐；透過多元智能課、聯課活動及境外交流，擴闊學生的學習經歷。
關鍵項目的發展 : 1. 透過專題研習課，發展學生研習、探究和自學能力，培養學生的共通能力。
2. 閱讀計劃：推行閱讀獎勵計劃；並設有早讀、圖書課、中英文廣泛閱讀課，培養學生閱讀興趣和教授閱讀策略。
3. 資訊科技教學：推行電子學習計劃，教導學生運用資訊科技進行互動及網上學習，培養學生資訊素養。
4. 透過德育及公民教育、宗教課、週會及成長課培養學生正確的價值觀，建立關愛文化。
共通能力的培養 : 透過課堂上的協作活動和跨學科的專題研習課，加強培養學生之溝通、協作、創造、批判性思考等共通能力。
學生支援
全校參與照顧學生個別差異 : 開設校本資優培育課程和全班式資優課程、輔導教學及加強輔導教學；採用全校參與模式支援有特殊學習需要學生；實施課程及評估調適。
課程剪裁及調適措施 : 推行跨學科專題研習課。
家校合作及校風
家校合作 : 1. 重視與家長聯繫，已成立家長教師會，所有新入學學生的家長均自動成為家教會會員。
2. 家長義工協助學校各方面的工作。
校風 : 採用全校參與模式，推行各項支援服務及跨組別協作計劃，為學生提供全面的支援服務。教師協力培養學生正面的價值觀，營造學生對學校的歸屬感，培養和諧有序的校園氣氛。 舉辦小一適應家長講座及朋輩輔導；為小六學生及家長安排升中選校指導及適應講座；透過聯課活動啟發學生潛能。本校已連續十一年榮獲「關愛校園」榮譽。
學校發展
學校發展計劃 : 1. 行政教學電子化
2. 實踐健康校園
3. 擴闊學生學習經歷
教師專業培訓及發展 : 邀請專家到校作專業培訓，加強校內外的專業交流，提升教與學效能。
學費減免 : 不適用。
其他 : 曾獲四年香港學校戲劇節小學廣東話組— 評判推介演出獎（組別冠軍）之殊榮。
全港小學創意數學遊戲設計比賽2016 — 冠軍。
