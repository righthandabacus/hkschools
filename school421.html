全方位學習活動
本校共有30多個課外活動組別，包括學術、藝術、體育、資訊科技及服務方面，校隊包括中樂團、管弦樂團、合唱團、集誦隊、球隊等。此外，開設多項課後興趣班，包括舞蹈、跆拳道等；另外，全學年安排各級全方位學習參觀及教育性參觀各一次，豐富學習生活及經歷。
辦學宗旨
致力讓學生過著健康、愉快、具學習效能的學校生活，充份發揮個人潛能，追求卓越，回饋社會，邁向豐盛人生。
學校特色
學校管理
學校管理架構 : 本校直隸教育局，由學校管理委員會協助學校訂立政策。老師由校長、副校長和主任領導。
法團校董會/學校管理委員會/校董會 : 學校管理委員會。
環保政策 : 環保大使協助推行活動，宣揚環保訊息，提高同學的環保意識和觸覺。主題為「源頭減廢」，活動包括舊衣回收、月餅罐回收、利用廚餘製造酵素、參觀廚房廚餘回收等。
學校關注事項 : 1. 培養自主學習，提升學習效能。
2. 強化自我管理，培養自律的品格。
3. 促進教師專業，提升教學效能。
教學規劃
學習和教學策略 : 以學生為本，課堂中鼓勵學生提問，透過師生及生生互動，培養學生樂於思考的學習態度和能力；另外，透過預習、數學科自擬應用題、專題研習等，培養學生自學能力和態度。
關鍵項目的發展 : 中央圖書館提供優良的環境和資源，推動閱讀風氣；各級設電腦認知課，鼓勵學生應用網上資源；在四至六年級推行專題研習，並由常識科作為平台，加入STEM學習元素（科學、科技、工程、數學）促進探索精神及學習能力；透過講座、班主任課及生活教育課，灌輸正面人生觀及價值觀。
共通能力的培養 : 透過正規課堂及課堂外多元化的學習活動，重點培養學生創意、溝通及思考能力。
學生支援
全校參與照顧學生個別差異 : 透過及早識別計劃幫助有特殊學習需要學生。與家長定期舉行「個別學習計劃」小組會議。為學生提供駐校言語治療服務、課後輔導、抽離式課程、協作教學。拔尖方面，為學生提供學科提升班及增潤課程。
課程剪裁及調適措施 : 1. 根據學生學習需要，進行課程增潤及剪裁。
2. 為有學習需要學生進行課業及測考調適。
家校合作及校風
家校合作 : 家長教育計劃、義工家長隊、親子義工隊，親子興趣班、家長興趣班、親子大旅行、故事爸媽。
校風 : 校風淳樸，學生努力學習，並致力發展多元化學習領域，發揮潛能，追求卓越。參與長幼共融活動營造了關愛長者的精神和文化。
學校發展
學校發展計劃 : 1. 培養自主學習，提升學習效能。
2. 強化自我管理，培養自律的品格。
3. 促進教師專業，提升教學效能。
教師專業培訓及發展 : 1. 參與教育局「地區支援網絡計劃」以發展校本數學課程。
2. 透過共同備課、觀課、教研等教學分享活動，提升教師專業。
學費減免 : 不適用。
其他 : 家長教師會獎學金、舊生會獎學金、香港學校音樂及朗誦節獎項、校際游泳校際排球、籃球、乒乓球比賽獎項、公益少年團2016年傑出團員交流團灣仔區代表。
