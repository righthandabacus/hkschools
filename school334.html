辦學宗旨
我們願以基督愛心為動力、以人為本的信念、積極進取的態度，提供優質教育，啟發學生潛能，分享整全福音；培育學生成為良好公民，回饋社會，貢獻國家。
學校特色
學校管理
學校管理架構 : 法團校董會領導學校的發展方向，由校長帶領行政團隊推行相關工作。
法團校董會/學校管理委員會/校董會 : 中華基督教會何福堂小學法團校董會於二零零六年八月二十二日成立，參與學校發展的方向。
環保政策 : 提倡使用環保酵素，設立有機農莊，節能減廢，循環再用。
學校關注事項 : 1. 優化組織效能，促進各科組持續發展。
2. 優化教學策略，提升學與教效能。
3. 推展生命教育，培養正面的價值觀及積極的人生態度。
教學規劃
學習和教學策略 : 我們著重通過師生與生生的互動過程，以探究、研習、同儕互評、協作、討論、表演、自學等不同模式來幫助學生建構知識。每節課最後的十分鐘鞏固學生所學，所以又稱為「黃金十分鐘」。在二至四年級推行「電子學習試驗計劃」，以加強學生的學習興趣，培養自學態度，並有系統地推行STEM。我們設置資源室如博聞室、語言室、圖書館等進行啟發式教學；我們亦帶學生走出校園，參觀博物館、科學館、立法會、公共事業機構等，讓學生體驗學習的樂趣。
關鍵項目的發展 : 1. 從閱讀中學習：設置中英文早讀課、英文寫作教學計畫、每週一寫、網上閱讀、親子伴讀、故事媽媽、假期閱讀、出版文集、課室字典、參觀公共圖書館等。
2. 德育與公民教育：藉著週會、早會、成長課、參觀國民教育中心、建立制服團隊、領袖生訓練、有品校園計劃等活動，讓學生的身心健康成長。
3. 運用資訊科技學習：設置訊息科技科使用資訊科技教學、開發網上學習資源、電子學習計劃以增強學習的興趣。
4. 專題研習：進行跨科協作專題研習，注重研習過程及匯報。
5. 跨課程閱讀：進行跨課程閱讀促進。
共通能力的培養 : 本校關注學生共通能力的培養，尤其注重溝通、創造與明辨是非能力的發展。我們採用融入的自主學習模式，與日常的學習作有機的結合。例如：就一些社會的熱門議題，讓學生合作研習、探究，從而議論與報告，使之成為會獨立思考、善於溝通、有創新思維的良好國民。
學生支援
全校參與照顧學生個別差異 : 電腦輔助學習、學習小組、學長計劃及課後功輔班，並設有駐校社工、言語治療及教育心理學家服務。
課程剪裁及調適措施 : 本校的課程寬廣而均衡，致力促進學生的全面發展。我們根據學生的學業水平與能向編班，積極照顧學習差異。對於能力稍遜的學生，教師除課後作個別的支援外，還設置鞏固班，分科分級作適切的輔導。對於能力高的學生，則提供不同的資優培訓，以培育及提升學生的潛質。
家校合作及校風
家校合作 : 家長大力支持學校發展，家長教師會設有獎學金，並協助學童午膳服務及家長教育。
校風 : 校風良好，學生普遍自信，樂於與人合作，朋輩相處和睦，師生彼此關懷。家長積極參與及支持學校發展，共建和諧校園文化。我校是一所基督教學校，教會值堂於本校，透過家校堂合作，基督化的品德和生命教育對學生的全人發展收事半功倍之效。
學校發展
學校發展計劃 : 歷年來，升中成績理想，家長對派位結果十分滿意。
學校推行小班教學多年，重視孩子品德發展，透過持續優化學與教的措施，不斷提升學與教的表現。
教師專業培訓及發展 : 教師的持續專業發展是配合學生成長，課程發展，教師個人發展及學校發展需要而制定。我們認為教學理論、技巧與實踐是環環緊扣，我們透過課堂學習研究、分享「學科教學知識」，豐富學校的「智慧資產」，聚焦於「承傳」、「轉換」及「創新」，成為教學專業中心的「實踐社群」。
學費減免 : 不適用。
其他 : 家長學堂（十月至五月）。
