全方位學習活動
星期三下午，全校進行圖書課、德育課或各類興趣小組活動，包括女童軍、合唱團、球類、游泳、田徑、手工藝、圍棋、中國文化基礎學習課程（包括粵曲及書法學習） 等。星期六下午，設有幼童軍活動。每週還有多元化的自費體藝培訓班及資優培訓班逾一百班。
辦學宗旨
我們致力創造優良的學習環境，以便培育學生德、智、體、群、美各方面均衡發展。協助孩子們充份發展潛能；培養他們積極進取的精神及關心社會的態度。
學校特色
學校管理
學校管理架構 : 教學人員：包括校長、副校長、主任、教師（包括外籍老師）及圖書館主任共51人。全部經教師專業訓練，且大多數已持有教育學士學位。
法團校董會/學校管理委員會/校董會 : 教育局首席助理祕書長（基礎建設及研究支援）擔任主席，其他委員包括校長、教師代表2人、家長代表2人、校友會代表1人、社會人士代表1人。
環保政策 : 除了教導學生關注環保之外，學校亦盡量節省使用空調及紙張。
學校關注事項 : （2015-2018）的學校重點關注事項：
1. 引發教學正能量，共建學習型社群。
2. 深化優質教育，啟迪多元潛能。
3. 建立正面價值觀，邁向積極人生。
教學規劃
學習和教學策略 : 每年特定教育主題，各科根據教育主題擬訂學習目標和活動，教師都善於運用活動教學法，透過校園電視台製作各科節目提高學生學習興趣。學校重視學生全人成長，於每天靈活運用課堂時間，為孩子提供多元化的體藝培訓活動，協助孩子培養良好的興趣及建立自信。
關鍵項目的發展 : 1. 致力推行各項閱讀計劃。語文教育及研究常委會研究報告顯示：本校學生之語文水平遠超全港學生平均成績。
2. 全校課室及特別室均具備科技化設施，曾獲教育局委任為「資訊科技卓越中心」，負責推廣全港學校間的資訊科技教育。
3. 本校重視學生德育之培訓，學校外評人員在學生學業、態度、行為及成就各方面均給予「優異」之評價。
共通能力的培養 : 教師透過各科校內校外活動，有效發展學生共通能力。
學生支援
全校參與照顧學生個別差異 : 1. 拔尖：實行資優普及化，設資優培訓班。
2. 支援：協作教學，課前輔導，小組功課輔導班，外購小組培訓服務。
3. 專業支援：校本教育心理服務，到校言語治療服務。
課程剪裁及調適措施 : 各科同級教師會因應需要調動和剪裁教材，提高學生學習成效。
家校合作及校風
家校合作 : 學校設有家長教師會及家長義工服務。
校風 : 本校悉心安排各項有關措施，包括公民教育及德育、訓輔、一生一體藝、資優培訓及家校合作等。學校外評人員在「校風及對學生支援」方面均給予「優異」評價。
學校發展
學校發展計劃 : 1. 透過閱讀計劃及資訊科技教育培養孩子自學能力。
2. 透過教師專業發展，課程改革及自評進一步提高教學效能。
3. 致力推行「一生一體藝」及「資優培訓」，協助孩子盡展潛能。
教師專業培訓及發展 : 每名教師每年進修時數不少於50小時。
學費減免 : 不適用。
