辦學宗旨
培養仁愛精神達致愛主愛人，掌握自學能力達致終身學習。
學校特色
學校管理
學校管理架構 : 中央校董會→法團校董會→行政組：管理與組織、學與教、校風及學生支援、學生表現。
設有家校一線通（朝六晚十學校電話熱線）。
法團校董會/學校管理委員會/校董會 : 校董會成員除了由辦學團體委任的校監、校長及校董外，更包括家長代表、舊生代表及獨立校董。
環保政策 : 校服及圖書回收、自備餐具、「地球一小時」行動等。
學校關注事項 : 人的價值。
教學規劃
學習和教學策略 : 1. 中、英、數、常推行電子學習計劃。
2. 英文科參加教育局校本支援服務計劃。
3. 中、英、數參加AQP計劃 。
4. 語境教學，加強兩文三語的能力。
5. 一至三年級推行英文識字計劃—閱讀/寫作。
6. 一至四年級教授漢語拼音，為普教中奠下基礎。
7. 透過專題研習及全方位學習活動，培養共通能力。
8. 推行合作學習，照顧個別差異。
9. 發展電子學習平台，提升自學能力。
10. 學生設有境外交流活動。
關鍵項目的發展 : 1. 閱讀：設閱讀獎勵計劃、晨光閱讀、伴讀大使；英文科推行「小學識字計劃」、設立"Reading Land"閱讀角，培養學生閱讀英文書的習慣 。
2. 專題研習：推行跨學科主題式學習活動。
3. 資訊科技：IT大使、電子學習平台、無線網絡校園、智能卡系統。
4. 德育及公民教育：透過宗教、成長課、傳媒教育及服務學習培養良好品德及公民意識。
共通能力的培養 : 透過合作學習，培養學生的溝通、創造及批判性思考等共通能力 。
學生支援
全校參與照顧學生個別差異 : 1. 分組輔導教學、課後功課輔導班。
2. 奧數課程、英語大使培訓。
3. 協作教學，照顧學生個別差異。
4. 學藝高飛獎勵計劃。
5. 新來港學童校本支援計劃。
課程剪裁及調適措施 : 因應學生能力設計不同程度的課業。
家校合作及校風
家校合作 : 1. 由家長及教師共同訂定小一學生的學習方向。
2. 出版雙月刊。
3. 家教會活動包括：週年大會暨親子旅行、新春掛飾創作比賽、健康美食製作比賽、出版會訊。
4. 組織家長義工協助學校活動。
校風 : 1. 校風淳樸，訓輔合一，校園氣氛和諧，師生關係融洽。
2. 透過「小一新生活體驗」、「跨級伴讀計劃」、「小導師計劃」及「愛心大使」等建立關愛文化。
3. 多次獲「關愛校園」榮譽獎。
4. 連續多年獲「活力校園」獎項。
學校發展
學校發展計劃 : 1. 提升學生的學習興趣、培養學生成為主動學習者。
2. 優化課堂教學，提升教學效能。
3. 認識天主教學校五大核心價值，在生活中加以實踐。
教師專業培訓及發展 : 1. 自主學習策略。
2. 電子學習及創新科技教育。
學費減免 : 不適用。
其他 : 1. 獲多項計劃∕基金資助：中西區區議會、福傳Visa卡「福傳經費」資助計劃、平和基金、兒童權利活動資助。
2. 提升學生成就感：公教報、星島日報、地區星報、兒童快報專訪、曾參與ATV "Let's Talk"及春晚、RTHK上網問功課、TVB文化新領域表演及ViuTV 學校節目。
