辦學宗旨
透過分享基督的愛和真理，讓學生在知識及個人成長上有所增長，並提供多元化的優質教育，發展學生潛能；另透過德、智、體、群、美、靈的培育，使學生成為良好的公民。學校致力提供完善的學習環境，讓學生愉快學習。
學校特色
學校管理
學校管理架構 : 本校由法團校董會管理及監察學校運作，在校長領導下，設有校務組、課程組、訓輔組、學生活動組、資訊科技組、危機小組及校園福音組。各科組及功能組互相配搭，共同管理及發展學校。
詳情請瀏覽本校網頁。
法團校董會/學校管理委員會/校董會 : 已於2009年8月31日成立。
環保政策 : 本校透過不同的環保活動如廢紙回收、舊衣物回收、參觀活動、環境保護大使計劃的等活動，培養學生環保意識，鼓勵學生綠化校園、保護環境、節約能源，實踐綠色生活。
學校關注事項 : 1. 自主學習．積極參與
2. 推行價值教育．建立正向人生
教學規劃
學習和教學策略 : 英文科Shared Book Approach；中文科設普教中班以讀帶寫；數學遊戲日；常識探究為本學習；全方位學習模式，如：全方位學習日、戶外學習等；各級中英文話劇表演；境外學習交流等。
關鍵項目的發展 : 1. 閱讀：館藏豐富，書類繁多；推行中、英文廣泛閱讀、親子閱讀獎勵計劃；「故事爸媽」、晨讀課、伴讀大使、閱讀大使等。
2. 資訊科技：運用資訊科技教學、建立資源庫及多元化網上學習平台。
3. 專題研習：各年級常識科開展專題研習，於各科課程內滲入研習元素，提升學生探究及研習能力。
4. 德育及公民教育：推行價值教育，每周進行德育及成長課，促進學生德育及情意的發展。
5. STEM教育︰設科技日及課外活動小組，培養學生科學探究的興趣、精神及技能。
共通能力的培養 : 強調高階思維培訓，培養學生自學能力；常識科建立探究為本的學習模式，以銜接中學的通識課程。
學生支援
全校參與照顧學生個別差異 : 利用小班合作學習策略提升學生探究及協助精神；設有校本資優課程、電子學習互動平台；開辦課後鞏固小組、拔尖班以照顧學生學習多樣性及推行Good K.I.D.S 好孩子獎勵計劃，提升學生學習動機。
課程剪裁及調適措施 : 課程統整，進行跨學科主題式學習；因應學生的能力及需要，部份課堂設小組教學，以提升學習效能及照顧學生學習多樣性。
家校合作及校風
家校合作 : 本校家長積極參與學校的活動，並組成龐大的義工隊伍，協助學校推行不同的活動，充分發揮家校合作的精神。為配合學校的關愛和環保主題，家教會每年均舉辦舊英文故事書、舊校服回收及轉贈服務，幫助有需要的家庭。
校風 : 本校重視學生紀律及品格的培養，以訓輔合一，全校參與的模式，推展訓輔工作。透過生活教育課程，校本主題活動、班級經營計劃、德育講座等建立學生正確的價值觀。同時透過全方位輔導服務，舉辦預防性、發展性及補救性的活動及推行不同形式的家長教育活動，以支援學生全面成長。
學校發展
學校發展計劃 : 1. 推動自主學習，促進學習效能。
2. 推行價值教育．建立正向人生。
3. 著重學生體藝的培育，促進全人發展。
教師專業培訓及發展 : 1. 優化同儕觀課文化，聚焦交流照顧學生的多樣性、自學能力。
2. 引入校外專業支援，優化校內教師的課堂策劃及教學技巧，提升教師教學效能。
3. 舉辦教師培訓工作坊、研討會及聯校教師專業發展日，提升教師專業水平。
學費減免 : 不適用。
