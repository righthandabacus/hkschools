班級教學模式
一至六年級進行小班教學。另設兩班加強輔導，以照顧學習差異。二至四年級每級設一班精英班（見備註），二至六年級每級設一班普教中班；外籍英語老師與一至三年級英文科教師進行協作教學。
班級結構備註
2015-16學年9月於二年級設精英班；由2015-16學年至2019-2020學年，精英班將由二年級逐年擴展至六年級。

多元學習評估 : 全年共有三學段，每學段設學期考試，學段期間進行中英數常四科持續性評估，以跟進學習進度，提供持續的回饋；評估形式多元化，包括紙筆評估、專題研習、口頭報告、朗讀、小組討論等。
分班安排 : 2015/16年度開始小二設精英班，將於2017/18學年擴展至小四；精英班課程具增潤元素，中文科以普通話授課。其他班別為混合能力班，在部份級別設加強輔導教學小組及非華語學生的中文教學小組。

辦學宗旨
本校秉承東華三院的辦學宗旨，提供全人教育，使學生能發揮個人潛能，具知識技能、獨立思考能力和勇於承擔的精神，日後服務社會。培育兒童有正確的價值觀和積極的人生觀，並以校訓「勤儉忠信」為進德修業的依歸，勉勵他們拓展豐盛而有意義的人生。
學校特色
學校管理
學校管理架構 : 設法團校董會。校長及副校長以下設學校發展組、學校及自評組、行政組及科組以統籌和管理全校行政、學與教、訓輔、活動及學生事務。
法團校董會/學校管理委員會/校董會 : 二零零六年二月一日成立法團校董會。成員包括辦學團體代表、校長、教師、家長、校友及獨立人士。
環保政策 : 1. 善用環保紙及進行廢紙回收，推行文件電子化。2. 遵守課室冷氣機使用守則，安裝一級能源標籤冷氣。3. 鼓勵學生自備可循環再用的午膳餐具。
學校關注事項 : 本校課程 1. 建立孩子兩文三語的能力。 2. 發展孩子的數理能力及探究精神。 3. 培養孩子閱讀和自學的習慣及能力。 4. 培養孩子成為能思考、會分析、愛溝通、會解難的學習者。
教學規劃
學習和教學策略 : 1. 推動科技創意學習，從常識科核心課程作起點，結合電腦科教學及「科學小先鋒」培訓，帶領學生參與科技發明計劃及比賽，培育其探究精神。
2. 於英文科推行教育局「小學識字計劃：閱讀及寫作PLP-R/W」及「Key Integration Programme」。
3. 藉資訊科技提供多元化的學習活動，如電子自學領域、編程、3D實物設計打印等。
4. 透過德育及公民教育課、成長課等，培養學生的正確價值觀。
關鍵項目的發展 : 1. 閱讀計劃：培養學生的閱讀興趣，活動包括早讀課、閱讀獎勵計劃、校園小記者、故事爸媽、外籍英語義工講故事等。
2. 資訊科技：提供內聯網系統及在線學習平台（遠端桌面），發展學生運用資訊科技之能力。
3. 專題研習：以常識科為起點進行專題研習，發展學生的共通能力。
4. 德育及公民教育：透過東華三院德公課程及各類講座活動，培養學生的正向價值觀，並營造和諧及關愛的的校園氛圍。
共通能力的培養 : 透過正規課堂及跨學科活動、專題研習、參觀、講座、校內外比賽等，培養學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 1. 調適部份課程、功課及默書；
2. 設課前輔導班、加強輔導班、課後功課輔導、個別學習小組；
3. 開設主題式的學科課外活動，以擴展能力較強學生的興趣及能力；
4. 開設課時內抽離式中文班及課後班，以改善非華語學生的中文能力；
5. 設教育心理學家、言語治療師及學習支援助理；提供教職員培訓。
課程剪裁及調適措施 : 1. 定期透過共同備課及進度會議，為學生設計適切的教學材料、學習活動及評估，以提升學習成效。 2. 二年級開始每級設一班精英班（至2019-20學年，精英班將擴展至六年級）；設加強輔導教學小組（主要為英文），以拔尖保底。 3. 設考試調適安排（例如試卷編排、加時等）。
家校合作及校風
家校合作 : 通過下列各項，學校與家長保持緊密聯繫及提供支援：
1. 舉辦各級家長會； 2. 與家長教師會緊聯繫積極推展各項活動，如親子活動、家長義工隊等； 3. 舉辦興趣班及家長學堂，透過家長或專家的講授，幫助家長掌握管教子女的方法。
校風 : 1. 本校校風淳樸，致力培養學生知書識禮、大方得體，懂得自尊、自重、自愛、自律，更能尊長愛友。 2. 學生純良受教，尊重師長，對人有禮，同學間相處融洽，他們喜愛校園生活，積極參與學校活動和服務。
學校發展
學校發展計劃 : 1. 透過科本重點課程發展及發展教師的專業能量，提升課堂教學的質素，從而提升學生的學習動機及學習效能，讓他們愛學習、能自學。
2. 推動「愛校園」文化，透過各式輔導活動、班級經營、價值教育等，培養學生關愛的精神及對學校的歸屬感。
教師專業培訓及發展 : 學校鼓勵教師參與不同類型的進修課程，同時參予教育局及其他教育機構的支援計劃。學校也因應每年的關注事項，組織相關的培訓活動，透過同儕觀課、共同備課、教學觀摩及教師分享等提升教師教學質素。
學費減免 : 不適用。
其他 : 1. 東華三院提供獎學金，以表揚於學術及體藝方面表現傑出的學生。
2. 開設多元化的興趣小組和課外活動，以發展學生個人潛能；培訓及推薦學生參加公開及校際比賽，在數學、科技創意、游泳方面，成績不俗。
3. 每年參加教育局內地交流活動，並配合校本課程發展適時舉辦海外交流活動，以擴展學生的視野，豐富其學習經驗。
