健康校園生活 : 本校推行全校早操、早讀、早禱活動。為了提高學生愛護環境的意識、加強對環保的重視，推行綠化課室及4R節約能源計劃。
備註 : 每週2次上課前早操及早讀；週三圖書課／成長課；週五興趣小組／多元智能活動；每週3次放學前設功課輔導。
全方位學習活動
本校注重多元化參與，發掘學生才藝潛能：學生團契、合唱團、朗誦組、木笛隊、手鈴隊、田徑、排球、足球、乒乓球、游泳、武術、公益少年團、西人英語、科學實驗、油粉彩畫、塑膠彩畫、水墨畫、小提琴、非洲鼓、數學小精英、小小科學家等及個人成長等興趣小組逾三十組。
辦學宗旨
提供靈、德、智、體、群、美六育並進之優質基督化全人教育：引導學生認識真理，接受主耶穌作救主；培養學生待人處事的品格，心存信、望、愛，邁向豐盛人生；並樂於服務人群，貢獻社會、國家；一生榮耀天父上帝。
學校特色
學校管理
學校管理架構 : 推行校本管理，校董會負責制訂辦學宗旨及重要政策，校長會同行政會議負責日常運作。
法團校董會/學校管理委員會/校董會 : 法團校董會。
環保政策 : 廢物分類回收 、一紙兩面用再回收 、攝氏25度以上開啟冷氣。
學校關注事項 : 1. 營造愉快學習環境。
2. 建立關愛共融校園。
3. 發展知識管理。
教學規劃
學習和教學策略 : 本校推行小班教學、普教中及合作學習，高年級引入創意思維教學元素。此外，全校推行思維訓練，以資優十三招為改善學與教的發展重點，教師透過同儕備課、觀課及評課，不斷反思教學策略，及運用資訊科技融入學與教，提升教學效能。家課重思考創意，鞏固課堂所學，學生自訂學習目標。
關鍵項目的發展 : 1. 以全校參與靈訓輔合一理念，推行生命教育，讓同學學會學習、愛中建立；慎思力行、愛神愛人。
2. 培養閱讀興趣，圖書課，廣泛閱讀、早讀、親子伴讀、閱讀獎勵計劃、圖書與學科協作，提升學生閱讀策略。
3. 全方位戶外學習、境外學習、專題研習，擴闊學習經歷。
4. 加強課室電腦網絡設施，資訊科技融入學與教，推行電子學習，使用平板電腦作小組學習。
共通能力的培養 : 課程以「六何法」、「六帽子」及心智圖(mind map)等思考法訓練，培養學生不同學習階段的九種共通能力。
學生支援
全校參與照顧學生個別差異 : 加強輔導教學計劃、功課輔導、提升能力輔導小組、個別輔導及成長的天空計劃。課後學習支援，扶助成績稍遜同學。為六年級同學設升中輔導工作坊、說話及面試技巧訓練。
課程剪裁及調適措施 : 補底：功課默書調適、讀寫障礙評估支援 、言語治療 、加輔計劃。
拔尖：各科尖子計劃 、才藝潛能比賽與活動、領袖生的培訓及各項才藝比賽。如：音樂、視藝創作、武術、電腦、數學小精英、文學菁英等。
家校合作及校風
家校合作 : 雙訓導、雙輔導及校園宣教士，家長教師會及家長學堂，促進家校合作，支援家長。
校風 : 本校推行「生命工程」。師生關係和諧，同學喜愛上學。勵行家長教育。設「成長的天空」計劃、學生團契及栽培小組。家長會、講座、家長成長小組及家長團契。
學校發展
學校發展計劃 : 培育學生靈育成長，建壯孩子心靈。
營造愉快學習環境，建立關愛共融校園。
開拓學生創意潛能，培養閱讀興趣。
發展智識管理，持續學校發展。
教師專業培訓及發展 : 教師領受召命、委身基督、高舉聖經真理、建立生命。
教師專業發展、以共同備課、觀課及評課建立學習型專業團隊。
共同參與教師團契，建立團隊互為肢體、個人生命及教學不斷更新。
學費減免 : 不適用。
其他 : 設循理會獎學金、家教會獎學金、校董會獎學金、柏立基獎學金。
