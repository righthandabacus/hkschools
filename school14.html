健康校園生活 : 根據《學校膳食安排指引》提供五穀類、蔬菜及肉類比例3:2:1的午餐推行健康的生活雙週、水果日、三項鐵人運動示範、眼鼻操等向學生傳遞健康訊息。
備註 : 學校重視學生身、心、靈發展，除了舉辨多項體育訓練外，還有早禱、詩歌分享、週會、成長課、團契、節日祟拜等。學校又組織學生參加社區文化、藝術和音樂活動，培養學生的欣賞能力。推行啟發潛能教育，營造正面文化。

辦學宗旨
提供全人教育，培育學生在基督愛中成長，建立其人格，自尊自重，追尋真理，關心社群，注目未來，並作一個有承擔及愛心的人。
學校特色
學校管理
學校管理架構 : 學校由聖公宗（香港）小學監理委員會有限公司掌管，校董會及校長則管理學校的日常決策和事務，並設有多個行政、科目及功能組別，推動行政及教學。
法團校董會/學校管理委員會/校董會 : 校董會成員共8人（包括校長），獨立人士（1位），辦學團體委任（6位）。
環保政策 : 1. 節省用紙。 2. 使用電子通告、內聯網及電郵通訊。
3. 行政工作電腦化。
學校關注事項 : 1. 優化教與學，照顧學生多樣性，讓學生喜愛學習。
2. 推動自主學習及資訊科技教學。
教學規劃
學習和教學策略 : 課堂上，老師致力推動四高（高展示、高參與、高回饋、高反思）和資訊科技互動教學，透過有效提問、追問和轉問來引起大班討論。還有小組活動、合作學習、探究學習(STEM)、閱讀策略。老師鼓勵學生預習，並把有關資料融入教學，老師亦重視課堂回饋和鼓勵學生反思所學。
1. 跨學科主題學習日。
2. 中文科：初小分層支援計劃、小一至小六「普通話教中文」計劃。
3. 英文科：校本寫作課程及Primary Literacy Programme - Reading/Writing。
4. 數學科：發展學生解難能力。
5. 常識科：推行專題研習和科學探究活動。
關鍵項目的發展 : 1. 多元化閱讀計劃：閱讀獎勵計劃、朋輩伴讀計劃、故事媽媽、好書齊分享及早讀課。
2. 資訊科技：圖書館電腦化、行政電腦化、校本電子教材、流動電子裝置教學、各科網上自學平台。
3. 常識科設專題研習、科學探究、時事專題分享、境內外戶外學習，培養學生共通能力。
4. 德育及公民教育：透過正規和非正規課程推行基督教全人教育及啟發潛能校園關愛文化。計有：班代表會，班級經營、校長的話、升旗禮、早會、周會、成長課、自我挑戰獎勵計劃、社區探訪和服務、公益少年團、交通安全隊、幼童軍、教育營、接待外賓訪校等。
共通能力的培養 : 透過課程設計、主題學習日和課堂教學，還有服務學習、專題研習及不同類型的學科活動和課業來培養學生的溝通能力、解決問題能力、創造力及高階思維能力。
學生支援
全校參與照顧學生個別差異 : 1. 設中、英、數輔導班。　2. 中、英文抽離小組教學。　　3. 伴讀天使計劃。
4. 伴讀父母計劃。　　　　5. 朋輩輔導。 6. 網上轉載資優課程。
課程剪裁及調適措施 : 測驗及考試會用抽離模式以照顧有特殊學習需要的學生。
家校合作及校風
家校合作 : 1. 家長教師會於2001年成立，協助舉辦各類型活動，以及監察校車和午膳的質素。
2. 不定期組織家長教育活動及每年兩次家長日。
3. 學生輔導老師為家長舉辦不同類型的小組。
4. 每月每班家長代表會議，聆聽及搜集家長對學校的意見。
校風 : 1. 學校著重基督化品格教育。
2. 學校參加國際啟發潛能教育聯盟，推動校園關愛及正面文化。
3. 本校歷史悠久，校風淳樸，學生知禮守規。
學校發展
學校發展計劃 : 1. 優化教與學，運用多元化教學策略，特別是資訊科技教學，照顧學生的多樣性，讓學生喜愛學習。　
2. 提升學生自主學習的能力。
教師專業培訓及發展 : 1. 為教師提供有關學與教及照顧有特殊需要學童的工作坊。
2. 鼓勵教師進修資訊科技教學。
3. 除了各科組有特定的教師專業培訓外，我們又與友校組成學習圈，鼓勵教學交流。
4. 參加香港大學「促進以自主學習開展STEM教學創新的多層領導網絡」計劃，促進教師專業及課程發展。
學費減免 : 不適用。
其他 : 1. 67屆香港學校朗誦節初小英文集誦季軍
2. 全港十八區小學數學比賽全場冠軍、急轉彎亞軍
