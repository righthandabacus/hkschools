班級教學模式
全新十一年直通模式，分基礎二年、拓展五年及通達四年共三個階段完成中小學。現小學部約有1000人，中學部約有1100人。完成十一年後可參加中學文憑試或國際預科文憑試。
註： 完成首五年後可原校升讀中學。
班級結構備註
採生活化自選教材，不過份倚賴課本，著重知識與生活結合，讓學生愉快及有效學習。批核小一名額198個。拓展一年級設電子書包班。

全方位學習活動
按年齡及不同的成長階段，每週定時設有聯課活動，讓學生有系統地接受各類文、康、體各類培訓。另於課餘時間提供多元化活動，包括興趣小組、學生團契、制服團隊、社會服務、校際比賽、戶外考察、宿營、本地及海外文化交流計劃，以及各類體育活動等。基階一設多元智能課。
辦學宗旨
建基於聖經的真理，以直資一條龍的辦學模式，提供嶄新十一年中、小學直通優質全人教育。透過靈、德、智、美、體、群，均衡發展學生的潛質，培養具國際視野的未來領袖。
學校特色
學校管理
學校管理架構 : 一般行政工作由校長、首席副校長、七位助理校長及非教學同工分擔，減輕教師非教學工作的負擔，使能專注教書育人，提升教學效能。校方亦聘請具豐富教學經驗的課程主任參與校本課程發展及教師專業培訓；同時又邀請海內外專家擔任顧問，協助學校完善各項專業發展。
法團校董會/學校管理委員會/校董會 : 香港華人基督教聯會真道書院校董會有限公司。
環保政策 : 設廢紙回收計劃，並鼓勵減少用紙。
學校關注事項 : 更有效的學與教，老師不斷追求專業成長，提升學生成為更主動和負責任的學習者。
教學規劃
學習和教學策略 : 每學年共分五學段，全年數學日共一百九十天。學校課程採用單元模式組成，靈活配合各學段的課時。學校重視學生在課堂內外的真實學習經歷，強調知識必須與生活結合，自基階一開始，所有學科均環環緊扣，科學及人文學科已採用綜合跨科形式設計課程。
關鍵項目的發展 : 致力推動閱讀風氣，購入數萬冊圖書鼓勵學生廣泛閱讀，另設家長伴讀計劃。各科於不同學習階段悉心安排專題研習，讓學生循序漸進地掌握研習方法。德育結合宗教及家庭生活教育，全方位培養學生成長。
共通能力的培養 : 營造自學、自理及自省文化。著重創意、批判性思考等共通能力的培養。評定學生成績時，學生如何應用共通能力亦為主要的考慮因素。
學生支援
全校參與照顧學生個別差異 : 本校聘得教育心理學家及數位社工駐校，對資優或有特殊學習需要的學生提供特別支援或輔導。學校亦特聘資深輔導老師，為校園生活組老師進行輔導技巧培訓，加強該組對學生的訓輔功能。
課程剪裁及調適措施 : 學校採用內地、英國及美國教材，揉合中西教育優點，不拘一格，因應教學需要而作校本調適，創立特色課程。
家校合作及校風
家校合作 : 每年舉辦多次家長聚會，家長均踴躍參加。成立家長閱讀社羣，鼓勵家長閱讀。亦出版多期家長通訊。通過家長教師會、家長義工、家長學堂、家長團契、家長支持小組及家長祈禱小組等活動加強夥伴關係。
校風 : 推行生本教育。鼓勵家長、學生與教師間加強溝通，在互信與尊重的夥伴關係下，共同建立和諧共融、和而不同的校園文化。
學校發展
學校發展計劃 : 建基於聖經真理，擁有國際級教育理念，結合中小學一條龍的辦學模式及直資的靈活性，本校實有優勢發展為別具特色、足以與世界各地優秀學府相比的學校。教育的核心是品格塑造與心念轉化，本校致力協助學生追求真、善、美，建立平衡豐盛人生。通過全人教育，讓學生不斷探索、思考、自學、自我解難、迎接轉變。校方積極推動以學習為中心的校園文化，鼓勵學生愉快和有效地學習。校方匯聚優秀人才同心興辦教育，加強對外聯繫交流，引進世界各國科研成果，保持與海內外大學緊密合作。透過不斷優化課程、教學法、評核及管理，堅定學校的發展方向。
教師專業培訓及發展 : 校方斥資讓全校教師加入世界性教師專業發展團體，每年出訪世界各地，參加研討會，結識同背景的學者，訪問當地學校。校方亦安排本地及各國資深學者到訪學校，共建環球性多元文化的支持網絡。
學費減免 : 本校設有學費全免或減免1/2學費關於學費減免詳情，請瀏覽網上版的超連結。有需要的學生(包括來自綜援家庭的學生及領取學生資助辦事處財政資助的學生)可提出申請學費減免。
- 本校學費減免計劃詳情包括家庭入息限額，請瀏覽網上版的超連結。
其他 : 本校設有獎學金、比賽獎項及交流計劃等，獎勵在學術或活動等方面有突出成就的學生。
