全方位學習活動
本校設有步操管樂及敲擊樂團、中樂團、音樂創藝、歌詠、粵劇、東西方舞、龍獅藝、非洲鼓、中國武術、陶藝、機械人工程師、魔術、話劇、校園小記者、奧林匹克數學、魔力橋、朗誦、書法、交通安全隊、男女童軍、獨輪車、花式跳繩及各種球類項目等四十多項活動。
辦學宗旨
本校秉承母會「道化教育」的辦學宗旨，奉行「以道為宗、以德為化、以修為教、以仁為育」 的辦學理想，以「明道立德」為校訓，推行品德和學業兼備的全人教育，使學生在德、智、體、 群、美得到全面發展。
學校特色
學校管理
學校管理架構 : 法團校董會負責監察及管理學校，轄下有「校政管理委員會」、「財務監察委員會」及「學與教委員會」，共同決定及執行各項發展項目。
法團校董會/學校管理委員會/校董會 : 本校法團校董會於2007年成立。由辦學團體代表、獨立專業人士、校監、校長、家長及教師組成，為學校制定發展方向，監察學校運作。
環保政策 : 1. 透過建立「環保校園政策」及環保課程，教育學生環保知識。
2. 避免使用即棄飯盒及減少廚餘，採用「現場派飯」的方式派發午餐，亦設有廚餘處理機。
3. 設廢物分類，鼓勵循環再用。
4. 舉辦「回收校服」計劃。
學校關注事項 : 1. 優化關愛文化，培養學生互助互愛的精神。
2. 利用閱讀策略和電子學習工具，提升課堂的學習效能。
教學規劃
學習和教學策略 : 本校推行小班教學，設有外籍英語老師計劃，尖子班以普通話教中文，及增設英文科學科。在自編校本電腦課程中，加入編程教學，運用平板電腦作電子教學，亦透過軟件「擴增實境」(Augmented Reality)電子學習程式，和舉辦統整周及以專題研習進行跨學科學習來提升學生的學習動機；又透過全方位學習和境外交流來豐富學生的學習體驗。
關鍵項目的發展 : 1.從閱讀中學習：透過閱讀活動，提高學生的閱讀興趣及培養閱讀習慣。
2.專題研習：透過跨學科進行專題研習，訓練學生建立共通能力： 批判性思考能力、運算能力、解決問題能力、研習能力、創造力、溝通能力、協作能力、自我管理能力、正面的價值觀和態度。
3.運用資訊科技進行互動學習：運用平板電腦作電子教學，設電腦編程教學，透過軟件「擴增實境」(Augmented Reality)電子學習程式，提升學生學習動機及教學效能。
4.德育及公民教育：以道化教育的理念滲入成長課及德育課，培養學生良好的品德和自律守規的行為。
共通能力的培養 : 透過專題研習、參觀活動、德育課、課外活動等培養學生批判性思考能力、運算能力、解決問題能力、研習能力、創造力、溝通能力、協作能力、自我管理能力、正面的價值觀和態度的共通能力。
學生支援
全校參與照顧學生個別差異 : 以融合教育的支援策略照顧學生學習上的個別差異，舉辦「齊齊做個好學生計劃」、「每日讀默寫獎勵計劃」、「及早識別和輔導有學習困難的小一學生計劃」等校本活動，以提升學生的學習能力。
課程剪裁及調適措施 : 由課程統籌主任、各科老師及專業人士（如輔導主任及教育心理學家）共同選出基礎學習重點進行課程調適，因應學生程度作出功課量和默書範圍的調適，並依據基礎學習重點設計研修班及輔導班的教學內容。
家校合作及校風
家校合作 : 本校成立家長教師會，成立一平台予家長及教師作緊密聯系，以提供優質及完善的教育服務。每年家教會亦提供多元化的家長教育課程，如子女管教工作坊、家長電腦班等。此外，為了令家長對學校的發展及學校環境有更深入認識，本校提供多樣的義工計劃（如義賣二手校服和午膳姨姨）及聚會讓家長參與。
校風 : 本校校風淳樸，著重培育學生建立良好的品德，學會自律，積極學習，敬師愛友。
學校發展
學校發展計劃 : 1. 優化關愛文化，培養學生互助互愛的精神。
2. 利用閱讀策略和電子學習工具，提升課堂的學習效能。
教師專業培訓及發展 : 因應學校的重點發展項目作專題培訓。與北京及廣州兩所重點小學結為姊妹學校，兩地老師進行語文教學觀課及交流，優化課堂教學。為了建立及推廣健康校園，校長及老師曾到澳洲柏斯及台灣的學校交流，分享推行健康校園的經驗。
學費減免 : 不適用。
其他 : 本校佳績無數，曾獲「閃耀之星」才華拓展獎學金、數學大王國際邀請賽「最佳培育獎」、亞太區小學數學奧林匹克邀請賽金獎、華夏盃狀元獎、全港十八區小學數學比賽（觀塘區）全場冠軍、環保杯全國邀請賽（香港區）總決賽冠軍、全港小學數學挑戰賽（九龍東區）六年級組團體賽冠軍、華夏杯全國數學創新思維決賽狀元獎、全港速算比賽六年級組全港冠軍及孔聖盃學界數學奧林匹克公開賽小五及小六組狀元獎。
