全方位學習活動
發揮全方位學習精神，激發學生潛能，建立自信，令心智與情意得以平衡發展，計劃中之活動分四大類：智能訓練、學術學會、興趣小組及運動學會。深受學生歡迎的包括東方舞班、體操班、戲劇興趣小組。
辦學宗旨
本著融合多元民族文化，發展多元智能的教育方針，開拓學生的世界觀，建構學生獨立思考、善於溝通及學以致用的能力，培養學生熱愛生活、樂觀積極、敢於創新、勇於承擔及參與服務的精神。
學校特色
學校管理
學校管理架構 : 屬地利亞教育機構管轄，具備明確循理的組織架構，實行高效益的管理。
法團校董會/學校管理委員會/校董會 : 校董會成員包括辦學團體和其他地利亞學校校長。校董會主要對學校提供支援及建議，並下放權責予學校自主權。
環保政策 : 「一人一花」運動；學生自備食具運動；綠化校園計劃。
學校關注事項 : (1) 提升學與教之成效
(2) 建立健的校園生活
(3) 加強德育與公民教育
教學規劃
學習和教學策略 : 1. 英語為主要教學語言，兼重法文、普通話及粵語。
2. 採互動教學，學以致用。
3. 全方位學習，實行跨科專題研習，如「我愛我的學校」，「香港主題公園」，「奧運會」等。
4. 廣泛閱讀計劃。
關鍵項目的發展 : 中、英文「悅目動聽讀書樂」計劃，鼓勵自選閱讀，提升興趣。推行網上學習，建立運用資訊科技能力。透過專題研習，學生掌握分析及整理數據的能力。鼓勵學生參與服務社區活動，培育公民意識。推行環保活動，應用在日常生活。
共通能力的培養 : 在日常生活中靈活運用三文四語：英文、中文、普通話及法語。
學生支援
全校參與照顧學生個別差異 : 為不同需要學生安排增潤及訂正活動；輔助學生在新的學習階段力求上進。
課程剪裁及調適措施 : 為非華語學生提供特別剪栽之中文調適課程學生可參加（英國）GCSE考試。
家校合作及校風
家校合作 : 家校緊密合作，例如：家長教師日，定期電話聯繫。
校風 : 1. 全體教職員營造一個共融環境，照顧所有學生的需要。
2. 修訂或擴闊正規課程以迎合不同需要。
3. 因材施教。
4. 課餘功課輔導班及「拔尖補底」計劃。
5. 暑假多元化活動，持續學習心態。
6. 新生適應班，消除恐懼，增加對學校的歸屬感。
7. 定期出版校報，邀請家長。
學校發展
學校發展計劃 : 1. 加強教師專業發展，提升學與教的質素。
2. 為學生開展「一生一體藝」計劃。
3. 訓練學生自學、解難、創造力及運用資訊科技能力。
教師專業培訓及發展 : 為教師籌組專業培訓及工作坊。
學費減免 : 本校設有學費全免或減免3/4、1/2、1/4學費及按個別家庭實際需要設有其他資助計。有需要的學生(包括來自綜援家庭的學生及領取學生資助辦事處財政資助的學生)可提出申請學費減免。
- 本校學費減免計劃詳情包括家庭入息限額，請瀏覽網上版的超連結。
其他 : 本校另設獎學金之學費減免計劃，如考獲全級第一名的學生有50%減免，而全級第二名的學生有40%減免，第三名的學生則有30%減免。
