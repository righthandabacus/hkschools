全方位學習活動
1. 設多元智能課，均衡發展興趣。 2. 舉辦全方位及大型學習活動。 3. 設不同興趣小組，包括：英語話劇、劍橋英語班、奧數班、小記者、手鐘、小提琴班、小結他班、欖球、籃球隊、Lego機械人小組、園藝組、舞蹈組等。 4. 舉辦境外交流，包括內地及國外。
辦學宗旨
以德智體群美五育均衡並進為宗旨；更透過多元化的課外活動，發揮學生的潛能，加強認識課本以外的知識和技能，建立正確的人生觀和價值觀， 從而貢獻社會和國家。
學校特色
學校管理
學校管理架構 : 設學校管理及質素保證工作會議統籌學校發展，轄下設學校行政組、課程發展組和學生成長組，重點發展全人教育。
法團校董會/學校管理委員會/校董會 : 於2008年成立法團校董會，成員包括辦學團體代表、教員校董、家長校董、校友校董及獨立校董。
環保政策 : 推行綠色校園計劃，設定學校環保政策，涵蓋教育、節能、減排、回收、採購等各方面。已超過十年獲「綠色學校」獎及連續多年獲得環保卓越界別優異獎及銅獎。
學校關注事項 : 1. 善用有效學習方法，提升自主自學的能力，促進學習成效。
2. 透過正向校園氛圍，建立個人自信，彼此互助互愛。
教學規劃
學習和教學策略 : 推行小班教學，重視班級經營和分組學習，輔以教學助理協作教學，以提高學生的課堂參與和個別照顧。學校重視透過課程領導推動自主學習，重視同儕觀課及共同備課，促進專業交流，以培育學生高階思維及其他共通能力。
關鍵項目的發展 : 1. 德育及公民教育：重視品德培育，依照校本六大核心價值發展課程，使學生能明辨是非，互愛互助，成就自利利他的人格。
2. 從閱讀中學習：每星期均設有主題式「午讀」課，引領學生進行跨課程的美文和德育兼備的閱讀。語文科發展校本閱讀策略課程，讓學生喜愛閱讀，學會閱讀。周會以故事帶動價值觀的涵養。
3. 專題研習：以普世關注的議題，運用STEM（數理科普）的知識，帶動專題研習，從而建立健康身心靈的生活態度。
4. 運用資訊科技進行互動學習：運用電子學習，加強互動學習，提高課堂參與和回饋，深化學習成果。
共通能力的培養 : 1. 實施課程統整，運用不同學習策略，營造多元化學習環境，培養學生各種共通能力。
2. 推行學生個人學習歷程檔案，著重培養學生的反思能力。
3. 著重培育學生批判、創意、解難、溝通和協作能力。
學生支援
全校參與照顧學生個別差異 : 1. 採取全校參與模式，按學生特殊學習需要以調適評估支援形式；
2. 學生支援小組制定政策及設立不同小組以支援情緒或學習上有特殊需要的學生；
3. 為有特殊需要的學生提供適切的服務。
課程剪裁及調適措施 : 1. 設加強輔導班，協助成績稍遜學生。
2. 設資優小組，培育資優學生，提升學生能力。
3. 按照專家建議，為有需要學生提供評估調適。
4. 設立不同形式之課後支援小組，協助新來港學童及其他有特別學習需要的學生加強學習。
家校合作及校風
家校合作 : 1. 每年辦多次家長會，與家長教師會合辦活動，確保緊密溝通。
2. 利用手機程式，鼓勵家校經常保持聯繫，關顧學生成長。
3. 法團校董會設有家長校董，讓家長參與校政事務。
4. 設有家長課程，協助家長協助子女的學習。
校風 : 1. 訓輔並重，訂立清晰獎懲制度，實施「優點累積」、「以功抵過」及校本全人計劃。
2. 重視培養學生的自我管理能力。
3. 設駐校社工及校本教育心理學家，並有情緒輔導小組及「大哥哥大姐姐計劃」，協助低年級新入學學生。
學校發展
學校發展計劃 : 1. 重點於主要學科發展校本自主學習課程。
2. 建立校內、校外的學習群體，促進專業成長。
3. 持續發展評估素養，有效回饋課程及學與教的發展。
4. 增加學生展示學習成果的機會和平台，提升學生的自信心。
教師專業培訓及發展 : 1. 每周為主要學科編訂共同備課時段，共同設計教學計劃。
2. 參與教育局及其他大專機構的課程發展計劃，完善校本課程。
3. 教師定期進行同儕觀課，彼此分享教學的成功經驗。
4. 與友校組織學習圈，定期交流和觀摩，擴闊專業視野。
學費減免 : 不適用。
