全方位學習活動
本校以發展一生一體藝為目標，為學生安排多元化的課外活動，包括中文小作家、劍橋英語、英文話劇、奧數、領袖訓練、書法國畫、國術、醒獅、跳繩、籃球、足球、田徑、小提琴、非洲鼓、童軍、公益少年團、義工組等。 本校每學年與啟歷國際學校進行6-8次的學生交流活動。
辦學宗旨
本校致力為南區兒童提供高質素、全面而均衡的基礎教育服務，並期望學生能透過學業與活動並重的學校生活，發展個人潛能，學會學習，為接受中學教育及個人成長奠下穩固基礎，並貫徹校訓勤儉忠信、立己立人的精神，使成為具備知識、技能，懂得關心社群、勇於承擔，有積極人生態度的良好公民。
學校特色
學校管理
學校管理架構 : 學校由東華三院及鶴山同鄉會共同營辦，由東華三院教育科及東華三院鶴山學校法團校董會負責管理。
法團校董會/學校管理委員會/校董會 : 已於2009學年9月1日成立法團校董會。
環保政策 : 推動環保及綠色校園的措施，例如：環保午膳、廚餘處理、節約用電及冷氣開放管制等。
學校關注事項 : 1.推動STEAM教育，發揮學生創意潛能。2.營造良好的英語學習環境，加強學生學習英語的興趣和動機。3.提升學生自我管理能力，培養他們成為一個能自學、懂自律、常自省，且具自信的學生。
教學規劃
學習和教學策略 : 1.推行小班教學，以學生為本的活動教學模式，提升學習技巧及培養正確的學習態度。2.營造兩文三語學習環境，提升學生語文學習能力。3.利用平板電腦輔助學習，讓學生善用科技，擴闊學習經歷。4.利用網上學習平台，培養學生自學技能與習慣。5.每年進行超學科專題研習活動，培養學生的共通能力，發揮學生的自主學習精神。6.推行全方位學習，帶領同學參加各類參觀、比賽及社會服務，豐富其學習經歷。7.在常規課程、專題研習和課外活動中滲入STEAM教育的元素，鼓勵學生動腦思考，動手操作，訓練並提升學生的邏輯和創意思維及其協作和解難的能力。8.善用提問技巧，促進學生學習。
關鍵項目的發展 : 1.從閱讀中學習：推行親子閱讀獎勵計劃、早讀及好書分享活動。2.運用資訊科技進行互動學習：將資訊科技融入課堂。3.專題研習：全校每年進行跨學科專題研習活動。4.德育及公民教育：每周舉行生活教育課及周會。
共通能力的培養 : 透過課程設計和課堂教學，還有服務學習、專題研習、科學探究學習、跨學科活動及不同類型的學科活動和課業來培養學生的溝通能力、解決問題能力、協作能力、創造力及高階思維能力。
學生支援
全校參與照顧學生個別差異 : 1.實施「雙班主任」制，加強對學生的關顧。部分課堂採用協作教學，增強師生互動。2.放學前設導修時段，課前及課後設功課輔導班、語文能力提升班等。3.發展資優教育，並於課時外提供尖子訓練班。4.提供院本心理學家及言語治療服務，及早識別和照顧學生的不同學習需要，為有需要學生進行家課、測考及課程調適。
課程剪裁及調適措施 : 1.合作學習/多感觀教學法。2.善用資訊科技融入教學活動，提升學習動機和興趣，加強課堂上的生生和師生互動。3.透過不同層次的問題，善用提問技巧，照顧不同學生的學習需要。4.多樣化評估，促進學生學習。
家校合作及校風
家校合作 : 多元化家校活動，如家長講座、親子興趣小組、親子旅行、家長義工小組等。
校風 : 培養學生「關愛互助」精神，營造「友愛和諧」氣氛，透過大哥哥大姐姐計劃、風紀領袖訓練、一人一職等活動，推動朋輩互助及服務學習。
透過參加「健康校園認證」及「護脊校園」計劃，宣揚健康生活的重要性，鼓勵學生養成健康的生活習慣，致力建構「關愛、和諧、健康、快樂」的校園文化。
學校發展
學校發展計劃 : 1.推動STEAM教育，發展學生的創意潛能。2.營造良好的英語學習環境，加強學生學習英語的興趣和動機。3.提升學生自我管理能力，培養他們成為一個能自學、懂自律、常自省，且具自信的學生。
教師專業培訓及發展 : 學校會因應每年的關注事項，透過不同形式的培訓活動，如推薦教師出席不同的工作坊、講座及課程、舉辦教師專業發展日，又或邀請大專院校提供到校支援服務，提升教師在學與教及學生發展兩方面的專業能力，使學生能得到全面的發展。
學費減免 : 不適用。
其他 : 1.獎學金：梁勁謀紀念獎學金、關超然太平紳士學業進步獎、香港鶴山同鄉會勵學獎、東華傑出學生獎、學科獎、學科挑戰獎、閱讀獎、傑出綜合體藝獎、跨越進步獎、劍橋小學英語考試獎學金。2.公開比賽：學校朗誦節、音樂節、戲劇節、西區田徑比賽、西區籃球賽、ICAS、中華武術精英賽、新秀武術錦標賽、全港醒獅大賽。3.學校參加了「促進香港與內地姊妹學校交流試辦計劃」，加強專業交流。
