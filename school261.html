備註 : 着重發展全人教育，培育學生「祟德尚學」、一生一體藝及樂於服務的精神；強調學生能主動學習，培養其終身學習的能力；關注幼小中銜接，舉辦小一迎新日、小一家長會、小一適應課程、「愛心學長」計劃、中學體驗計劃及小六升中選校家長會等。
全方位學習活動
開設多個活動小組及課後學藝班，涵蓋學術、體育、藝術、思維及技能訓練等。制服團隊有幼童軍、紅十字會少年團、小女童軍、交通安全隊、升旗隊及步操樂團。特色團隊包括管樂團及樂器小組、公益少年團及少年警訊。陸運會、旅行、參觀、教育營等豐富學生學習經歷。
辦學宗旨
本校致力推行普及教育，為學生營造舒適愉快的學習環境，提供優質的全人教育，使學生在德、智、體、群、美五育上有均衡的發展。
學校特色
學校管理
學校管理架構 : 本校施行校本管理，由校長帶領行政小組共同策劃和執行各項政策。
法團校董會/學校管理委員會/校董會 : 「學校管理委員會」監察及推動校務的發展。
環保政策 : 知識與實踐並重，除宣揚環保信息外，亦推行綠化校園、節省能源、廢紙及舊衣物回收計劃。
學校關注事項 : 1. 優化校本課程，發展資訊科技教學，提升學與教效能。
2. 培養學生盡責守規，積極自信的良好品格。
教學規劃
學習和教學策略 : 採用適切的教學策略及設計以學生為本的活動，提升教學效能。讓學生掌握自學技巧，培養良好的學習習慣。
關鍵項目的發展 : 1. 推行早讀課、閱讀活動及獎勵計劃，培養學生良好的閱讀習慣。
2. 推動電子學習，加強教學效能。設網上學習平台，提升學生的自學能力。
3. 透過主題學習日及專題研習，建構學生的共通能力。
4. 透過正規課程、班級經營、生活教育課、週會及訓輔活動，推行德育及公民教育。
共通能力的培養 : 透過課堂及課外活動培養學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 1. 及早識別有特殊教育需要的學生。
2. 提供校本教育心理及言語治療服務。
3. 提供不同類型的課餘支援服務、課程及調適工作。
4. 提供全方位學生輔導服務。
5. 為尖子學生提供增潤課程及支援。
課程剪裁及調適措施 : 同級科任教師按需要調動及剪裁課程；又為有需要的學生進行家課及評估調適。
家校合作及校風
家校合作 : 1. 家長參與家教會、學校管理委員會、商業活動監察委員會等，為學校發展提供意見。
2. 家教會每年舉辦會員大會、親子旅行、家長日、興趣班、家長講座及親子活動等。
3. 組織家長義工隊，協助學校舉辦大型活動。
校風 : 培育學生「律己愛人」的精神，建立正面的價值觀。推行「進步之星」獎勵計劃，讓學生實現自我，建立自信。舉辦領袖生訓練活動，啟發學生領導潛能。團隊積極服務社群，貢獻社區，從而培養歸屬感及責任感。
學校發展
學校發展計劃 : 1. 優化校本課程，發展資訊科技教學，提升學與教效能。
2. 培養學生盡責守規，積極自信的良好品格。
教師專業培訓及發展 : 按學校發展計劃訂定校本培訓計劃；教師亦積極參加校外課程，以強化專業能量。
學費減免 : 不適用。
其他 : 本校歷史悠久，多年來為社會培育人才，校友在不同崗位上發揮所長，貢獻良多。本校將繼續致力塑造關愛校園，提供優質的全人教育，使學生在德、智、體、群、美五育上有均衡的發展，迎接未來挑戰。
