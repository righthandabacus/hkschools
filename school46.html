全方位學習活動
安排同學於星期五下午課外活動節參加不同的週會或多元化活動，同學將分別按照自己的興趣參加三十多組課外活動。此外，本校設跆拳道、電腦、圍棋、中樂、小提琴、藝術體操及油畫班等十多項課餘興趣班供同學付費參加。
辦學宗旨
本校致力為學生營造一個愉快的學習環境，從而啟發他們的認知，解難及自學能力，以達到德、智、體、群、美五育並重的理想。我們期望學生勇於承擔責任，與人和睦相處及獲得健康的身心，日後成為社會上良好的公民。
學校特色
學校管理
學校管理架構 : 學校管理委員會轄下設有學校自評及發展小組和行政小組，專責學校的發展及行政決策。
法團校董會/學校管理委員會/校董會 : 學校管理委員會主席為教育局首席助理秘書長，其他成員包括校長、教師、校友、社會人士及家長，定期舉行會議反思、前瞻及檢討學校的運作與發展，提升教學質素。
環保政策 : 制定相關措施，鼓勵節約能源，循環再用。透過不同的學習領域、活動、德育及公民教育灌輸環保訊息，提升學生環保意識。
學校關注事項 : 提升學生的自信心、自學精神、解難能力、服務精神及創新思維能力，培育學生成為「成績好、品德好、身體好」的「三好五進」學生。
教學規劃
學習和教學策略 : 本校推行小班教學，運用自主學習、合作學習、互動電子學習等教學策略，讓學生從活動、探究、實踐中學習，幫助學生建構知識、提升學習動機、促進自學能力。
本校致力推動閱讀，每天設閱讀好時光，透過教授閱讀策略及閱讀獎勵計劃，鼓勵學生多閱讀，藉此增強學生閱讀能力。
透過專題研習、跨課程學習、全方位學習活動、德育成長課、跨境學習等，為學生提供多樣化的學習經歷，培養他們成為自律守規的好公民。
初小部分班別以普通話教中文，強化學生的普通話口語能力。
關鍵項目的發展 : 1. 優化品德及公民教育。
2. 舉辦多元化的閱讀活動。
3. 小一至小六設「通識教育」校本課程及專題研習等活動。
4. 把資訊科技輔助教學融入課堂教學之中。
共通能力的培養 : 透過課外活動、全方位學習活動及跨科學習活動培養學生的協作、溝通、解難、創意及批判性思考等共通能力。
學生支援
全校參與照顧學生個別差異 : 本校以「全校參與模式」支援有特殊學習需要的學生。老師會按學生的需要而設計合適的課程，並為學生作適當的教學調適。此外，學校亦會為有需要的學生安排課後學習班、課前小組訓練和言語治療等，以提升學生的學習效能。
課程剪裁及調適措施 : 學校設有清晰的機制，以識別不同學習需要的學生，為有需要學生作出調適。透過不同的拔尖及保底小組，照顧學生的多樣性；本校亦為小一學生安排「小一啟航適應活動」、推行中文及英文科銜接課程。
家校合作及校風
家校合作 : 家長教師會舉行多項不同類型的活動，促進家校夥伴合作的關係。本校舉辦親子開心學習天及多元化的家長教育課程，提升家長管教子女的技巧，促進親子關係。
校風 : 本校十分重視學生的品德教育和全人發展，學生關愛合群、待人有禮，師生關係融洽。透過「晨光悅耳聲」早會活動、校本輔導計劃、教育性講座和通識課等，全方位培養學生正確的態度，建構正面的價值觀和確立良好的品格。
學校發展
學校發展計劃 : （2015-2018年度）
1. 提升學生自主學習的能力；促進學與教效能。
2. 推廣健康生活習慣，建立正面積極人生觀，活出身心健康成長之路。
教師專業培訓及發展 : 進行多元化的校本教師持續專業發展：如共同備課、同儕觀課、評課及教研等，促進教師專業交流；透過啟導教師計劃，培育新入職老師。另外，按學校發展方向及學生需要有策略地推展對焦的教學發展及培訓課程，以掌握教育趨勢，提升專業能量。
學費減免 : 不適用。
