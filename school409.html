全方位學習活動
本校設有幼童軍、小女童軍、紅十字少年團、交通安全隊、升旗隊、公益少年團、基督小先鋒、各類球類校隊、田徑隊、游泳隊、劍擊校隊、各類樂器班、合唱團、牧童笛隊、弦樂團、管樂團、話劇、集誦隊、奧數班、英語班及科學小實驗等。
辦學宗旨
秉承天主教學校的教育使命，以「全人教育」作為辦學宗旨，本著「堅毅力行」的校訓，引領學生追求知識、培養積極的學習態度及發展潛能，從而對社會作出貢獻。
學校特色
學校管理
學校管理架構 : 辦學團體、校董會、校長、行政組及功能小組。
法團校董會/學校管理委員會/校董會 : 校董會。
環保政策 : 1. 午膳飯堂，用可清洗的食具。
2. 「惜食卡」獎勵計劃，鼓勵學生珍惜及減少浪費食物。
3. 小花園種植區。
學校關注事項 : 1. 落實天主教五個核心價值之「愛與生命」，使其融入課程中。
2. 開展校本 STEM 教育，優化課程，加強連繫常識，數學及資訊科技科的學習元素。
教學規劃
學習和教學策略 : 本校採用全方位教學方針，學生通過參加每週的多元智能課及參與不同的課外活動，豐富其學習經歷；透過參觀活動、專題研習及跨科課程，擴闊其他學習經驗。
關鍵項目的發展 : 1. 閱讀計劃：「我的閱讀記錄冊」閱讀獎勵計劃。邀請「故事爸爸媽媽」到校為初小學生講故事。
2. 資訊科技：電腦堂教授各類軟件應用，提升學生資訊科技能力，以開展校本STEM教育。
3. 專題研習：安排專題研習匯報日，讓學生可分享成果，從而提升表達能力，增強自信。
4. 德育及公民教育：每週設有週會，以英語、粵語及普通話進行，藉以提升學生兩文三語的能力；內容包括學生帶領祈禱及教師週訓短講，其他學生亦即場作交流互動，加強學生的自信心。此外，另有生命教育課、宗教及道德教育課，培養學生良好的品德。
共通能力的培養 : 在課堂中進行專題研習，學生彼此討論，從而培養學生各項共通能力。本校更把資優元素融入課堂中，提升學生的創造力及解難能力。
學生支援
全校參與照顧學生個別差異 : 1. 推行校本全校式及抽離式資優教育課程。
2. 輔導主任及教育心理學家駐校，提升學生的學習效能。
課程剪裁及調適措施 : 為有特殊學習需要的學生在考評過程中加以調適，舉行定期會議訂定跟進計劃。
家校合作及校風
家校合作 : 高主教書院小學部家長教師會於2014年成立。作為家長與學校交流的橋樑，協助學校籌劃各項活動，達致家校合一。
校風 : 校風淳樸，訓輔合一。學校注重培養學生的品德，教導學生依法守規，建立關愛文化，發揮堅毅的精神。
學校發展
學校發展計劃 : 1. 按照天主教學校的五個核心價值—「真理、義德、愛德、家庭、生命」辦學，讓學生獲得良好的學科學習及品德培育機會，彰顯愛德，達致生命的圓滿。
2. 學校重點發展的教學計劃，包括提升學與教，擴闊語言學習環境。
教師專業培訓及發展 : 校監為中、小學部天主教老師進行靈修，培養教友老師的靈性生活，體現中、小一家親的精神。
學費減免 : 不適用。
其他 : 第十二屆全港小學數學比賽（灣仔區）全場亞軍， 第十二屆香港中、小學數學創意解難比賽（初賽）金獎， 港島東區小學校際羽毛球比賽男子組冠軍（四連冠）。
