學校特色
學校管理
學校管理架構 : 本校已成立法團校董會，落實校本管理措施。
法團校董會/學校管理委員會/校董會 : 本校已於2015年3月2日成立法團校董會，法團校董會成員包括辦學團體代表、校監、校長、老師、家長、校友及教育界人士，統籌及決策校政。
環保政策 : 廢紙回收計劃、提倡節約能源計劃。
學校關注事項 : 持續完善課程，推動學生自主學習：提升學習表現：持續優化教學、加強照顧學習差異；加強天主教教育核心價值培育，讓學生建立正面的價值觀。
教學規劃
學習和教學策略 : 推行全方位學習活動以豐富學生的學習經歷；籌組海外交流活動，以擴闊學生視野；配合專題研習，啟發學生共通能力；發展藝術教育，啟發學生潛能；推行電子學習，於課堂運用平板電腦進行學習活動，即時回饋，提升學與教的效能；發展戲劇教育，以培養學生自信及批判性思考能力。
關鍵項目的發展 : 透過不同的閱讀活動幫助學生學會閱讀、享受閱讀，並從閱讀中學習；推行電子學習幫助學生善用資訊科技進行學習；透過專題研習發展學生的溝通、創意、協作及自理能力，並從中學會學習；透過宗教教育、戲劇教育及特定德育及公民教育活動幫助學生學習正確的價值觀。
共通能力的培養 : 藉提問、探究及討論等課堂活動發展學生的批判性思考及解難能力；藉小組特定任務發展學生的協作、溝通和創造能力；藉電子學習發展學生運用資訊科技及自我管理能力。各級均會進行主題教學，幫助學生從不同角度認識主題及進行學習，教師亦會設計特定課業以培養學生的共同能力。
學生支援
全校參與照顧學生個別差異 : 善用小班優勢加強學生在課內參與學習及展示成果的機會；為有需要的學生提供課內的加強輔導教學、課後支援及成長小組；為有特定需要的學生提供個別教學計劃、功課、默書及測考調適。透過多元化的教學活動、設計不同程度的課業及朋輩支援，照顧學生的學習差異，並營建關愛及共融校園。
課程剪裁及調適措施 : 拔尖補底措施、校本課後支援、協作教學措施、抽離教學措施、分層課業、功課及測考調適。
家校合作及校風
家校合作 : 學校著重家校合作，班主任透過陽光電話定期與家長聯絡，以加強與家長的溝通及合作，並透過多元化的活動，如親子工作坊、家長講座、家校旅行、親子同樂日、名人講座、傑出學生計劃、家長義工凝聚活動等，與家長建立良好的伙伴關係，為本校學生成長及發展共同努力。
校風 : 1. 校風淳樸，培養學生優良品德，營造關愛文化，發揚校訓明理祟德，愛主愛人的精神。
2. 訓輔並重，從多方面推行德育及生命教育活動，如白天智多星小冊子、領袖訓練及個人成長課，協助學生建立正面的價值觀。
3. 家校合作，舉辦不同的家長講座及工作坊，以提升家長協助學生學習及管教技巧，並提供不同類型的親子活動等。
學校發展
學校發展計劃 : 1. 加強學生自主學習，提昇學習成效。
2. 持續優化教學，加強照顧學習差異。
3. 培養學生正面價值觀，迎接成長過程中的挑戰。
教師專業培訓及發展 : 與教育局及專業團體協作，教師透共同備課、同儕觀課、課研及教學經驗交流，優化課堂學與教及提升教師的專業能量。
學費減免 : 不適用。
其他 : 全港數學挑戰賽、亞洲國際奧數公開賽獲、「華夏盃」、港澳盃奧數比賽獲多個獎項。朗誦節粵語朗誦獲冠軍、榮譽成績。香港國際音樂節獲傑出金獎、鋼琴組獲優良成績、動感聲藝展銅獎。九西區學界田徑女甲跳高亞軍、男初60米殿軍；深水埗青年盃足球比賽殿軍；校際「舞蹈藝術家」大賽、標準舞及拉丁舞大賽牛仔舞冠軍。戲劇：校際戲劇比賽獲多個獎項。
（本校將於2020年搬往東京街新校舍）
