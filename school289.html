辦學宗旨
以學生為中心，提供良好學習環境，運用適合的教育策略，啟發學生潛能，協助學生不斷進步。
學校特色
學校管理
學校管理架構 : 設有「學校管理及領導」組，統籌各項學校改善及發展工作，另按不同校務工作成立各行政小組，由專責主任及老師推動各項學校發展。
法團校董會/學校管理委員會/校董會 : 學校已成立法團校董會，由校監、辦學團體校董、獨立校董、校長、家長校董、校友校董和教師校董組成。法團校董會就學校管理、運作及發展等工作進行討論及檢討，以期校務得以順暢發展。
環保政策 : 積極推行環保教育。
學校已安裝節能光管及節水器具。
學校關注事項 : 優化課堂教學，強化英語學習，關顧學生成長。
教學規劃
學習和教學策略 : 施行小班教學，運用有效的教學策略，指導不同的學習方法，讓學生習得知識，發展自學能力。
學校鼓勵學生參加校外比賽，體驗不同的學習經歷，以促進多元發展。
編排閱讀課、多元學習課及專題研習週等。透過科技日、英語日、普通話日、才藝表演日、中國傳統文化日、外出參觀，組織跨科全方位學習。
關鍵項目的發展 : 1. 推行早讀計劃、閱讀獎勵計劃、網上中英文閱讀，有系統地訓練閱讀技巧。
2. 推行IT學習獎勵計劃，鼓勵利用IT自學。
3. 設專題研習週讓學生建構各種共通能力。
4. 推行一人一服務。
共通能力的培養 : 主力發展學生溝通、自我管理、解難創意等能力。
學生支援
全校參與照顧學生個別差異 : 以TARGET六個導向模式作為提升學習動機的教學策略。推行全校共同參與輔導計劃，設輔導課、加強輔導班、成長的天空等。
課程剪裁及調適措施 : 1. 於高小開設「英語文法課」，並由外籍教師任教「英語閱讀課（Reader）」。
2. 閱讀策略教學規劃。
3. 常識科校本探究學習。
4. 校本資優課程規劃。
家校合作及校風
家校合作 : 學校重視家校合作，已成立家長教師會，作為與家長渠道。
校董會有1名家長代表，加強家長在學校發展的參與度。
定期以致家長函形式報告學校近況。
校風 : 學校連續八年獲「關愛校園」獎。我們鼓勵及讚賞學生，校內遍掛學生的校園生活照片及視藝作品。推行「跳躍飛升學生獎勵計劃」、「和諧大使計劃」等。
學校發展
學校發展計劃 : 2015至2018年︰
1. 培養自學能力，讓學生樂學善學。
2. 培養自律精神，展現學生才華。
教師專業培訓及發展 : 鼓勵教師持續發展，與時並進，組織教師發展活動。
學費減免 : 不適用。
其他 : 1. 推行中文「喜閱寫意」計劃。
2. 初小校本英文語音及圖書閱讀課程。
3. 英語話劇拔尖班。
