全方位學習活動
學校籌辦多元化的課外活動及服務團隊供學生參加，另設多項收費活動，由專業導師執教，如舞蹈、劍擊、圍棋、跆拳道、樂器班、球類活動和視覺藝術等。
辦學宗旨
本基督精神，發展全人教育；藉宣講福音，培育豐盛生命。
學校特色
學校管理
學校管理架構 : 由各持份者組成。
法團校董會/學校管理委員會/校董會 : 法團校董會。
環保政策 : 省能綠化。
學校關注事項 : 1. 加強培育生命素質，提升學生自我形象。
2. 持續優化學與教，鞏固學生自學能力。
3. 完善行政架構，強化團隊效能。
教學規劃
學習和教學策略 : 1. 為培育學生整全的發展，追求卓越，本校透過高階思維和多元認知的教學策略，引導學生建構知識，加強他們的明辨性思考和解難能力，並提升他們的自學能力。
2. 透過全方位學習活動，豐富學生的學習經歷，如：服務學習、境外學習、社區參觀等。
3. 善用資訊科技進行互動學習。
關鍵項目的發展 : 1. 推行生命教育，培養學生的生命素質。
2. 推行閱讀獎勵計劃，擴闊學生的閱讀層面。
3. 活用電子學習平台，有助學生自學。
4. 各級設專題研習，學生透過有效的資料搜集、整理、分析、綜合及反思，在探究及解難的過程中，主動建構新知識或從新角度看事物。
共通能力的培養 : 1. 連繫各組科，培養學生的共通能力。
2. 推行資優教學，發展學生的高層次思考能力、創造力和情意社交能力。
學生支援
全校參與照顧學生個別差異 : 1. 開設棋盤遊戲、奧數班和寫作班等，以發展資優學生的多元智能。
2. 開設支援小組，跟進有個別學習需要的學生。
課程剪裁及調適措施 : 1. 剪裁和調適課程，以配合不同學習能力的學生。
2. 幼小銜接和小六升中銜接課程，有助學生適應學習上的過渡。
家校合作及校風
家校合作 : 1. 設立家長教師會，籌組家長義工小組，協助推行各項活動。
2. 定期舉行家長會和校長茶座，以了解家長的需要，更特設家教會專用電郵信箱，以蒐集家長的意見。
3. 透過家校/堂校的合作，舉辦家長講座、親子活動和家長教育課程等，以加強學校與家長、學校與堂校的溝通。
校風 : 1. 以「生命教育」的宗旨建立學校文化。
2. 校風淳樸，師生關係融洽，注重學生品德培養。
學校發展
學校發展計劃 : 1. 培育孩子有優質的生命素質，活出彩耀的人生，盡展潛能。
2. 致力保持學生學業成績的優勢。
3. 建立教學分享文化。
4. 借助教育局和大學的專業力量，優化校本課程和課研的質素。
教師專業培訓及發展 : 邀請本港大專院校提供校本培訓，並鼓勵教師參與交流活動，積極推動教師專業發展。
學費減免 : 不適用。
其他 : 提供不同範疇的獎學金，以獎勵學業成績優良或體藝表現優異的學生。
