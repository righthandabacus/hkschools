班級教學模式
一至三年級推行「英文識字及寫作計劃」。一至六年級推行小班教學。四至六年級以普通話教授中文。一至六年級活動教學。
班級結構備註
以兒童為中心，從活動中學習，並鼓勵師生、生生互動，強調學生齊參與、多互動、高展示。本校以腦科學及思維為本的模式進行學習，並透過多元化的教學模式，包括單元教學，協作教學，課程統整等，加強教學效能。
學習評估
全年全科測驗次數 : 2
全年全科考試次數 : 2
多元學習評估 : 全學年共有兩次測驗及考試，包括筆試及口試。
各科另安排多元學習評估，包括：課堂觀察、口頭匯報、專題研習、主題報告、體適能獎勵計劃及各科其他進展性評估。
分班安排 : 一至三年級學生將按考試成績，平均編入三班。四至六年級實行精英班制，首三分之一名次的學生將編入精英班（A班），其餘學生平均編入B、C班。

辦學宗旨
我們本著有教無類及因材施教的宗旨，為學生提供優良的學習環境及編訂均衡的課程，讓學生愉快地學習，充分發展個人潛能，培育學生成為五育兼備的好學生和積極向上、獨立思考、不斷學習、富責任心的好公民。
學校特色
學校管理
學校管理架構 : 本校由英皇書院同學會法團校董會管理。
法團校董會/學校管理委員會/校董會 : 於2008年成立
環保政策 : 成立環保組，舉辦各類型環保活動及比賽，包括環保講座、參加康文署的「綠化校園資助計劃」、「校園齊惜福」計劃等。校內舉行與環保有關比賽（如課室清潔比賽等），並由學生擔任「環保大使」，提升學生的環保意識，培養學生保護環境的責任感，並鼓勵全校學生積極實踐環保生活。
學校關注事項 : 1. 深化品德教育，促進學生價值觀及態度培養。
2. 推動自主學習，提升學生學習效能。
3. 優化課堂教學，提升學生的高階思維。
教學規劃
學習和教學策略 : 本校推行小班教學，著重教師提問技巧，強調齊參與、多互動、高展示，並透過不同的學與教策略，包括課程調適、以腦基礎為本的教學技巧等，鼓勵學生合作學習，培養學生的自學能力及明辨性思維，促進師生及生生互動。學校靈活運用課時，進行全方位學習，鼓勵學生善用內聯網及資訊科技進行自學，為學生提供多元化的學習經歷。
關鍵項目的發展 : 1. 從閱讀中學習：設立星級閱讀計劃、閱讀分享劇場、齊讀樂趣多、伴讀爸媽、大姐姐哥哥伴讀計劃、親子閱讀獎勵計劃、中英文網上閱讀計劃等。2. 學校運用資訊科技進行互動學習：推行運算思維教育計劃、運用資訊科技進行互動學習、數碼音樂創作、數碼藝術家計劃、雕塑漫遊、數學遊蹤、認識我們的花圃及網上圖書館等。3. 專題研習：一至四年級進行主題報告；高年級進行專題研習。4. 德育及公民教育：推行「I CARE」計劃，透過生命教育及跨學科課程，培育學生正面價值觀和積極態度。
共通能力的培養 : 通過不同科目或學習領域的教學活動，培養學生的共通能力。低年級以溝通、協作能力為主，並逐步引入創作、明辨性思維、研習能力、自我管理及解決問題等共通能力。
學生支援
全校參與照顧學生個別差異 : 學校成立「學習支援小組」，負責策劃、統籌和推動學生支援工作，並採用「全校參與」的模式，定期檢視成效。本校不單著重因材施教設計校本課程，同時亦照顧學生需要而特設各種活動及計劃，並推行「拔尖補底」計劃，分別為成績優異或稍遜的學生，舉辦不同類型的輔助課程。
課程剪裁及調適措施 : 拔尖措施：創意編程拔尖班、奧數、中英文寫作拔尖、視藝拔尖班等。補底措施：課前學科輔導、課後功課輔導、教學助理支援計劃、校本心理學家服務、小一及小二助苗班、言語及聽覺治療計劃，並進行課程剪裁及評估調適等。
家校合作及校風
家校合作 : 家教會活動包括：家教會會員大會暨學生文藝表演、親子旅行、親子競技日及教育講座等。
校風 : 建立淳樸、關懷、友愛的校風，師生關係和諧。注重及培養學生的品德發展，舉辦各項德育及輔導活動。另設立進步獎及獎學金鼓勵優異生。
學校發展
學校發展計劃 : 1. 深化品德教育，促進學生價值觀及態度培養；2. 推動自主學習，提升學生的學習效能；3. 優化課堂教學，提升學生高階思維；4. 教育局「課程領導專業計劃（中文科）」；5. 教育局PLP-R/W Programme Space Town Pilot Scheme（英文科）；6. 教育大學「促進實踐社群以優化小班教學支援計劃（常識科）」；7. 教育局「於小學實踐視覺藝利課程計劃（視覺藝術科）」；8. 賽馬會運算思維教育計劃（電腦科）。
教師專業培訓及發展 : 本校關注課程改革的趨勢，十分重視教師的專業發展。本校設立教師個人學習檔案及組織「學習圈」，進行有重點的同儕備課、觀課、評課、教研課及集體評課，並安排科組分享會，發展學習型的教師團隊。本校積極參與大專院校的協作計劃，引入教育局專業支援，進行教研活動。本校更擴闊交流網絡，積極到本地及內地學校進行教育交流及考察。
學費減免 : 不適用。
其他 : 「香港FLL創意機械人大賽(2017)」獲得「機械人設計冠軍」及「機械人表現賽冠軍」；「WRO世界奧林匹克機械人競賽-香港區選拔賽2016」常規賽冠軍；學校音樂創藝展「最佳音樂獎銀獎」及「最佳意念獎銅獎」；校際音樂節「小學中文教堂音樂組別」合唱冠軍。
為鞏固學生課堂知識，擴闊學生視野，增加對祖國的認識，於本年四月到湖南進行「湖南歷史文化交流之旅」，詳情可瀏覽本校網頁。
