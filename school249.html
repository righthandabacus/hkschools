班級教學模式
小一至六小班活動教學、小一至四普通話教中文，小一至小三英文科小學識字閱寫計劃(PLP_R/W)，數學及常識科探究式學習，英文、數學設勵學班及奮進班，創意思教班（資優）等。
班級結構備註
小班能照顧學生個別差異，配合成長的天空、新來港學童課程、加強言語治療、感統及情緒小組，學校靈活運用資源促進全校參與支援不同學習需要的學生；推行資優普及化，成立人才庫，學生於資優教育學院接受培訓。
學習評估
全年全科測驗次數 : 2
全年全科考試次數 : 2
多元學習評估 : 加強進展性評估元素，按單元持續評估學生。採用專題研習匯報、科技學習報告、生活技能、閱讀報告、課堂觀察、學生自評、同儕互評和家長評鑑等多元化及多角度評估學生的學習。
分班安排 : 以小班教學，合作學習理念異質分班，讓不同學習能力的學生能在和諧共融的氣氛下，發揮所長，不同特質和能力的學生互相啟發，彼此幫助， 一同學習。

健康校園生活 : 健康政策：重視身心靈健康，以福音陶成人格，提倡健康飲食，舉辦水果月、環保午膳、 監察餐盒營養質素，推廣體適能獎勵計劃，鼓勵學生多做運動。
備註 : 以建設安全校園、環保校園和健康校園為綱領，讓學生在愉快和優雅的學習環境下學習。設香草園、蝴蝶園及小農夫耕種園地，增強學生環保意識，養成健康的生活習慣和態度，本校為香港中文大學健康軍團南區支部，負責聯絡區內中、小學，一起推廣健康訊息。

辦學宗旨
秉承基督博愛及服務精神，實踐校訓「信、望、愛、敬」，將學校發展為學習型組織。發展六育，透過多元學習，提昇學生素質，熱愛中華文化，培養正確價值觀，勇於服務他人，成為有學養及愛國愛港的良好公民。
學校特色
學校管理
學校管理架構 : 法團校董會下設校監、校長、校友會及家教會，學校發展委員會，行政會等，校政具民主化及透明度。
法團校董會/學校管理委員會/校董會 : 辦學團體校董、獨立人士校董、校長、教師校董、家長校董、校友校董。
環保政策 : 設再生能源儲電系統；綠化校園：校園小農夫、有衣食計劃、校園零廚餘計劃、蝴蝶園、香草園、推行校園4R、綠色校園、獲減廢標誌銀獎，另設三色回收箱及自備環保餐具；水果月，鼓勵學生每天吃早餐，不設小食部，學生自備綠燈小食，推動師生使用果皮酵素代替洗潔精。
學校關注事項 : 本年度主題是「主賜生命常感恩，華寶校園顯愛心」。讓學生懂得珍惜身邊的一切，以服務表示對別人的愛心。關注事項：1. 開展自主學習，加強學教自主意識與能力。2. 探討生命意義，培育學生正向生命價值。
教學規劃
學習和教學策略 : 推行小班教學，以學生為本的活動教學模式，並進行課程統整。提升資訊科技設備，運用網上學習，豐富學生學習經歷，發展學生共通能力。各級中、英、數、常教師共同備課，運用網上學習資源，全校學生可使用英文科網上學習平台、E-CLASS內聯網平台加強學習，又設PLP_R/W小學識字閱寫計劃，外籍英語教師締造英語學習環境。推行電子學習，教師以平板電腦教學，全校共66部平板電腦，及Wi-fi 100計劃，可供兩班學生同時上網學習。圖書課使用電子書，配合圖書獎勵計劃。
關鍵項目的發展 : 設每天晨讀、故事媽媽及友校伴讀、推行閱讀獎勵計劃。小一至六運用電子白板及平板電腦進行互動學習，增加學生的學習興趣。並設全校跨學科專題研習及分享匯報、周會德育講話、優化「感恩」、「寬恕」計劃，推行校本「生命教育課」、成長課及國情教育。每年均舉辦主題學習、境外遊學、科學及科技學習日、中華文化日、親子學習日，家長與學生一起參與全方位學習；學習分享日，STEM DAY等全校家長都有機會到校觀課，加強老師和家長之間的聯繫。
共通能力的培養 : 發展共通能力，著重溝通、解難，創意及批判性思考，提升自學能力。小班教學，推行合作學習模式，提高師生及生生互動，學生分組協作，照顧個別學習差異，推行資優普及化，以資優13式提升高階思維能力。
學生支援
全校參與照顧學生個別差異 : 全校參與照顧不同學習需要：設有駐校心理學家及言語治療服務，為小一學生進行及早識別和照顧學生的不同學習需要，為不同需要學生開辦針對性的小組，課前及課後設勵學班、奮進班、讀寫能力提升課程，家課、測考及課程調適，發展資優教育，設思考創意班及人材庫。
課程剪裁及調適措施 : 教師透過共同備課及參考專家意見，根據學生興趣和能力，以單元組織教學，調適教材。每完成一個單元，進行進展性評估，以監察學習情況。
家校合作及校風
家校合作 : 學校十分重視與家長溝通，設各級校長與家長早餐會，每月出版華寶家書，收集家長意見，並歡迎家長出席早會。家教會每年舉辦多次大型活動，如盆菜宴、新春團拜、家長興趣班、親子旅行、親子同樂日和周年大會，而家長義工經常到校服務。
校風 : 校風純樸，學生謙恭有禮。秉承寶血會辦學宗旨，建立和諧校園，彼此關愛，以基督博愛精神，推行全人教育，為學生創造成功舞台，使學生能發揮所長，建立自信。
學校發展
學校發展計劃 : 1. 推行以福音為最高價值的生命教育，陶成學生美好人格。2. 建立綠色校園及安全、健康和環保的學校環境。3. 全校無線網絡、遠程教室、平板電腦和網上學習平台，提升學習效能。4. 深化校園正面文化，創造孩子成功舞台，提升學生自信與能力。5. 強化雙語學習，追求更高語文表現。6. 駐校心理學家策劃推行資優教育。7. 豐富學生健康知識和生活技能，發展身心成長，增強自信及抗逆能力。8. 建立正面的價值觀，服務學習，表揚犧牲及修和精神，培養勇於承擔的精神。9. 重視小一銜接，建設關愛校園。
教師專業培訓及發展 : 中文大學支援人員到校與老師進行課堂學習及研究，推動教師在自主學習和STEM方面的發展照顧個別差異、資優教學、專科專教和兩文三語的發展計劃，鼓勵教師進修及進行校內專業分享，鞏固教師團隊。辦學團體每年均舉辦多元化的工作坊，讓教師掌握教學新技巧，減壓及彼此分享。
學費減免 : 不適用。
其他 : 推行正面文化，架設成功平台，提升完善能力，推行服務學習、Superkid獎勵計劃、舉辦萬千星輝耀華寶頒獎典禮，我的溫習行動、學習歷程放大鏡家長日。小記者，發揮創意；主題學習、科技學習日等。本校為香港中文大學健康軍團南區支部，負責聯繫區內中、小學，同推廣健康訊息。
