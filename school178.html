辦學宗旨
推行「全人教育」，提供一個優良的學習環境，發揮個人的潛能，日後成為具備知識、技能、有獨立思考能力、勇於承擔責任和關注社會事務的良好公民。培育學生有正確的價值觀和積極的人生觀，勉勵他們拓展豐盛的人生。
學校特色
學校管理
學校管理架構 : 設學校管理、課程發展、訓輔、學生支援、課外活動及資優教育六大組。
法團校董會/學校管理委員會/校董會 : 已成立法團校董會，成員包括辦學團體代表、校長、教師、家長及獨立人士。
環保政策 : 設環保大使、安排環保講座、參與環保活動及比賽、進行廢物回收、採用可循環再造的飯盒及實踐低碳生活。
學校關注事項 : 1. 深化學生學習的積極性及主動性，從而提升學生學習效果。
2. 培養學生的正向價值觀。
教學規劃
學習和教學策略 : 學校靈活運用課時進行大班、小班或小組教學，採用「合作學習」教學模式，提升學生學習動機及互動性，訓練學生共通能力及照顧學生個別差異。
關鍵項目的發展 : 【著重兩文三語】於初小加入「英數」及「英常」校本課程。英文科採用「青田陽光英語綜合教學法」。中文科推行「以普通話教授中國語文校本計劃」。
【提升學生運用資訊科技學習能力】透過應用電子學習計劃，提昇學生的學習興趣及建立互動的學與教模式。
【發揮學生的潛能】透過不同的課外活動及比賽，擴闊學生視野以豐富他們的學習經歷。
共通能力的培養 : 透過不同形式的學習活動，如專題研習、服務學習等，培養學生共通能力、提升學生的學習效能及豐富學生的學習經歷。
學生支援
全校參與照顧學生個別差異 : 透過小組教學、加強輔導班及輔導教學，照顧學習能力稍遜的學生；設語言特工隊發展學生潛能；另設有中文科分層支援模式及小老師伴讀計劃，鞏固低年級學生的基本語文能力和提高識字量；為有需要的學生提供言語治療及教育心理學服務，安排學習能力評估、社交技巧小組及遊戲治療。
課程剪裁及調適措施 : 於課堂教學、功課量、默書範圍及評估時間作調適。
家校合作及校風
家校合作 : 家長教師會舉辦親子活動、興趣班及家長學堂，另設家長交流角收集家長的意見，與學校建立良好的夥伴關係。
校風 : 學校重視學生群性發展，整體氣氛融洽和諧，以「用心學、悉心教、愛心導」為願景，著重以「愛心」及「耐心」去教導孩子，使成為積極學習、知禮守規、尊己愛人的未來棟樑。
學校發展
學校發展計劃 : 1. 在數學及常識科加入英語教學元素，加強升中銜接。
2. 引入多元化的英語學習活動，提升學生的英語水平。
3. 善用遠程教室，擴闊學生視野，為學生提供不同類型的全方位學習活動。
教師專業培訓及發展 : 校方重視教師專業發展，為教師安排多元化培訓課程，提升教師教學及輔導學生品行的技巧。
學費減免 : 不適用。
其他 : 設有院本不同獎學金，嘉許學業成績卓越、有進步或課外活動表現出色的學生；另外，亦會推薦有潛能的學生參與校外的獎勵計劃，至於成績優異學生會被推薦參與國際聯校學科評估。此外，本院設有東華三院董事局考試費獎勵計劃，鼓勵英語學習的尖子參與具知名度的校外英語考試。
