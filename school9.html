全方位學習活動

辦學宗旨
嘉諾撒教育團體， 以十字架上基督為典範， 以會祖聖嘉諾撒瑪大肋納的「心靈陶育」為依歸， 實踐全人教育， 培育學生熱愛生命、積極學習、關懷弱小及服務社群的精神。
學校特色
學校管理
學校管理架構 : 法團校董會轄下依次是校監、校長、中層、各級統籌及科目統籌小組，制訂、監察、執行及評估校本發展方向及政策。
法團校董會/學校管理委員會/校董會 : 2013年9月成立。
環保政策 : 實施「學校廢物分類及回收計劃」。
學校關注事項 : 1. 優化學校行政架構，提升管理效率。
2. 進一步提升學生自主學習能力。
3. 活出聖心人友愛共融、謙和仁愛的特質。
教學規劃
學習和教學策略 : 靈活安排課時及學習活動；推行全方位探究式學習及專題研習，注重學習過程，學生自我建構知識；著重多層次提問技巧、課堂互動；善用促進學習的評估，以培養學生高階思維；發展電子學習，促進學生自主學習及個別化學習；拓寬閱讀面，發展跨課程閱讀，擴闊學生視野，提升自學能力。
關鍵項目的發展 : 1. 跨課程閱讀、中英文網上閱讀計劃、美文博覽角、家長伴讀、大姐姐伴讀、英文閱讀室、操場閱讀小天地。
2. 電子學習促進學與教活動、電子學習平台、專題設計、電子論壇、資訊科技比賽。
3. 全方位探究式學習、跨領域專題研習、STEM教育。
4. 生命及品德教育、個人成長教育課、生命成長手冊、心靈培育、校本輔導計劃。
共通能力的培養 : 注重培養學生各種共通能力，特別著力提升學生探究學習及自主學習的能力。
學生支援
全校參與照顧學生個別差異 : 由學生支援小組統籌及推行校本學習支援計劃：設功課輔導班、校本言語治療服務，並引入資源老師及專業服務支援有需要之學生。
課程剪裁及調適措施 : 翱翔小組（中英數學科輔導）、中英數拔尖課程，額外資源老師照顧能力稍遜及有特殊學習需要的學生。
家校合作及校風
家校合作 : 重視家校合作，定期茶聚加強溝通。家教會舉辦多元化家長教育及親子活動，發展家長義工團隊協助推展學校活動。
校風 : 學校宗教氣氛濃厚，著重心靈及品德培育，透過全方位學生輔導活動、心靈培育、生命及品德教育、成長課及各項活動，協助學生建立積極的人生觀及正確的價值觀。推行校本獎勵計劃鼓勵學生自我挑戰、力求進步。推動學生參與義務工作，主動關懷社區，發揮會祖仁愛謙遜的服務精神。重視幼小銜接，安排幼稚園學生到校參觀，舉辦「小一新生日」，並適切剪裁小一課程。
學校發展
學校發展計劃 : 1. 跨課程閱讀。
2. 善用電子學習提升學與教的成效。
3. 全方位探究式學習。
4. 思辨課程。
5. 「聖心小天使」獎勵計劃。
6. 促進學習的評估。
7. 資優教育。
教師專業培訓及發展 : 配合學校周年計劃及關注事項，為教師舉辦相關的工作坊和分享會，配合校本發展。每位教師訂定個人進修計劃，持續提升專業能量。
學費減免 : 不適用。
其他 : 新西蘭遊學團、台灣環保學習交流團。
