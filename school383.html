辦學宗旨
提供全面的優質教育服務，培養學生良好的品格和正確的人生觀。
學校特色
學校管理
學校管理架構 : 法團校董會、一位校長、副校長、行政小組及各科教職員。
法團校董會/學校管理委員會/校董會 : 法團校董會於2008年8月成立。
環保政策 : 廢物分類、紙張雙面印刷、自備餐具、自備手帕、舊校服及舊課本轉贈、回收舊電子用品。
學校關注事項 : 本年度學校發展關注項目：
1.發掘領袖才能 以愛服務社群
2.加強學生自主學習能力，照顧學生學習多樣性
3.拓闊閱讀視野，推廣分享文化
教學規劃
學習和教學策略 : 跨學科課程，以生活體驗配合不同學科課題，實行全方位學習。英語科進行PLP-R/W教學計劃，數學科校本課程研習、內聯網系統、靈活運用課時及共通能力的培養。發展電子學習：讓學生通過使用平版電腦，配合電子教學新模式，發展高階思維學習。
關鍵項目的發展 : 1. 從閱讀中學習︰中、英文廣泛閱讀計劃、親子閱讀計劃、早讀、班級讀書會、好書推介、讀報心聲。
2. 運用資訊科技進行互動學習。
3. 利用跨學科模式進行專題研習。
4. 成長課、德育主題演講。
共通能力的培養 : 共通能力的培養滲透在各科的教學活動中。
學生支援
全校參與照顧學生個別差異 : 除了設有加強輔導班之外，還設有課後提升班、非華語課後輔導班及學習小組（語文、情緒、社交技巧）。
課程剪裁及調適措施 : 課程剪裁及調適配合校本課程發展、學生學習的需要。設有課後各科能力提升班、英語加強訓練活動。
家校合作及校風
家校合作 : 家長教師會經常舉辦聯誼活動如親子旅行、興趣班、廚藝班等。校方亦為家長開設多項教育子女的工作坊。
校風 : 1. 培養和諧的師生關係，學生以週記作為與老師溝通的渠道。
2. 提升學生對學校的歸屬感，安排新生適應活動，設學校與家庭通訊。
3. 加強家校合作，設有家長教師會，定期舉辦活動。
學校發展
學校發展計劃 : 1. 部份四、五、六年級學生利用平板電腦全面進行電子學習。
2. 推行流動電子互動教室。
3. 英語科參加教育局「DTS seed project」。
4. 延伸英語閱讀及寫作計劃。
5. 優化常識科校本課程及發展資優教育課程。
6. 學生能利用電子學習平台定期進行自學。
教師專業培訓及發展 : 按照學校發展方向，提供教師培訓課程。
學費減免 : 不適用。
