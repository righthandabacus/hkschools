辦學宗旨
忠於天主教教育原則及母佑會辦學精神，以仁愛、理智、宗教，愛而不溺、嚴而不厲的預防教育法，合情理的溝通交談，並以宗教知識及情操推動學生積極善用生命，愛護、尊重自己及他人，以信心及喜樂面對生命的順逆境。
學校特色
學校管理
學校管理架構 : 辦學團體及校董會成員以慈幼教育精神辦學校，兩位副校長及各工作小組協助校長推行各種教育方案。
法團校董會/學校管理委員會/校董會 : 聖母小學校董會
環保政策 : 環保為週訓主題，學生作專題研習、短劇、回收等活動。
學校關注事項 : 1. 提升學生學習效能、創意思維和自學能力。
2. 加強學生宗教及品德教育，培養正面價值觀。
3. 提升教師專業質素。
教學規劃
學習和教學策略 : 參觀講座。外籍人士會話班。全方位協作學習、溝通、創造、批判性思考、解決問題、自我管理等跨科目活動。
實施下午時間後進行閱讀、活動及導修時間，讓學生盡可能在學校完成大部分功課。
關鍵項目的發展 : 1. 全年十份報告。編輯《創思集》第十六集 "Enjoy Your Reading" 第十五集、《創藝美集》。
2. 推行電子教學及多元智能的發展。
3. 編定教育大綱，各科以此為主題。公民及品德教育，老師同學以多元化形式傳遞。
4. 注重學生全人發展，除注意學業成績外，也鼓勵學生在音、體、藝各方面的發展。
共通能力的培養 : 教育大綱以「培養自我管理的能力」為主題，校本輔導活動配合，請家長評估。
學生支援
全校參與照顧學生個別差異 : 教師課後輔導。同輩輔導。
課程剪裁及調適措施 : 教師共同備課後增刪教材並設計補充資料。
家校合作及校風
家校合作 : 家教會成立十九年，家長積極熱誠，親子活動600多人參加。心聲：聖母小學是一間與時並進的優質私校。優質的老師教出品學兼優的好學生。我的女兒很喜歡上學，愉快地學習。
校風 : 慈幼愛的教育法。簡樸衣飾、和諧友愛、喜樂精神；學生們喜歡老師，高年級喜歡低年級，低年級喜歡高年級姐姐的帶領。舉行家長講座及小一先修班及迎新日。參觀校園、模擬上課及集隊，讓小朋友盡早適應。學校特別實施「校園特務」計劃，讓高年級同學幫助小一生。
學校發展
學校發展計劃 : 1. 本校所有老師均不斷進修。
2. 每個課室提供電腦及投影設備，學生以電腦搜集及製作專題研習報告為通識學習作好準備。
3. 本校中小學部教師作定期交流，以配合學生升讀直屬聖母書院；同時也和幼稚園有密切聯絡及分享，讓幼稚園學生能盡快適應小學生活。
4. 教師帶領學生及家長到北京、西安及山東作全方位學習。
教師專業培訓及發展 : 慈幼教育法及教師身心健康之教育講座及旅遊。每年教師發展日：多元化主題，與本港、澳門、台灣及新加坡交流。
學費減免 : 不適用。
其他 : 發展學生音樂、體育及視藝方面才能。增加普通話及外籍英語老師課節。帶領學生作海外交流及學習。
