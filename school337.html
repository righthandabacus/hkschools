班級教學模式
小班教學，每班25人，課堂以小組學習方式，以「高參與、高讚賞、高展示」的三高課堂，讓學生積極投入學習。推行資優教育，安排抽離式課程。
班級結構備註
小班教學。各科組採用不同的教學模式和策略，配以老師優化的提問技巧，即時回饋，協助學生解難。

全年全科測驗次數 : 0
全年全科考試次數 : 4
多元學習評估 : 各科運用多元化評估模式檢視學生的學習情況。中文、英文、普通話科包括說話及聆聽評估，常識及音樂科則包括專題研習。學業成績以外，每學期均有《學生學習表現報告》，讓家長了解學童上課表現及品行情況。
分班安排 : 各級平衡班制。

健康校園生活 : 1. 推行健康校園，鼓勵學生均衡飲食，參加一生一體藝，培養多元智能發展。2. 全校參與至營學校認證、中大健康校園網絡、體適能獎勵及護脊校園計劃。

辦學宗旨
在地區上為學童提供一個積極、高質素、富創意、和諧融洽、校風淳樸的學習環境接受教育。
學校特色
學校管理
學校管理架構 : 採取全校參與的科層管理模式，共同訂定學校發展計劃及關注事項，具備透明度及問責性。
法團校董會/學校管理委員會/校董會 : 31/8/2007成立法團校董會，共有法團校董13人。成員來自辦學團體、家長、教師、校友及獨立人士，定期商議校政，推動學校發展。
環保政策 : 積極參予環保活動及綠化校園計劃，並安裝多項節能及可再生能源設施。
學校關注事項 : 1. 優化學與教策略，促進學生的自主學習。
2. 優化學生支援策略，提升學生自律自勉的素質。
教學規劃
學習和教學策略 : 1. 以學生為本。
2. 重視高階思維培養。
3. 推展合作學習。
4. 幫助學生學會學習。
5. 培養學生自主學習能力。
6. 發揮學生多元智能。
關鍵項目的發展 : 1. 有圖書課、中、英早讀課、親子伴讀及「網上閱讀」等計劃。
2. 設有兩個電腦室，全校課室安裝電腦及投影機，並增設60部平板電腦，加強資訊科技的發展，推展電子學習。
3. 一至六年級設專題研習，五年級超學科專題研習由全校老師帶領小組進行探究學習。
4. 訓輔合一，以班主任課、生活課、成長小組、服務學習及校本輔導獎勵計劃培育學生。
共通能力的培養 : 培養學生批判性思考能力、溝通能力、協作能力、創意思考能力，亦透過專題研習培養學生共通能力。
學生支援
全校參與照顧學生個別差異 : 1. 設有加強輔導教學及中英數學科輔導。
2. 提供免費及收費託管服務。
3. 三層式資優教育課程。
4. 言語治療師駐校服務。
5. 教育心理學家駐校服務。
課程剪裁及調適措施 : 中英數常各科因應學生學習能力進行課程剪裁並調適課業，以提升學習效能。
家校合作及校風
家校合作 : 家長教師會作為家校溝通的橋樑，配合學校發展，推行多元化的校內及校外活動。家長代表加入法團校董會，參與校政。
校風 : 1. 校風淳樸，師生關係良好。
2. 建立關愛文化，支援學生成長。
3. 重視學生品德教育、責任感的培養，常規課加入生活課及班主任課。
4. 推行校本輔導獎勵計劃，培養學生建立正面價值觀。
學校發展
學校發展計劃 : 1. 優化學與教策略，促進學生的自主學習。
2. 優化學生支援策略，提升學生自律自勉的素質。
教師專業培訓及發展 : 1. 舉行教師發展日。
2. 教師專業進修。
3. 推行集體備課。
4. 同儕及公開觀課。
5. 舉辦教師工作坊。
6. 教研分享。
7. 學校交流。
8. 推行教師夥伴計劃。
學費減免 : 不適用。
其他 : 注重教研工作，配合學校發展需要，參與校本專業支援計劃，引入大學及教育專業團體，亦與國內小學結成姊妹學校，優化教學，積極與校外專業團體交流，追求卓越。
