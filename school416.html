辦學宗旨
以基督的愛為根，以禮仁勤信為訓，使學生能在愉快的校園氣氛下成長，培養他們成為具學習知識、反思、批判性思考、自理和領導能力的人，能接納自己，尊重別人和大自然，具同理心，能與人建立互信，誠實正直，不怕冒險，願意從失敗中學習，承擔責任，邁向卓越，願意服務他人和陪伴受服務者。
學校特色
學校管理
學校管理架構 : 學校由法團校董會管理，行政團隊包括校監、校長、副校長和助理校長。中層管理由行政會議和科組領導負責，與包括家教會和校友會的各持份者經常保持溝通。機構內的不同成員透過知識型領導，以信任為基礎的關係，在天主教價值氛圍和家長參與的前提下，為學生提供具耶穌會特色的教育。
法團校董會/學校管理委員會/校董會 : 按章程成員包括不多於七位辦學團體校董、校長校董、一位教員校董、兩位家長校董（上下午校）、一位校友校董和一位獨立人士校董。替代校董包括一位辦學團體替代校董，一位替代教員校董和兩位替代家長校董（上下午校）
環保政策 : 制定環保政策，加強全校師生、職員及家長的環保意識，著重保護環境，重複、循環及減少使用。本年度參加了香港教育大學的「塑膠資源教育─3Rs&3Cs」計劃，在校園增設了八格塑膠回收箱，把不同的塑膠類別分別回收。在課室加設小型回收箱，鼓勵學生把廢紙、塑膠及金屬回收。
學校關注事項 : 1. 提升學習效能和學業成績（特別是中文、英文和數學），並開展資優教育的跨科組和科本發展。
2. 提升學生品德質素，特別是與同理心、仁愛和功利心態有關的價值觀。
3. 提升學生的自理能力。
教學規劃
學習和教學策略 : 為培育學生具有華小畢業生的特質，兼具知識和良好的品德，推行中小銜接的資優導向校本課程：如：四至六年級中文科校本課程、五、六年級英文寫作課程、專題研習等。
關鍵項目的發展 : 優化評估促進學習的發展。推行校本德育課程，以配合本校畢業生特質，以提升學生自理能力及強化正確的價值觀，以實現本校的遠景。
共通能力的培養 : 以課前預習培養學生的自學習慣，課堂教學則以提問提升學生思維能力，以小組學習培養協作能力。
學生支援
全校參與照顧學生個別差異 : 1. 甄選及訓練資優生參與抽離式課程及比賽。
2. 設學生、家長及校友人才庫。
3. 學生支援小組與科任老師協作，為個別學生安排適合的教學和輔導策略及調適。
4. 外購服務小組和言語治療服務。
課程剪裁及調適措施 : 中文科四至六年級校本課程： 五、六年級英文寫作課程；數學一、二年級躍進課程、常識科專題校本課程。
家校合作及校風
家校合作 : 家教會每年舉辦家校活動，參加者極踴躍。學年初為初小學生家長安排導引講座。家長義工協助學校活動，全年無間。家教會家長委員與教師全年協作，支援學務。
校風 : 重視學生德育與靈育，培養學生自律守規的態度。通過各種獎勵計劃，激發學生自我完善。五、六年級學生全體均擔任風紀工作，表現積極投入。家長參與義務工作，並不時與教師交流，共同攜手，使學生愉快地成長發展。我們訂定了畢業生特質，期望學生在畢業時能達到追求卓越的目標，學生更能透過各種的義工訓練及服務，做到服務他人，貢獻社會。
學校發展
學校發展計劃 : 1. 提升學習效能和學業成績。
2. 開展資優教育的跨科組和科本發展。
3. 提升學生品德質素。
4. 提升學生的自理能力。
教師專業培訓及發展 : 配合當今教育發展，深化分組學習、提問技巧等小班教學策略，並加強多角度思考教學策略培訓。
學費減免 : 不適用。
其他 : 2017世界紙飛機飛行大賽小學組全場總亞軍、最長飛行時間冠軍、最長飛行距離亞軍；第69屆香港學校音樂節教堂音樂中文歌曲13歲或以下冠軍、外文歌曲男子童聲季軍；2017香港花卉展覽小學組最佳盆栽種植／培植優勝盃、仙人掌類植物、賞葉植物及肉質植物冠軍、蕨類植物季軍；港島東區小學游泳比賽團體男甲冠軍、男乙殿軍、男丙亞軍；香港學校朗誦節獎項，校際田徑、乒乓球、排球比賽獎項。
