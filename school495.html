班級教學模式
全校小班教學及活動教學，以合作學習提升學生參與課堂學習機會。一至六年級逾一半班級實行雙班主任制，另外學校有五名英語外籍老師協助英文教學。
班級結構備註
小一推行普教中；小四至小六推行電子學習計劃；小三及小五設延展課程，小六設補課。另設課後的區本輔導班、小豆苗班、伙伴計劃等。

備註 : 屢獲關愛校園及活動校園殊榮，近年更致力發展資優教育，課程設計也著重啟發學生思考、培育創造力及發展社交能力，充分發揮個別潛能。校園生活多姿多采，設生日會、遠程教學、活力廣場、護眼操，以及課後四十多項體藝訓練及輔導。
全方位學習活動
為促進學生均衡及具個性的發展，安排資優教育培訓、公益活動、校園電視台、田徑及球類訓練、游泳、劍擊、制服團體、時代舞、英語話劇、STEM 小組、珠算應用、中樂團及合唱團等，啟發學生多元智能，體驗成功。
辦學宗旨
本校遵照政府的教育政策，秉承校董會「以學生為本」的辦學精神，以校訓「博學篤志」為經，「德智體群美」為緯，提供全面的優質教育服務，培養學生良好的品格和正確的人生觀。
學校特色
學校管理
學校管理架構 : 學校管理架構的設定參照教育局的學校表現指標架構，由校長和副校長領導三大範疇的主任及老師，並輔以各委員會，制定學校政策及發展方向，決議後交各科組商討施行細則及執行方法。
法團校董會/學校管理委員會/校董會 : 全港第一間成立法團校董會之小學。校董會在資源上支持學校，如提升資訊科技教學資源等。
環保政策 : 設環保大使、舉辦廢紙回收比賽等。全校廣泛張貼隔熱膜，減少陽光照射，節省冷氣用電，增設感應燈。
學校關注事項 : 2015-16至2017-18年度：善用資訊科技、促進自主學習、做個堅毅公益人。
教學規劃
學習和教學策略 : 一至六年級推行小班教學，安排不同的課堂活動讓學生透過合作學習建構知識。早會透過校園電視台作全校廣播，早讀課安排中英文篇章閱讀，午間開放活力廣場及各科特別室，課後設輔導、補課及訓練活動。一年級設「動感藝術課」，讓學生愉快學習。各級課堂教學及課業著重發展學生的高階思維。每年安排學生進行境外交流及到本港不同地點進行戶外學習活動，擴闊視野。
關鍵項目的發展 : 於小一至小六舉行專題研習活動，四至六年級更配合服務學習課程。透過早讀課、中文科、英文科及圖書科日常教學，教授學生閱讀策略。積極推展資訊科技教育，培養學生自主學習。校園電視台配合德育科培養學生正面的價值觀和態度。
共通能力的培養 : 於一至六年級設專題研習及科技日，發展學生共通能力及科探能力。各科日常教學中，提升學生溝通、協作及解難能力，並注重批判性思考訓練。
學生支援
全校參與照顧學生個別差異 : 在一至六年級推行小班教學，加強學生照顧；另設學生支援小組，並鼓勵家長參與支援工作，令學生能愉快學習。本校重點推行資優教學，為照顧資優學生，學校設有「English Drama Elite」、校園電視台等資優課程，以提高他們的高階思維能力、創造力和領導力。
課程剪裁及調適措施 : 安排適當的學習調適予不同需要學生。在各科校本工作紙中設有不同程度題目，提高學生學習動機。設評估調適措施，讓學生得到適切的評估。
家校合作及校風
家校合作 : 教育局外評小組指出本校獲家長支持及信任，家校合作表現優異。學校視家長為學校的重要夥伴，家長支持和認同學校發展，對學校有強烈的歸屬感。本校透過家長會、工作坊、講座等渠道，增強聯繫，並讓他們更了解本校發展資優及資訊科技教學的策略及方向。
校風 : 獲外評報告讚揚校風淳樸、學生待人以誠、有禮。學生熱愛校園，尊敬師長，睦友愛群。校長關顧教職員和學生，凝聚校園動力，上下齊心，力求改善。教師欣賞及關愛學生，師生關係良好。教師團隊積極進取，互相扶持，相處愉快融洽，團隊精神良佳。本校連續八年榮獲由香港基督教服務處主辦的關愛校園獎，反映本校重視關愛文化，建立一個校風純樸的校園。
學校發展
學校發展計劃 : 科技發展，一日千里。為順應時代趨勢，並配合教育局發展資訊科技教育的策略，我校以「善用資訊科技，促進自主學習，做個堅毅公益人」作為三年發展計劃的關注事項。我校於2015年獲得政府撥款四十多萬，成為首批鋪設能覆蓋全校的無線網絡的學校，讓我們有足夠配套設施以資訊科技實踐課堂教學，藉此提升學生的自主學習和解難能力。我們期望學生即使面對困難，亦能借助資訊科技的力量及自主學習的精神跨越重重難關，成為學習表現卓越的堅毅公益人。
教師專業培訓及發展 : 教師持續發展能讓教學工作日益精進，教師得以持續發展，同時也能改善學生學習。為有效推行教師發展及培訓工作，學校特意成立教師專業發展委員會，製作教師專業發展記錄冊。本校教師發展及培訓包括：集體備課、工作坊或講座、同儕觀課、教研課、校內專業進修分享會、教師發展日、短期培訓課程、復修課程、交流營、教育局支援、聯校學習及申請支援計劃。而全校老師亦已進行資優教育培訓。
學費減免 : 不適用。
