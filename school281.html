全方位學習活動
課程設才華班，發掘不同潛能。須參加一項制服團隊如男、女童軍、教育局公益少年團、交通安全隊或少年警訊；另有舞蹈組、朗誦、合唱團、交響樂、球類及學科學會。透過活動或比賽獲得成績，增強自我形象及自信心。
辦學宗旨
本校致力建構國際化校園，推行「全人教育」，提倡以愉快學習之道，讓學生盡展潛能，成為具備卓越溝通能力、積極心態及國際視野之人才，迎接廿一世紀之挑戰。
學校特色
學校管理
學校管理架構 : 本校之辦學團體為聖瑪加利教育機構（非牟利慈善團體），由學校管理委員會管理及監察學校。學校管理委員會成員包括主席、校監、辦學團體委員、校長、家長委員、教師委員、舊生委員及獨立人士委員。
法團校董會/學校管理委員會/校董會 : 學校管理委員會。
環保政策 : 本校致力推廣綠色文化，考慮不同校園環保節能方案，另舉辦不同的環保活動，例如響應「地球一小時」節能活動及學校食物部合作舉辦「無飲管日」，獲環保署撥款購入廚餘機並教導學生「惜食」概念及推行循環再造廚餘計劃。另曾獲綠色學校獎及「惜水學堂」計劃獎項。
學校關注事項 : 三大關注事項：強化資源、優質教學、關愛品格。
教學規劃
學習和教學策略 : 重視三文四語。主要科目包括英文、中文（以普通話教授）、現代語言（包括法文、德文、日文或西班牙文）、數學、常識及資訊科技，學習領域以專題研習及多元化的學習活動提高學生的學習興趣及培養其自學能力。主要科目同一時間授課，配合不同學生需要。閱讀計劃、學習角、探訪、專題報告有多方面探討、以終身學習為目標。跨科活動與生活體驗結合，鼓勵學生獨立思考，培養其創意及協作能力。
關鍵項目的發展 : 設早午閱讀及閱讀獎勵計劃。資訊科技：設學會學習課，實踐學習範疇。專題研習：定期跨科專題研習，培養高階思維及協作能力。德育及公民教育：透過早、週會及生活價值教育課，在德智體群美靈有均衡發展；培養責任感及毅力，成為21世紀良好公民。
共通能力的培養 : 設參觀交流及海外遊學團，走出校園，步入社區以至世界，主動尋求知識，培養獨立思考，提升解難能力，並發展溝通及社交技巧。
學生支援
全校參與照顧學生個別差異 : 中英數科設增潤及輔導課、創意寫作、趣味數學及劍橋英語課，鞏固及深化所學。「大姐姐，大哥哥」計劃，中學生提供功課輔導。
課程剪裁及調適措施 : 推行英語正音計劃：小一至小四教授英語拼音，小六至中二教授國際音標法；設完善的資優教育計劃，擁有相應培訓的師資，按資優生需要而安排其課堂，貫徹推行拔尖補底政策；制作校本英語教材，照顧不同程度的學習需要；各科均作課程調適，助學生能更快適應課程，提昇學習效能。
家校合作及校風
家校合作 : 家長明白雙方溝通對學生成長有一定的幫助。本校協力與家長改善學生之品德及學業；並推動家長教育。設雙班主任制。每年舉行一至兩次家長教師聚會，以促進雙方之了解。
校風 : 校風淳樸，注重學生品德修養，「訓輔合一」，令學生能積極地面對困難；教師以愛心及耐心作經，家長以關懷及期望作緯，交織成「以學生為本」的教育。特設生活價值教育課，培養學生積極人生觀、情緒健康及基本生活技能。
學校發展
學校發展計劃 : 1. 視英語為學校生活語言，由小一開始奠定良好基礎。透過活動和互動的教學形式，引發學生的自學精神。2. 為配合學生的學習需要及學校的五年發展計劃，學校正籌劃在現址進行校舍發展，目標是增加一個游泳池及學與教之空間，有利小班教學、新高中分組及「一生一體藝」之推行。學校將可更靈活地為學生繼續提供優質教育。
教師專業培訓及發展 : 鼓勵老師視乎學生需要及課程改革進修，以最少三年共150小時為目標。
學費減免 : 本校設有學費全免或減免1/2學費。有需要的學生(包括來自綜援家庭的學生及領取學生資助辦事處財政資助的學生)可提出申請學費減免。
- 新生可於開學後的2個星期內遞交申請，其申請結果會在1個月內獲得通知。
- 本校學費減免計劃詳情包括家庭入息限額，請瀏覽網上版的超連結。
其他 : 1. 本校重視三文四語，英文及中文（以普通話教授）為必修科，非華語學生可於小二修讀中文為第二語言。小一開始修讀現代語言；小一至小二分別學習法文、德文、日文或西班牙文，讓其淺嘗學習不同語言的趣味，小三階段起可按興趣選修其一。2. 設學科獎學金、體藝獎學金、品行獎、進步獎、科學獎等。並獎勵於體藝方面成績優異的學生參與國際性的交流。
