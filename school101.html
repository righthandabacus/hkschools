班級教學模式
設中英數小組輔導，課前輔導成績落後的學生，另加設加強輔導計劃小組，課內輔導成績稍遜學生及課後托管班，指導學生做功課。全校學生均可參加。

多元學習評估 : 小組討論、專題研習、口頭報告、觀察、進展性評估及總結性評估等方法來評估學生，然後調整教學進度及策略。利用小測及進展性評估，幫助學生學習。

辦學宗旨
以基督真理培育學生，讓學生在信、望、愛中全面地發展，為未來升學及服務社會奠下良好基礎。
學校特色
學校管理
學校管理架構 : 校董會 → 校長 → 副校長 → 主任及學位教師 → 教師
法團校董會/學校管理委員會/校董會 : 本校法團校董會於二零零八年八月三十日正式成立。
環保政策 : 鼓勵學生節約能源，學校每年舉辦舊書回收及舊衣回收等活動。
學校關注事項 : 1.推動中層管理人員的專業發展，強化中層領導角色。
2.善用評估資料回饋教學策略，提升各級同學的學習水平。
3.推動學校各項德育及健康活動，致力建構具關愛文化的健康校園。
教學規劃
學習和教學策略 : 刪改課程以配合學生需要，舉辦不同形式活動讓學生透過多元化的學習模式學習。跨學科課程、戶外學習、專題研習、珠心算班、奧數班、電腦班、西人英語班。
1. 專題研習推行跨科課程。
2. 大型活動推行跨科課程：中國文化雙週。
3. 各科課程因應學校目標互相配合。
關鍵項目的發展 : 1. 中央圖書館藏書豐富。
2. 推行親子伴讀計劃。
3. 高年級學生成為伴讀小天使。
4. 全校課室寬頻上網，配合投影機，方便教師授課。電腦室更能提供一人一機；圖書館則已電子化，學生可透過電腦查閱圖書館資料及可透過網上學習。
5. 一至六年級全面推行專題研習。
6. 推行公民教育，更加入生活教育元素。
7. 訓輔合一，積極鼓勵學生。
共通能力的培養 : 推行創意寫作、批判性思維及專題研習等計劃，培養學生共通能力。
學生支援
全校參與照顧學生個別差異 : 1. 中英數輔導課。
2. 開設加強輔導小組。
3. 調適課程配合不同能力學生。
4. 開設拔尖小組幫助能力強之學生。
5. 課後開設托管班，指導學生做功課。
課程剪裁及調適措施 : 課程調適、拔尖及補底計劃。個別學生功課及考試調適。
家校合作及校風
家校合作 : 家長、學生及教師活動：郊遊、講座、參觀。
校風 : 校風良好，推行小一新生適應、風紀培訓及訓輔合一等活動，配合家長與學校的良好關係，培養學生自動自覺，敬師孝親的精神。
學校發展
學校發展計劃 : 未來重點在課程改革，教導學生思維及記憶方法，用創意教學及創意學習幫助學生進一步提高水準。
教師專業培訓及發展 : 教師培訓重點：思維教學，從閱讀中學習及高效能小班教學。
學費減免 : 不適用。
