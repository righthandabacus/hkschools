健康校園生活 : 本校成立午膳監察委員會，監察供應商所提供食物的質素。透過多類型健康生活推廣計劃，宣傳健康飲食訊息。每周設全校早操活動鼓勵學生鍛鍊身體。
備註 : 本校每星期設一節多元智能課，推行一生一體藝計劃，發掘同學不同潛能；每位學生最少參加一項體藝活動；鼓勵同學愉快學習，彼此建立，以基督精神，學習律己愛人。
全方位學習活動
本校舉辦多項活動，發掘學生多元智能：團契、圍棋、科學探究、歌詠隊、中樂、弦樂、視藝、跆拳道、中國舞、朗誦、辯論隊、奧數、書法、游泳、足毽、園藝、校園電視台、中、英語話劇、公益少年團、少年警訊、基督少年軍、幼童軍、各項球類活動等。
辦學宗旨
以聖經的真理作為教育的基礎，結合現代教育理論與方法，提供優質全人教育，使學生能明辨是非，追尋真理，在靈、德、智、體、群、美六育得到均衡發展。
學校特色
學校管理
學校管理架構 : 本校為香港九龍塘基督教中華宣道會屬下小學。由法團校董會管理，校長、兩位副校長及三位學位主任帶領，設立不同工作小組，共同訂定及實行學校發展計劃。
法團校董會/學校管理委員會/校董會 : 2008年成立法團校董會。
環保政策 : 綠化校園。以工作坊、回收活動向學生灌輸環保意識。
學校關注事項 : 實踐基督教全人教育；建立溫馨關愛校園。
教學規劃
學習和教學策略 : 推行資優教育，啟發學生潛能，通過全班及抽離式的資優教育課程如科學探究、戲劇教育、領袖訓練等活動，提升學生的創意和批判思考能力。靈活運用電子學習互動活動提高學與教效能。又以關愛提升學生自尊感及學習動機，養成自主學習精神。
關鍵項目的發展 : 學生掌握閱讀策略，從閱讀中學習；推行電子學習，培養學生資訊素養，建立網上教學資料庫，鼓勵學生進行互動學習；進行全方位德育及公民教育活動，推動生命教育，培養學生正面價值觀和態度；常識科發展專題研習、科技教育、縱向課程以促進學習。
共通能力的培養 : 透過領袖生培訓、境外交流活動、戲劇教育、專題研習、科學及科技教育、合作學習等培育學生共通能力。
學生支援
全校參與照顧學生個別差異 : 於課堂進行合作學習及濃縮課程，以加廣、加深、加速原則推行適異性課程，如普及思維策略培訓、抽離增潤課程、校外支援拔尖課程等。一至六年級均設外籍英語老師協作課堂，中、英文科設小組教學，加強學生的中、英語能力。
課程剪裁及調適措施 : 為不同學習能力學生進行課程剪裁，為有特殊學習困難之學生作功課及評估調適。
家校合作及校風
家校合作 : 定期舉辦小一家長班、家長教育活動及興趣小組等，為家長建立學習社群及聯繫網絡。家長定期參與不同的校內和校外義務工作，如故事姨姨、午膳姨姨、親子義工服務等。家長教師會設家長資源中心，並出版家長教師會會訊。定期舉行家長接見日及出版校訊，加強家校溝通協作。學校以電子通告與家長聯繫。
校風 : 以靈育培養學生品格：有禮守規、尊敬師長、律己愛人、服務關懷、積極人生。訓輔合一，設德育成長課、小一新生適應課程及升中輔導。
學校發展
學校發展計劃 : 2015-2018學校發展項目：
善用多元策略，培養自學能力。
營造正向文化，共建愉快校園。
優化知識系統，延續專業分享。
教師專業培訓及發展 : 透過研討會及工作坊，提升教師的專業知識。組織學習圈，透過觀課、共同備課及友校交流活動促進教學效能。
學費減免 : 不適用。
