辦學宗旨
秉承校訓「勤、誠、禮、愛」的精神，推行全人教育，啟發學生潛能，讓學生在優美的環境中愉快地學習，全面汲取知識，掌握技能，成為一個具備美德、高尚人格及積極人生觀的良好公民。
學校特色
學校管理
學校管理架構 : 法團校董會為最高管理組織，校長管理學校的日常決策和事務，並領導校內各行政小組、科目及功能組別，推動行政及教學。
法團校董會/學校管理委員會/校董會 : 成立於2009年，成員包括：辦學團體校董、教員校董、獨立校董、家長校董、校友校董及當然校董。
環保政策 : 1. 鼓勵循環再造及回收物資。
2. 校園設環保回收箱。
3. 全年定期有舊書及舊校服回收活動。
4. 與午膳供應商簽訂環保協議，減少使用即棄餐具及器皿，並提供環保午膳餐具予學生使用。
5. 設有學生環保長。
6. 設有師生節省電力及節省紙張的指引。
學校關注事項 : 1. 培養學生律己愛人的態度，建立正向價值觀。
2. 營造積極學習的校園氣氛，提升學生學習能力。
教學規劃
學習和教學策略 : 1. 全人教育 正向思維 - 以「正向教育」，建立學生正面價值觀，並重視體藝發展，推動全人教育。
2. 互動學習 啟發思考 - 以小組學習模式，帶動學生互動學習，並推動專題研習，培養思考及共通能力。
3. 重視基礎 強化閱讀 - 重視兩文三語及數理基礎： 以推行晨讀課、分層閱讀計劃及多元化閱讀活動，建立恆常的閱讀習慣。
4. 結合生活 體驗經歷 - 以活動教學模式及跨科課程，帶動學生全方位學習，並透過海外遊學活動擴闊學生視野。
5. 善用科技 探究學習 - 透過科技探究課程，讓學生善用科技擴闊學習經歷。
關鍵項目的發展 : 1. 閱讀計劃：全校參與，拓寬學生的知識領域與視野。
2. 資訊科技：發揮資訊科技學習中心的功能，並與香港大學及中文大學合作，提升資訊科技教育功能。
3. 專題研習：設跨科專題研習，發揮自主探究精神，培養共通能力。
4. 德育及公民教育：通過聯科及日常活動，培養學生積極價值態度及國民身份認同。
共通能力的培養 : 1. 以小組學習模式，帶動學生互動學習，培養溝通及解難能力。
2. 推動專題研習，培養思考及共通能力。
3. 透過科技探究課程，推動學生探究學習能力。
學生支援
全校參與照顧學生個別差異 : 1. 以「全校參與」模式，支援有特殊教育需要的學生。
2. 營造共融的校園文化，促進全校師生互相尊重個別差異。
3. 及早識別有特殊教育需要的學生，採用「三層支援模式」，因應學生不同的需要而提供適切的支援。
4. 透過各方面的調適，照顧個別差異，提升學生的學習效能。
課程剪裁及調適措施 : 按學生能力及需要進行課程剪裁及調適：中英數均設拔尖班，另有輔導班、促進學習班：設分層課業設計、默書及評估調適以照顧能力稍遜學生。
家校合作及校風
家校合作 : 校政民主開放，家長意見及心聲容易表達。家長、教師及學生互信及尊重，校園成為一個充滿關愛及民主的園地。
校風 : 本校校風純樸，家長師生關係融洽。關注小一學生適應及小六學生升中問題，舉辦迎新日，升中講座，升中銜接及適應課程。並定期舉辦家長講座及親子活動，重視培養學生成長。
學校發展
學校發展計劃 : 1. 培養學生律己愛人的態度，建立正向價值觀。
2. 營造積極學習的校園氣氛，提升學生學習能力。
3. 改善校舍環境，包括粉飾課室及重鋪籃球場等。
4. 推行電子教學，提升教學效能。
5. 家校合作，支援家長，並建立教師學習社群。
教師專業培訓及發展 : 每學年安排三天教師專業發展日，進行專業培訓。此外教師每學年參加不少於四項校外專業培訓，以提高專業素養。
學費減免 : 不適用。
其他 : 舊生會獎助學金、陳英琴獎助學金、彭祖耀獎學金、周王二院獎學金。
