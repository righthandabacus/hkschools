辦學宗旨
本著基督精神，以「孝悌忠信、服務人群」的精神，培養學生優良品德、愛心及責任感，身心健康發展、勤奮上進。透過學校生活，讓學生懂得珍惜自己，且能關心別人，實踐所學以回饋社會。
學校特色
學校管理
學校管理架構 : 聖公宗（香港）小學監理委員會有限公司、聖公會聖匠堂
法團校董會/學校管理委員會/校董會 : 校董會
環保政策 : 學校廢物分類回收計劃、綠化校園計劃、魚菜共生計劃。
學校關注事項 : 「重品德、勤學問、嚴而有愛」
培養學生自律、自省、自信、自主學習的精神。
教學規劃
學習和教學策略 : 1. 課堂上運用不同的視像工具，如腦圖、時序圖、解難步驟圖等，發展學生的高階思維。2. 課堂設計著重互動，推行合作學習，提高學習的積極性。3. 安排不同的專題學習及跨科活動，靈活運用各種學科的知識，培養學生的共通能力。4. 以資訊科技支援課堂內外的互動與協作，促進教師與學生之間的互動，使學習更為有效。5. 推行「全方位學習」，豐富學生的學習經歷，包括探訪、戶外考察、境外交流、參觀活動等。6. 透過進行手腦並用的 STEM 相關活動，培養學生的創造力及解難能力。
關鍵項目的發展 : 1. 透過早讀、伴讀加強從閱讀中學習，課堂上教授閱讀策略。2.透過WiFi900(第二期)計劃，全校建立無線網絡，並利用60部平板電腦，試行各種教學應用程式，逐步讓老師和學生體驗電子教與學，同時也將傳統的紙筆教學模式兼容並進。3. 培養學生自尊、自重、自律、自省、自強不息的抗逆精神，建立面對科網世代誘惑的正確態度。4. 推行「校本輔導活動」，著重學生身心靈及社交能力的發展。透過德育講座，培養公民意識，關心社會和國家。5. 透過專題研習，開展探究性的活動，促進學生主動學習。6. 以繪本閱讀方式推動生命教育。
共通能力的培養 : 透過預習、閱讀、專題研習、課程統整、科學探究活動、編程、課堂內合作學習及課後延伸學習，建立學生創造力和共通能力。
學生支援
全校參與照顧學生個別差異 : 1. 津貼部分課外活動及補習班，提倡均等學習機會
2. 課後輔導小組
3. 小組／個別中英數學習計劃
4. 英文拔尖／提中／補底班
5. 週六英文補習、會話班、升小銜接課程及多元智能活動
6. 升中輔導 7. 言語治療/社交小組 8.「閃亮之星」計劃
9. 聖公會小學輔導服務處全方位學生輔導服務
課程剪裁及調適措施 : 1. 備課會調適教學策略 2. 默書加分制
3. 學習態度評估 4. 為有特殊學習需要學生設家課及測考調適
家校合作及校風
家校合作 : 1.家長教師會參與學生福利政策制訂 2.家長教師會參與學校政策制訂
3.家長講座包括：「如何協助子女建立良好生活習慣」、「父母適當地控制情緒」、「解開子女的情緒密碼」、「兒童情緒百寶袋」、「管教子女有妙法」4.家長義工 5.親子賣旗活動、家長支援小組 6.聖公宗「生命、心靈、價值教育」計劃
校風 : 1. 以基督精神，培養學生「德、智、體、群、美、靈」的健康成長。
2. 「行公義，好憐憫，存謙卑的心」，珍愛自己和家人，並發揚推己及人的精神。 3. 五大關注（品德、價值觀、生活習慣、成績、體藝教育）。 4. 設雙班主任制，加強關顧學生。 5. 「2016關愛校園」榮譽。
學校發展
學校發展計劃 : 1. 提升團隊領導效能，發揮教師才幹，建立專業文化。2. 優化課堂策略，提升學生共通能力，推動自主學習。3. 提升學生素質及自信心、自理能力，發展學生潛能。4.推動STEM教育，發揮學生的創意潛能。
教師專業培訓及發展 : 本校與各大教育機構建立網絡，參加教育局「校本支援服務計劃-專業發展學校計劃(中國語文科)」及NET Section導師支援英文科種籽計劃Developing Text Sets (DTS)，透過共同備課、集體觀課、評課、教學經驗分享等互動活動，優化課堂教學效能，並促進校內外的協作交流文化。
學費減免 : 不適用。
