辦學宗旨
以聖經真理為基礎提供全人教育；悉心教導、愛護和見證，以體驗基督教的真理；提供完備及適切課程和德育培訓；協助最有需要學生，完成辦學使命；培養學生好學自律、有禮合群、熱愛生命、勇於承擔、貢獻社會。
學校特色
學校管理
學校管理架構 : 本校為基督教中華傳道會轄下學校，設有法團校董會監察學校、擬定政策，由校長領導各功能組別及學科組別，管理及推動學校各種政策。當中包括有學校發展組、校本管理組、校政及校務、課外活動、訓導及輔導、宗教組、基督教生命教育組、資訊科技組、課程發展及各學科組別。
法團校董會/學校管理委員會/校董會 : 辦學團體代表、校長、教師、家長。
環保政策 : 環保大使、生態導賞團、環境教育課程及工作坊、校內及社區綠化活動。
學校關注事項 : 每日下午新設啟發潛能時段，除功課導修外、全面發展學生多元智能，包括語文、邏輯、空間、肢體運作、音樂、人際、靈德（內省）及自然探索：藉多元化活動，開拓孩子全人發展，舒展孩子身心，啟發孩子潛能。
教學規劃
學習和教學策略 : 課程的安排及規劃，配合學校的發展主題：「熱愛生命、凡事謝恩、享受學習、齊展潛能」。中、英文科推行圖書閱讀教學法、跨科專題研習、資優培訓、多元智能活動課、校本科技數理研習(STEM)、並透過藝術的教育去啟發學生的潛能及正向思想。推動適異教學，增加學生在學業上的成功經驗；與此同時，優化電子學習以配合適異教學，亦提升學生學習興趣；繼續推行學習態度獎勵計劃，培養學生良好的學習態度；推行自我挑戰潛能獎勵計劃，幫助學生提升自我管理和自我反省的能力；培養同學勤學盡責的態度，並且鼓勵同學發揮個人潛能，全力以赴，做到最好。
關鍵項目的發展 : 積極發展電子學習，並已發展英語網上學習平台、eClass及電子圖書，以電子學習加強學生學習能力，並設有科技研習室、電腦室及校園電視台。此外，本校致力發展STEM教育課程，包括衣物纖維、電學世界、食物安全、生活環境、3D電腦打印、分子料理等，培養學生的學習興趣，提昇創意和解難能力。參與教育局「專業發展學校計劃」，透過顆伴學校的支援，構建電子學習平台，透過平板電腦及小裝置，於數學科進行電子教學和學習，提升學生資訊科技能力，各學科也運用電子學習平台進行活動。
共通能力的培養 : 透過不同的課堂設計及學習活動，訓練學生九種共通能力，包括：協作能力、溝通能力、創造力、批判性思考能力、運用資訊科技能力、運算能力、解決問題能力、自我管理能力、研習能力。
學生支援
全校參與照顧學生個別差異 : 學生支援小組與校本教育心理學家合作，透過小一及早識別及分班會議，及早識別及支援有不同學習需要的學生，並透過學生輔導服務、駐校言語治療、課程剪裁、功課及評估調適、小組輔導、朋輩輔導、電腦輔助學習等不同活動，讓不同能力的學生能夠投入學習，發揮所長。
課程剪裁及調適措施 : 學校設有課後功課輔導服務、樂學小組、言語治療、成長的天空、精靈小特工、升中適應活動、並有教育心理學家、言語治療師及駐校社工跟進學生的學習及情緒發展等，全方位支援學生成長和學習。
在課程剪裁及調適措施方面，設有資優培訓，拔尖、進深等課程，並為有特殊學習需要的學生安排考試及學習調適。
家校合作及校風
家校合作 : 設家長教師會，為家長提供多元化的活動及講座，安排小一迎新日及適應班、推行「愛．學習」家長課程、家長藝術活動，舉辦親子遊戲日，親子旅行，並推動家長作義工，強化家校溝通，共同培育孩子成長。
校風 : 設基督教生命教育課、成長的天空、風紀訓練、各科小老師領袖生、和諧大使、班級經營計劃、領袖訓練、正向訓輔及義工服務等，讓學生在愛、被重視、正面鼓勵與關懷中成長。
學校發展
學校發展計劃 : 推行三年發展計劃：從全校規劃，以全人教育角度，建構正向校園，培育學生正向生活態度，幫助學生在身心上學業上正向成長，使學生能熱愛生命、凡事謝恩、享受學習、齊展潛能。
本校參加國際啟發潛能計劃，在專家指導下優化學校整體運作。學校重視學生的身心靈發展，推展基督教全人生命教育，以學生生命成長、學習態度和能力的培養為核心。
教師專業培訓及發展 : 為教師安排教師發展日，各學科複修課程及特殊學習需要的培訓。此外，亦鼓勵教師參與和學科有關的研討會，工作坊等。
學費減免 : 不適用。
其他 : 本校曾連續三年成功申請「優質教育基金」及參加加強公營學校行政管理計劃。另外，設有境外遊學交流計劃，擴闊學生視野。此外，在這幾年來，本校榮獲的獎項其中包括：香港學校朗誦節冠軍及獲多項優良獎，葵涌區校際田徑賽獲多項獎項，葵涌區校際足球賽冠軍，學校舞蹈節甲級獎，另有多項學術及體藝的獎項，未能盡錄，詳情請參考本校網頁。
