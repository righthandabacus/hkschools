健康校園生活 : 推行「至『營』學校計劃」及環保教育，小食部售賣食品及午膳餐單均符合認證要求。為提升學生的體質，學校更推行體適能計劃及多元化有益身心健康課程。
備註 : 為配合健康校園生活，學校更注重學生及老師的關懷，因此學校積極推行關愛校園文化活動，除了重視學生的校內健康的飲食及衞生環境，學校亦經常為學生，家長及老師舉辦有益身心活動，例如親子比賽活動，境外學習及各種大使計劃，積極培養學生良好品格。
全方位學習活動
本校設有於81個課外活動小組，其中包括：英文進階班、英語話劇班、中英朗誦隊、奧數班、校園電視台、小提琴、合唱團、街舞舞班、舞獅舞龍隊、跆拳道、游泳隊等活動。除此之外，還有定期全校活動，例如：旅行日、親子同樂日、親子學習參觀計劃等親子的活動。
辦學宗旨
本校以「篤學力行」為校訓，致力培養學生良好品格，改善學習環境，使學生在五育方面，能夠有均衡的發展，並能獨立思考、發展潛能。校方鼓勵學生積極參與社區活動和服務社會，做個樂善勇敢的良好公民。
學校特色
學校管理
學校管理架構 : 辦學團體公立興德學校有限公司是最高決策組織。下設法團校董會，其下由校長管理行政主任、科主任、老師及職工。
法團校董會/學校管理委員會/校董會 : 法團校董會共有16位校董，9位來自辦學團體（包括校監），其他校董包括校長1名、家長校董2名、教員校董2名、獨立校董1名及校友校董1名。
環保政策 : 校園是綠色學校，遍植黃龍船花、翡翠朱蕉及花葉沿階草。校方使用環保光管及智能洗手設備。此外，本校使用的紙張均採用雙面印刷，學生午膳亦必須使用環保餐具。
學校關注事項 : 提升教學效能；建立資訊科技自學平台；培養學生藝術興趣及創造力。
教學規劃
學習和教學策略 : 在課後 3:30 開始課後延展課，內容有功課輔導及體藝訓練，延展學生學習及體藝方面的發展，並增加他們對學校的歸屬感。
關鍵項目的發展 : 1. 全面推行各級閱讀計劃及建立學生專題研習能力架構。
2. 全面推廣德育及公民教育。
3. 建立關愛校園，以生命教育元素滲入各科。
4. 全面推行資訊科技教學。
5. 全面推行校本課程。
6. 引入「境外學習」元素，擴闊學生國際視野。
共通能力的培養 : 滲入9種共通能力的訓練於各年級的學習領域，培養學生學會學習的治學態度。在9種共通能力中，本校致力發展學生的溝通能力、創造力及批判性思考。
學生支援
全校參與照顧學生個別差異 : 開設加強輔導班，為各年級成績稍遜的學生開設輔導課及學科輔導小組。
課程剪裁及調適措施 : 五、六年級增加中英數上課課節。
家校合作及校風
家校合作 : 本校透過家教會，凝聚家校力量，以促進家長與學校之間的聯繫、溝通和合作；積極推動家長教育、親子活動及家長義務工作，讓學生在愉快的環境下健康快樂地成長。過去學校與家長舉辦多項活動，例如家長烹飪班、家長電腦班、家長英語班、家長書法班，另外我們亦組織家長推動多項活動，例如故事媽媽，敬師日及開心生果日等。
校風 : 1. 重視學生品德培育，致力培養學生自律自覺，積極進取的精神。
2. 在開學前舉行小一適應課程及小一新生家長會。
3. 其他支援小一新生活動包括新生自理能力訓練、家長茶座、親子活動、小一適應講座、讀寫活動及培育組和駐校社工合辦之活動。
4. 學校一直參加關愛校園計劃，給予學生建立一個關懷環境，以培養學生良好品格。
5. 學校推行領袖訓練計劃，從而培養學生領導才能。
6. 每班推行一生一職計劃，每名學生最少有擔任一個職務，例如班長，科組組長，風紀，環保大使，學校親善大使，以培養學生責任感。
學校發展
學校發展計劃 : 1. 持續優化課程及教學，提升學習效能。
2. 持續培養積極及正面的價值觀。
3. 強化教師專業，建立學習型組織。
教師專業培訓及發展 : 每年最少三次教師發展日，藉以改善教學技巧及提升教學效能；定期舉行各類教學研究活動，例如同儕備課、同儕觀課、資訊科技分享會、行動研究等，通過集體分析和反思來提升教師的教學效能。本校設有新教師導師制，協助新入職教師儘快熟習學校運作。本港大學及大專院校更是本校的長期合作伙伴，大家共同進行教師專業發展及研究，為本校帶來很多創新的教學意念。
學費減免 : 不適用。
其他 : 1. 本校與英國及中國著名小學已締結為姊妹學校，舉辦交流學習活動。
2. 本校與教育局進行校本英文科課程統整計劃，不斷提升學與教的效能。
3. 本校為阿仙奴足球學校的指定合作學校，將足球訓練融入校本體育課課程。
4. 本校與多所大學協作，發展校本課程，為學生提供課堂以外的學習。
