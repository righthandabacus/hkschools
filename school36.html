班級教學模式
小一至小三不按成績分班，以混合形式把學生平均分為五班；小四至小六以學生學習能力及成績分班；並在中英數常四科施行「校本式小班教學」。

健康校園生活 : 本校參與健康飲食計劃，監察午膳質量；舉辦主題講座及不同活動，鼓勵學生參加體適能獎勵計劃，營造健康愉快的校園環境。

全方位學習活動
本校推行「一生一體藝」政策，鼓勵學生參與體育或藝術活動，發揮德、智、 體、群、美的精神，達至全人發展。學校的聯課活動涵蓋學術、藝術、體育、興趣及服務五大範疇；課餘興趣班著重各項體育運動及中、西樂器的培訓。
辦學宗旨
本校貫徹「有教無類」的教學精神，以「全人教育」為目標，提供均衡教育；培養學生「德、智、體、群、美」的良好品格，致力培育優秀的下一代。
學校特色
學校管理
學校管理架構 : 學校管理委員會下設包括各職級及範疇老師的學校發展小組，探究學校發展的方向。按管理與組織、學與教及學生成長支援三大範疇設立行政小組，各組主任就學校的關注事項及工作範圍訂下清晰的工作目標，統領老師執行職務。
法團校董會/學校管理委員會/校董會 : 成員包括教育局代表、兩位獨立人士、校長、兩位家長、一位校友及兩位教師代表。
環保政策 : 將「環保」課題融入各科的教學，訂定「環保」為德育主題，提倡減廢和惜物，鼓勵學生擔任環保小先鋒，以提升環保意識。午膳安排：全校實施課室分飯模式運作，以減少廚餘。經常透過電郵分配工作及發放指引；雙面列印文件以減少用紙。
學校關注事項 : 提升學生主動學習的能力，培養學生成為有教養的孩子，增強學生對學校的歸屬感。
教學規劃
學習和教學策略 : 設計多元學習活動、激發學生的學習動機，培養學生主動學習及多角度思考的能力。中文科以讀寫結合的方式，提升學生的說話及寫作能力。英文科透過掌握常用字、語音及腦圖、並閱讀優質篇章的習慣，提昇學生運用英語的能力。數學科以數學化(Mathematising)觀點組織教學設計，在過程中，教師讓學生參與數學「再創造」(re-invention)的過程，並以數學語言發表己見。常識科以探究式學習鼓勵學生主動探索，引導他們進行引證。另透過探討時事，提升學生的時事觸角及明辨性思維。各科同時利用電子白板教學平板電腦及網上學習平台延伸學習。
關鍵項目的發展 : 德育及公民教育採取訓輔合一方向，設成長加油站時段，注重品德培育及價值教育。將「成長課」、「環保教育」及「公民教育」有系統地融入課程，也透過雙周會及班級經營活動，提升學生的自我管理的能力。推動閱讀：設早讀時段，提升閱讀興趣；並於小息時段進行大哥哥大姐姐伴讀計劃，讓學生可以從不同的平台與同學分享閱讀的樂趣。圖書組亦會支援各科閱讀素材及活動，提升學生的閱讀理解能力。
共通能力的培養 : 除透過探究周活動及經歷學習活動，發展學生的共通能力外，各級更將各種共通能力滲透於課程中，使不同級別能發展不同的共通能力。
學生支援
全校參與照顧學生個別差異 : 學校制定支援服務的發展方向；不論在班級結構、教學設計、課堂活動、課業及評估機制，均重視照顧學生的個別差異和不同的學習需要；此外，又以新資助模式為有特殊學習需要的學生提供適切支援，使他們融入校園生活，得到平等的教育機會。
課程剪裁及調適措施 : 因應學生不同的學習能力及教師專業的共同備課，製定校本的教學策略、內容和課業。配合不同課堂學習活動及適切的回饋以協助不同能力的學生學習。
家校合作及校風
家校合作 : 家校關係密切，家長組織龐大的義工隊，協助學校日常的運作及活動。家教會還會有系統地舉辦家長教育課程和講座、親子活動及外訪交流，並透過校訊、會訊、網頁及家長網絡，提升家校溝通。家教會還會協助學校監察學生福利的商業活動。
校風 : 學校關注學生的品德培育，也著重發掘他們的潛能。有計劃地編排價值教育，也經常為學生安排講座、活動及比賽等，藉此建構校園關愛及彼此欣賞的文化，提升學生自我管理能力，培養他們建立積極的人生觀，並能尊敬師長，與同學和睦共處，彼此支援，學會感恩。
學校發展
學校發展計劃 : 發展中、英、數和常的校本課程，積極發展電子學習，提升課堂互動。分組學習及編修分層課業，照顧學習多樣性。深化品德和價值教育課程，培養學生個人及群性發展的價值觀和態度。安排不同的平台，讓學生展示學業和體藝的潛能。發展家長教育，拓寬家長的視野，配合學校的發展方向，以使孩子身心健康成長。
教師專業培訓及發展 : 全校的老師均具備教師專業資格，學校按校本及老師的發展需要，安排專業的培訓活動，又鼓勵老師參與校外課程和講座，提高其專業水平及將最新的教育資訊帶進學校。學校積極引入校外支援，以提升教師專業知識和能力，發展校本課程和支援學生成長需要。編定同級同科老師共同備課、同儕觀課及課後研討，以提升學與教的成效。
學費減免 : 不適用。
