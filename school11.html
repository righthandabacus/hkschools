辦學宗旨
學校本著基督精神，為學生提供全人教育，使學生的身心靈得以均衡發展。在關愛的文化下，學校致力幫助學生發展潛能，培養完美的品格。
學校特色
學校管理
學校管理架構 : 校董會負責監察學校運作。校內設不同功能小組，統籌學校自評、訓育輔導、課程發展、課外活動等工作。
法團校董會/學校管理委員會/校董會 : 本校之辦學團體為「聖士提反女子中學校董會」，校董會成員共15人，其中包括聖公會選出之代表、聖約翰學院選出之代表、聖士提反堂代表、聖士提反女子中學校長、家長教師會會長、舊生會會長及獨立人士。
環保政策 : 1. 培養學生注重環保的意識，並鼓勵她們在生活中實踐。活動包括：「地球一小時」、「綠色聖誕」、「校服回收/送贈計劃」、「課外圖書分享計劃」、廢物分類、廚餘回收活動等。
2. 為響應環保，節約用紙，學校以電子通告形式與家長通訊。
學校關注事項 : 1. 加強資訊科技教學，促進課堂互動。
2. 培養學生自學能力，發展自主學習。
3. 發展學生自我規劃及管理學習生活的能力，持續提升自信。
教學規劃
學習和教學策略 : 課程以發展五育及靈的修養為整體目標，著重培養解難和創新的能力及積極、主動的學習態度。本校著力提升兩文三語，協助學生養成閱讀習慣及愛好。我們相信每位學生都是獨特的，希望發掘她們的潛能和發展所長。我們善用下午時段安排聯課活動，豐富學生的學習經歷。
關鍵項目的發展 : 1. 從閱讀中學習：設圖書課及閱讀活動，如「悅」讀樂、小一家長伴讀計劃等。
2. 資訊科技教育：除設電腦課外，亦以平板電腦輔助教學，裝備學生適應時代的需要。
3. 專題研習：各科利用專題研習，協助學生建構知識，培養共通能力。
4. 德育及公民教育：以「紫荊花獎勵計劃」培養學生的自律及品德，輔以生命教育課、班主任課等，培養學生正確的價值觀。
共通能力的培養 : 於不同科目的學與教活動及全方位活動中，發展學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 學校為在語文、數學及科學方面具優厚潛質的學生提供在課後進行的精英班活動，亦推薦學生參加各項比賽，讓他們的潛能得以發揮。另一方面，我們為有需要的學生提供在課後進行的增進班，亦有在課時內進行的各項輔導活動，並利用課堂活動和課業照顧個別差異。
課程剪裁及調適措施 : 為能力稍高的學生提供中、英、數、常增益課業，另分別為能力較高及能力稍遜的學生提供精英班及增進班。
家校合作及校風
家校合作 : 本校家教會歷史悠久，架構完善，家校衷誠合作，致力推動學校發展，其手工藝組為學校的籌款活動貢獻良多。
校風 : 學生純品受教，尊師愛友，樂於學習，具自信，願意與別人溝通和協作。家長關心女兒在學情況，支持校政，除擔任家長義工外，亦踴躍出席學校大型活動。校友的凝聚力強，為推動校務發展，不遺餘力。中小幼聯繫緊密，每年定期舉行交流活動，讓學生適應不同階段的校園生活。
學校發展
學校發展計劃 : 1. 加強資訊科技教學，促進課堂互動。
2. 培養學生自學能力，發展自主學習。
3. 發展學生自我規劃及管理學習生活的能力，持續提升自信。
教師專業培訓及發展 : 教師積極參加各類進修活動，以提高專業水平，同時通過定期的共同備課、同儕觀課及教師專業分享會，共建互相學習及分享的文化。
學費減免 : 不適用。
其他 : 1. 本校設多項獎學金，以肯定學生在學術、服務、音樂、體育、美術等各方面的表現
（內容詳見本校網頁www.ssgps.edu.hk/m3p8.php）。
2. 本校學生積極參與公開或校際比賽
（獎項詳見本校網頁www.ssgps.edu.hk/m7p1.php）。
3. 本校每年均帶領全級六年級學生參加由教育局主辦的境外交流活動，擴闊學生的視野。
