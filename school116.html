班級教學模式
按平均能力分班，混合教學模式，按學科授課需要設計活動（小一綜合課程、專題研習評估及特別活動）。
班級結構備註
小一專題研習評估、雙班主任制、小組教學、特設圖書課、全人成長課及多元智能課、早會（家長可參與）。

分班安排 : 按性別及成績平均編班，而中文科會分為華籍及外籍中文班，部份年級數學科會按能力分班。

本校有多項課外活動：長笛、色士風、管樂隊、弦樂隊、小提琴、無伴奏合唱、陶笛、敲擊樂、合唱團、爵士舞、乒乓球、花式跳繩、田徑、小型網球、羽毛球、籃球、足球、跆拳道、排球、輪滑、木球、冰球、集誦隊、視藝會、幼童軍、學生成長小組、法文、日文班、英語話劇班等。
辦學宗旨
遵循政府教育政策，秉承保良局辦學方針，以校訓「愛敬勤誠」為橫軸，以五育「德智體群美」為縱軸，以多元文化為側軸，貫徹「跳出框框、育出非凡；同一團隊、同一信念」的核心價值；致力為學生經營一個優質英語學習氛圍，培養學生高雅的品德情操及積極人生觀，成為以關愛為本的全人教育基地。
學校特色
學校管理
學校管理架構 : 透過法團校董會、行政部門、科組及功能小組，以「策劃－施行－評估」模式監測工作計劃的推行及評估其成效，積極向持份者收集及匯報學校表現，以增加管理的透明度及問責精神，加強員工溝通，促進學校的持續發展，同心為學生提供優質教育。
法團校董會/學校管理委員會/校董會 : 辦學團體校董九名，非辦學團體校董六名。
環保政策 : 利用廚餘作花肥在學內培植有機植物、地球一小時、保護水資源大使選拔賽、無冷氣日、一人一花計劃、花圃、廢物分類箱、課室更換LED燈、風力發電、舊校服故事書回收等。
學校關注事項 : 強化教學效能及教學法；把資訊科技工具運用於自主學習中；持續健康、關愛及環保校園。
教學規劃
學習和教學策略 : 校本課程，以英語為主導教學語言，以普通話學習中文，透過專題研習、口頭報告等，實踐探究學習。
關鍵項目的發展 : 強調學生語文運用、品德培養及自控能力。
共通能力的培養 : 透過全方位學習、課外活動及學科活動，滲透共通能力的培育，包括溝通、運用資訊科技、創意及自學能力等。
學生支援
全校參與照顧學生個別差異 : 註冊社工駐校支援學生全人發展； 舉辦不同形式的學生支援小組及家長工作坊；課後設不同科目的拔尖補底班及輔導班，致力照顧學生個別差異
課程剪裁及調適措施 : 為照顧華語、非華語學生不同的學習需要，中文科分流上課，數學科會按學生能力分班。
家校合作及校風
家校合作 : 學校開放早會給家長參與、學校定期舉辦家長日、專題講座、天才表演、與校長茶敘及工作坊等親子教育活動，讓家長掌握子女的學習進度及加強家校合作和溝通。家長教師會作為溝通橋樑，讓家長們建立支援網絡，本校邀請家長於各學校活動擔任家長義工，以支援學校的運作，彼此齊心扶助孩子健康快樂地成長。
校風 : 本校連續七年獲頒「關愛校園榮譽」，本年更獲牽引共融優異獎，關顧學生的成長需要，提供多樣化的支援計劃，與家長及社區保持緊密的夥伴關係。透過德育及公民教育、健康校園活動、學生聯課活動等，建立關顧及兼融的校園；設立及早鑑別學生的特殊教育需要的機制，建立共融文化；並在多元文化的薰陶中，培養學生平等尊重及友愛的品德情操，提昇學生對學校歸屬感。
學校發展
學校發展計劃 : 1. 強化教學效能及教學法。
2. 把資訊科技工具運用於自主學習中。
3. 持續健康、關愛及環保校園。
教師專業培訓及發展 : 每年舉辦最少一次教師境外專業發展交流，拓闊視野，裝備自己，定期舉行講座、研討會、專題工作坊、發展日營、教師持續發展計劃、教師啟導計劃、共備會等，學校鼓勵教師進修及出席校外工作坊等。
學費減免 : 本校設有學校減免計劃，詳情請瀏覽本校網站。有需要的學生(包括來自綜援家庭的學生及領取學生資助辦事處財政資助的學生)可提出申請學費減免。
- 本校學費減免計劃詳情包括家庭入息限額，請瀏覽網上版的超連結。
其他 : 保良局獎學金、學校獎助學金計劃、家長教師會獎學金計劃、校外各推薦計劃等。
交流計劃包括服務學習、多元文化學習。
