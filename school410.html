班級教學模式
除普通話及中文科外，其餘各科以英語為主要授課語言。學生以小班教學形式上課，校方按學生中文能力編組。課前及課後設中文、英文及數學「奮進班」及「課後功課輔導班」，以小組模式上課，照顧學生個別學習差異。
班級結構備註
本校運用新資助模式進行融合教育，安排半職老師及教學助理照顧有特別教育需要學生。中文科為非華語學生設計課程，程度參考本地中文課程並作調適，另設程度較高的課程予全級中文學習能力較高的學生。
學習評估
全年全科測驗次數 : 0
全年全科考試次數 : 2
多元學習評估 : 運用多元化評估方式，主科定期依單元進行進展性評估。全校學生都以英文閱讀能力評級。另安排跨科專題研習、學生自評及互評等，適時給予學生回饋，提升學習效能。
分班安排 : 小一至小四不按成績，以混合能力形式分班；小五及小六按學生的考試總成績分班。

午膳安排 : 由指定供應商提供、由家長安排及學生自攜午餐。
健康校園生活 : 以「Be a Good Kadoorie」為中心，積極推廣學生培養正面價值觀本。另鼓勵學生培養良好生活習慣，包括參加水果月及安排每周一天全校早操時段。
備註 : 本校特設第十堂為家課堂，讓欠缺家長支援的學生亦能在老師的指導下完成較高層次的家課。
各班設立自助閱讀閣，提供故事書、雜誌及期刊等不同類別的閱讀材料，讓學生在轉堂自行借閱。
全方位學習活動
學校透過課後興趣小組及週五CCA活動推動學生全人發展。 活動組別包括制服團隊、戲劇、舞蹈、音樂、視藝、體育、小記者、STEM/科學探究、語文趣味活動、Hindi及Urdu 課後班等。 校方經常安排或鼓勵學生參加校內外比賽，以增廣見聞及提升自信心。
辦學宗旨
營造愉快學習環境，提供優質的學與教，培養學生五育均衡的發展，達致「德、智、體、羣、美」的目標。
學校特色
學校管理
學校管理架構 : 學校以學校管理委員會為首，負責管理及監察學校發展。
法團校董會/學校管理委員會/校董會 : 學校管理委員會成員包括教育局首席助理秘書長、校長、兩名教師代表、兩名家長代表、兩名校友及一名獨立人士。學校管理委員會監察學校有效運作。
環保政策 : 本校現有30位環保大使。他們會利用星期五的課外活動時段學習更多環保訊息。
學校關注事項 : 透過電子學習提升學與教的效能，培育學生正面的價值觀及自信心，建立良好的學習態度及學習技巧。
教學規劃
學習和教學策略 : 主科均有校本課程，按學生能力及學習需要設計課程。主科發展電子學習，通過不同軟件或程式提升學與教效能。不同科目設網上自學資源，方便學生上網自學。訂定家課政策及評估政策，為學生提供適當的評估方法。外籍英語老師與低小英文老師共同進行閱讀課堂。中文及英文科均安排學生擔任中文或英文大使，協助帶領語文活動。課前及後設班主任課，照顧學生全人發展。課前及後進行體藝活動訓練，興趣小組及功課輔導等。積極鼓勵學生參與校內及校外比賽及活動。透過安排探訪及服務，擴闊學生眼界。
關鍵項目的發展 : 學校重視培育學生的共通能力及自主學習能力。關注德育及公民教育，通過主題講座、校本設計學習材料及探訪活動，培養學生正面的價值觀和態度。中英文科及圖書館主任推行多項閱讀活動以提升學生的閱讀能力和興趣，從而提升學習能力。
共通能力的培養 : 推行各科小型專題研習及為小一至小六學生安排跨學科專題研習，因應學生各級的能力教導學生處理及分析數據，培養學生各種共通能力，讓學生學以致用。
學生支援
全校參與照顧學生個別差異 : 採取全校支援模式，為有特殊學習需要的學生提供支援。設功課堂及課後功課輔導小組，協助學生完成功課。逢周末安排中、英文課程予新來港學童，協助適應學習。於暑假期間為小一至小四開設適應課程，以增強同學學習中文的能力。
課程剪裁及調適措施 : 中文科科組以「中國語文為第二語言學習架構」進行課程調適，設計校本課程以照顧非華語學生及中文能力較高的學生的學習能力和需要。英文科安排低小學生參加「小學英語讀寫計劃」，以提升英語能力；高小推行校本英文寫作課程，讀寫並重。數學科獲校本課程發展組到校支援數學老師，一同研究有效的教學策略和課堂活動。
家校合作及校風
家校合作 : 本校學生家長均為家長教師會會員。會方每年舉辦會員週年大會、興趣小組、講座及親子旅行。會方與校方共同安排工作坊及講座予家長，分享教導子女的方法，以及加強家長之間的聯繫。家教會鼓勵家長積極參與家長義工服務。家長義工隊積極參與學校大型活動，每天均有家長義工照顧學生午膳，以及每周一次為有需要家長提供翻譯。
校風 : 著重融和與互相尊重，教育學生共同建立和諧互信的校園文化。推行全校參與獎勵計劃，以正面文化鼓勵學生學習堅持，為進步而努力。
學校發展
學校發展計劃 : 通過三年為一周期的學校發展計劃，聚焦學校的發展重點：
1) 透過電子學習提升學與教的效能
2) 培育學生正面的價值觀及自信心
3) 建立良好的學習態度及學習技巧
教師專業培訓及發展 : 鼓勵教師積極進修，以提升教師的專業發展。通過定期的共同備課、同儕觀課及教師專業分享，共同建立互相學習及分享的文化。
學費減免 : 不適用。
其他 : 本校是一所以英語為主要教學語言的多元文化官立小學，著重融和及互相尊重。本校積極推薦學生參加不同的獎勵計劃，例如融和獎學金、教育獎學基金、胡禧堂獎學金、香港學生運動員獎及灣仔區傑出學生獎等，以獎勵成績優異或積極參與學校、社區活動及服務的學生。
