班級教學模式
學生來自本港和全球三十個不同民族，以小班教學模式， 中文及英文為授課語言： 一、二年級注重學習能力和態度的培養，三、四年級鞏固學習基礎；五、六年級重視學業成績和個人成就，為未來升學做好準備。
班級結構備註
為照顧學生的獨特性，部分班級， 中、英、數科目分能力組別上課。
學習評估
全年全科測驗次數 : 4
全年全科考試次數 : 2
多元學習評估 : 小一及小二採用進展性評量，並以描述式學習報告記錄學生學習進展；其他各級定期以紙筆測考評估學生學業進度。各級學生透過平時習作、電子學習平台、專題研習及學習歷程檔案，以瞭解學習表現。
分班安排 : 學生以學業成績平均分班，主要科目以學習能力分組，並輔以資優培育計劃。

備註 : 學校設有早會、德育及公民教育課、圖書課及課外活動課。參與香港環保健康學校獎勵計劃、中文大學協辦的開心學英語計畫及香港大學協辦的喜悅寫意中文學習支援計劃。小息、午息及放學時段均開放電腦室及圖書館予學生使用，方便學生自主學習。
全方位學習活動
興趣活動包括資訊科技獎勵計劃、英文及普通話大使訓練、中、英語話劇、小女童軍、基督小先鋒、醒獅組、中國舞組、中國笛子班、小提琴班、書法組、劍橋英語班、游泳訓練班、足球組、合唱團等，中英文集誦採用全班式訓練。
辦學宗旨
傳揚天主福音，用愛心、接納與關懷體現基督精神。重視學生個人價值和尊嚴，以及多元文化的背景，對學生絕對信任，予以信任及尊重：因材施教，為兒童提供優質的基礎教育。

教學規劃
學習和教學策略 : 本校率先推行小班教學，輔以合作學習法，以加強課室互動及提升學生創意思維。各級全方位學習以常識科的專題研習題目為切入點，帶動各科進行跨學科學習活動。中文科推行普教中及活潑多元語文課程；英文科強化英文兒童文學欣賞，由低年級開始推動小學英語校本讀寫計劃，以發展互動學習及從閱讀到寫作計劃；數學科進行嘗試教學法，更積極推動數理教育和電子學習。
關鍵項目的發展 : 1. 閱讀：中英文廣泛閱讀計劃、每日早讀時間、早會設好書推介、英、普故事媽媽、圖書課、中英文網上閱讀計劃等。
2. 資訊科技：各級有電腦課，參加「資訊科技挑戰獎勵計劃」，開發校本教學軟件及校園社交平台，推動電子學習。
3. 跨學科專題研習：常識科以「STEAM」為主題，配合各科進行全方位學習。
4. 德育及公民教育：課程配合個人成長，並滲透於日常校園生活。
共通能力的培養 : 重點發展閱讀、探究和電子學習等三項學習策略；透過時事新聞討論，擴闊學生全球視野；推行全校演講訓練，增強學生自信和表達能力，積極實踐全方位培育學生多種共通能力。
學生支援
全校參與照顧學生個別差異 : 小一有甄別有特殊需要學生服務；設有資優政策及人才庫，推行精英培育計劃；另設課後功課輔導班及非華語學生支援服務。
設有卓越人才幼小銜接課程，為幼稚園學生升小一提供適應學習。另外，為畢業班提供升中強化計劃，加強學生升中信心和適應能力。
課程剪裁及調適措施 : 靈活的課程剪裁和增潤政策，為各班級訂定優質的課程，並安排特別測考給與有特殊需要的學生，另於課後特別抽離非華語學童以加強照顧。
家校合作及校風
家校合作 : 家教會配合學校舉辦學生學習活動、家長教育、親子活動及興趣小組等，更參與學校校服及午膳供應招標，以及協助監管校車等工作。
校風 : 設新生家長會、課後功課輔導班、大哥哥大姐姐計劃、新來港學童英文適應課程、新小一適應課程、德育及公民教育。

教師專業培訓及發展 : 積極發展優秀的教育專業團隊，強化教師學校行政及兩文三語教學能力。
學費減免 : 不適用。
其他 : 家教會每年都贊助體藝及服務獎學金予除學業以外在體育、藝術及服務有優秀表現的同學。學校每年均參與國際聯校學科評估試（英文、數學及科學科）。學校每年都帶領學生參加跨境交流活動。
