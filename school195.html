本校提供多元化的課外活動，包括：空手道、小提琴班、英文民歌組、魔術班、攝製隊、結他班、園藝、舞蹈、非洲鼓、硬筆書法、毛筆書法、陶笛班、手鐘隊、合唱團、小童軍、博物調查隊、非洲鼓班、基督徒團契、田徑訓練班及球類訓練等。
辦學宗旨
德智體群美五育並重。注重學生品德培養，身體力行，學習與人相處合作、明辨是非、承擔責任；教導學生汲取各科知識和技能，發揮專長；鍛鍊學生有強健的體魄及增進藝術修養，使他們成為一個良好公民，服務社群。
學校特色
學校管理
學校管理架構 : 法團校董會監察學校。由校長及行政人員統籌各行政組別維持學校運作。
法團校董會/學校管理委員會/校董會 : 法團校董會由辦學團體、校長、老師代表、家長代表及校外人士組成。
環保政策 : 1. 各種節能減廢及循環再用建設及活動。
2. 獲環保基金59萬，建立嶺英森林學校，以宣揚物種多樣性價值，保育大自然。
3. 獲賽馬會撥款294萬建設生化淨水系統教育中心。
學校關注事項 : 關顧學生身心健康及學習需要，提供多元化課程及活動，啟發學生創意和潛能。
教學規劃
學習和教學策略 : 正面思考、欣賞文化、小班教學、合作學習、體驗歷奇等，鼓勵學生積極參與體藝活動及社會服務，為學生設計豐富的學習經歷，發展全人教育。
關鍵項目的發展 : 1. 設生命教育課，推廣德育、公民及國民教育。
2. 從閱讀中學習，設圖書課，在中英文科融入閱讀策略教學。
3. 主題研習，常識科設專題研習週。
共通能力的培養 : 在各科備課會、各種校外活動及境外學習中注入協作、溝通、創造、研習、批判思考、解難及自理等，培養學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 設加強輔導教學、言語治療、課後功課輔導及新來港適應班以照顧學生的學習需要。
課程剪裁及調適措施 : 學校引入教育局支援以發展校本課程，備課會研製各科校本調適課程，在考試及學科上發展不同的調適措施，鞏固學生的學習。
家校合作及校風
家校合作 : 家長教師會定期籌劃活動，組織義工隊支援學校，發揮伙伴及家校橋樑的角色。
校風 : 1. 教師接受NLP身心語言程式學執行師培訓，致力使學生思想正面、身心健康，並獲和諧校園獎。
2. 師生關係融洽，重視學生輔導工作。
3. 照顧跨境學生的需要。
4. 發揮學生個人潛能，成為學習的動力。
學校發展
學校發展計劃 : 1. 發展校本課程。
2. 賦能追求卓越。
教師專業培訓及發展 : 學校重視教師專業發展及培訓，經常對外分享教學成果，亦屢獲教學獎項。每一成員都終身學習，邁向學習型團隊。
學費減免 : 不適用。
其他 : 本校榮獲「卓越關愛校園獎」最關顧新來港學生適應主題大獎。
本校榮獲三屆綠化校園工程獎園圃種植─小學組冠軍。
