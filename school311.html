辦學宗旨
本校乃聖公宗（香港）小學監理委員會有限公司屬下之津貼小學，創校於一九七六年。透過教學活動，使學生在德、智、體、群、美、靈六育上得到均衡發展，以「非以役人、乃役於人」為校訓，期望學生將來對社會作出貢獻。
學校特色
學校管理
學校管理架構 : 在辦學團體帶領下，設法團校董會推動學校發展。學校亦設改善小組、行政組及科組推行校務。
法團校董會/學校管理委員會/校董會 : 已於2015年8月成立法團校董會。成員包括辦學團體校董、校長、教員校董、家長校董及獨立校董。
環保政策 : 學校重視環保教育。透過專題研習、展覽、「校服回收」計劃、「一人一花」活動、環保餐盒等措施，推動學生支持環保。
學校關注事項 : 關注事項(2015-2018)
1. 培養學生自主學習的習慣及能力。
2. 培養學生運用資訊科技的能力，促進互動學習。
3. 推動價值教育，優化學生的品德情意。
教學規劃
學習和教學策略 : 各學習領域的課時靈活地安排特別時段進行協作教學、專題研習等及全方位學習的課程統整。更設有專題學習，以常識科為主幹，而其他學習領域的教師則選材或自編教材以配合有關之學習範疇，進行跨科的研習。
關鍵項目的發展 : 設有英語活動室以配合一至三年級「小學識字計劃–閱讀/寫作」教學的推行，以提升學生學習英語的能力；一至三年級各級兩班及四年級至六年級各班推行「普教中」計劃。 常識科設有「專題研習」，戶外觀察、資料搜集及科學探究，提升他們的團體合作精神。德育及公民教育︰透過週會和週訓進行德育的培訓。
共通能力的培養 : 常識科、數學科進行專題研習以發展學生的共通能力。一、二年級發展溝通能力、協作能力及創造力；三、四年級發展明辨性思考能力、運算能力及解決問題能力；而五、六年級發展研習能力、自我管理能力及資訊科技的運用能力。另外，運用高層次思維學習策略，提升學生自主學習的精神。
學生支援
全校參與照顧學生個別差異 : 設有支援小組，照顧個別差異的學生；校本課後學習支援計劃為有需要的學生提供課後的功課輔導；成長的天空計劃的輔助及發展課程提升四至六年級的學生的抗逆力。
課程剪裁及調適措施 : 1. 推行校本跨學科課程統整，以提升學生的學習效能。
2. 設有「人才庫」及「優才班」以發展各科資優生的潛能。
家校合作及校風
家校合作 : 家長教師會舉辦「親子旅行一天遊」、「親子廚神比賽」、「家長義工訓練活動」及「家長天地」。通過這些活動，加強家校合作。
校風 : 1. 學生知禮守規及受教。
2. 師生關愛校園，相處融洽。
3. 師生家長彼此體諒支持。
4. 學生喜學習，愛探究。
學校發展
學校發展計劃 : 加強教師發展及培訓，以資訊科技教學融入課程，將教學過程轉化為互動和以學生為本的模式。推動「普教中」計劃及英文PLP-R/W計劃，以提升學生學習中英語文的能力。
教師專業培訓及發展 : 教師專業發展著重於團隊之建立。此外，學校安排共同備課、協作教學及教師分享會以提升教師之專業水平。
學費減免 : 不適用。
