辦學宗旨
1. 以「愛、敬、勤、誠」為校訓。
2. 教授知識、訓練技能及鍛鍊體格。
3. 着重培養學生優良的品格、高雅情操及正確的人生觀。
4. 注重學生在德、智、體、群、美之均衡發展。
學校特色
學校管理
學校管理架構 : 校長之下，設副校長兩名，主任八名。在校長帶領下，副校長及主任共同策劃、執行學校發展政策，處理日常事務。
法團校董會/學校管理委員會/校董會 : 本校法團校董會于2013年成立，有成員14人，包括校監一人，辦學團體校董六人，替代辦學團體校董一人，當然校董（校長）一人，獨立校董一人，教員校董及替代教員校董各一人，家長校董及替代家長校董各一人。
環保政策 : 1. 回收廢紙、膠樽、鋁罐及碳粉盒。
2. 參加：
A. 水務處：節約用水整合式教育計劃。
B. 中電：綠優校園認證計劃。
C. 世界自然（香港）基金會：米埔濕地小學教育項目。
D. 香港聖公會麥理浩夫人中心林植宣博士老人綜合服務中心： 「老幼好薯」計劃。
學校關注事項 : 1. 於課堂上推行電子學習。
2. 提升學生的基層思維能力（理解及分析力）。
3. 營造愉快的學校氣氛。
教學規劃
學習和教學策略 : 1. 運用合作學習、腦基礎科學、高階思維教學策略來提昇學生學習效能。
2. 將早操、眼操、護脊操、中樂、弦樂、舞蹈、綜藝、游泳、體操、武術融入教學中。
3. 三年級設綜合藝術科，培育學生的在音樂、視覺藝術和表演藝術的潛能。
關鍵項目的發展 : 1. 提升學生的基層思維能力（理解及分析力）。
2. 營造愉快的學校氣氛。
共通能力的培養 : 透過九大學科，培養學生的九大共通能力。
學生支援
全校參與照顧學生個別差異 : 1. 設學生培育組及學生支援小組。
2. 設兩名駐校社工。
課程剪裁及調適措施 : 1. 設功課及考試調適。
2. 設輔導、支援訓練及伴讀課，照顧個別差異及有特殊學習需要學生。
3. 設中文、英文、數學、科技、體藝精進課程，培養能力較強學生。
家校合作及校風
家校合作 : 1. 推行家長教育：舉辦家長講座。
2. 促進家校和諧：安排文娛交流活動。
3. 加強家校合作：邀請家長參與學校活動，協助訓練學生。
4. 加強家校溝通：每年舉行兩次全校家長日、最少一次分級家長會。
校風 : 為促進師生關係和提升學生的歸屬感所作的安排：
1. 師生組隊參加社際球類比賽。
2. 「愛心大使」計劃加強輔導。
3. 安排高年級學生擔任中、英科小老師。
4. 加入學生主持早會、週會。
5. 午膳時間，設「師生撐檯腳」計劃，師生一邊吃飯，一邊聊天。
6. 設「交齊功課」計劃，鼓勵學生每天完成家課。
7. 設生日會，促進學生的凝聚力。
8. 設「零功課日」，舒緩學生的功課壓力。
學校發展
學校發展計劃 : 1. 提升教師教學效能。
2. 在教學策略上推行合作學習、電子學習、腦基礎教學。
3. 提升教師之間的凝聚力和團隊精神。
4. 透過多元學習提升學生的分析、理解及溝通能力。
5. 營造愉快的學校氣氛。
教師專業培訓及發展 : 1. 給予新教師入職輔導。
2. 參加教育局或大學支援服務。
3. 發展合作學習、腦基礎教學、高階思維。
4. 發展同儕觀課，交流學習。
學費減免 : 不適用。
其他 : 成功申請以下的基金及推行的計劃：
1. 教育局WIFI900。
2. 香港大學-MDM計劃。
