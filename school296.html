全方位學習活動
發展學生多元智能，活動及興趣班如：語言（英文話劇）、數學邏輯（奧數、lego機械人創建班）、音樂（中國鼓）、肢體動作（棒球、醒獅）、自然環境（環保小組）、個人內省（義工小組）、人際溝通（男、女童軍）、視覺空間（陶藝）等。
辦學宗旨
提供優質教育，重視學生五育及多元智能發展，培養學生積極進取、終身自學的精神，讓學生接受全人教育，將來成為服務社會的好公民。
學校特色
學校管理
學校管理架構 : 法團校董會、校長、副校長、行政主任、老師。
法團校董會/學校管理委員會/校董會 : 2007/08學年成立法團校董會。
環保政策 : 1. 設環保小組，培養環保意識。
2. 課室有廢紙回收及環保收集箱。
3. 午膳時要求學生自備餐具。
4. 環保圖書及file回收計劃。
學校關注事項 : 培育品德新世代，追求卓越新里程。
教學規劃
學習和教學策略 : 運用合作學習、拔尖補底計劃，以照顧學生學習差異。英文科設有外籍老師，課堂上進行大圖書教學計劃及process writing；中文科以發展寫作教室、普通話教學；數學科發展解難教學；常識科加入思維教學策略及自學探究小專家活動；視藝科有陶藝課程；體育科有棒球、游泳、木球等課程；跨學科活動，以自我管理、環保、棒球、中國文化等主題，發展學生共通能力。每年更舉辦境外學習交流團，曾到日本、韓國、台灣、馬來西亞、新加坡、北京、廈門、廣州等地考察，擴展學生之國際視野。
關鍵項目的發展 : 1. 設早讀課、閱讀越開心獎勵計劃、每月閱讀之星。
2. 課室安裝電腦、投映機、電子白板等，加強師生互動學習。
3. 透過全方位專題研習、自學及探究小專家比賽，發展學生共通能力及自學探究精神。
4. 推行成長課，重視學生全人發展。
共通能力的培養 : 各科發展學生的創意、溝通及解難能力。
學生支援
全校參與照顧學生個別差異 : 安排小六參與升中講座、面試班及參觀中學，以便中小銜接的落實；透過開放日、多元智能課及小一適應日，落實幼小銜接課程。課堂合作學習及課後功課輔導，照顧學生不同學習需要。
課程剪裁及調適措施 : 1. 拔尖課程，如：奧數、Buddy Reading、中文拔尖。
2. 保底課程，如加強輔導班、輔導小組、小老師計劃、新來港學童適應課程。
家校合作及校風
家校合作 : 家長義工到校為學生講故事及進行小豆豆伴讀計劃，協助中國文化日、頌親恩活動、敬師日、專題研習戶外考察、親子運動會等活動。每年亦會舉行周年大會及旅行。
校風 : 提倡「關愛」 及 「自學」文化 ， 舉行「自我管理計劃」，鼓勵學生養成自理的好習慣；DIY自學及探究小專家比賽等，培養學生有自學探究精神，匯報技巧、互相欣賞文化，積極自主學習的態度，以追求卓越的成就。
學校發展
學校發展計劃 : 每年，為擴闊學生視野，學校會為小五、小六學生安排「境外交流」活動；而學業成績及術科表現優異的學生，更會獲頒獎學金，以示鼓勵。
教師專業培訓及發展 : 深化合作學習，將電子白板及平板電腦應用於教與學，優化小班教學、思維策略及提問技巧。
學費減免 : 不適用。

2. 傑出舞台效果獎
3. 傑出演員獎
