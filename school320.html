辦學宗旨
以基督耶穌精神，透過基礎教育，使兒童在德、智、體、群、美、靈育各方面獲得均衡發展，養成良好習慣，建立健全人格，踏上豐盛人生。
學校特色
學校管理
學校管理架構 : 學校法團校董會以下是校監，校長，兩位副校長及兩位學位主任(PSM)，其下設十三個小組以便進一步管理學校各範疇事務。
法團校董會/學校管理委員會/校董會 : 馬鞍山靈糧小學法團校董會。
環保政策 : 1. 透過參與綠色校園計劃、環保大使計劃及環保課程，教育學生環保知識。
2. 設廢物分類，鼓勵循環再用。設現場派飯，減少廚餘，亦有廚餘處理機。參加可持續海鮮計劃落實支持環保。
3. 校園安裝環保T5光管及窗戶加裝隔熱膜。
4. 設有溫室及氣象站。
學校關注事項 : 虛己納善，發揮恩賜，謹慎自守，互相建立。
教學規劃
學習和教學策略 : 1. 設English Monday、星期五普通話日及狗教授活動，為學生營造兩文三語的學習環境。
2. 設校園電視台、專題學習日如愛心義賣活動、跨學科專題學習及舉辦不同的境外交流活動，以擴闊學生學習體驗。
3. 校本及特色課程如普教中、六年級英文數、四至六年級英常課程及校本英文網上學習平台等。
4. 發展電子教學，四年級開始推行"BYOD"自攜裝置電子學習計劃，以促進學習效能及培養學生自主學習。
5. 運用不同學與教策略如合作學習、元認知等。
6. 設有資優課程。
關鍵項目的發展 : 設圖書館、課室圖書、雨天操場閱讀區，推行網上閱讀、全校靜讀（明日閱讀計劃）、故事爸媽及各項閱讀獎勵計劃； 每班設電腦及實物投射機，以進行多媒體教學。 每年進行兩次專題研習，培養學生思考及組織能力。 設成長課及周會培養學生正面價值觀。
共通能力的培養 : 早會靈糧思考區、復活節尋蛋活動，讓學生發揮探究及創意；透過精英展藝教育博覽，讓學生學習不同領域的知識。設學生會及校園小記者發展學生溝通，組織及領導能力。
學生支援
全校參與照顧學生個別差異 : 設有輔導、課後溫習及加強輔導班，幫助有特殊學習需要的學生； 設中、英、數、普拔尖班、各項培訓及課外活動，發展學生各方面專長。
課程剪裁及調適措施 : 提升數學的進度，P6設英數，P4-P6設英常。
家校合作及校風
家校合作 : 本校透過家長講座、校訊、家長義工隊及家教會活動，建立緊密之家校合作關係，家長義工給予學校不少支援，協助學校進行不同活動如運動會、升中面試模擬班、故事爸媽、午膳爸媽、教具製作、節日學校佈置等。
校風 : 訓輔並重，推行「星級榮譽榜」、「學生成就獎勵計劃」、「每月一星」等校本輔導計劃來表揚學生不同成就及表現，另設功過相抵及大哥哥大姐姐計劃。
學校發展
學校發展計劃 : 虛己納善，發揮恩賜，謹慎自守，互相建立。
1. 優化教學效能。
2. 陶塑謙和文化。
3. 建構團隊網絡。
教師專業培訓及發展 : 鼓勵教師進修，定時進行共同備課、觀課及課研。舉辦提升教學效能的講座及境內外的教學交流。
學費減免 : 不適用。
其他 : 榮獲聯合國兒童基金會頒發之「友兒學校」，IAIE之「國際啟發潛能學校成就大獎」，基督教服務處之「關愛校園」，環境保護運動委員會之「香港綠色學校獎」及「香港環保卓越計劃金獎」，學校戲劇節評判推介演出獎。
