全方位學習活動
星期一至星期五下午及星期六上午，安排學生參加體藝活動、全方位學習活動、多元智能學習活動、基督教及生命教育活動及制服團隊活動，包括：球類、田徑、藝術創作、中樂團、合唱團、手鐘、中國舞、朗誦、英語話劇、小女童軍、幼童軍、基督少年軍等，合共約40多項活動。
辦學宗旨
按聖經真理提供全人教育，讓學生在五育和靈育上，得到全面優質栽培，以致學能得以提高，個人潛能得以確立和發揮，明白真道，建立優良品格，樹立律己精神，長大後能立身處世，熱愛國家，關心社會，榮神益人。
學校特色
學校管理
學校管理架構 : 法團校董會監察學校的發展，校長秉承法團校董會的辦學宗旨，領導教職員推動學校發展。
法團校董會/學校管理委員會/校董會 : 辦學團體邀請熱心推動基督教教育人士及獨立人士擔任校董，教師、家長及校友校董經選舉產生，共同推動學校的發展。
環保政策 : 培養學生對「可持續發展」的認知和態度，在校內推行各種環保活動，以提高學生的環保意識，培養學生對環保的正確態度，並且在生活上實踐。
學校關注事項 : 1. 建立關愛的校園交化及「樂善勇敢」的品格素質。
2. 發展校本高動機模式課程，以照顧差異，提升教學成效。
教學規劃
學習和教學策略 : 學校提供多元化的學習活動，如：配合學習主題的參觀活動、專題學習、全方位學習活動、服務活動和體藝活動，全面培養學生不同的閱讀策略和從閱讀中學習；音樂科和視藝科逐步發展校本數碼藝術課程，讓學生在各科進行電子學習，實踐自主學習和延伸學習。
關鍵項目的發展 : 有系統地設計閱讀活動，專題研習、電子學習及宗教及生命教育活動，發展和確保四個關鍵項目融合在學習生活中。
共通能力的培養 : 透過重新編訂的校本課程，視藝科課程加強學生欣賞能力，中文科加強美文學習，英文科的閱讀計劃，數學科及常識科的高思維課程，培養學生評賞、創造、說話、思維、協作和溝通能力。
學生支援
全校參與照顧學生個別差異 : 於一年級推行校本小幼銜接課程，引導學生愉快融入和適應小學的學習生活。小二至小六設有中、英、數不同的優化和輔導小組。另設課後功課輔導班。為新來港學童，提供校本支援計劃。發展資優課程和全腦培訓，提升尖子的學習機會，發揮潛能。
課程剪裁及調適措施 : 1. 小幼調適及升中銜接課程。
2. TARGET高動機學習模式，照顧差異學習校本課程。
家校合作及校風
家校合作 : 積極推動家校合作，舉行各級家長教師聚會，透過家長通訊，加強聯繫。重視家長教育，舉辦家長學堂，發展基督教家庭教育。家教會及校友會與學校保持良好伙伴關係，積極支持學校活動及方向，讓學生得益。本校連續8年獲「關愛校園」榮譽獎項。
校風 : 以基督教及生命教育為核心，建構關愛校園，培養學生成為「樂於凡事感恩、善於以愛待人、勇於面對挑戰、敢於經歷困難」的新一代。
學校發展
學校發展計劃 : 1. 建立關愛的校園文化及「樂善勇敢」的品格素質。
2. 發展校本高動機模式課程，以照顧差異，提升教學成效。
教師專業培訓及發展 : 基督教教育，優化學與教和照顧差異的專業發展。
學費減免 : 不適用。
