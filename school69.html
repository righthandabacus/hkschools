辦學宗旨
「敬、愛、勤、慎」為校訓，培養學生良好品格，致力推行 「德、智、體、群、美、靈」六育的全人教育，發揚基督精神。提供愉快的學習環境及多元化課外活動，透過德育及公民教育，使學生建立正確的人生觀與價值觀。
學校特色
學校管理
學校管理架構 : 辦學團體是中華基督教會長洲堂。
法團校董會/學校管理委員會/校董會 : 錦江小學法團校董會已於30/8/2008成立，法團校董會成員包括有七名辦學團體委任校董、校長、教師校董兩名、家長校董兩名、校友校董一名及獨立校董一名。
環保政策 : 推動環保教育，讓學生能從理論的認知，而實踐於學校及家庭生活中。學校制訂能源使用守則，減少浪費能源，如減少使用課室空調、換上節能光管，又推行校園廢物分類活動，設置三色回收箱。
學校關注事項 : 關愛校園
教學規劃
學習和教學策略 : 中英數三個主科多以連堂上課，增加學習機會。輔導課安排於課前或課後進行，以免影響原班課程進度。本校提供全方位學習機會給學生，讓他們獲得智能發展、德育及公民教育、社會服務經驗、體藝發展及與工作有關的經驗。全校推行專題研習。全校課室皆為多媒體電子系統教室，電腦室內學生可以一人一機學習。
關鍵項目的發展 : 德育方面，全力推行義工服務活動；策劃國民身份認同活動。 閱讀方面，推行早讀及中、英文廣泛閱讀計劃。
共通能力的培養 : 透過常規課程及全方位學習，培養學生的九種共通能力。
學生支援
全校參與照顧學生個別差異 : 本校以「全校參與」新模式推行加強輔導教學計劃，照顧學生的不同學習需要。成立學生支援小組，成員包括校長、社工、班主任、輔導老師等，於課後開辦「功課輔導班」聘用專業導師協助成績稍遜學生。
課程剪裁及調適措施 : 推行校本課程時，將部份原有課程作適量剪裁；為有特殊教育需要的學生提供功課及考試調適。為小一學習進行課後支援學習，如個別輔導及小組輔導等。
家校合作及校風
家校合作 : 1. 出版家長教師會會訊。
2. 舉辦親子歷奇活動和親子營。
3. 舉辦家長講座。
校風 : 營造關愛校園，並以基督教訓，推行有教無類的優質教育，啟發學生不同的潛能及興趣，達至「全人教育」。為這時代培養出勇於承擔、樂於服務、甘於委身的良好公民。
學校發展
學校發展計劃 : 發展計劃項目
1. 推動正向教育
2. 善用評估數據 - 以提升學與教效能
教師專業培訓及發展 : 學校成立教師發展小組策劃及推行教師發展計劃。
學費減免 : 不適用。
其他 : 與佛山海口小學締結為姊妹學校，推動教師和學生交流學習。
