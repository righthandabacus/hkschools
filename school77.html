備註 : 除了課堂學習外，我們十分重視學生不同的學習經歷，為他們提供多元化的學習活動，例如：「環保年宵」、「全方位學習日」、「我做得到---生活體驗課程」及

全方位學習活動
本校設有多元智能課程，讓學生參與五個範疇（趣味英語、通識、藝能、資訊科技、創意解難），十九個學習項目，例如：英語話劇、童軍式自理課程、園藝、環保教育、單車、游泳等。學生能透過經歷不同的學習活動，培養自己的興趣，訓練多維思考及博雅涵養。
辦學宗旨
有教無類，全人教育，五育均衡發展；為學生提供愉快的學習環境，奠定良好的學業基礎，以達到終身學習、貢獻社會的目標；為員工提供和諧互助的工作環境及專業發展機會；與家長和社區合作，共同培育學生健康地成長。
學校特色
學校管理
學校管理架構 : 分三個行政組：學校發展、課程發展、學生發展。
法團校董會/學校管理委員會/校董會 : 法團校董會已於2012年成立。
環保政策 : 鼓勵學生實踐環保4R，例如：在校園內設廢紙回收箱、鼓勵學生使用環保食具。本校成功申請環保及自然保育基金，裝置太陽能及風力供電系統及發電單車。校內環保大使積極宣傳環保訊息，帶領同學參與環保活動，身體力行。校內亦定期舉行環保主題活動，從活動中學習環保的意義。
學校關注事項 : 「教育一個孩子，需要整條村的努力。」
詳情請參考本校網頁。
教學規劃
學習和教學策略 : 為了鞏固學生語文能力，一年級中文科採用「喜悅寫意」教材，英文科參加PLP／RW 「小學識字計劃」，為初小學生打好語文基礎；本校亦申請了語文教育及研究常務委員會的「提升小學英語水平津貼計劃」，發展校本英語課程。學校積極推行閱讀文化，設有早讀課及圖書分享活動，此外，老師在設計活動時，常鼓勵同學從圖書當中找資料，實踐從閱讀中學習。為了擴闊學生學習經歷，學校舉行全方位學習日、科學探究日、文化日及境外交流等活動，增長學生見聞。
關鍵項目的發展 : 1. 設有早讀課，鼓勵師生同閱讀，分享閱讀的樂趣。
2. 通過全方位學習日及科學探究日活動，培養學生的研習能力。
3. 通過週會課，個人成長及公民教育課，培養學生正面價值觀。
4. 資訊科技結合學科，推行多元化活動。
5. 開展「多元智能課程」，擴寬學生學習經驗。
共通能力的培養 : 以合作學習模式促進協作、溝通、解決問題、自我管理和研習技能等能力的發展；創造力、批判性思考、運用資訊科技和運算能力則廣泛結合於各科學習活動中。再輔以課外活動，則所有共通能力的培養均得到保證。
學生支援
全校參與照顧學生個別差異 : 本校以「全校參與」模式支援有特殊教育需要學生，利用教育局增撥的額外資源包括：小學加強輔導教學計劃；加強言語治療津貼及「小學校本教育心理服務」為學生安排適切的訓練小組。課後設有不同支援小組，如非華語班、功輔班及新來港學童適應班等。本校又參與中文大學「優質學校改進計劃」提升老師在照顧個別差異的教學技巧。
課程剪裁及調適措施 : 1. 提供家課及評估調適予有需要學生。
2. 設課後中文、英文及數學輔導課和增潤課，照顧不同學習需要的學生。
3. 設課後非華語班、新移民班，幫助學生適應學校生活。
家校合作及校風
家校合作 : 家長教師會支持學校的各類活動，包括設立獎學金、開辦興趣班、發動家長義工支援學校活動等。
校風 : 推行級本管理，塑造和諧校園文化。設學生發展組，成員包括訓輔導老師和學校社工，策劃個人成長活動。
學校發展
學校發展計劃 : 關注事項（一）：統整中英數常課程，優化課堂教學，照顧學習差異，提升學習效能。
關注事項（二）：培養三品葛小人（品德/品質/品味），推動關愛文化，培養積極人生態度。
關注事項（三）：推動資訊科技校園計劃，提供升學校行政效率及建構以資訊科技輔助學與教的學習環境。
教師專業培訓及發展 : 1. 以集體備課和同儕觀課，提升教與學的質素。
2. 引入大專院校的專業支援，協助推行各類專業發展活動。
3. 研習多元策略，培養學生建立良好學習習慣，強化學習效能。
學費減免 : 不適用。
其他 : 1. 本校鼓勵學生參與各項比賽，開拓視野，例如：香港校際戲劇節，香港校際朗誦節及香港校際音樂節等。
2. 本校舉辦優才計劃「明日領袖—北京學習之旅」，計劃包括前置課程、境外學習及成果展示，實踐「讀萬卷書走萬里路」的精神。
3. 本校推薦優才生參與校際學術比賽及傑出學生獎勵計劃，從而培訓優秀人才。
