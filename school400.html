班級教學模式
本校運用電子學習及透過小組學習的模式讓學生從互動中學習。二、三年級及四至六年級部分班級以普通話教授中文；而二至六年級設加強輔導班，以抽離形式為學習能力稍遜的學生進行小組教學。
班級結構備註
四至六年級會按照學生學習的需要分班。

分班安排 : 四至六年級按學生的學業成績分班，一至三年級平均分班。

辦學宗旨
通過全人教育宣揚福音精神，讓學生認識「愛與生命的價值」，建立積極的人生觀，並透過「靈、德、智、體、群、美」的培育，使學生成為良好的公民，貢獻社會。
學校特色
學校管理
學校管理架構 : 法團校董會轄下由校監、校長、副校長及各科組統籌組成學校管理層，並策劃、統籌和監管學校的組織與管理、學與教、校風及學生支援和學生表現。
法團校董會/學校管理委員會/校董會 : 梨木樹天主教小學法團校董會
環保政策 : 園圃種植、環保大使、廢物分類回收、節約能源、用水、用紙、滅蚊巡查等；並藉講座及活動培養學生的環保意識。
學校關注事項 : 欣賞生命‧擁抱挑戰
教學規劃
學習和教學策略 : 1. 提升學生兩文三語的能力：推行「普教中」、「喜閱寫意」計劃、校本英文課程及語境教學，加強學生運用語文的能力。
2. 加強學生高階思維及自學能力：數學科解難教學、「奧數」培訓、常識科自學小冊子及高層次的提問，進一步幫助學生發展思維及學習能力。
3. 推行主題學習及全方位學習：透過跨學科主題活動、STEM教育活動、多媒體藝術綜合學習及全方位學習活動，豐富學生學習經歷及提升學生共通能力。
關鍵項目的發展 : 1. 從閱讀中學習：設早讀課、中英文廣泛閱讀計劃、網上閱讀活動、校本閱讀獎勵計劃及閱讀分享等活動，提升學生閱讀興趣及閱書量。
2. 資訊科技教育：運用電子白板及平板電腦推行電子教學，發展校本電子教材，並推動學生於網上進行自學及交流。在電腦科課程中加入編程教學，提升學生邏輯思維能力。
3. 專題研習：設校本研習課程及研習週，教導學生運用共通能力進行研習。
4. 德育及公民教育：透過宗教培育及生活教育、早會、週會、班級經營、服務學習活動、制服團隊及「活出彩虹獎勵計劃」等，培養學生正確的價值觀。
共通能力的培養 : 透過課堂上小組學習，以及不同科目的課程設計及跨學科活動，加強培養學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 一至三年級推行初小中文分層支援模式計劃，藉以鞏固學生的語文基礎。另外透過小一及早識別計劃，校本教育心理服務及言語治療服務，儘早支援有學習困難的學生。本校亦開設加強輔導班、中、英、數輔導班、功課輔導班、新來港學童適應課程及大哥哥大姐姐計劃，並為有特殊教育需要的學生安排聽說讀寫小組及喜伴同行小組。
課程剪裁及調適措施 : 為加強輔導班剪裁中文科的課程，以鞏固學生對基本字詞的掌握，並為有讀寫困難的學生提供功課、默書及評估調適，從而改善學生的學習效能及提升他們的學業表現，讓他們建立自信。
家校合作及校風
家校合作 : 家長校董、標書審批、家教會周年大會、家教會會議、親子旅行、講座、工作坊、家長興趣班、義工服務。
校風 : 學校透過為學生提供不同的學習經歷、活動和講座等，讓學生懂得珍惜自己的生命，發揮自我潛能，而且懂得尊重及關懷別人。
學校發展
學校發展計劃 : 1. 配合天主教學校的五大核心價值中的「真理」及「義德」，培養學生欣賞生命，積極面對挑戰的良好人生觀。
2. 優化學教策略，提升學生自主學習的能力。
教師專業培訓及發展 : 1. 鼓勵教師參與配合學校發展及任教科目的教師培訓活動，達至3年150小時的目標。
2. 設教師專業發展日、共同備課、同儕觀課、教師分享時段及教學資源分享，讓教師互相交流、反思，促進教學效能。
3. 參與教育局、校外支援機構及內地交流等計劃，拓闊教師視野，並掌握教學新趨勢。課程發展計劃更有助教師設計適切的校本課程及掌握多元化的教學策略。
學費減免 : 不適用。
其他 : 1. 家教會設立獎學金鼓勵積極求進步的學生。2. 校友會設立獎學金，獎勵表現傑出的學生。3. 校本及區本課後學習支援協助學生提升學業成績及發展多元智能。4. 重視英語培育，致力提升學生的英語能力，開設英文劍橋試班、升中英語面試班及英文話劇組等。5. 幼小銜接︰訪校日、中英數適應課程及興趣班。6. 中小銜接︰中學觀校日及升中講座。7. 積極改善學校環境，增添教學設施。
