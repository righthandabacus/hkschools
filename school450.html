辦學宗旨
本校重視良好的學習風氣，鼓勵學生在學業、品德及個人潛能的發展；培養學生獨立思考的能力、樂善合群，成為良好公民；提供完善的學習環境，優良的師資，配合資訊科技，使學生愉快學習、成長，達至全人教育的使命。
學校特色
學校管理
學校管理架構 : 學校管理主要分成三個委員會：
1. 校務委員會（學生福利、學校資源管理等）；
2. 學與教委員會（各科目、校本支援課程等）；
3. 校風及學生培育委員會（課外活動、訓輔組等）；
4. 對外聯繫委員會 （學校編輯組、家教會、校友會）。
法團校董會/學校管理委員會/校董會 : 已於2007年成立法團校董會。
環保政策 : 節約用電計劃，學生自備午膳餐具，廢紙回收及校園綠化計劃。
學校關注事項 : 1. 持續發展多元化的課程，培養學生和諧共融及自主學習的素質。
2. 透過多元化的評估方式，培養學生自我反思及追求臻善的態度。
3. 積極推動教研及深化各科組的自評，促進學校發展。
教學規劃
學習和教學策略 : 德育課程、聯課、電子學習、全方位學習、跨科課程統整。
關鍵項目的發展 : 1. 加強資訊科技配套設施，落實電子學習模式。
2. 透過班本晨讀、校本閱讀獎勵計劃，專題閱讀報告培養學生閱讀習慣。
3. 透過多元化的教學策略，培養學生高階思維能力、溝通、創造等共通能力。
4. 透過全方位學習、領袖團隊、繪本教學等，培養學生正面價值及態度。
共通能力的培養 : 透過校本課程及不同的學習活動，培養學生九種共通能力。
學生支援
全校參與照顧學生個別差異 : 小一生適應活動、朋輩輔導計劃、輔導班、課後功課輔導班、照顧有特殊學習需要學生計劃及言語治療。
課程剪裁及調適措施 : 與教育局、本港大學及香港教育大學合作，按學生能力及生活經驗，剪裁校本課程，重組教學單元，並設計不同教學活動以提高學生的學習興趣。
家校合作及校風
家校合作 : 家教會委員會議、家長講座、工作坊、親子旅行、家長義工隊。
校風 : 1. 訓育及輔導：德育及公民教育課程、全方位輔導計劃、成長的天空計劃。
2. 新生適應：舉辦小一新生輔導日、朋輩輔導計劃、伴讀計劃。
3. 多元化活動：戶外考察、科技日、中華文化日、跨學科專題活動、專題研習、親子日營。
學校發展
學校發展計劃 : 1. 透過各種語文活動，創設豐富中、英、普並重的生活語境 提升學生兩文三語的基礎。
2. 落實推行電子學習及STEM教育計劃。
3. 透過繪本教學推行生命教育。
4. 與教育局及專上教育機構等協作發展及優化校本課程，提升中、英、數學科課程規劃及教學策略。
5. 透過網上學習、創意寫作、專題研習等策略推動自主學習。
6. 發展教師伙伴教研，提升教師專業發展。
教師專業培訓及發展 : 1. 教師自評。
2. 同儕互評。
3. 小班教學教師培訓。
4. 照顧有特殊學習需要教師進修。
5. 教師發展日。
6. 教師進修分享會。
學費減免 : 不適用。
其他 : 1. 中華回教博愛社獎學金。
2. 鮑伯濤夫人獎學金。
