班級教學模式
全校均採用活動教學、聲情教學、合作學習模式學習。小一至小六推行小班教學，小三至六年級設加強輔導教學，以單式課節授課。
班級結構備註
推行全校參與模式支援每個學生，因應學生在學習上的多樣性而作出教學調適。

辦學宗旨
秉承基督教香港信義會的辦學宗旨：提供全人教育，服務社會，見證基督，培養學生德、智、體、群、美、靈均衡發展。本校核心價值：以學生為本、愉快學習、愉快教學，重視學生的整體健康成長及發展。
學校特色
學校管理
學校管理架構 : 法團校董會、校董教師諮議會、校監、校長、學校管理委員會、學教發展委員會、校風及學生發展委員會、各科組統籌。
法團校董會/學校管理委員會/校董會 : 基督教香港信義會葵盛信義學校法團校董會
環保政策 : 1. 舉辦環保主題的遊學團，讓學生到台灣進行考察活動和專題研習。
2. 培訓環保先鋒宣揚環保訊息。
3. 設置回收箱回收廢物。
4. 舉辦有機耕種、回收廚餘、魚菜共生等活動。
學校關注事項 : 1. 拓展資優教育，讓學生發展潛能；
2. 優化學習模式，以學生為本，提升教學效能；
3. 加強學生的責任感，培育學生的承擔精神。
教學規劃
學習和教學策略 : 1. 通過合作學習及聲情教學，讓學生高參與、多展示，透過多元化的活動擴闊學生的學習經歷。2. 透過超學科專題研習，讓學生擴闊學習經歷及建構知識，提升共通能力。3. 推行電子學習，增加師生、生生互動的機會，培養學生自主學習的能力和習慣。
關鍵項目的發展 : 1. 推行閱讀計劃，透過分享，培養學生的語文能力及閱讀興趣。
2. 提供網上學習資源及獎勵計劃，鼓勵學生運用資訊科技學習。
3. 透過統整課程、生命教育、講座、週會、成長課，培養學生正確的價值觀及道德觀。
4. 環繞主題推行超學科專題研習，發展學生的共通能力。
共通能力的培養 : 各學習領域課程均滲入共通能力的培養，訓練學生的創造力、批判性思考及溝通能力。
學生支援
全校參與照顧學生個別差異 : 本校為照顧有特殊教育需要的學生，利用全校參與照顧個別差異模式，進行及早識別和支援，讓不同潛能的學生發揮所長。本校提供的學習支援包括：小一適應課程、識字策略小組、言語治療服務、小老師朋輩輔導計劃、新來港學童適應課程、課後功課輔導、家長工作坊、加強輔導班和課程、課堂教學、家課、默書及評估調適。
課程剪裁及調適措施 : 課程和評估的調適、多元化的評估、小班教學。
家校合作及校風
家校合作 : 家長與學校保持緊密合作關係，透過舉辦多元化活動，如親子旅行、講座、工作坊、聯歡會等家長活動，促進家校間的溝通，強化親子關係。家長義工團隊積極參與和協助學校舉行各項活動，建立良好的夥伴關係。學校同時透過「家．長．愛」學院計劃及與外間機構的合作，按需要提供家庭教育，為每位學生家長提供合適的教育及支援。
校風 : 1. 透過班級經營，營造班級溫馨、和諧的氣氛。
2. 推行「關愛校園計劃」，照顧學生身心全面發展，建立關愛文化。
3. 以基督的愛為基石，透過福音工作，培養學生健全的人格。
4. 透過傑出及模範學生選舉計劃，發展學生的個人潛能。
5. 引入外間資源：
．「魔術治療小組」，識別學生是否早期焦慮，以及及早介入。
．「乘風航」風紀訓練，協助學生建立自信心及自我挑戰的能力。
．「理好營」自理成長訓練營，提升學生的自理能力。
學校發展
學校發展計劃 : 1. 照顧學生的個別差異，培育資賦優異的學生。
2. 善用電子學習，提升學與教的效能。
3. 推展STEM教育，提升學生創意及解難能力。
4. 優化學與教的策略，提升學生的語文能力。
5. 營造學校關愛文化，培育學生的正確價值觀。
6. 學生能在愉快的氣氛中學習。
7. 優化校園設施，讓學生在優質的環境下學習。
教師專業培訓及發展 : 1. 逢星期三放學後設教師專業發展時段，因應關注的主題、老師的需要及學科發展而進行不同的校本專業發展活動及培訓。
2. 透過共同備課、觀課交流、教學分享平台和課堂學習研究，提升學與教的效能。
3. 積極鼓勵及推薦老師參與不同範疇的培訓課程，提升教師專業水平。
4. 為教師設立團隊活動，建立一個關係密切及有團隊精神的隊工。
學費減免 : 不適用。
其他 : 2009-2016參與「關愛校園獎勵計劃」，連續八年獲頒發「關愛校園」的榮譽。
