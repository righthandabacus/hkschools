辦學宗旨
秉承保良局辦學宗旨，為廣大巿民提供優質的教育服務，五育並重，全人發展。
學校特色
學校管理
學校管理架構 : 實行校本管理，設立學校發展委員會及各行政小組。校長及教師每年就學校的現況，共同訂立來年的關注事項，因應學校的需要，制定發展計畫。
法團校董會/學校管理委員會/校董會 : 依法團校董會規章組成。
環保政策 : 學生透過參與學校舉辦的環保活動，提升環保意識。
學校關注事項 : 照顧個別差異，啟發學生潛能，建立健康、關愛校園。
教學規劃
學習和教學策略 : 除正規課程外，透過全方位學習活動，豐富學習經歷。推行專題研習，培養學生各種共通能力以及成為一位積極主動、富責任感的人。於小一至小六進行小班教學，以25人為限。全校實行合作學習模式，讓學生得到更多照顧，課堂發揮更佳效果。
關鍵項目的發展 : 1. 閱讀計劃：採用圖書教學，推行中英文廣泛閱讀計劃。
2. 資訊科技：推廣電子學習。
3. 專題研習：每月兩至三次專題研習技巧課。
4. 德育及公民教育：隔週安排生活教育及德育講座，舉辦不同類型的輔導性及預防性活動，參與公民教育系列活動。
共通能力的培養 : 透過專題研習、小組活動及戶外活動，發展學生共通能力。
學生支援
全校參與照顧學生個別差異 : 設分組教學、輔導及功課輔導班，為學生提供學習支援；為有特殊學習需要之學生提供個別化支援計劃、讀寫能力提升小組等。為初小學生提供分級成長小組、伴讀、伴寫及伴數計劃。
課程剪裁及調適措施 : 按學生不同需要及專業判斷，制定不同調適政策，如家課及考試調適等。
家校合作及校風
家校合作 : 學生對學校歸屬感強，愛學校、愛師長、愛同學。學校與家長有足夠的溝通渠道，成為親密的伙伴。設「關心一線」、「家訪」及「家長工作坊」等，並成立「家長義工隊伍」及「家長資源中心」，加強家校合作。
校風 : 建立關愛校園及服務社區為目標，鼓勵每一個學生積極參與，參與教育局家長伴讀及朋輩伴寫計劃；並連續八年獲基督教服務處頒發「關愛校園」榮譽獎。
學校發展
學校發展計劃 : 1. 培育「兩文三語」人才，推廣「普教中」，聘外籍英語教師，提供良好語文學習環境。
2. 加強學生溝通協作、解難及思維能力；培養學生主動學習的精神。
3. 培養學生建立正面的價值觀，健康的自我形象。
4. 提高學生的公民責任感，關心社會及身邊事物，服務社群。
5. 建立健康校園，促進學生全人發展。
教師專業培訓及發展 : 注重教師專業發展，鼓勵持續進修。參與教育局及學術機構的專業發展計劃，與友校交流，致力推廣同儕觀課文化，鼓勵教師互相觀摩，提升專業知識及教學技巧。
學費減免 : 不適用。
其他 : 推行小班教學，提供優質多元及均衡的課程，建立健康、關愛的校園。
