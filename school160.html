辦學宗旨
以「非以役人、乃役於人」為校訓，致力培育學生六育的均衡發展，使學生的潛能得以發揮，日後成為具獨立思考能力、關心社會、具備知識和技能、有正確待人處事態度的良好公民。
學校特色
學校管理
學校管理架構 : 學校管理架構由法團校董會、學校行政會、教師校務會及各功能組別組成。
法團校董會/學校管理委員會/校董會 : 法團校董會由校監、辦學團體校董、獨立校董、教員校董、家長校董及校長組成。
環保政策 : 設環保校園組推廣環保活動，包括：善用資源，綠化校園；發揮創意，廢物再生；珍惜節約，全校採用環保照明燈具；循環再用，裝置雨水收集系統，以灌溉園圃；天台鋪設隔熱塗層，降低室內溫度；推廣綠色能源，設太陽能、風力發電設施及單車發電系統。
學校關注事項 : 1. 品德：有禮守規，學會服務；
2. 學術：主動積極，創新求進；
3. 活動：提供機會，啟發潛能。
教學規劃
學習和教學策略 : 1. 重視培育學生的創意、解難、探究及自學能力；
2. 重視照顧學生個別差異，並發展資優培育策略；
3. 重視朋輩互動學習，推行課堂合作學習及小老師計劃；
4. 重視課程統整，自擬中、英、數科教材及推行跨科協作計劃。
關鍵項目的發展 : 1. 推廣閱讀風氣，優化閱讀環境，提升閱讀策略，重點發展中、英、數、常閱讀計劃。
2. 參加WiFi100電子學習學校支援計劃；全校鋪設無線網絡；擔任教育局資訊科技教育卓越中心；設編程課程以配合STEM教育的發展。
3. 強化專題研習內容，培養閱讀報章習慣，提升九種共通能力。
4. 推動德育及公民教育，設立生活教育課，舉辦健康小先鋒獎勵計劃。
共通能力的培養 : 重視服務學習、專題研習及網上學習，舉辦校外及境外實地考察等活動，以提升學生的共通能力及自學能力。
學生支援
全校參與照顧學生個別差異 : 設學生支援小組，由駐校言語治療師、駐校教育心理學家及學生輔導主任協助有特殊需要的學生及策劃輔導服務。為四至六年級學生舉辦「成長的天空」計劃，提升抗逆能力，建立正向生命及價值觀。
課程剪裁及調適措施 : 課後設拔尖補底課程，包括學科輔導、功課輔導、數學精英培訓、電腦及體藝精英培訓等，另為有特殊學習需要學生安排小學加強輔導班及考試調適。
家校合作及校風
家校合作 : 透過家長教師會活動、講座、親子旅行、家長日、家長參與教學活動和義工服務，促進家校的溝通和聯繫。
校風 : 以有禮守規及學會服務為品德培育中心，引導學生建立主動積極的學習態度及創新求進的學習精神。
學校發展
學校發展計劃 : 1. 培養自律自學自信的新一代。
2. 強化教師團隊的專業分享及交流文化。
3. 提升反思自評文化，強化質素保證機制。
教師專業培訓及發展 : 1. 訂立教師專業進修策略。
2. 組織共同備課及互相觀摩教學。
3. 組織教師專業發展日及交流活動。
4. 邀請專家及學者進行校本專業分享。
學費減免 : 不適用。
