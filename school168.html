辦學宗旨
本校秉承中華基督教會香港區會以結合傳道服務精神為宗旨，悉心培育學生身、心、靈三方面的均衡發展；承諾以愛心及關懷締造良好學習環境，讓學生有機會認識基督，培養求知自學能力，發展個別潛能。
學校特色
學校管理
學校管理架構 : 行政老師協助學校管理工作。設立學校行政核心組策劃及監察校務進行。
法團校董會/學校管理委員會/校董會 : 已於2011年9月1日成立法團校董會。
環保政策 : 定期進行綠化及回收活動，組織學生環保大使協助推行，並推行班級綠化種植活動。
學校關注事項 : 誠信有禮 自省自愛 散佈亮光 活出生命
教學規劃
學習和教學策略 : 運用多元智能理念，學校提供不同學習經歷，讓學生獲得均衡發展及成功感。每年進行全方位學習活動，提供跨學科學習體驗。部分科目亦嘗試進行協作教學設計，以豐富學生的學習經驗。
關鍵項目的發展 : 在中、英文科推行廣泛閱讀，發展分級閱讀技巧培訓，提升同學閱讀技巧，全校推行閱讀獎勵計劃，並推行跨科圖書閱讀協作，邁向從閱讀中學習；利用資訊科技進行學與教，安排網上學習資訊及練習，午膳及課後開放電腦室供學生研習；各級進行專題研習；在德育及公民教育方面，推展生命成長課，培育學生對個人、社會、國家的認識及承擔。
共通能力的培養 : 高階思維能力以滲透式融入課堂教學，通過學科、專題研習及成長課中培養各種共通能力。透過不同聯課活動，培育學生的自學能力。
學生支援
全校參與照顧學生個別差異 : 推行資優教育，發展適異教學課程，並在課外活動增添資優學科及領袖培訓活動，安排不同的拔尖課程，啟發學生潛能。安排課後輔導小組及小組培訓，為後進生提供支援；安排教師、家長工作坊，深入認識如何支援具有不同學習需要的學生。
課程剪裁及調適措施 : 透過單元教學，進行課程剪裁，照顧不同學生的需要；對「有特殊學習需要」學生按識別結果調適課業模式及考試配置。
家校合作及校風
家校合作 : 學校有兩隊熱心服務的家長義工隊伍和彩虹讀書會伴讀家長義工，家長教師會與學校合作無間。
校風 : 校訓「純樸自律，愛主愛人」，學生純樸守規，透過德育及靈育培訓，加強學生自律勤學的品格。
學校發展
學校發展計劃 : 配合基督教精神發展全人教育，發展學生不同的興趣及技能，並加強生命教育推廣；加強中層人員的領導能力；優化課堂教學，提升學生學習能力；提升學生的學習自信及主動學習精神。
教師專業培訓及發展 : 配合課程發展及學生學習需要，建立中、英、數、常學習社群，共同備課及進行課研，提升教師專業。安排全校及中層人員培訓，令學校成為專業教育團隊。
學費減免 : 不適用。
其他 : 戶外學習及境外交流學習計劃︰每年均舉行國內及英語國家的交流學習團，令學生不但能實地運用兩文三語，而且親身體驗當地文化。
