辦學宗旨
本校秉承基督的精神，提供全人教育，使他們在德、智、體、群、美、靈六育方面得到均衡的發展。學習與活動並重，使學生能愉快而有效地學習及心智健康地成長，栽培他們成為一個有責任感，明辨是非的公民。
學校特色
學校管理
學校管理架構 : 除法團校董會外，學校還設立九個行政部門負責校內各項事務。
法團校董會/學校管理委員會/校董會 : 法團校董會由辦學團體成員及其他公眾或專業人士組成。
環保政策 : 制定環保政策，執行環保學校和辦公室指引，舉辦種植、綠色聖誕聯歡會和無污染旅行等活動。
學校關注事項 : 建立數碼校園，推行STEAM教育，提升學生表現，深化生命教育。
教學規劃
學習和教學策略 : 1. 推展「普教中」和「英語電子學習」，提升學生兩文三語。
2. 透過全方位學習及跨學科活動，提升學生學習興趣和自主學習能力。
3. 推展校本資優課程和啟迪課程，照顧學生學習的多樣性。
4. 透過境外學習活動，擴闊學生視野。
5. 推展STEAM教育，培養學生探究事物的興趣，豐富學習經歷。
6. 推展 「一生一體一藝計劃」 、 「好學生星級獎勵計劃」 和 「服務學習」，促進學生健康身心靈發展。
關鍵項目的發展 : 各學習領域滲入「從閱讀中學習」的元素。推展電子學習、專題研習和科學探究，並持續發展宗教、德育及公民教育和健康校園計劃，培育健康身心靈。
共通能力的培養 : 透過專題研習、主題學習活動、互動學習及服務學習發展學生共通能力。
學生支援
全校參與照顧學生個別差異 : 設學生支援組及資優教育組，及早識別和幫助個別有特殊學習需要的學生，為學生提供駐校言語治療服務及校本教育心理服務，適切地提供功課、情緒及社交輔導。
課程剪裁及調適措施 : 小二至小六年級有中英數科輔導小組、啟迪課程 、資優課程、每週設兩節課內活動，以豐富學習經驗。
家校合作及校風
家校合作 : 鼓勵家長參與子女教育，加強家校溝通。透過家教會組織家長義工，協助學校舉辦家長教育及安排故事/午膳/保健爸媽等服務。
校風 : 透過主題集會，「好學生星級獎勵計劃」及「成長的天空」，培養學生正確的價值觀及態度，推行「服務學習」，培養學生成為各項主題大使，彼此關懷尊重，建立其自尊自信，發揮潛能。
學校發展
學校發展計劃 : 1. 持續優化課程規劃，提升學生兩文三語能力及科學探究等能力。
2. 推行小班教學，優化教學策略，提升學與教效能。
3. 加強校本資優課程，發掘學生潛能。
4. 深化「一生一體一藝計劃」，發揮學生的多元智能。
5. 推行電子學習，提升學習動機及自主學習的能力。
6. 推展STEAM教育，強化學生的綜合應用知識和技能。
7. 深化生命教育，培養學生多角度思考及解決問題的能力。
教師專業培訓及發展 : 加強有關「生命教育」及「STEAM教育」的培訓，推動校內和校外交流活動，建立專業的教學團隊。
學費減免 : 不適用。
其他 : 由2009年開始，在校內安排課餘託管服務，由聖公會聖雅各福群會專業社工及導師主持，以照顧有需要的學生及家長。
