辦學宗旨
我們重視學生德、智、體、群、美、靈六育的均衡發展，並透過靈性教育，使學生對人生、家庭、社區和國家有正確的認識及態度，以達至全人教育的目的。
學校特色
學校管理
學校管理架構 : 法團校董會監督學校運作，校長帶領學校發展組策劃學校的發展，透過校務會議商討學校事工和評估學校發展計劃的成效。設教師諮議會促進教師與法團校董會的溝通，設家長教師會及校友會促進家校合作與溝通。
法團校董會/學校管理委員會/校董會 : 法團校董會由辦學團體校董、獨立校董、教員校董及家長校董共十三人組成。
環保政策 : 環保小組舉辦不同活動以提升全校員生的環保意識，推動簡約生活，保持校園清潔及綠化。
學校關注事項 : 提升教學效能，增強學生個人責任感及服務精神。
教學規劃
學習和教學策略 : 推行全方位學習，推動多元智能活動，培養共通能力。另設資優教育計劃、校本輔導活動、資訊科技教育，發展多媒體教室。設協作課，配合週年學習主題，提升學與教的效能。
關鍵項目的發展 : 透過獎勵計劃、閱讀分享、自助圖書角等，培養校園閱讀風氣。各科能善用資訊科技學習中心及多媒體教室進行互動學習。推行專題研習活動，培養學生的共通能力。以校本輔導計劃、成長課及宗教課推行德育及公民教育，培養學生正確的價值觀及態度。
共通能力的培養 : 推行「一生一體一藝一服務」、設多元智能活動課及設四十多項興趣小組和課外活動，發展學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 全校參與模式，照顧有特殊教育需要的學生。設雙班主任制，課堂設協作教學，課後有不同的學習小組及支援計劃。
課程剪裁及調適措施 : 推行拔尖保底教學及活動，為能力有差異的學生設計不同課業及學習活動。
家校合作及校風
家校合作 : 舉辦家長教育課程及講座。透過不同活動如全校家長日和小一新生輔導日加強家校溝通。家長也可透過電郵及家教會執委向學校反映意見。家教會成立10多年以來，曾籌辦不少教育性聯誼活動：家長義工隊在多方面為學校服務。
校風 : 校風淳樸，是實幹型的專業團隊。重點培養學生的責任感和強化其自省能力，讓學生從服務中學習，體現校訓「非以役人，乃役於人」的精神，又培養學生國民意識、發展德育及國民教育。
學校發展
學校發展計劃 : 提供優良的學習環境，讓學生學會學習，奠定其終身學習的基礎。積極推動專科專教，全體教師具備專科學位。學校有計劃地發展自評工作，建立校本的質素保證機制。持續提升教師的專業發展，增強照顧學生個別差異的能力。
教師專業培訓及發展 : 透過共同備課、觀課、校本教師專業發展，及參與教育局培訓課程，讓教師更專業成長。
學費減免 : 不適用。
其他 : 設聖公會傑出學生獎等獎學金以獎勵在品、學、服務和活動各方面表現突出的學生。學生每年參加校際陸運會、水運會、球類、朗誦、演講、寫作、音樂和奧數等比賽超過370多項，屢獲殊榮，去年有超過230個學生獲得1156個獎項，實屬難得。每年又安排學生參加內地交流，提升學生對國民身份的認同和對國家的認識。亦會安排學生參加新加坡交流活動，與海外學生分享 STEM 的學習經驗。
