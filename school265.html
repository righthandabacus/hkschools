除了為一年級至六年級同學小組分科輔導外，本校更有隨堂抽離式的小組學習班 ，在課堂上為有需要的學生加強師生教學互動，還有課後功課輔導班。

全方位學習活動
1. 課外活動：幼童軍、救傷隊、瑪小園丁、創意教育、校園小記者、棋藝。 2. 各展所長班：黏土樂、中國民族舞、英語戲劇、乒乓球、籃球、足球、羽毛球、奧數、空手道、Lego機械人、辯論。 3. 音樂興趣班：口風琴、敲擊樂、歌詠、小提琴、大提琴。
辦學宗旨
本校秉承天主教辦學精神，以基督的愛為本，以「全人教育」為目標，致力拓展學生的靈性幅度，並以「真理忠貞」作為校訓，幫助學生建立積極進取的人生觀和價值觀，並在靈、德、智、體、群、美六育的薰陶下健康地成長。
學校特色
學校管理
學校管理架構 : 包括學校發展組、學生發展組及課程發展組。
法團校董會/學校管理委員會/校董會 : 法團校董會已於2014年3月1日成立。
環保政策 : 1. 透過綠化校園，培養及提高師生的綠化環境意識，並加強師生的歸屬感。
2. 推廣環保教育，讓學生認識再生能源並了解人類面對的能源危機，從而教導學生愛護環境，珍惜資源的重要性。
學校關注事項 : 1. 通過學習不同的思考方法，持續培養學生自主學習的能力。
2. 通過學習不同的閱讀策略，持續培養學生自主學習的能力。
3. 通過推行校本輔導活動，培養學生的承擔精神。
教學規劃
學習和教學策略 : 善用課內及課外時間安排全方位的學習活動，如語文自學、早操、校園電視台、各展所長班、音樂聽我show、普通話攤位遊戲、專題研習、跨學科學習活動日、國情教育、創意課程等。
關鍵項目的發展 : 1. 德育及公民教育：推行成長課及國情教育，協助學生建立良好的人際關係、正確的價值觀及增強對國家的歸屬感。
2. 從閱讀中學習：除進行不同的閱讀獎勵計劃及閱讀活動外，並在語文課及圖書課中教授閱讀策略，提升學生的閱讀能力。
3. 專題研習：推行探究式專題研習，提升學生的研習能力及共通能力。
4. 資訊科技：各科運用電子學習，為運算思維教育重點發展學校，於高小開設運算思維教育課程。
5. STEM教育：通過科學探究活動，提升學生運用科學過程技能的能力，加強他們綜合應用不同範疇的知識來解決日常生活問題的能力。
共通能力的培養 : 通過不同科目及跨學科的學習活動（如專題研習，單元式主題學習活動日，創意思維課程等）以發展學生的溝通能力、創造力及明辨性思考能力為基礎，以至運用資訊科技能力、運算能力、解決困難能力、自我管理及研習能力。
學生支援
全校參與照顧學生個別差異 : 透過合作學習、資優領袖訓練課程、資優聯課活動、小組學習班、功課輔導班、小組分科輔導班、奧數培訓班，強化拔尖保底功能。
課程剪裁及調適措施 : 各科均為校本課程，由本校老師精心設計或剪裁，並提供校本的創意課程，此屬於全班模式的資優教育課程。另外，為資優學生舉辦領袖訓練、領袖潛能啟發課程及英語專題學習工作坊及暑假資優班，以發揮學生潛能。同時，為有需要的學生安排課後功課輔導班及小組學習班等以增加師生的互動。
家校合作及校風
家校合作 : 家長教師會成立超過19年，定期會舉行講座、工作坊及親子日營。
校風 : 本校設有學生發展組，重視學生品德培育，致力建構和諧健康的校園文化，支援學生成長，經多年的培育，學生都能自律守規，對人有禮，自我管理良好。同學間相處融洽，彼此友愛，積極投入學校活動和服務。
在不同的獎勵計劃如「瑪小存摺」及「傑出學生獎勵計劃」等配合下，正面強化了學生的良好品行。
本校培養學生建立健康生活、健康的飲食習慣，提升學生對環境衞生的注意，並身體力行地保持學校環境衞生。
學校發展
學校發展計劃 : 1. 通過不同的學習策略，培養學生自主學習的能力。
2. 加強品德教育，培養學生自主學習的個人特質。
教師專業培訓及發展 : 制訂持續專業發展指引，鼓勵教師不同範疇的發展。另外，透過教師發展活動，提升教師專業素質。
學費減免 : 不適用。
其他 : 本校設立多項學科獎學金，除鼓勵同學在學業及體育方面積極投入外，並發揮中小學薪火相傳的精神。此外，本校學生在校際田徑、球類、游泳、舞蹈、中英及普通話朗誦、奧數、敲擊樂、口風琴及歌唱比賽中屢獲殊榮。
