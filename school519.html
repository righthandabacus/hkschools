辦學宗旨
學校本著有教無類、因材施教的精神，以發揮不同階層、不同種族學生的個別才能。學校以禮、義、廉、恥四維為校訓，冀望學生能恪守四維，作為將來立身之本。
學校特色
學校管理
學校管理架構 : 本校已成立法團校董會，在管理與組織，學與教，校風及學生支援，學生表現四個範疇推展校務。
法團校董會/學校管理委員會/校董會 : 油蔴地街坊會學校法團校董會成員包括：1位校監，6位校董，2位獨立校董，1位教員校董，1位替代教員校董和1位家長校董。
校董會共同釐定學校的發展項目，審批財政預算及人事安排，並檢討學校各種計劃的成效。
環保政策 : 學校推行節約能源，設立綠化校園課外活動小組，並積極參與各團體舉辦的環保活動。
學校關注事項 : 1. 提升學生的學習效能。
2. 培養學生自學能力，提高閱讀興趣。
3. 校內推廣關愛文化，建立和諧校園。
教學規劃
學習和教學策略 : 以英語為主要教學語言，並按課程需要，靈活運用課時，為學生提供多元化的學習經歷，以配合不同的學習需要，有效地達成學習目標。
關鍵項目的發展 : 1. 閱讀計劃：設不同的閱讀活動及獎勵計劃，鼓勵學生閱讀。
2. 資訊科技：改善IT設施、設立e-class平台，發展學生運用資訊科技自學的能力。
3. 專題研習：以常識科作平台進行專題研習，培養學生的多元能力。
4. 德育及公民教育：透過不同的主題活動，讓學生身心均能健康地成長，並營造一個和諧校園。
共通能力的培養 : 透過分組學習、專題研習及全方位學習，培養學生的共通能力。
學生支援
全校參與照顧學生個別差異 : 1. 學校成立學生支援小組，及早識別及幫助有學習需要的學生。
2. 課後設輔導教學小組、提升專注力小組，以改善學生的學習表現。
課程剪裁及調適措施 : 1. 學校設校本課程發展小組，與校外專業課程發展的專家，為非華語學生剪裁中文及數學課程，以提升他們的學習興趣及能力。
2. 中、英、數三科的科任老師，因應同學的能力，調適課程及評估，為成績稍弱的學生，提供更多學習上的照顧。
家校合作及校風
家校合作 : 學校與家長保持緊密聯繫，每年均舉辦一至六年級家長會、小一新生家長會及小六升中選校家長會。
校風 : 設小一新生適應班，讓學生在輕鬆及關愛的環境下，盡快適應小學的學習生活。
學校發展
學校發展計劃 : 1. 配合學校的發展重點，加強教師專業發展，提升學與教的成效。
2. 推廣閱讀文化及加強閱讀策略訓練，從而提升學生的自學精神。
3. 發展校本中文課程。
4. 推廣自評文化。
5. 與社區機構合作，向少數族裔人士推廣健康教育。
教師專業培訓及發展 : 校方因應校本需要，為老師提供專業交流的機會，並鼓勵老師持續進修，以提升學與教的效能。
學費減免 : 不適用。
