班級教學模式
小班教學：一至六年級。
加強輔導教學計劃：三、五、六年級中、英、數科目額外增加一班，為學習能力稍遜之學生作加強輔導。
班級結構備註
學校著重學生全人發展，安排不同的課外學習經歷予學生，例如每年為期三天，旨在培養學生研習技巧及共通能力的專題研習週，每年的境外交流團、畢業營及去年首辦的「十年樹人」計劃等，均以全方位幫助學生成長為目標。

全年全科考試次數 : 3
多元學習評估 : 聽、說、讀、寫能力，學習態度與技能表現均作評估。中文、英文、數學、常識、視覺藝術、體育及資訊科技教育等科目會按學生在各單元之習作或表現作進展性評估。
分班安排 : 小一平均分班。小二、小三按能力設一班精英班，其餘平均分班。小四至小六按能力設一班精英班，一班次精英班，其餘平均分班。

全方位學習活動
敲擊樂隊、合唱團、校園電視台製作組、英語話劇、中英文朗誦隊、田徑、足球、排球、乒乓球、欖球、跆拳道、中國舞、爵士舞、視藝學會、電腦學會、數學精英組等，制服團隊有：幼童軍、小女童軍和公益少年團。
辦學宗旨
配合政府教育政策，秉承保良局辦學方針，以校訓「愛敬勤誠」為經，五育「德智體群美」為緯，提供優質教育服務，讓學生在優良的環境學習及發展潛能，同時重視培養學生以「禮義廉恥」為立身之本。
學校特色
學校管理
學校管理架構 : 校本管理模式，全校參與訂立關注事項及發展計劃。設改善小組及自評小組，提升教育質素。
法團校董會/學校管理委員會/校董會 : 設法團校董會，由辦學團體校董、校長、教員校董、獨立校董及家長校董組成。
環保政策 : 校內廣泛植樹，修建花糟、花架及魚池，營造優美環境。午膳採用現場派飯措施，洗手間安裝節水器具，藉此節約資源。學校透過不同的渠道及舉辦不同的活動，向學生宣揚環保訊息。
學校關注事項 : 1. 幫助學生掌握有效的學習策略，建立良好的學習習慣，並推動以學生為中心的教學模式，以促進學習的效能。
2. 培養學生「愛己樂群」的價值觀。
教學規劃
學習和教學策略 : 與教育局、大專院校協作推展電子學習課程、推行BYOD電子學習模式、規劃初小英語拼音課程於讀寫中、統整初小中文科單元教學、深化規劃高小校本戲劇英語課程、配合課改最新發展於高小推展「科普教育」、在高小資教課堂加入「運算思維教育」、規劃各科的整體及抽離式資優課程、組織校內觀課及評課學習圈。
關鍵項目的發展 : 1. 全校網絡光纖化，提供靈活的學習環境，支援電子學習的發展。
2. 以校本輔導計劃及跨科課程推展德育及公民教育。
3. 提升教師的專業水平，以多元教學策略引入學教環境中。
共通能力的培養 : 除了恆常的課堂外，本校又透過施行校本專題研習技能框架，透過圖書課及專題研習建構學生的共通能力；以課程統整及校本課程，有策略地培養學生的高階思維能力及良好公民的正確態度。
學生支援
全校參與照顧學生個別差異 : 實施小班教學，透過合作學習，爭取師生與生生間互動學習的空間；按二至六年級中、英、數不同科目需要額外增加共9班的小班教學。
課程剪裁及調適措施 : 校本訂定基礎及拔尖課程、提供多元評估及設立調適機制，以切合不同學生的學習需要。
家校合作及校風
家校合作 : 著重家校合作，致力推展家長教育，裝備家長支援子女成長。
校風 : 培養學生自律精神，為學生創設成功經歷，營建自律、讚賞氛圍。
學校發展
學校發展計劃 : 1. 以學生為本位，透過不同的策略及建立學生的習慣，提升學習效能。
2. 組織學習圈，促進教師專業成長，改善學教素質。
3. 善用多元教學策略及評估，以照顧學生學習多樣性。
4. 善用資訊科技創締學教環境，提升學教質量。
5. 提升學生的「愛已樂群」的價值觀，建立良好校風。
教師專業培訓及發展 : 1. 透過校外進修及不同院校的協作交流，提升專業素質及教學技巧。
2. 制訂長遠專業發展計劃，讓老師能有系統地參與專科及照顧學生學習多樣性的培訓工作。
學費減免 : 不適用。
其他 : 1. 本校音樂及科普方面的課外活動表現尤為突出。
2. 每年舉辦境外交流學習計劃，擴闊學生視野及學習領域。
