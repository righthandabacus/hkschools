辦學宗旨
以德、智、體、群、美五育並重為原則，著重兒童的均衡發展，培育兒童達至身、心、社、靈、品學之至善，使將來成為社會和國家有用的人才。
學校特色
學校管理
學校管理架構 : 由法團校董會賦權予校長負責管理學校，設不同行政組別，由資深中層管理人員統籌，副校長負責監管，以期各司其職，有效推動校務之運作及發展。
法團校董會/學校管理委員會/校董會 : 由一群熱心教育的地區人士組成，其中亦包括、辦學團體董事、校長、教師代表、家長代表、校友代表及獨立人士，甚具代表性。
環保政策 : 1. 培養師生、家長及職工的環境保護知識及責任感。
2. 鼓勵學生參與改善環境的活動，並參加「綠色學校獎勵計劃」。
3. 節約能源
4. 減少產生廢物，推廣循環再用、回收及再造。
5. 與家長教師會合作，舉辦校服回收運動。
6. 遵守有關
學校關注事項 : 1. 深化自主學習。
2. 建立正向思維。
教學規劃
學習和教學策略 : 1. 重視學生課堂內、外之學習經驗。
2. 語文科專科專教。
3. 以普通話教授中文。
4. 專題研習、德育及公民教育、環境教育、全方位學習及境外體驗，以培養學生共通能力。
5. 發展校本之小班教學計劃。
關鍵項目的發展 : 1. 舉辦閱讀計劃，推廣跨科閱讀文化。
2. 利用資訊科技強化學生的學習能力和興趣。
3. 推行跨學科專題研習活動。
4. 成立學生義工隊，服務社區。
5. 積極推展生活教育及公民教育。
共通能力的培養 : 課堂及學習活動中滲入發展共通能力的元素。
學生支援
全校參與照顧學生個別差異 : 1. 開辦學習支援小組。
2. 舉辦家長講座/工作坊，提升家長支援子女的技巧。
3. 額外聘請支援教師，為有學習困難的學生提供輔導。
4. 開辦全班式的資優教育培訓課程。
課程剪裁及調適措施 : 開辦拔尖補底課程及實施功課評估調適。
家校合作及校風
家校合作 : 透過定期舉辦的分級家長教師座談會、親子旅行日、親子日營等，使家長和老師加深認識。
校風 : 1. 訓輔合一，培養學生積極人生觀。
2. 透過不同渠道，與家長保持聯繫。
3. 成立家長義工隊，協助學校推行政策及活動。
4. 成立健康促進組，與專業團體合作推行健康政策。
5. 為小一學生及將升中之學生舉辦活動，讓學生適應新環境。
6. 透過多元化的校園活動，促進學生全人發展。
學校發展
學校發展計劃 : 1. 提升學與教的效能，促進學習。
2. 建立健康關愛社群。
教師專業培訓及發展 : 1. 教師參與學校或教育局舉辦之工作坊及分享會。
2. 透過進修機制，教師全學年需參加持續專業發展工作坊或講座。
學費減免 : 不適用。
