興趣班：樂器、歌詠團、網球班、花式跳繩、圍棋、英語話劇、跆拳道及乒乓球校隊班。 星期六活動：免費電腦班、宗教活動及各類興趣班。 星期五下午：課外活動。
辦學宗旨
秉承天主教辦學目標，以愛及熱誠培育學生，使其在溫暖愉快的環境中成長。提供優質教學，培養學生獨立思考，提高公民意識及使其在靈、德、智、體、群、美各方面有均衡發展。
學校特色
學校管理
學校管理架構 : 由法團校董會管理，加入教員、家長及校友校董，推動學校發展。校長下設副校長及各行政小組策劃及推行校務。
法團校董會/學校管理委員會/校董會 : 天佑小學法團校董會
環保政策 : 鼓勵學生自備水瓶、餐具及自備手帕。
學校關注事項 : 1. 以天主教教育核心價值貫連靈訓輔，培養學生正確價值觀及態度，營造愉快的校園生活。
2. 持續提升學與教效能，增強學生學習成就感。
教學規劃
學習和教學策略 : 著重發展中、英、數學習能力及培養各種共通能力，建立自學能力。透過全方位學習活動，豐富學生的學習經驗。課堂設計小組協作活動，鼓勵學生互動交流，和諧相處。推行全校性英語及普通話語境教學，小一至小六用普通話教中文，小一至小三推行英語閱讀計劃(PLP-R/W)，奠定良好的語文學習基礎。
關鍵項目的發展 : 1. 推行校本「悅讀升章獎勵」計劃及每年以主題圖書，推動閱讀風氣。
2. 各級以常識科為中心，發展專題研習及滲入STEM學習活動。
3. 以天主教社會內倫理為中心，推展生活教育課、成長課，並配合校本宗教課程及價值教育課程，培養學生正確的態度及價值觀，愛主愛人。
4. 提供不同的電子學習資源及學校內聯網，鼓勵學生主動自學。
5. 本年度新設「天佑至GOOD KIDS計劃」，藉此獎勵計劃培養學生建立優良的品行。
共通能力的培養 : 著重課堂師生互動，透過小組討論、匯報、圖書分享、專題研習等，提升學習興趣，照顧差異，發展溝通、協作、創意思維及明辨性思維能力。
學生支援
全校參與照顧學生個別差異 : 課餘設拔尖補底課程。本校積極發展校本中英數常資優課程，如趣味科學探索班、英文創意寫作班、中文創意寫作班、數學資優班。設有「校本教育心理服務」。
課程剪裁及調適措施 : 教師因應學生能力，設計教學內容及發展不同能力課業。本校亦實施評估調適和功課調適。
家校合作及校風
家校合作 : 家教會為家長與學校之溝通橋樑，與校方合作緊密，舉行各種活動如親子旅行、義賣舊校服、故事爸媽等。家教會委員代表為買賣業務委員會成員，為校務發展提供意見。
校風 : 學校重視培養學生的紀律，保持良好校風。學生知禮守規，關愛同學。學校積極提供切合學生成長需要的輔導計劃，強化學生的正確態度和行為。
學校發展
學校發展計劃 : 營造一個健康愉快的學習環境，積極培養學生良好語言能力，並建立正面的價值觀和積極上進的生活態度，步武基督。
教師專業培訓及發展 : 1. 透過教師發展日、科本課程發展工作坊、共同備課及觀課後分享，提升教學效能。
2. 參與教育局校本支援計劃、課程領導學習社群及P1-3PLP-R/W英文閱讀及寫作計劃。
學費減免 : 不適用。
其他 : 1. 與校友合辦「蝴蝶保育計劃」
2. 與家教會合辦親子旅行、日營、烹飪班、毛巾健康操、黃昏營及講座
3. 與明愛賽馬會社會服務中心（荃灣）合作推行區本計劃，提供課後功輔班及興趣班
4. 卓越護脊校園
5. 參與由教育局委託香港大學教育應用資訊科技發展中心(CITE)為本校五年級常識科進行「電子學習支援科學科自主學習以達至照顧學生個別差異及升中過渡(SDLS)計劃
