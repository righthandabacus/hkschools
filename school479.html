全方位學習活動
實行「一生一體藝」，活動多元化，包括：粵劇小豆苗、花式跳繩、英語及普通話戲劇、書法、國畫、跆拳道、東方舞、乒乓球、籃球、足球、排球、游泳、幼童軍、公益少年團、外籍教師英語班、珠心算及奧數班等。
辦學宗旨
為本港兒童及青少年進行「全人教育」，提供一個優良的學習環境，使能發揮個人的潛能，日後成為具備知識技能、有獨立思考能力、勇於承擔責任和關注社會事務的良好公民。
學校特色
學校管理
學校管理架構 : 本校隸屬於東華三院，在校董會領導下，設不同科組統籌全校教學、行政及學生事務。
法團校董會/學校管理委員會/校董會 : 2006成立法團校董會，加強持份者的參與及透明度，法團校董會運作已上軌道。
環保政策 : 組織綠色大使，協助老師推行各項環保活動。定期進行校服及廢紙回收，實踐環保生活。校園內設有「農耕樂」園地，進行有機耕種活動，又種植大量植物，優化環境。本校學生積極參與各項環保活動，以增進環保的知識和養成環保的生活習慣。
學校關注事項 : 1. 優化自評工具運作機制，提高校政的透明度及問責性。
2. 透過推行不同的學與教策略，培養及提升學生創造力和解決問題的能力。
3. 推行品德教育，實踐關愛感恩校園文化。
教學規劃
學習和教學策略 : 1. 透過學習活動及課業，培養學生自主學習能力。
2. 透過推動STREAM教育以培養學生學習科學與科技、閱讀、文藝及數學的興趣、提升創意和解難能力，以及發展學生的創新思維。
3. 廣泛採用合作學習，以同儕力量促進學習。
4. 透過學習活動、社區服務培養德育及公民教育。
5. 每年舉行文化歷史實地考察活動。
6. 科與科之間緊密配合，促進跨科發展，使關鍵項目元素及共通能力應用於課堂中。
關鍵項目的發展 : 閱讀策略：中文、英文、圖書課。
STREAM 教育：電腦、常識及各科。
專題研習：中文、常識、音樂、普通話、視藝、體育。
德育及公民教育：各科。
共通能力的培養 : 共通能力已融入正規課程，並在課堂和活動中實踐。
學生支援
全校參與照顧學生個別差異 : 照顧學習多樣性：設分層課業、資優課程及適異課程培育不同能力的學生。一般課堂設多元化學習活動，照顧不同學生的學習需要。參與各類型公開試及國際聯校學科評估以拔尖展能。提供全校參與式輔導教學計劃。
課程剪裁及調適措施 : 1. 參與《促進學生自主學習：小學中國語文科閱讀規劃與策略運用》「種籽」計劃，以優化學與教安排。
2. 小二至小六每級最少一班以普通話教授中文。
3. 英文小學識字PLPRW計劃。
4. 常識科著重科學學習。
家校合作及校風
家校合作 : 法團校董會、家教會、家長領袖課程等架構推動家長積極參與各類家校合作活動。
校風 : 尊己、關愛、國民身份認同：為學生提供服務的機會，培養正面價值觀。早會、週會、午膳廣播和級任課推行德育活動；德育元素滲入課程，全校參與。
學校發展
學校發展計劃 : 1. 優化自評工具運作機制，提高校政的透明度及問責性。
2. 透過推行不同的學與教策略，培養及提升學生創造力和解決問題的能力。
3. 推行品德教育，實踐關愛感恩校園文化。
教師專業培訓及發展 : 全年進行不同範疇的教師培訓，包括東華三院聯校教師發展日、全體教師工作坊、各科組工作坊及教學交流講座、觀課交流等。
學費減免 : 不適用。
其他 : 東華三院提供學業、體藝、進步等各範疇的獎學金，本校另有李東海獎學金鼓勵在下列五大範疇中有優秀表現的學生：社會服務、語文能力、體育運動、美術創作及表演藝術。
