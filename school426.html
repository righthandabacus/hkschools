班級教學模式
一至六年級實施小班教學，加強課堂上之師生及學生互動。另設有2個加強輔導教學計劃及課後輔導教學，以加強補底支援。
班級結構備註
融合教育以協作及抽離模式，促進融合學生之學習。

多元學習評估 : 除利用每年的測驗及考試外，本校會利用不同的評估方式，如電子化「進展性評估」，日常課業及專題研習等，亦已採用多方參與評估模式，如同儕互評及家長評估等，有效回饋學生學習。
分班安排 : 小一至小六平均分班。

健康校園生活 : 積極鼓勵學生均衡飲食，推動每週一天的「環保素食餐」計劃；同時在校內設置體育角及「護脊操專區」，參與「SportACT」運動獎勵計劃。
備註 : 一至六年級實施小班教學。
全方位學習活動
為學生提供多元化課外活動，包括學術、體育、藝術、興趣小組及制服團隊等三十多項課外活動。本校亦聘請專業教練作導師，包括籃球、小型網球、木笛、足球、羽毛球、舞蹈組、跆拳道、芭蕾舞蹈組及中國民間敲擊樂等。
辦學宗旨
秉承天主教教義，引導學生認識福音真理，推行愛的教育，使學生在德、智、靈、體、群、美各方面得到均衡的發展。「愛心教化在溥仁，六育均衡啟潛能」。
學校特色
學校管理
學校管理架構 : 在法團校董會的監督下，學校事務由三個組別負責。
（一）管理與組織
（二）學與教
（三）機構文化及給予兒童的支援
法團校董會/學校管理委員會/校董會 : 法團校董會成員包括辦學團體成員、校監、校長、家長、教師、校友及獨立人士等各持份者。
環保政策 : 本校定期推行環保活動，如舊衣回收運動、參觀香港有機農莊及推展「溥仁再生能源動能供電」、「節能教室」計劃等，著力推動環保教育。
學校關注事項 : 1. 培養學生的堅毅精神。
2. 透過課程規劃，以電子學習促進有效學習。
3. 透過專業發展提升教師運用電子學習的能力，以優化學生學習。
教學規劃
學習和教學策略 : 1. 學習和教學策略：課程靈活有彈性，除十科外，尚設有圖書課及德公課。各科課程與全方位學習及聯課活動靈活統整。
2. 教師運用高層次的提問與回饋，內容扣連學生生活經驗，採取合作學習及活動教學策略，有效照顧學習差異。
關鍵項目的發展 : 1. 一至六年級進行專題研習。
2. 德育及公民教育方面滲透各科進行價值教育，並配合週會、主題講座及成長課等。
3. 師生善用資訊科技進行學與教活動。
4. 本校設定每天早讀時段及READING FRIDAY，並教授有效閱讀策略，再加上一系列閱讀獎勵計劃，培養學生養成良好的閱讀習慣。
共通能力的培養 : 回應教育局建議優先發展的「3C」，已取得良好的成果。未來將加強學生的專題研習能力及解難能力。
學生支援
全校參與照顧學生個別差異 : 本校設有「加強輔導班」、「教育心理服務」、「言語治療服務」、「融合教育」、「校本及區本課後支援服務」、「幼小銜接計劃」及升中支援配套等服務。
課程剪裁及調適措施 : 本校按照學生的學習需要，進行課程剪裁及調適，如「個別教學計劃」(IEP)，此外，編製「校本電腦課程」、「校本通識課程」、「科本英語滲透計劃」、英文科PLP-R/W及校本中文說話課程等。
家校合作及校風
家校合作 : 家教會每年均舉辦不同類型的活動，包括：家長教育講座、故事爸媽、新春團拜、親子大旅行等。大型家長義工隊支援學校舉辦各項大型活動。
校風 : 校風淳樸，嚴謹，師生關係融洽。強調學生在有紀律、關愛、包容的環境下成長，被校外評核隊稱許為「儼如一個大家庭」。
學校發展
學校發展計劃 : 1. 全面推行電子學習及落實BYOD政策；善用電子工具發展探究式學習及自主學習。
2. 持續推展校園電視台 "Po Yan Super TV"。
3. 裝備學生成為廿一世紀的全球化公民，終身學習，與時並進。
教師專業培訓及發展 : 逢週三舉辦教師專業發展活動，內容緊扣本年度學校關注事項。另積極鼓勵教師作專科進修，全校老師共100%獲學士學位，62%取得碩士或雙碩士學位。
學費減免 : 不適用。
其他 : 每年均頒發學業成績獎、進步獎及「英蘭獎學金」；此外，亦有參與校際朗誦節及音樂節、九龍北區各項田徑及球類比賽。
