辦學宗旨
本著有教無類精神，提供全人教育，讓學生在愉快的氣氛下學習；為社會培養德、智、體、群、美、靈兼備的好公民。
學校特色
學校管理
學校管理架構 : 1999年成立學校管理委員會。教師組成學校行政委員會及各行政小組，參與學校決策，集思廣益。
法團校董會/學校管理委員會/校董會 : 學校管理委員會成員包括教育局代表、校長、老師代表、家長代表、校友代表及校外獨立人士。
環保政策 : 推動綠色校園，各持份者身體力行，愛護環境、節能減費。
學校關注事項 : 正向思維及自主學習能力。
教學規劃
學習和教學策略 : 以「ARCS」理念融入課程與教學中，提升學生學習動機。推行元認知教學策略，設校本思維課，培養學生科創思維及自學精神。此外，重視培養學生正向思維及感恩包容之心，以積極樂觀的態度去管理自己、關懷別人，並通過多元化的全方位學習活動，促進學生全人發展，發掘個人潛能、服務社會。
關鍵項目的發展 : 1. 開展WIFI900計劃，於課堂活動加入電子學習元素，鼓勵學生透過網上資源進行預習。
2. 開展Cool-think 賽馬會電腦運算思維課程。
3. 設校本思維課，培養學生科創思維及自學精神。
4. 發展學生專題探究及研習能力。
共通能力的培養 : 透過課堂活動、專題研習、日常課業，課外活動、校本思維課程及組織制服團隊，培養學生批判性、創意、解難、溝通、協作及自我管理等共通能力。
學生支援
全校參與照顧學生個別差異 : 為照顧學生學習的差異及多樣性，本校設有學生支援小組制定政策和服務，老師培訓已達到教育局要求。
課程剪裁及調適措施 : 通過共同備課，按學生需要，設計校本課程。
家校合作及校風
家校合作 : 家長教師會及家長義工隊協助帶領學習活動，家長教師會積極參與和學生福利有關的政策。
校風 : 校風淳樸，注重品德培養；為不同需要學生開設小組；設領袖生及不同服務團隊等組織，培養學生服務精神及對學校的歸屬感。
學校發展
學校發展計劃 : 1. 加強學生正向思維，培養學生感恩包容之心，以積極樂觀的態度去管理自己、關懷別人。
2. 聚焦學生對學習的「興趣」、「能力」和「應用」三個方向，善用資訊科技及深化學習策略，提升學生學習能力。
教師專業培訓及發展 : 學校定時為老師安排專業發展活動；透過共同備課、觀課及參與大學或教育局的校本支援教學計劃，提升老師教學效能。
學費減免 : 不適用。
其他 : 每年舉行境外學習活動，例如星加坡、韓國遊學團等。
