班級教學模式
利用新資助模式撥款於課後開設不同強化小組，學科小組，於星期六聘請導師開設學科加強班，教導成績稍遜的學生學習技巧。一至三年級推行普教中，四至六年級各級有兩班用普通話教中文。
班級結構備註
星期一至四上課時間 8:10至3:30，逢星期四2:45至3:30為全校活動時間，星期五上課時間8:10至2:10，每天12:45至1:35午膳。
學習評估
全年全科測驗次數 : 2
全年全科考試次數 : 2
多元學習評估 : 語文方面包括聽說讀寫評估，各科因應本科特色作多元化評估，並有學生互評、自評及家長評鑑。視藝、體育設學習檔案，常識科分小組進行專題研習，並為有特殊教育需要的學生提供測考及功課調適。
分班安排 : 一年級隨機分班，升二按成績較佳編入B、D班，其餘平均編入A、C及E班。升三至六年級按成績較佳順序編入B、D、E 班，其餘平均編入A、C班。一至二年級全級及三至六B、D班用普通話教中文。

健康校園生活 : 重視同學的均衡發展，參加體育推廣計劃，推行健康小食，環保午膳，採用樓層派飯。成立校園清潔隊，舉辦班際清潔比賽，各類身心成長健康講座等。
備註 : 利用早會時間舉行「晨讀」、圖書分享，德育訓練及心靈培育，設午間英語廣播。逢星期一至三設導修課，由老師指導同學完成功課。每天午間及課後設不同的聯課活動，同學可自由參加。周四設綜合活動，星期五舉行崇拜週會和生活指導課。
全方位學習活動
聯課活動：棒球、英語話劇、小記者、奧數、電腦、舞蹈、國術、跆拳、球類、田徑、游泳、射箭、美工、詩班、管弦樂、敲擊樂、中樂、團契、童軍、環保、種植小先鋒、創意科學、升旗隊、陶藝、小領袖培訓等體藝活動。
辦學宗旨
本校老師以愛心及關懷締造一個溫馨有序的學習環境，讓學生有機會認識基督，啟發自己的潛能，鍛鍊強健的體魄和培養高尚的情操，學習領導及服務，成為一個良好公民，回饋社會，貢獻國家。
學校特色
學校管理
學校管理架構 : 由法團校董會管理，秉承區會的辦學精神，領導各行政小組實踐已制定的發展目標及政策。
法團校董會/學校管理委員會/校董會 : 由不同的持份者來管理學校。
環保政策 : 設環保大使，空中花園、惜食計劃及綠化校園。
學校關注事項 : 照顧學習差異、營造創意校園生活、培育學生自主學習及做個盡責好學生。
教學規劃
學習和教學策略 : 鼓勵自學，透過共同備課及課程剪裁，設計校本課程，推行不同教學策略和活動。
關鍵項目的發展 : 1. 致力培養學生的閱讀習慣和策略，設中英文閱讀課，低年級由外籍老師帶領小組閱讀，舉辦主題閱讀月及各項閱讀獎勵計劃。
2. 推行學科專研、跨科專題設計及電子學習計劃，透過網上學習及互動通訊。
3. 推動STEM教育，加強學生創意。
4. 透過德育及公民教育培養學生愛己愛人的態度，鼓勵學生注意個人衞生，灌輸環保觀念及加強學生的責任感。
共通能力的培養 : 透過課堂學習、創意日、專題研習、匯報及境外交流等，鼓勵學生從實踐中學習，從以培養學生的共通能力，如批判性思考、創造力和合作能力等。
學生支援
全校參與照顧學生個別差異 : 透過三層支援模式照顧學生的個別差異，於課後及周六舉辦中英數科的拔尖補底課程。利用校本課後學習支援計畫撥款開辦各類增益課程。設有幼小銜接、升中適應及外購輔導服務等。
課程剪裁及調適措施 : 因應學生的學習情況而調整教學方法及家課量。
家校合作及校風
家校合作 : 本校重視家校溝通，與家長攜手合力建設關愛和諧校園。家教會致力促進家校合作，協辦學術及親子活動及出版會訊，推動親職教育及為學校發展提出寶貴意見。
校風 : 校風淳樸，學生好學守規，注重關愛校園文化。推行優質小息活動，讓學生從遊戲中培育自信及積極進取的精神。設立50多項聯課及康樂活動和豐富的暑期參觀及學習活動，定期舉辦綜藝匯演。每週推行綜合教育課，將專研技能，個人成長及生命教育等元素融入課程內。
學校發展
學校發展計劃 : 促進教師持續專業發展，培育學生樂、善、勇、敢的學習態度及能成為愛主愛人，承責有禮的學生，又致力建立關愛校園，培養學生正面積極的人生觀。
教師專業培訓及發展 : 校本培訓，辦學團體聯校教師發展日，教育局和大專院校培訓課程等。
學費減免 : 不適用。
