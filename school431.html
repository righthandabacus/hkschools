班級教學模式
全校採小班及活動教學模式授課。四年級起採用一人一機的電子書包教學模式，並推展Flipped Classroom 的教學法，提升學生自主學習能力。
班級結構備註
本校致力推動創新教學模式，學校特設「立體打印室」、「數碼21創意教室」以電子學習設備，讓學生善用創新科技，以支援學習，提升學習興趣及效能。

全方位學習活動
課外活動包括宗教、學術、音樂、體育、演藝、興趣小組、制服團體及社會服務等。本校每年均參與校際田徑、各項球類、音樂節和朗誦節等比賽，藉此與各友校互相切磋，提升同學的體藝才能。
辦學宗旨
以基督的精神作為教育方針，六育並重，為未來社會培養人才，服務社群。
學校特色
學校管理
學校管理架構 : 由法團校董會管理，不同行政組負責推動學校發展。家教會及校友會是我們的合作伙伴。
法團校董會/學校管理委員會/校董會 : 已成立法團校董會，讓家長及校友參與學校發展，提高管理的透明度及問責性。
環保政策 : 以可持續校園為目標，建立環境管理系統，善用資源；建構素質文化，舉辦環教活動及組織環保大使；綠化校園。
學校關注事項 : 愛學習 學習愛。
學校在各科及各組的計劃中，均會以這個主題貫通各項教學活動，讓學生對學習產生更大的喜悅，提升學習興趣及動機，同時亦學習如何以愛活出生命。
教學規劃
學習和教學策略 : 建立知識基礎，啟發思考，戶外學習及全方位學習提供實際學習經歷。教學及評估照顧學習差異。靈活運用課時，進行學科及跨科活動。
關鍵項目的發展 : 1. 推行中、英文閱讀獎勵計劃。
2. 推行閱讀課，學習閱讀策略，從閱讀中學習。
3. 設立網上教與學平台，有利教與學。
4. 課室裝置電腦、投映機及電子白板，方便學習。
5. 進行專題研習，提升學生共通能力。
6. 推行成長課，培養德育及公民教育。
7. 利用評估數據作分析檢討，並針對學生的學習難點，回饋日常教學。
共通能力的培養 : 各科學習過程中培養共通能力，常識科設共通能力分級學習目標。
學生支援
全校參與照顧學生個別差異 : 1. 學習支援小組，支援有特殊學習需要同學，如「識字2課程」、「創意思維課程」。
2. 功課輔導班，指導同學完成課業。
3. 為小一學生進行「及早識別和輔導有學習困難的小一學生」計劃。
課程剪裁及調適措施 : 精進課程為成績落後的同學提供針對性支援。
家校合作及校風
家校合作 : 透過家長網絡，家長團契、義工計劃及親職教育，加強家校的伙伴關係。
校風 : 1. 關愛為核心，建立同學自信，培養良好操守和完備人格。
2. 周年主題突出行為目標，全校推動。
3. 暑期適應班，幫助同學盡早適應小一生活。
4. 升中選校及適應講座，面試訓練，為小六同學升中作準備。
學校發展
學校發展計劃 : 1. 建立智能化校園。
2. 創新電子學習計劃。
3. 3D立體打印技術。
4. 「五餅二魚 ‧午餐變魚」–廚餘再造魚菜共生系統計劃。
5. 實踐BYOBit 「機械創客」計劃。
6. 推行DreamStarter 的課程。
7. 「賽馬會運算思維教育」計劃之資源學校。
教師專業培訓及發展 : 持續專業發展方向，切合學校發展和老師需要。籌劃校本培訓，提升教師專業素質，對學生及學校帶來正面積極的影響。
學費減免 : 不適用。
