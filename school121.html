班級教學模式
全校推行小班教學，為照顧學生學習需要，設拔尖及補底小組輔導、加輔班、校本小班及課後功課輔導班。三 、四年級設「自攜平板電腦」電子教學班，並於各級運用不同的教學策略，推動學生自主學習。
班級結構備註
設小一惜福學堂，由時間表、課業至評估為小一課程作改革，讓學生學習生活「留白」，推動全人教育，培養具良好品格，抗逆能力的新一代。全校推行校本STEM教育，並於高小推行資優教育網絡，訓練學生高階思維技巧。

學校特色
學校管理
學校管理架構 : 設有法團校董會、學校發展委員會、課程核心小組、價值教育核心小組、不同行政組別及科組，共同推動學校的校本管理及發展。
法團校董會/學校管理委員會/校董會 : 2006年4月1日成立。
環保政策 : 開展「健‧營‧綠」校本計劃，透過多元化活動及飲食政策，培養學生更健康及更綠色的生活習慣，在可持續發展理念下，珍惜地球資源。教導學生善用資源，並制定環保政策，宣揚及實踐環保，讓學校教職員及學生攜手合作，採取各種善用資源的措施，避免浪費。
學校關注事項 : 推行自主學習教學策略，發揮專科專教優勢及建立學生自主學習能力，各級推行校本STEM課程，提升學生探究及解難能力﹔推行心靈環保的價值教育，以惜福惜緣為重心，將學科與全人生命教育融合，提升人格素質。
教學規劃
學習和教學策略 : 本校採用多元化的教學方法與策略，以推動自主學習。在教學上，以閱讀為基石，大量滲入閱讀元素，並配合電子教學策略，提升學生的學習質量。另為讓學生得到全面學習，本校十分重視學生在音、體、藝的發展，透過聯課活動、專題研習、境外交流等方式協助學生擴闊眼界，發掘學生的潛能。又利用生命教育課推動價值教育。
關鍵項目的發展 : 1. 全方位閱讀活動、設閱讀時段、親子閱讀工作坊，家長閱讀會，推動家校合作，共同培養學生閱讀興趣。2. 佛化教育融入校園生活，培養正向價值觀、人生觀；加強生命教育、班級經營，共建健康和諧的綠色校園。3. 專題研習，提升研習技巧。4. 課室設無線網絡，學生透過網上學習平台學習，培養自學能力。
共通能力的培養 : 通過科本活動、STEM活動及跨科活動如語文日、數學解難日、藝術日、境外交流等，提升學生的基礎能力（溝通、數學、運用資訊科技）、思考能力（創造、明辨性思考、解決問題）、個人及社交能力（自我管理、自學、協作）。
學生支援
全校參與照顧學生個別差異 : 設學習支援組及特殊教育需要統籌主任，協助統整資源及協調所有支援服務。教育心理學家及言語治療師提供專業支援。增聘學生輔導員，及早支援有需要的學生。推行「小學加強輔導教學計劃」及「朋輩輔導計劃」，並設學科輔導班，為學生提供輔導。設拔尖課程，培育資優生。設幼小銜接課程，為小一學生適應小學生活作好準。
課程剪裁及調適措施 : 1. 設計全方位學習課程，讓學生培養自學及共通能力，發揮多元智能。
2. 設有分層工作紙及課業，以照顧學生的多樣性。
3. 為有特殊教育需要的學生提供家課及評估調適。
家校合作及校風
家校合作 : 每月設家長茶聚、每周故事爸媽活動及家長義工大使等，加強與家長的溝通和聯繫。透過GRWTH應用程式發放學校及學生在校學習的最新資訊。舉行家長日、座談會及觀課日讓家長到校了解學生學習。提供不同渠道讓家長與老師聯繫，如：陽光熱線、內聯網、Facebook等。
校風 : 將全人生命教育融合於學校生活，推動佛化、心靈、價值、生命教育；培育學生的良好品格，懂得知福、惜福；學習訂立自己的人生目標，努力建設健康、美好的人生；也懂得慈悲護生，願意服務他人。
學校發展
學校發展計劃 : 1. 推行自主學習 2. 推動電子學習 3. 持續發展校本佛化德育及價值教育
4. 持續優化課堂教學-評估促進學習 5. 裝備學生以銜接新高中學制課程
6. 持續發展家長教育
教師專業培訓及發展 : 加強學校行政管理津貼；優化學校投訴管理計劃；促進與內地姊妹學校交流試辦計劃；課程領導學習社群；資優教育社群；中國語文科閱讀規劃與策略運用；「促進實踐社群以優化小班教學」支援計劃（中文）PLPR/W (Space Town)；與內地教師交流及協作計劃（數學科）；小學科學教育學習社群；「校園音樂大使展關懷」資助計劃；第九屆校園藝術大使計劃。
