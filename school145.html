辦學宗旨
本校秉承區會辦學宗旨，培養學生具備優良品德、高雅情操、豐富學識、健康的人生態度、正確的價值觀，以及強烈的社會意識。
此外，本校以「並肩培育豐盛生命，攜手見證基督大愛」為學校願景。
學校特色
學校管理
學校管理架構 : 設立自評及發展、行政、課程、德訓等組別拓展校務，另設家教會及校友會。
法團校董會/學校管理委員會/校董會 : 07年成立法團校董會。
環保政策 : 參與美化校園及「一人一花」計劃。午膳採用環保餐盒。
學校關注事項 : 優化學習團隊，強化兩文三語，品德學藝兼備，發展多元才能。
教學規劃
學習和教學策略 : 重視閱讀及寫作，加入普教中教學；重視公民及國民意識，培養正面價值觀；重視體藝活動及境外學習，拓闊生活體驗；重視創意，批判性思考，協作及溝通3C、資訊科技等共通能力的發展。
關鍵項目的發展 : 1. 從閱讀中學習：透過圖書課、閱讀獎勵計劃、故事大使及網上閱讀平台，營造閱讀文化；
2. 資訊科技：透過互動網上平台、加入資訊素養於恒常教學中，於各科滲入IT教學元素，以促進學生自學及互動學習；
3. 專題研習：進行跨科專題研習，建立研習能力架構，以培養學生專題研習能力；
4. 德育及公民教育：透過週會、成長課等，建立關愛文化，培育學生國民身份的認同。
共通能力的培養 : 透過跨科專題研習、全方位學習、境外交流、種籽計劃、校本課程等，培養各種共通能力。
學生支援
全校參與照顧學生個別差異 : 1. 運用課程及評估調適策略。
2. 設有支援小組、英文及數學尖子小組、抗逆活動、成長的天空計劃及區本支援計劃等，以照顧學生不同需要，未來重點發展資優教育。
課程剪裁及調適措施 : 按學生學習能力，調適有關教授內容。
家校合作及校風
家校合作 : 1. 家教會：親子旅行、親子遊戲日等。
2. 家校合作：各級家長會、IMC包括家長代表等。
3. 家長心聲：PTA會訊、福音聚會、團拜及家長團契。
4. 家校合作：設優質家長學堂，提供管教及學習技巧教育。
校風 : 校風純樸，學生守規。奉行德訓輔三合一政策。活動包括基法小學特別任務、關愛校園計劃、小一適應課程、小六升中適應課程。
學校發展
學校發展計劃 : 1. 建立完善有效管理模式，提升中層領導力。
2. 優化學與教，照顧差異，發展學生潛能。
3. 建立關愛校園，培育學生全人發展。
教師專業培訓及發展 : 優化團隊教學質素；深化教師專業發展（如共同備課）；強化自評文化。
學費減免 : 不適用。
