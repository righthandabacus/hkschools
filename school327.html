全方位學習活動
類別：體育、音樂、藝術、制服團隊、領袖生會、各學會、社會服務及各興趣活動（如棋藝、園藝、書法、辯論、話劇、歌劇）。
辦學宗旨
培僑書院以英語及普通話作為授課語言，中西薈萃，培育能為香港、為中國走向世界作出貢獻的出類拔萃新生代。我們期望學生能：
1. 擁有廣闊的國際視野。
2. 具備深厚的中華文化涵養和中國情懷。
3. 通曉中英語文。
4. 思辯敏銳、好學篤行、勇於承擔。
學校特色
學校管理
學校管理架構 : 培僑書院法團校董會。
法團校董會/學校管理委員會/校董會 : 培僑書院法團校董會於2008年8月30日成立，由8名辦學團體校董、2名家長校董、2名教師校董、1名獨立校董及1名當然校董組成。
環保政策 : 本校成長科課程透過不同學習活動提高學生的環保意識。
學校關注事項 : 1. 發展十二年一貫課程。
2. 提升學生的寫作水平。
3. 強化學生自我管理能力；培養關愛及尊重他人的精神；培育學生領袖。
4. 優化教師考績制度及同儕專業交流制度。
教學規劃
學習和教學策略 : 我們根據學童的發展特點，統整課程，設計學生學習經歷。
學校致力建立豐富的語言環境，英語和普通話為主要學習和溝通語言。
關鍵項目的發展 : 把閱讀興趣及能力的培養作為英文、中文課程核心，引導學生掌握各種閱讀策略，並透過閱讀，讓學生學會學習，樂於學習。
共通能力的培養 : 透過課堂內外活動及學習環境，培養學生解難、溝通、協作、研習、自我管理及思辯等共通能力。
學生支援
全校參與照顧學生個別差異 : 學校聘有專業人士，例如教育心理學家、語言治療師、社工；通過小組活動、個案輔導，對學生作出支援。
課程剪裁及調適措施 : 個別有特殊學習需要的學生獲評估調適。
家校合作及校風
家校合作 : 使用eClass系統，設立家長討論區。家長義工團強大。成立家教會。
校風 : 小一新生及七年級銜接課程，透過豐富有趣的活動，讓學生逐步適應校園生活規律及語言學習環境，培養自信心和積極投入的學習態度。
學校發展
學校發展計劃 : 1. 提升學與教的效能。
2. 提升學生個人素質，推動學生關心國家社會，強化其服務社群的精神，促進全人發展。
3. 促進教師的專業發展。
教師專業培訓及發展 : 1. 提高專業知識。
2. 加強相互專業交流，完善考績制度。
3. 加強中層人員培訓。
學費減免 : 本校設有學費全免或減免1/2學費。有需要的學生(包括來自綜援家庭的學生及領取學生資助辦事處財政資助的學生)可提出申請學費減免。
- 新生可於開學前的2個月內遞交申請，其申請結果會在3個月內獲得通知。
- 本校學費減免計劃詳情包括家庭入息限額，請瀏覽網上版的超連結。
其他 : 1. 本校設有「獎學金計劃」，獎勵在學業、品德、體育、音樂、視覺藝術及其他才藝領域有傑出表現的同學。每年派發獎學金超過200萬元。
2. 沙田區小學全年總錦標（男子組）2011-12，2012-13，2013-14。
3. 沙田區小學全年總錦標（女子組）2012-13，2013-14。
