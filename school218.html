班級教學模式
引入不同教學策略，如自主學習、合作學習、電子教學、協作教學及小組互動，在小組中優差並進。輔導策略︰因應學生學習需要，在課程上作出調適，設導修課。
班級結構備註
全校採用小班教學及雙班主任制。中文科一至四年級實施普通話教學，五及六年級設普通話科；英文科推行PLPRW課程及Space Town，以小組形式上課；數學科運用ELPS模式，以提升學生學習效能。

全年全科測驗次數 : 2
全年全科考試次數 : 2
多元學習評估 : 1. 課程施行與學教評估平行進行，利用多元化評估，多角度了解學習情況，回饋教學。
2. 加強學生自評、互評的元素。
3. 按教學進度，訂定多次進展性評估。
分班安排 : 常模分班，運用分組活動，加強互動，使不同能力學生能夠在學習中優差並進。在拔尖補底方面，另設啟發潛能課程及小組輔導，以配合學生的多元化發展。

全方位學習活動
多元興趣小組，全年可參加最少四項：如 3D 創意打印、 STEM 智慧型電子積木、航拍小組、陶土、舞蹈、戲劇工作坊、樂器； 另設訓練小組：如獨輪單車隊、田徑隊、球類活動、義工隊、圖書館領袖生、電視台節目製作班等，另設有視藝學會及數學學會。
辦學宗旨
以「德、智、體、群、美」五育為課程綱領，以「自治、自學、自省」為教育目標，以發展優質、完善及均衡的教育服務為宗旨。以學生為本，著重發展個別潛能，透過不斷的自學、體驗和實踐，建立積極的人生態度，熱心服務的助人精神和貢獻社會的豁達胸懷。
學校特色
學校管理
學校管理架構 : 本校於2006年成立法團校董會，管理學校；日常事務由校長及不同行政小組負責，包括校務、學校推廣組、課程組、訓輔組、課外活動組及資訊科技組、學生事務組及學生支援組等。
法團校董會/學校管理委員會/校董會 : 本校的法團校董會共有十二位成員，成員包括：辦學團體校董、教員校董、家長校董及獨立人士校董等。校董會定期舉行會議，就校務進行討論及決策，貫徹校本管理精神。
環保政策 : 設廢物回收箱，定期舉行舊衣回收、熄燈慳電活動。
學校關注事項 : 加強學生學習效能；提升教師專業培訓；加強學生品德教育，並培養學生自治、自學、自省的能力，及服務社區的態度。
教學規劃
學習和教學策略 : 採用多元化教學策略：小組學習、戶外學習、學習營、合作學習、電子學習及主題教學。透過研習課、德育課、成長課、專題研習、戶外學習，進行跨學科學習。讓學生高參與，多展示作品及才藝潛能。
關鍵項目的發展 : 1. 訂定具體的考測目標，培養學生的自主學習能力。
2. 推行電子學習，運用電子書平版電腦及不同的應用程式進行教學、自行編製網上練習、設學科網站提供網上學習資源等。
3. 設本科及跨科全年學習主題。STEM融入常識及科學科，推動跨課程閱讀。
4. 每年設不同德育及公民教育主題，另設校本輔導活動，提倡自我管理及服務學習。
共通能力的培養 : 1. 設校本戲劇課、陶土課，發展學生共通能力。
2. 推動STEM課程，提升學生對科學探究的興趣及能力，讓學生發揮創意潛能。
學生支援
全校參與照顧學生個別差異 : 1. 學習支援：協作教學、小組輔導、新移民英文班、課後支援、大哥哥大姐姐計劃等。
2. 成長支援：成長小組、家訪、雙班主任制、香港樹仁大學輔導及支援小組。
3. 有駐校的言語治療師、專業社工及教育心理學家。
4. 設各項小組訓練，包括：專注力訓練、讀寫訓練、社交小組等。
課程剪裁及調適措施 : 學校特色共分五個範疇，分別為兩文三語、對外交流、戲劇教育、服務學習及陶土藝術。並設各科各級的學習指標，因應學生的學習表現加以增潤及調適，為能力較佳的學生開設各科拔尖課程，為成績稍遜的學生設輔導班及課後補課。
家校合作及校風
家校合作 : 透過家教會及法團校董會，令校政透明化。為促進家校緊密的聯繫，定期舉辦講座、聯誼活動、聯校家長活動、出版會訊及於校網刊載家長心聲。另設校長家長Whatsapp群組促進家校合作和溝通。
校風 : 本校以啟發潛能教育為發展方向，以學生為本，提升學生自治、自學及自省能力。推展各項關愛校園政策、服務學習計劃及德育主題活動，鼓勵學生關心別人，服務別人，使校園洋溢著溫馨關愛的氣氛。
學校發展
學校發展計劃 : 培養學生英語能力，鼓勵學生以英語及普通話溝通；將戲劇融入教學，幫助學生以英語作思考及表達。發揮校園電視台視象的效能，加強學生說話的能力。發展跨地、跨校的對外交流活動，擴闊學生視野。優化電子學習及發展校本特定的STEM課程。
教師專業培訓及發展 : 透過聯校教師專業發展日、校本專業培訓及大專院校的支援計劃，重點發展教師的電子教學、校本STEM課程、合作學習及照顧學生學習差異能力。學校更鼓勵教師參加教育局的進修課程，以提升他們的專業素質。另外，本校亦安排教師參觀本港及海外學校，讓他們進行考察及交流。教師透過共同備課及課研計劃提升教學效能。
學費減免 : 不適用。
其他 : 1. 辦學團體設獎學金，獎勵同學敦品勵行，力學向上。
2. 參與各類校際運動比賽、朗誦節、戲劇節、音樂節等，發揮學生不同潛能。
3. 進行教育營、境外交流計劃，以擴展學生學習視野。
