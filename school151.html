班級教學模式
小三至小六設加強輔導班，由已接受特殊教育培訓教師教授數學及英文科成績稍遜的學生。
班級結構備註
1. 小一至小六推行英文閱讀及寫作計劃。 2. 小一至小三校本英文拼音課程。 3. 小一至小三中文科悅讀悅寫意計劃。 4. 小一至小六中文科校本圖書課程。

全方位學習活動
學生逢星期三參加校內活動小組，每年均為各級安排戶外參觀活動。午膳後及放學後開放圖書館及電腦室。設有各項球類小組、泳隊、獅藝、節奏樂、小提琴、中國舞、英文話劇組、奧林匹克數學、武術、珠心算、園藝組、拉丁舞班、跆拳道、校園小記者及十鼓班。
辦學宗旨
本校注重學生品德的培養，啟發學生的智慧，五育並重，使學生成為一個品學兼備的良好公民。
學校特色
學校管理
學校管理架構 : 設法團校董會、校監、校長及副校長，由學校改善小組管理以下四個範疇：學生支援、管理與組織、學與教及課程、學生表現。
法團校董會/學校管理委員會/校董會 : 已於2008年成立法團校董會。法團校董會由14人組成，包括辦學團體校董、獨立校董、當然校董、家長校董及名譽校董。
環保政策 : 環保大使計劃、 節約能源、節約用紙及舊校服回收。
學校關注事項 : 完善學與教策略，以促進學生的學習。
優化校本生命教育。
教學規劃
學習和教學策略 : 調配教學時間，安排各級學生進行戶外學習。調配上課時間表，以便舉行全級聯課活動。全方位的德育及公民教育活動。佛化德育及價值教育課。校內個別學科及跨學科的學習活動及課餘體藝活動。中英數常各科均有公民教育的元素。
關鍵項目的發展 : 1. 閱讀：閱讀獎勵計劃。
2. 資訊科技：全校課室安裝電腦及電子白板，設網上自學平台，並利用iPad學習英文拼音。
3. 專題研習：跨學科專題研習。
4. 各級設一節公民教育課、成長課、佛化德育及價值教育課。
共通能力的培養 : 本校透過各學習範疇、多元化的學習活動和課外活動，如：專題研習、小記者訓練班、各類校隊、制服團隊等，從中發掘學生的內在潛能和培養共通能力。
學生支援
全校參與照顧學生個別差異 : 1. 語文強化班。
2. 奧數班。
3. 視藝精英班。
4. 創意思維小組。
5. 新來港學童補習班。
6. 非華語學童中文學習班及廣東話班。
7. 輔導班。
8. 朋輩輔導。
9. 小導師計劃。
10. 小組聊天室。
11. 言語治療。
12. 個別加強輔導計劃。
13. 專注力小組訓練。
課程剪裁及調適措施 : 1. 中文科「悅讀悅寫意」讀寫支援計劃。
2. 英文科佛聯會會屬小學學習圈。
3. 非華語「改善非華語學生的中文學與教」計劃。
4. 常識科「促進實踐社群以優化小班教學」計劃。
5. 「生活化的中國經典學習：品德與生命教育小學試行」計劃。
家校合作及校風
家校合作 : 家教會每年舉行多項活動，並與輔導主任合辦家長講座、家長工作坊及家長義工服務等。
校風 : 學校重視學生關愛文化，透過早會、午間廣播及校內活動建立學生關愛共融，尊重生命的情操。
學校發展
學校發展計劃 : 1. 優化課堂教學–本校與香港中文大學教育學院合作建立「英文拼音教師專業發展平台」（網站：http://caite.fed.cuhk.edu.hk/phonics）。
2. 完善學與教策略，以促進學生的學習。
3. 優化校本生命教育
教師專業培訓及發展 : 參與會屬小學組織的聯校教師發展日及學習社群：課程主任會、訓輔教師會、中英數分科學習圈等。校內舉行與發展計劃相關的培訓工作坊。
學費減免 : 不適用。
其他 : 妙蓮長老佛學科獎學金。
林秀棠榮譽校董「進步之星」獎學金。
林秀棠榮譽校董電腦科獎學金。
